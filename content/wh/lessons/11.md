---overview---
395年、ローマ帝国が東西に分割されました。西ローマ帝国は１世紀も経たずに滅ぼされましたが、東ローマ帝国ことビザンツ帝国はおよそ1000年続きました。ビザンツ帝国ではローマ帝国からキリスト教を国教として継承し、独自に発展させました。
---

## キリスト教
### キリスト教の成立
::top
ローマの属州となったパレスチナでは、ユダヤ教祭司や利法の実行を重んじるパリサイ派が、 ユダヤ人を支配する層として存在しました。
::
::last
彼ら支配層は、貧困に苦しむ民衆の声に応えようとしませんでした。
::
---arrow---
::top
[[イエス]]は、祭司やパリサイ派を批判し、貧富の区別ない神の愛を説きました。
::
::middle
民衆はイエスを救世主（メシア）と信じて彼の教えに従いました。
::
::middle
ユダヤ教祭司やパリサイ派がイエスをローマに対する反逆者として訴えたため、イエスは処刑されました（30年頃）。
::
::last
弟子たちはイエスの復活と贖罪を信じ、この信仰を中心に[[キリスト教]]が成立しました。
::
::sup
メシア
:::lead
この語のギリシア語訳が「[[キリスト]]」
::
::sup
キリスト教の教典
:::lead
の教典『旧約聖書』と、コイネーという[[ギリシア]]語で記された『新約聖書』
::
### 伝道活動
::top
ペテロや[[パウロ]]などの使徒によって伝道活動が始まりました。
::
::last
特にパウロは、神の愛がユダヤ人以外の民族にも及ぶとして、パレスチナ以外の地域にもキリスト教を布教しました。
::
### 迫害から国教化へ
::top
ローマの宗教は多神教で、皇帝も神の一人とされました。
::
::middle
皇帝崇拝が強化されると、唯一絶対神を信じるキリスト教徒は反社会集団とされました。
::
::last
64年の[[ネロ帝]]による迫害や303年の[[ディオクレティアヌス帝]]による迫害が有名です。
::
---arrow---
::top
迫害の一方でキリスト教は拡大を続け、信仰を禁止すれば帝国の維持ができない状況となりました。
::
::last
次のようにキリスト教の扱いが変化しました。
::

- 313年、[[ミラノ勅令]]
  :::lead
  [[コンスタンティヌス帝]]がキリスト教信仰を==公認==した勅令
  :::

::sup
アタナシウス派
:::lead
ス派後に神・キリスト・聖霊を同一視する[[三位一体説]]を確立
::
---arrow---
::top
392年、[[テオドシウス帝]]がアタナシウス派キリスト教を国教としました（他宗教は厳禁）。
::
::last
キリスト教が国家権力と結びつくと、聖職者身分が成立し、教会の組織化が進みました。
::
