---overview---
古代以来続いてきた地中海世界は、フン人の西進、西ゴート族の移住に端を発するゲルマン人の大移動や西ローマ帝国の滅亡によって、政治的・文化的統一性を失いつつありました。そこにイスラーム勢力がイベリア半島から侵入し、喪失は決定的なものとなりました。
---
## ゲルマン人の大移動
### 大移動に先立つ時代
::top
ゲルマン人は、バルト海沿岸を原住地としました。
::
::middle
ゲルマン人は、西に勢力を拡大させ、アルプス以北のヨーロッパに住んでいたケルト人を圧迫しました。
::
::last
紀元前後頃には、ゲルマン人はライン川から黒海沿岸まで広がり、ローマ帝国の領域と接するようになりました。
::
---arrow---
::top
紀元前後頃のゲルマン人には、==社会の階層分化がある程度進み、すでに奴隷もいました ==。
::
::last
この頃のゲルマン人の生活・慣習は、[[カエサル]]の『ガリア戦記』や[[タキトゥス]]の『[[ゲルマニア]]』に記されています。
::
### ゲルマン人の大移動
::top
ローマ帝政後期頃、ゲルマン人の耕地が不足し、後のゲルマン人の大移動の内的原因を形成していました。
::
---arrow---
::top
４世紀後半、アジア系の[[フン人]]（[[フン族]]）が西に進み、ゲルマン人の一部族である東ゴート族を征服、西ゴート族を脅かしました。
::
::middle
東ゴート族の大半は征服され、また、==フン人の西進に圧迫された ==[[西ゴート族  ]]は、376年にドナウ川を渡ってローマ帝国内に移住しました。
::
::last
西ゴート族の移動は、他のゲルマン諸部族のローマ帝国内への移動を引き起こし、約200年におよぶ[[ゲルマン人の大移動]]を始めました。
::
::gazo
![フン人の侵入](img/33/Hun.webp){.border-1.border}
フン人の侵入
::
::top
410年、西ゴート族はガリア西南部（スペイン・南フランス）に[[西ゴート王国]]を建国し、[[イベリア ]]半島を支配しました。
::
::last
西ゴート王国のイベリア半島支配は、711年にイスラームの[[ウマイヤ朝 ]]に滅ぼされるまで続きました。
::
## 他民族・他部族の動向
### フン人の動向
::top
５世紀、フン人（フン族）の[[アッティラ ]]がパンノニア（ハンガリー）を中心に帝国を建国しました。
::
::sup
ドイツに成立した『ニーベルンゲンの歌』は、この帝国がブルグント族を攻撃したことが題材の一部
::
---arrow---
::top
451年、カタラウヌムの戦い
:::lead
カタラウヌムの戦いアッティラの軍が、西ローマ帝国・西ゴート王国・フランク王国の連合軍に敗れた戦い
::
---arrow---
::top
アッティラの死後、帝国は崩壊しました。
::
::sup
476年、西ローマ帝国滅亡
::
### 西ゴート族以外のゲルマン人の動向
::top
大移動後には、西ゴート族以外にも、次のゲルマン諸部族が建国しました。
::

- フランク人（族）がガリア北部（フランス北部）に[[フランク王国]]を建国
- [[アングロ＝サクソン人（族）]]が大ブリテン島（イングランドやスコットランド）に[[七王国]]を建国

## フランク王国の成立・発展
### メロヴィング朝
#### メロヴィング朝の成立
::top
５世紀末、フランク人でメロヴィング家出身の[[クローヴィス]]が、[[フランク王国]]をガリア北部（フランス北部）に建国しました。
::
::middle
この時代のフランク王国を、クローヴィスの出自に基づき、[[メロヴィング朝]]と呼びます。
::
::last
481年、クローヴィスは全フランク人を統一し、フランク王国を強国にしました。
::
#### 正統派への改宗
::top
当時、ゲルマン人の多くが、キリスト教の異端派とされたアリウス派を信仰していました。
::
::last
クローヴィスは、キリスト教正統派の[[アタナシウス派 ]]に改宗し、ローマ人との関係を強化しました。
::
#### メロヴィング朝の衰退
::top
６世紀、フランク王国はブルグンド王国などを滅ぼし、全ガリアを支配しました。
::
::last
しかし、８世紀にメロヴィング朝は衰退し、王朝の行政・財政の長官である{{宮宰|きゅうさい}}が実権を握りました。
::
::top
711年、イスラーム王朝の[[ウマイヤ朝]]が、イベリア半島にわたって[[西ゴート王国 ]]を滅ぼしました。
::
::last
ウマイヤ朝は、さらにピレネー山脈を越えてガリアに侵入しようとしました。
::
---arrow---
::top
732年、[[トゥール・ポワティエ間の戦い ]]
:::lead
[[トゥール・ポワティエ間の戦い ]] メロヴィング朝の宮宰[[カール＝マルテル  ]]が、フランク王国領内に侵入したウマイヤ朝軍を撃退した戦い
::
### カロリング朝
::top
751年、カール＝マルテルの子[[ピピン ]]は、ローマ教会に認められてフランク王国の王位につき、[[カロリング朝  ]]を開きました。
::
::last
ローマ教会の名のもとフランク王国は特別に扱われ、見返りにローマ教会への協力を期待されました。
::
---arrow---
::top
ピピンは、イタリアのランゴバルド王国から土地を奪い、==後の教皇領の起源となる領地をローマ教皇に寄進しました ==（ピピンの寄進）。
::
::last
このように、フランク王国側からも利害の一致からローマ教会と結びつきを強めていきました。
::
## ローマ＝カトリック教会の動向
### ローマ教会の独自の歩み
::top
ローマ帝政末期（３世紀末～）、初期キリスト教の五大教会（五本山）が、ローマ・コンスタンティノープル・アンティオキア・イェルサレム・[[アレクサンドリア ]]にありました。
::
::last
五大教会のなかでも最有力は、ローマ教会とコンスタンティノーブル教会でした。
::
---arrow---
::top
476年の西ローマ帝国滅亡後、ローマ教会は、コンスタンティノープル教会とは異なる独自の活動を展開しました。
::
::middle
例えば、６世紀末のローマ教会教皇[[グレゴリウス１世 ]]は、==ゲルマン人への布教をおこないました  ==。
::
::last
結果、ローマ教会は西ヨーロッパに勢力を拡大し、教皇の権威が高まりました。
::
### ビザンツ帝国との対立
::top
初期キリスト教では偶像崇拝が禁じられていました。
::
::middle
しかし、布教の手段としてイエス・聖母マリア・聖人の聖像は都合がよく、多くのキリスト教徒が聖像を礼拝していました。
::
::last
またローマ教会も、ゲルマン人への布教に聖像を必要としていました。
::
::gazo
![聖像をめぐる対立](img/33/icon.webp)
聖像をめぐる対立
::
::top
726年、ビザンツ帝国皇帝[[レオン３世 ]]は、次の２つの理由から[[聖像禁止令  ]]（[[聖像崇拝禁止令]]）を発布しました。
::

- 初期キリスト教の教理に反するため
- 偶像を否定するイスラームに対抗するため

::middle
ローマ教会はこの禁止令に反発し、ビザンツ帝国・コンスタンティノープル教会との対立を強めました。
::
::last
ローマ教会は、ビザンツ帝国皇帝に対抗する政治勢力の協力を求めました。
::
### フランク王国への接近
::top
ビザンツ帝国と対立したローマ教会は、政治勢力の協力が必要になりました。
::
::middle
折しもフランク王国宮宰カール＝マルテルがイスラームからキリスト教世界を守った頃でした。
::
::last
ローマ教会はフランク王国に接近し、カール＝マルテルの子ピピンが王位を継承することを認めました。
::
---arrow---
::top
ピピンは、イタリアのランゴバルド王国から土地を奪い、==後の教皇領の起源となる領地をローマ教皇に寄進しました ==（ピピンの寄進）。
::
::last
このように、フランク王国側からも利害の一致からローマ教会と結びつきを強めていきました。
