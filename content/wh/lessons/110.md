---overview---
1989年、東ドイツの政権が崩壊し、東西を分断していたベルリンの壁が壊されました。40年の分断を経て、ようやくドイツが１つに統一されました。そしてこの一か月後、米ソによって冷戦の終結が宣言されました。
---
## 東欧の民主化
### 内政干渉の否定
::top
1988年、新ベオグラード宣言
:::lead
1988年、新ベオグラード宣言
:::
、新ベオグラード宣言ソ連書記長ゴルバチョフが、東欧の社会主義国への内政干渉を否定した宣言
::
### 民主化の波及
::top
1980年、[[ポーランド ]]で[[ワレサ  ]]を指導者とする自主管理労働組合「[[連帯  ]]」が組織され、共産党の一党独裁政権に改革を求めました。
::
::middle
1989年、複数政党制（多党制）での選挙が実施され、「連帯」を中心とする連立政権が成立しました。
::
::last
「連帯」は、ポーランドの民主化を推進しました。
::
::sup
ワレサ
:::lead
ワレサ
:::
iv class="lead">「連帯」の指導者で、1990年、==ポーランドの大統領に選出 ==
::
---arrow---
::top
1989年、ハンガリー・チェコスロヴァキアにおいて、共産党一党体制が複数政党制に移行しました。
::
::last
民主化の流れは、これらの東欧諸国以外にも及び、ルーマニアを除いてほぼ平和的に体制が移行しました。
::
#### ルーマニア
::top
1989年、[[ルーマニア ]]では[[チャウシェスク  ]]の独裁体制が崩壊しました。
::
::last
ルーマニアの民主化は、チャウシェスク夫妻が処刑されるなど流血を伴う革命でした。
::
### 東西ドイツの統一
::top
1989年10月、東ドイツでは西側へ脱出する者が増加し、書記長のホネカーが解任されました。
::
::last
1989年11月（ソ連の解体==前 ==）、[[ベルリンの壁  ]]が開放され、東西ドイツ間の自由な往来が可能になりました。
::
---arrow---
::top
[[1990 ]]年、西ドイツが東ドイツを吸収して、東西ドイツが統一されました。
::
### 東欧社会主義圏の消滅
::top
1991年、コメコン・[[ワルシャワ条約機構 ]]が解体され、東欧社会主義圏が消滅しました。
::
## ソ連の解体
### ソ連の改革
::top
冷戦の終結後、ソ連では[[ゴルバチョフ ]]が次の改革がおこないました。
::

- 1990年、[[大統領]]制の導入
  :::lead
  初代大統領に[[ゴルバチョフ]]が就任
  :::
- 計画経済から市場経済への移行
- スターリン体制下で犠牲になった人々の名誉回復

### 東欧の民主化の影響
::top
ポーランドにはじまる東欧の民主化は、ソ連に影響を与えました。
::
::last
ソ連の改革・解体の過程で、まず[[バルト３国 ]]が、相次いで独立を宣言しました。
::
---arrow---
::top
1991年、==ソ連共産党の保守派がゴルバチョフの改革に反対してクーデタを起こしました ==。
::
::last
クーデタは失敗に終わり、[[ソ連共産党 ]]が解散されました。
::
---arrow---
::top
1991年、==バルト三国を除く ==ソ連内のロシア共和国（現ロシア連邦）・[[ウクライナ  ]]・ベラルーシなどの国々が、[[独立国家共同体  ]]（[[ＣＩＳ]]）を結成しました。
::
::last
結果、1991年に[[ソ連 ]]が解体しました。
::

::last
