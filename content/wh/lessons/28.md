---overview---
非アラブ人を差別するウマイヤ朝の政策は、『コーラン』の教えに背くと次第に批判されました。かわりに成立したアッバース朝は、首都バグダードを中心に発展し、ハールーン＝アッラシードの時代に最盛期を迎えました。しかし、彼の死後には徐々に力を失い、アッバース朝の君主以外にもカリフを自称する君主が現れました。
---
## 新たな王朝の形成
### ウマイヤ朝とコーランの教え
::top
『コーラン』は、全ての信者が平等だと説きます。
::
::middle
しかし、ウマイヤ朝の征服地では、非アラブ人はイスラームに改宗しても差別されました。
::
::last
ウマイヤ朝の在り方は『コーラン』に{{背|そむ}}くという批判が出てきました。
::
### 新たな王朝
::top
ウマイヤ朝を批判する人々は、ムハンマドの{{叔父|おじ}}の子孫アッバース家の挙兵に参加しました。
::
::middle
750年、ウマイヤ朝は滅び、新たに[[アッバース朝]]が開かれました。
::
::last
アッバース朝は、==ティグリス川の河畔に、 ==円状の首都[[バクダード]]を造営しました。
::
### アッバース朝の政策
#### アラブ人と非アラブ人
::top
イスラーム教徒であれば、非アラブ人であっても人頭税（ジズヤ）は課せられませんでした。
::
::middle
また、征服地に土地をもつ場合には、==アラブ人にも ==[[地租]]（[[ハラージュ]]）が課せられました。
::
::last
つまりアッバース朝では、アラブ人と非アラブ人との差別は、==イスラーム教徒である限り廃止されました ==。
::
#### 政治
::top
アッバース朝のカリフの政治は、イスラーム法（シャリーア）に基づき、実施されました。
::
::sup
イスラーム法
:::lead
イスラーム法
:::
法『コーラン』や預言者の伝承をもとに９世紀頃までに整えられた、生活の規範や政治の基本を示す法
::
#### 知恵の館
::top
アッバース朝の時代、「知恵の館」において==多くのギリシア語文献がアラビア語に翻訳されました ==。
::
::top
中央アジアの遊牧民トルコ人は、幼い頃から乗馬に親しみ、騎射に優れていました。
::
::middle
アッバース朝が中央アジアに進出すると、トルコ人は奴隷としてイスラーム世界に流入しました。
::
::last
アッバース朝のカリフは、トルコ人の力を認め、奴隷身分出身の兵士[[マムルーク ]]として用いました。
::
### アッバース朝の最盛期とその後の衰退
::top
アッバース朝は、５代目カリフ[[ハールーン＝アッラシード]]の時代に==最盛期を迎えました ==。
::
---arrow---
::top
ハールーン＝アッラシードの死後、アッバース朝の領域から独立する王朝が現れました。
::

  :::
- [[サーマーン朝]]
  :::lead

---arrow---
::top
さらに、アッバース朝以外にもカリフの称号を用いる王朝が現れました。
::
::last
具体的には、後述するファーティマ朝・後ウマイヤ朝の君主が使用し、同時代に３人のカリフが分立する状態となりました。
::
### 軍事政権への移行
::top
946年、ブワイフ家がバグダードに入城し、アッバース朝のカリフから、軍司令官（アミール）のなかの第一人者を意味する大アミールに任じられました。
::
::middle
さらに、アッバース朝のカリフにかわってイスラーム法を施行する権限を与えられました。
::
::last
アッバース朝のカリフは名目上存在するものの、実際の政権を担う[[ブワイフ朝]]が成立しました。
::
---arrow---
::last
ブワイフ朝が創始したこの制度を[[イクター制 ]]と呼びます。
::
