---overview---
「カールの戴冠」は歴史的に重要な意義をもっています。西ローマ帝国の皇帝が復活することで、ローマ教会・西ヨーロッパは、ビザンツ帝国皇帝の干渉から切り離されいきました。ここに、ローマ帝国以来存続した地中海世界が、西ヨーロッパ世界・東ヨーロッパ世界、そしてイスラーム世界の３つに分かれ、独自の歴史を歩むことになっていきます。
---
## 中世世界の誕生
### フランク王国の強大化
::top
８世紀末、ピピンの子[[カール大帝]]（[[シャルルマーニュ]]）は、イタリアの[[ランゴバルド王国 ]]を滅ぼしました。
::
::last
結果、大多数のゲルマン諸部族が統合され、ローマ＝カトリックに改宗させられました。
::
::sup
『ローランの歌』はカール大帝の時代が題材
::
---arrow---
::top
さらにカール大帝は、東では[[アヴァール人 ]]を、南ではイスラーム勢力を撃退しました。
::
::last
西ヨーロッパの主要部分はフランク王国によって統一されました。
::
---arrow---
::top
フランク王国は国内を州に分け、各地の有力者を州の長官[[伯 ]]として置きました。
::
::last
また、伯を監督する[[巡察使 ]]も派遣しました。
::
### 教会の政治的保護者
::top
西ローマ帝国の滅亡後、ローマ教会は政治的保護をビザンツ帝国皇帝に頼らざるを得ませんでした。
::
::middle
そのためローマ教会は、カール大帝にビザンツ帝国皇帝に代わる政治的保護を期待しました。
::
::last
800年、ローマ教会教皇[[レオ３世]]は、カール大帝にローマ皇帝の冠を授け、西ローマ帝国復活を宣言しました。
::
---arrow---
::top
{{所謂|いわゆる}}「カールの{{戴冠|たいかん}}」は、次の重要な歴史的意義をもちます。
::

- 西ローマ帝国の再興を意味すること（ローマ教会がビザンツ帝国皇帝への従属から独立）
- 西ヨーロッパ世界が、ビザンツ帝国皇帝（東ヨーロッパ）からの干渉に自立を主張できるようになったこと
- ローマ文化・ゲルマン文化・キリスト教の融合した西ヨーロッパ中世世界が誕生したこと

::sup
ローマ教会がビザンツ帝国から完全に分裂するのは11世紀
::

## 西ヨーロッパ中世世界と外部
### 外部勢力の侵入
::top
８～10世紀、外部勢力が西ヨーロッパへの侵入を繰り返しました。
::
#### 東部
::top
東方からはスラヴ人やマジャール人が侵入しました。
::
::last
これら侵入に対して、[[オットー１世]]がマジャール人を撃退しました。
::
#### 南部
::top
南部からはイスラーム勢力が侵入しました。
::
::top
北欧にはヴァイキングと呼ばれる[[ノルマン人]]（ゲルマン人の一派）が居住していました。
::
::last
９世紀頃から、ヴァイキング（ノルマン人）は海賊行為を伴う交易を繰り返し、北欧や西ヨーロッパ各地に進出して次の国家を建設しました。
::

- 10世紀、北フランスに[[ノルマンディー公国]]を建国
- 12世紀前半、南イタリアとシチリア島に、[[両シチリア王国]]を建国
- ９世紀、[[リューリック]]に率いられたノルマン人の一派が、ドニエプル川流域に[[ノヴゴロド国]]を建国しました。[[リューリック]]の一族が南下し、ドニエプル川流域に[[キエフ公国]]（ロシアの起源）を建国
- イングランドのアングロ＝サクソン王国へ侵入９世紀末、アングロ＝サクソン王国のアルフレッド大王がこれを撃退1016年、デンマーク地方のノルマン人（デーン人）の王[[クヌート]]（[[カヌート]]）がイングランドを征服1066年、ノルマンディー公[[ウィリアム]]（[[ウィリアム１世]]）が==イングランドに==ノルマン朝を建国
- ==アイスランド、さらにはグリーンランドやアメリカに到達==
