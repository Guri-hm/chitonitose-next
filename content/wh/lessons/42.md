---overview---
唐滅亡後の中国は、趙匡胤が建国した宋によって再び統一されました。宋は、唐末期から五代十国時代の節度使の台頭を招いた「武断政治」を改め、文人官僚を重用する「文治主義」を推進しました。この転換で皇帝の中央集権化が進みましたが、反面、文人官僚への俸給による国家財政の圧迫、北方民族に抗う軍事力の低下などの問題を生みました。これは金による華北の支配、宋の南渡に繋がりました。
---
## 宋の建国と南渡
### 宋の建国
::top
960年、五代の後周の武将[[{{趙匡胤|ちょうきょういん}} ]]は、宋（北宋）を建国しました。
::
::last
宋は、乱立していた中国各地の節度使を抑え、唐滅亡後の中国を再び統一しました。
::
### 文治主義
#### 節度使の弱体化
::top
宋（北宋）の趙匡胤は、文治主義をとり、学識ある==文人官僚の登用を促進しました==。
::
::middle
節度使に欠員が出るたびに文官をあて、その兵力や財力を奪いました。
::
::last
他方で皇帝の親衛軍を強化し、中央集権の確立に努めました。
::
#### 科挙の整備
::top
隋・唐の時代におこなわれた科挙を整備し、官吏登用のほぼ唯一の方法としました。
::
::middle
宋代以降、==皇帝自らがおこなう ==最終試験[[{{殿試|でんし}}  ]]が始まり、==皇帝と官僚の結びつきが強化されました  ==。
::
::middle
科挙には特別な受験資格は必要なく、男性であれば階層を問わずに受けられました。
::
::last
儒学・詩文を学んだ新興地主層が、科挙の主な合格者を輩出して[[形勢戸 ]]（[[官戸  ]]）と言われ、貴族にかわって勢力を伸ばしました。
::
### 北方民族からの圧迫
::top
宋は、契丹・西夏など北方民族の圧迫を受けました。
::
::last
北方民族への防衛費の増大は、国家財政の窮乏を招きました。
::

::top
11世紀後半の皇帝[[{{神宗|しんそう}} ]]は、王安石を宰相に任命しました。
::
::middle
宋の[[王安石 ]]は、[[新法  ]]と呼ばれる富国強兵の改革をおこないました。
::
::last
新法は、次の諸法で農民や中小商工業者の生活を守り（＝==地主や大商人の利益を抑え ==）、国家財政の再建を目指しました。
::

  :::lead

  :::
- 均輸法
  :::lead

  :::
- [[{{市易法|しえきほう}}]]
  :::lead
  中小商人への低利貸付け政策
  :::
- 募役法
  :::lead
  労役のかわりに免役銭を出させて希望者を雇用する方法
  :::
- 保甲法
  :::lead
  成年男子に軍事教練や治安維持を担わす政策
  :::

::
::last
新法党と旧法党の対立によって、宋は国力を弱めました。
::

## 宋の社会と経済
### 開封－宋の都
::top
趙匡胤によって建国された宋（北宋）の都である[[開封 ]]は、==黄河と大運河が合流する交通の要衝に位置します  ==。
::
::last
水路という利点を活かし、中国各地を結び付ける中枢として機能しました。
::

### 江南の開発
::top
南宋が建国されると江南の開発が進みました。
::
::middle
水はけの悪い低湿地が多かった長江下流域には、堤防で土地を囲んで干拓した囲田が造成されました。
::
::middle
開発された長江流域で、日照りに強い早稲種の[[占城稲 ]]が栽培されました。
::
::last
結果、==宋代には長江下流域は生産力が大いに高まり、中国経済の中心地となりました ==。
::
::top
宋代に長江下流域が穀倉地帯として発達し、==「{{蘇湖|そこ}}（{{江浙|こうせつ}}）熟すれば天下足る」と称されるようになりました ==。
::
::sup
蘇州・湖州一帯を蘇湖、江蘇・浙江一帯を江浙、湖北・湖南一帯を湖広と呼称
::
::sup
明代に穀倉地帯は長江中流域に移り、「[[{{湖広|ここう}} ]]熟すれば天下足る」と表現
::
