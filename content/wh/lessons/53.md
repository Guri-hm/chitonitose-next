---overview---
ローマ＝カトリック教会は、長くにわたって西ヨーロッパの精神的・文化的支柱でした。以前から教皇や聖職者への批判はありましたが、ルネサンス期にはルター・カルヴァンなどの改革者が、聖書をもとに信仰が個人の問題であると説き、教会や教皇の権威を否定しました。一方、教会側は自らの権威を回復するために対抗宗教改革に取り組みました。
---
## 宗教改革の開始
### 善行か信仰か
::top
16世紀、教皇[[レオ10世 ]]は、[[サン＝ピエトロ大聖堂  ]]の建築資金を調達するため、[[{{贖宥状|しょくゆうじょう}}  ]]を販売しました。
::
::last
贖宥状の購入は、教会にお金を寄付する善行であり、これを積めば受けるべき罰を{{赦|ゆる}}されるとされました。
::
::sup
サン＝ピエトロ大聖堂

::

::
::last
ルターは、「[[95か条の論題 ]]」を発表して、贖宥状の販売を批判しました。
::
::gazo

## 宗教改革の広がり
### スイスの宗教改革
::top
15世紀、[[ツヴィングリ ]]がスイスの都市[[チューリヒ  ]]で宗教改革を開始しました。
::
::last
追って、『[[キリスト教{{綱要|こうよう}} ]]』を著した[[カルヴァン  ]]が、スイスの都市[[ジュネーヴ  ]]で宗教改革をおこないました。
::
::gazo
::
::last
そしてカルヴァンは、==職業・動労に禁欲的に励むことを勧め、勤労の成果としての富の蓄積を容認しました ==。
::
---arrow---
::top
予定説を信じる人々は、自分が救われるかの確証を、職業・労働に励むことで得ました。
::
::sup
「職業・労働に励める人」＝「神に救われる人だろう」
::
#### 長老主義
::top
カルヴァンは、信者のなかから信仰の厚い者を長老に選び、牧師を補佐させる長老主義を採りました。
::
### カルヴァン派の広がり
::top
16世紀後半、カルヴァン派はフランス・ネーデルランド・イギリスなどに広がり、ルター派と並ぶ宗派になりました。
::
::last
ルター派やカルヴァン派など、ローマ教皇の至上権を認めず、聖職者の特権を否定する宗派を新教徒（プロテスタント）と総称します。
::
### イギリスの宗教改革
#### ヘンリ８世の治世
::top
国王ヘンリ８世は、スペイン王家出身の王妃との離婚を認めない教皇と対立し、宗教改革を始めました。
::
::middle
1534年、[[ヘンリ８世 ]]は国王を教会の首長とする[[イギリス国教会  ]]を創設しました。
::
::last
これに伴い、ヘンリ８世は==カトリックの修道院を解散し、その財産を没収しました==。
::
---arrow---
::top
結果、イギリスのキリスト教はカトリック世界から離脱しました。
::
::gazo
::
#### メアリ１世の治世
::top
女王メアリ１世はカトリックへの復帰を強行し、新教徒を迫害しました。
::
::last
メアリ１世が病死し、カトリックへの復帰は中止されました。
::
#### エリザベス１世の治世
::top
1559年、[[エリザベス１世 ]]は[[統一法  ]]を制定して礼拝の統一を図り、イギリス国教会を確立しました。
::
::gazo
::
::last
しかし、司教制や儀式など、カトリックを維持する点も残しています。
::
---arrow---
::top
イギリス国教会にカルヴァンの教えの徹底を求める信徒も存在し、ピューリタン（清教徒）と呼ばれました。
::
## 対抗宗教改革
### 思想統制の強化
::top
宗教改革に直面したカトリック教会は、勢力を建て直す改革[[対抗宗教改革 ]]をおこないました。
::
::last
1545年、[[トリエント公会議 ]]
:::lead
1545年、[[トリエント公会議 ]]
:::
、[[トリエント公会議 ]] ==教皇の至上権とカトリックの教義を再確認し、宗教裁判所による異端の取締り強化を決定  ==
::
### 教団の結成
::top
1534年、==スペイン人 ==[[イグナティウス＝ロヨラ  ]]がイエズス会を創始しました。
::
::middle
イエズス会は厳格な規律をもち、ヨーロッパ内外で布教してカトリック勢力の回復に努めました。
::
::last
海外での布教活動は、大航海時代の通商・植民活動と密接に関係しました。
::
---arrow---
::top
1549年、[[フランシスコ＝ザビエル ]]が日本に来航してキリスト教を伝えました。
::
### 社会的な緊張
::top
旧教徒と新教徒の対立が激化し、ヨーロッパ各地で宗教戦争がおこりました。
::
::last
この社会的緊張のなかで、「魔女狩り」がおこなわれた地域もありました。
