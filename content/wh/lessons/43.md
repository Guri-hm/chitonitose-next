---overview---
宋代の文化は、これまでの中国王朝の文化とは少し異なる特徴を見せました。儒学から派生しつつも大義名分論などの理論をもつ宋学（朱子学）が創始されたり、紀伝体であった歴史の記述が編年体で記されたりしました。
---
## 宋代の文化
### 磁器
::top
宋代に磁器は重要な貿易品として輸出され、竜泉や[[景徳鎮  ]]などが代表的産地となりました。
::
::last
また、宋の磁器は朝鮮半島へも輸出され、当時の磁器に大きな影響を与えました。
::
### 儒学
::top
北宋の{{周敦頤|しゅうとんい}}が始めた宋学は、儒学の{{経典|けいてん}}を哲学的に読み込み、==宇宙の原理や人間の本質などを探究しました  ==。
::
::last
南宋の[[{{朱熹|しゅき}}  ]]（[[朱子]]）は、経典の中でも『大学』『中庸』『論語』『孟子』の==四書を重視して、宋学を大成しました  ==。
::
::last
このように宋学は、朱熹（朱子）によって大成されたので[[朱子学  ]]とも呼ばれます。
::
---arrow---
::top
朱熹（朱子）は、==華夷や身分の別を重視する  ==[[大義名分論  ]]を説きました。
::
---arrow---
::top
宋代以降の支配階級は、概ね地主で、儒学の古典に精通して官僚となる士大夫と呼ばれる人々でした
::
::last
大義名分論は、これら士大夫に受け入れられました。
::
