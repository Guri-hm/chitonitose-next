---overview---
インドの人々は、大きくアーリア人とドラヴィダ人にわかれます。古くから異民族が侵入をくり返しましたが、侵入した民族は、それまでの社会の影響をうけつつも完全に同化されてしまうことはありませんでした。こうしてインド社会は、多くの民族・言語・宗教が共存する独自の世界を形成するようになりました。

## ヴェーダ時代以降
### 新たな宗教の成立とバラモン教の改革
#### 仏教の成立
::top
仏教は、[[ガウタマ＝シッダールタ]]（尊称：{{仏陀|ぶっだ}}）に創始されました。
::
::middle
ガウタマは、動物を犠牲に捧げる供犠、難解なヴェーダ祭式、バラモンを最高位とみなす==ヴァルナ制を否定しました==。
::
::last
ガウタマは、バラモン教の教えとは別に、輪廻転生からいかに脱却するかを説きました。
::
#### ジャイナ教の成立
::top
ジャイナ教は、[[ヴァルダマーナ]]に創始されました。
::
::last
ジャイナ教は、ヴェーダ聖典の権威や==バラモン教を否定し==、苦行と不殺生を強調しました。
::
#### バラモン教の改革
::top
バラモン教を否定する動きに対して、バラモン教に改革運動が生じました。
::
::last
祭式至上主義から内面の思索を重視する探究[[ウパニシャッド哲学]]が始められました。
::
### 都市国家の成長
::top
前６世紀頃、城壁で囲まれた都市国家がいくつか生まれました。
::
::last
その中から[[コーサラ国]]やガンジス川流域を統一した[[マガダ国]]が有力となりました。
::
---arrow---
::top
マガダ国では、この頃に成立した新しい宗教[[仏教]]や[[ジャイナ教]]を保護しました。
