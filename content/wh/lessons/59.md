---overview---
ヨーロッパ諸国は、アジアでの貿易の独占と植民地経営を許した東インド会社を設立し、海軍力を背景にアジアに進出していきました。各国は互いに活動の拡大を競い、時には現地での覇権をめぐって武力衝突を起こしました。
---

## 奴隷貿易と近代分業
### 奴隷貿易の開始
::top
16世紀以降、ブラジル・西インド諸島で、[[サトウキビ]]の大農園[[プランテーション]]が経営されました。
::
---arrow---
::top
アメリカの==スペイン植民地において ==、先住民インディオの人口が次の理由で減りました。
::

- 鉱山・農園での過酷な強制労働
- ヨーロッパからの伝染病

::last
減った労働力を補うため、アフリカの黒人奴隷が輸入されました。
::
---arrow---
::top
17世紀、砂糖・綿花・タバコ・コーヒーのプランテーションの経営で、より多くの黒人奴隷が必要になりました。
::
### 奴隷貿易のシステム
::top
黒人奴隷貿易は、次のような流通で成り立っていました。
::

- 西ヨーロッパから火器などを西アフリカで奴隷と交換
- 奴隷を西インド諸島などの植民地に運び、砂糖などを生産
- 砂糖などを西ヨーロッパに送って販売

::last
①～③の循環からこの貿易を三角貿易と呼びます。
::
---arrow---
::top
三角貿易は、イギリスなどに産業革命の前提条件である資本蓄積を促しました。
::
