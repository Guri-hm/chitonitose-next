---overview---
ローマの政治家カエサルは、独裁的な権力への野心を抱きました。この動きは、共和政の伝統を守ろうとする者たちによって阻止されました。カエサルの死後、オクタウィアヌスが台頭し、ローマ皇帝となって政治・軍事の実権を握ります。彼はカエサルと異なり、共和政の伝統を尊重しました。
---
## 地中海の平定
### 第１回三頭政治
::top
前１世紀、次の争いが続発し、ローマの混乱は頂点に達しました。
::

- 前１世紀初め、平民派のマリウスと閥族派のスラが互いに私兵を率いて争いを開始
- 前91～前88年、イタリア半島の同盟都市は、ローマ市民権を求めて反乱
- 前73～前71年、剣闘士（剣奴）[[スパルタクス]]による反乱

## ローマ帝国
### 事実上の皇帝独裁
::top
前27年、オクタウィアヌスは、元老院から[[アウグストゥス]]の称号を与えられ、事実上のローマ皇帝となりました。
::
::sup
アウグストゥス
:::lead
ゥス「尊厳者」の意
::
---arrow---
::top
アウグストゥスは、==共和政の伝統を尊重しつつも、政治・軍事の実権を握りました==。
::
::last
このように、名目的には元老院などの共和政の伝統を尊重する帝政を[[元首政]]と呼びます。
::
---arrow---
::top
以降約200年間の時代は[[ローマの平和]]（パクス＝ロマーナ）と呼び、繁栄と平和が続きました。
::
### ローマの平和（パクス＝ロマーナ）
::top
最盛期の98～180年には、次の５人の名君が現れ、[[五賢帝]]時代と呼ばれています。
::

- ネルウァ（在位96～98）
- トラヤヌス（在位98～117）
- ハドリアヌス（在位117～138）
- アントニヌス＝ビウス（在位138～161）
- マルクス＝アウレリウス＝アントニヌス（在位161～180）

---arrow---
::top
五賢帝の１人[[トラヤヌス帝]]のときにローマの領土は最大となりました。
::
::last
ローマ風の都市が国境付近まで建設され、後にロンドン・パリなど近代都市になったものも多くありました。
::

### ローマ帝国の支配と活動
::top
ローマは都市を通して属州を支配しました。
::
::middle
都市の上層市民は、ローマ市民権を与えられるかわりに帝国支配に貢献しました。
::
::last
他方、ローマ支配のもとで重税に苦しむ属州下層民もいました。
::
#### 全自由人の市民化
::top
212年、[[カラカラ帝]]のときには、帝国の全自由人にローマ市民権が与えられました。
::
### 帝国の瓦解
::top
五賢帝最後の治世末期、ローマ帝国のいきづまりが露わになりました。
::
::middle
３世紀末、軍人出身の皇帝の擁立・退位が頻発し、軍人皇帝の時代と呼ばれます。
::
::last
また、異民族が国境に侵入し、戦乱が止みませんでした。
::
#### 農奴制の先駆
::middle
大土地所有者は、==没落した自由農民やかつての奴隷などで構成された ==小作人（[[コロヌス]]）を用いるようになりました。
::
::last
この生産体制を[[コロナトゥス]]と呼び、従来のラティフンディアにとってかわりました。
::
---arrow---
::top
コロヌスには、==移動の自由が認められませんでした ==。
::
::middle
上述のように、==奴隷供給が困難となったことが、コロヌス制発展の一因となりました ==。
::
::last
コロヌス制は、西ヨーロッパ中世世界の==農奴制の先駆形態となりました ==。
