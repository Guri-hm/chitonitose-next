---overview---
1814～15年、ウィーン会議が開催され、ナポレオンによって混乱したヨーロッパ秩序の再建が目指されました。会議では、各国の政治体制をフランス革命以前に戻すことと各国の利害調整が協議されました。円滑に会議を進めるために親睦を深める舞踏会が催されましたが、各国の思惑は複雑に絡み、会議自体はなかなか進展しませんでした。「会議は踊る、されど進まず」です。
---
## ウィーン会議
### 会議開催
::top
1814～15年、ウィーン会議が開かれ、オスマン帝国を除く各国指導者が列席しました。
::
::last
会議では、オーストリア外相[[メッテルニヒ ]]を中心に、フランス革命とナポレオン戦争後のヨーロッパ秩序の再建が協議されました。
::
::gazo
::
#### フランス
::top
フランスのタレーラン外相が、革命前の王朝と旧制度の復活を目指す正統主義を主張しました。
::
::last
この主張で、フランスやスペインでブルボン王家の王朝が復活しました。
::
::sup
革命の反動として、旧体制に押し戻そうとする政治を「反動政治」とも表現
::
#### ロシア
::top
ポーランド王国が成立しましたが、ロシア皇帝が==ポーランド国王を兼ねました==。
::
::sup
ポーランド王国
:::lead
ポーランド王国
:::
王国ナポレオンが建国したワルシャワ大公国の大部分を再編した国
::
#### イギリス
::top
イギリスは、[[セイロン島 ]]（[[スリランカ]]）・ケープ植民地を[[オランダ  ]]から正式に獲得しました。
::
#### ドイツ

ライン同盟が廃止され、オーストリア・プロイセン以下35の君主国と４自由市で構成される[[ドイツ連邦 ]]が成立し、[[オーストリア  ]]が盟主となりました。
::
#### スイス

### ウィーン体制
#### 体制の成立
::top
ウィーン会議の結果、フランス革命とナポレオンの支配で広まった自由主義とナショナリズムが抑止されました。
::
::last
各国では、政治的現状維持を目指す保守的な政治（反動政治）がおこなわれました。
::
---arrow---
::top
列強の協議で勢力均衡と平和を維持する体制が定着し、ほぼ20世紀初めまで継続されました。
::
::last
この体制をウィーン体制と呼びます。
::
#### 四国同盟
::top
会議を主導したロシア・オーストリア・プロイセン・イギリスは[[四国同盟 ]]を結び、ウィーン体制を強化しました。
::
::sup
1818年、フランスも加わって五国同盟
::
#### 体制の２柱
::top
ウィーン体制の柱は、イギリス・ロシアの２国でした。
::
::top
19世紀、イギリスは圧倒的な経済力と軍事力をもち、その力で国際的な平和が保たれました。
::
::last
この状況は[[パクス＝ブリタニカ ]]と表現されます。
::
::sup
パクス＝ロマーナ（ローマの平和）になぞらえた言葉
::
::top
1815年、ロシア皇帝[[アレクサンドル１世 ]]が[[神聖同盟  ]]を提唱し、ヨーロッパ諸国を参加させてウィーン体制を補強しました。
::
## 体制の動揺
### 自由を求める運動
::top
ウィーン体制下、自由と統一を求める次の運動が続きました。
::

- [[ドイツ]]で、==学生同盟である==[[ブルシェンシャフト]]が結成され、運動を展開
- [[イタリア]]で秘密結社[[カルボナリ]]が結成され、反動政治に抵抗
- 1825年、[[デカブリストの乱]]
  :::lead
  西欧の自由主義を学び、自国の後進性を痛感したロシアの青年将校が改革を求めて蜂起
  :::
- 1821～30年、ギリシア独立戦争
  :::lead
  [[ギリシア]]がオスマン帝国から独立した戦争
  :::
- ラテンアメリカ諸国での独立運動

---arrow---
::top
ウィーン体制下、オーストリア外相メッテルニヒは、==民族運動を抑圧し ==、==ラテンアメリカ諸国の独立に反対しました  ==。
::
::middle
一方、イギリスの外相カニングは、==市場の拡大を狙い、ラテンアメリカ諸国の独立を援助しました ==。
::
::last
イギリスは、独立に関する考えの違いから、メッテルニヒの政策から次第に距離を置きました。
::
### 革命後のフランス
::top
ナポレオン戦争後、フランスは立憲君主政に戻りました。
::
::middle
ルイ18世の後を継いだ国王シャルル10世は、旧体制への復帰を目指し、制限選挙を採り、貴族・聖職者を重視しました。
::
::last
シャルル10世の政治は、フランス国民の不満を高めました。
::
---arrow---
::top
1830年、シャルル10世は国民の不満をそらすために、オスマン帝国支配下の[[アルジェリア ]]に派兵し、植民地化しました。
::
::last
また、反政府派が制限選挙で圧勝すると、未招集のうちに議会を解散しました。
::
