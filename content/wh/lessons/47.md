---overview---
17世紀、満州人の国家「清」が成立し、明にかわって中国を支配しました。清の前半期には有能な皇帝が続き、独裁的な権力をふるいました。明の制度の継承しつつも、独自の制度や民族の慣習を導入しました。
---
## 清の成立
### 女真人の国
::top
10世紀以後、中国の東北地方（満州）に住む[[ツングース ]]系の[[女真人  ]]（[[女真族、満州人、満州族]]）は、遼の支配下で半農半猟生活を営みました。
::
::middle
1125年、女真人が建国した[[金]]は、遼を滅ぼしました。
::
::middle
金は華北を支配しましたが、後にモンゴル帝国に滅ぼされました。
::
::last
以降、女真人は元・明の支配をうけました。
::
---arrow---
::top
1616年、女真人の[[ヌルハチ]]が明の支配から脱して、[[後金]]を建国しました。
::

- [[満州]]文字の制定

### 明の滅亡
::top
16世紀後半の明では、幼少の皇帝{{万暦帝|ばんれきてい}}を補佐する[[{{張居正|ちょうきょせい}} ]]の下で、財政再建などの政治改革が進められました。
::
::middle
再建に成功した反動で財政が放漫になり、また、宦官による政治への介入が目立ちました。
::
::middle
官僚の一部は、政治の腐敗や宦官の横暴を批判する[[東林派 ]]（[[東林党]]）というグループを作りました。
::
::last
==官僚は、東林派と非東林派に分かれて党争を展開し、政治の混乱を招きました==。
::

::
---arrow---
::top
政治の混乱に反発した民衆は、各地で暴動を起こしました。
::
::last
1644年、[[{{李自成|りじせい}} ]]の農民反乱軍が北京を占領し、明は滅亡しました。
::
### 清の中国統一
#### 順治帝の治世
::top
明の滅亡後の1644年、順治帝の清軍が明の武将[[{{呉三桂|ごさんけい}} ]]の協力を得て、北京に入城しました。
::
::middle
清は、北京に遷都し、中国全土へと支配を広げました。
::
::last
雲南・広東・福建の支配は、清の中国平定に協力した呉三桂含む３人の明の武将（三藩）に委ねました。
::

::

- 1661年、[[{{鄭成功|ていせいこう}}]]がオランダ人の勢力を駆逐して[[台湾]]を占領し、ここを拠点に清に抵抗
- 1673～81年、[[三藩の乱]]
  :::lead
  康熙帝が三藩を撤去しようとすると、[[呉三桂]]らが起こした反乱
  :::
- 康熙帝率いる軍がオイラトの一部族ジュンガル部と戦争
- モンゴル人に影響力をもつチベットを支配

#### 雍正帝の時代
::top

::

- 1727年、[[キャフタ条約]]
  :::lead

  :::
- 軍事・行政上の最高機関[[{{軍機処|ぐんきしょ}}]]の設置

#### 乾隆帝の時代
::top
1758年、６代皇帝[[乾隆帝|けんりゅうてい]]は、[[ジュンガル部 ]]を征服しました。
::
::middle
乾隆帝は、ジュンガル部・回部（ウイグル人イスラーム教徒）を平定し、東トルキスタンを「[[{{新疆|しんきょう}} ]]（新しい土地）」と呼びました
::
::last
18世紀半ば、清の最大領域は、ほぼ今日の中国の領土の原型をつくりました。
::
## 清の制度・統治方法
### 清の皇帝
#### ２つの面
::top
清の皇帝は、次の２つの面をもちました。
::

- 中国歴代王朝の伝統を継ぐ皇帝
- モンゴル帝国のハンの伝統を継ぐ北方遊牧民の君主

::last
清の前半は、有能な皇帝が続き、上記２つの面を兼ね備えて独裁的な権力をふるいました。
::
#### 宮殿

### 清の制度
::top
清は、明の制度を継承する一方で、独自の制度も創設しました。
::
#### 思想
::top
清は科挙・儒学を振興し、中国王朝の伝統を守る姿勢を示しました。
::
::middle
文化面では、『[[康熙字典 ]]』『[[{{四庫|しこ}}全書  ]]』を編纂させ、学者を優遇しました。
::
::last
一方で、禁書の設定や、反満州人・反清の文章を書いた者を処罰する[[{{文字|もんじ}}の獄 ]]をおこない、思想統制しました。
::
::sup

科挙
:::

::
#### 官制
::top
雍正帝の時代に、軍事・行政上の最高機関[[軍機処|ぐんきしょ]]が設置されました。
::
#### 軍制
::top
==漢人で組織する ==[[緑営  ]]と、満州人・モンゴル人・==漢人の  ==３軍で編制する[[八旗]]を、要地に駐屯させました。
::
#### 風習
::top
支配下の漢人男性に対して、満州人の風習である髪型{{辮髪|べんぱつ}}（弁髪）を==強制しました ==。
::
::last
辮髪の拒否は、清への反抗を意味しました。
::
#### 宗教
::top
白蓮教などの民間宗教は、邪教として弾圧されました。
::
### 統治方法
::top
清は、直接統治・間接統治を組み合わせて領土を支配しました。
::
#### 直轄領
::top
中国内地・==中国東北地方 ==・台湾を直轄領としました。
::
#### 藩部
::top
==モンゴル ==・==チベット  ==・新疆・{{青海|せいかい}}を[[藩部  ]]（≠直轄領）としました。
::
::last
清は[[理藩院 ]]を設け、藩部を管理・監督させました。
::
---arrow---
::top
藩部の実際の支配は、現地の支配者と清の派遣する監察官に任されました。
::
::last
清は藩部の習慣・宗教にあまり干渉せず、支持を得ながらの支配を目指しました。
::
::top

::top

::
::top

## 16～17世紀の東アジア
::top
東アジアや東南アジアの諸国は、清の盛期に朝貢をおこない、清から属国として扱われました。
::
::last
その一方で、諸国は内部で独特の国家意識を形成していきました。
::
### 朝鮮
::top
16世紀以降の朝鮮では、[[{{両班|ヤンバン}} ]]と呼ばれる支配階層が形成され、政治的実権を握っていました。
::
::sup
両班
:::lead
両班
:::
v class="lead">文官（文班）・武官（武班）の官職に就いている者の総称
::
::last
両班は、==党派に分かれて争うようになり、政治の混乱を招くこともありました ==。
::
---arrow---
::top
16世紀後半、日本の豊臣秀吉は、領土の拡大を目指しました。
::
---arrow---
::top
ホンタイジの時代、朝鮮は清に服属して朝貢関係に入りました。
::
::middle
しかし、朝鮮は自国こそ中国文化の継承者という「小中華」を意識し、北方民族出身の清に対抗心を抱きました。
::
::last
そのため、両班は清以上に==儒教の儀礼や規範を重視しました ==。
::
### 琉球王国
::top
17世紀初め、琉球王国は日本の薩摩藩の大名島津氏に服属しました。
::
::middle
琉球王国は==中国への朝貢を続け ==、日本と中国の両方に服属する状態となりました。
::
::last
そのなかで、日本・中国両方の要素を含む文化が形成されました。
::
### 日本
::top
17世紀初頭、徳川家康が江戸幕府を開きました。
::
::last
家康は朱印船貿易を促進し、日本人が==東南アジアに ==日本町を形成しました。
::
---arrow---
::top
東アジアでは、日本・中国・ポルトガル・オランダが銀と生糸の貿易を盛んにおこないました。
::
::middle
やがて江戸幕府はキリスト教禁止や貿易統制を強化しました。
::
::middle
1630年代、江戸幕府は日本人の海外渡航やポルトガル人の来航を禁じる鎖国をおこないました。
::
::last
ただし鎖国後も、長崎を通じて中国・オランダ、対馬を通じて朝鮮、琉球を通じて中国、というように==隣接諸国との国交は継続しました ==。
