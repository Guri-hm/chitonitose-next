---overview---
イギリスがインド支配を強めたことをうけ、西欧列強は東南アジアの植民地化を狙いました。ジャワ島、マレー半島、フィリピン、インドシナ半島が植民地化されていき、東南アジアのうちでタイのみが植民化を回避できました。
---
## 植民地化
### ジャワ島
#### ジャワ島の征服
::top
1623年、[[アンボイナ事件 ]]
:::lead
、[[アンボイナ事件 ]] インドネシアのモルッカ諸島で、オランダ人がイギリス商館員を虐殺した事件
::
::sup
モルッカ諸島
:::lead
島別名は香料諸島（スパイス諸島）
::
---arrow---
::top
アンボイナ事件後、[[オランダ ]]はジャワ島へ進出し始めました。
::
::middle
18世紀半ば、オランダはマタラム王国を滅ぼしました。
::
::last
これにより、オランダはジャワ島の大半を支配下に置きました。
::
#### ジャワ島の経営
::top
19世紀、オランダはジャワ島の住民に==コーヒー・サトウキビ・藍を ==生産させました。
::
---arrow---
::top
1825～30年、ジャワ戦争
:::lead
30年、ジャワ戦争オランダの暴政に対するジャワ島の大反乱
::
---arrow---
::top
1830年以降、オランダはジャワ島の住民にコーヒーなどを生産させ、安価で買い上げる[[強制栽培制度 ]]を導入しました。
::
::last
オランダは==莫大な利益をあげ ==、一方でジャワ島は飢饉で苦しみました。
::
### マレー半島
#### マレー連合州
::top
18世紀末からイギリスはマレー半島に進出しました。
::
::last
さらに、オランダが支配するジャワ島を一時占領しました。
::
---arrow---
::top
1824年、イギリスはオランダと協定を結び、ジャワ島を返還して、今後のマレー半島での支配拡大を認めさせました。
::
::last
1826年、[[イギリス ]]はマレー半島沿岸のマラッカ・ペナン・シンガポールを拠点に、[[海峡植民地  ]]を形成しました。
::
::sup
海峡植民地
:::lead
当初は東インド会社の管轄で、1867年以降にイギリス本国の==直轄領 ==
::
---arrow---
::top
1895年、イギリスはマレー半島内部の国々を保護国化し、[[マレー連合州 ]]（[[マライ連邦]]）を結成させました。
::
#### マレー半島の経営
::top
[[イギリス ]]は、マレー半島に[[ゴム  ]]栽培のプランテーションを形成しました。
::
::last
プランテーションの労働者として、南インドから大量の移民が導入されました。
::
### ビルマ
::top
1752年、[[コンバウン朝 ]]がビルマ（現ミャンマー）におこりました。
::
---arrow---
::top
1824～86年、[[ビルマ戦争 ]]
:::lead
86年、[[ビルマ戦争 ]] イギリスとコンバウン朝の３回にわたる戦争
::
### フィリピン
::top
16世紀後半以降、フィリピンは[[スペイン ]]の植民地とされました。
::
::last
スペインは、フィリピン人をキリスト教の宗派の一つ[[カトリック ]]に改宗させました。
::
---arrow---
::top
1834年、スペインは自由貿易をおこなうため、マニラを開港しました。
::
::last
商人や高利貸しが土地を集め、サトウキビ・麻・タバコなどのプランテーションを形成しました。
::
### ベトナム・カンボジア・ラオス
#### 阮朝の成立
::top
16世紀以降、ベトナムでは黎朝の名目的支配のもとで鄭氏・阮氏が対抗を続けました。
::
---arrow---
::top
1771年、西山の乱
:::lead
、西山の乱鄭氏・阮氏、黎朝を滅ぼし、ベトナムを統一させた農民反乱
::
---arrow---
::top
1802年、阮氏の生き残りの[[{{阮福暎|げんふくえい}} ]]が、フランス人宣教師ピニョーの協力を得て西山政権を倒し、[[阮朝  ]]（[[越南  ]]）を興しました。
::
::last
1804年、阮福暎は清によってベトナム国王に封ぜられました。
::
#### フランスの進出
::top
19世紀半ば、フランスはカトリック教徒への迫害を理由に、ベトナムに軍事介入しました。
::
::last
1867年、フランスはベトナム南部を奪いました。
::
::top
ベトナム北部では、[[{{劉永福|りゅうえいふく}} ]]が組織した[[黒旗軍]]が、[[フランス  ]]の植民地支配に抵抗しました。
::
::last
フランスはこの抵抗も口実にして北部に進出しました。
::
---arrow---
::top
1883年、[[ユエ条約 ]]
:::lead
、[[ユエ条約 ]] フランスがベトナムを保護国とした条約
::
::top
1884～85年、[[清仏戦争 ]]
:::lead
85年、[[清仏戦争 ]] [[ベトナム  ]]への宗主権を主張する清朝が、==フランスがベトナムを保護国化したことに  ==反対して宣戦布告
::
---arrow---
::top
1885年、[[天津条約 ]]
:::lead
、[[天津条約 ]] 清朝が、フランスによるベトナムの保護国化を認めた条約
::
::top
1887年、フランスは保護国化したベトナムと[[カンボジア ]]（1863年、保護国化）を併せて[[フランス領インドシナ連邦  ]]を成立させました。
::
::last
フランス領インドシナ連邦成立後の1899年、[[ラオス ]]が編入されました。
::

ラタナコーシン朝国王[[ラーマ５世 ]]（チュラロンコン）は、次の状況と政策で植民地化を回避しました。
::

- タイがイギリス・フランス両勢力の緩衝地帯に位置
- 外国留学や外国人専門家の招致で近代化政策を推進
