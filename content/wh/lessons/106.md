---overview---
冷戦における米ソの対立は、朝鮮戦争などの代理戦争や核開発競争というように間接的なものでした。しかし1962年、ソ連がアメリカに狙いを定めたミサイル基地をキューバに建設すると、米ソの軍事的緊張が一気に高まりました。いわゆるキューバ危機です。米ソ首脳の交渉で核戦争の危機は回避され、両国は関係改善に乗り出しました。
---
## 米ソの高まる緊張と緩和への転換
### キューバ危機
#### アメリカ大統領ケネディの登場
::top
1961年、アメリカでは民主党の[[ケネディ ]]が、初の==カトリック教徒  ==の大統領として就任しました。
::
::last
ケネディは==「ニューフロンティア」というスローガンを掲げ、積極的な国内改革を推進しました ==。
::
#### キューバ危機と核戦争の回避
::top
1962年、ソ連がキューバにミサイル基地を建設していることが発覚しました。
::
::middle
この事態に、アメリカ大統領[[ケネディ ]]は、ミサイル資材を運搬するソ連船を海上で封鎖しました。
::
::last
米ソともに譲歩せず、核戦争に発展しかねない状況、{{所謂|いわゆる}}[[キューバ危機 ]]が発生しました。
::

::top
キューバ危機は、ケネディとフルシチョフによる次の合意で回避されました。
::
::top
アメリカ：キューバの内政への不干渉
::
::top
ソ連：キューバのミサイル基地の撤去
::
### 核管理体制の強化
::top
キューバ危機を経験した米ソは、核戦争の危機に直面したことで、次のように核管理体制を強化しました。
::

- 1963年、部分的核実験禁止条約
  :::lead
  アメリカ・ソ連・イギリスが、地下実験を除き、核兵器実験を禁止した条約
  :::
- 1968年、[[核拡散防止条約]]（ＮＰＴ）
  :::lead
  核兵器保有国をアメリカ・ソ連・イギリス・フランス・中華人民共和国の五ヵ国に限定し、非保有国への核兵器譲渡や製造援助を禁止した条約
  :::
- 1969年、第１次[[戦略兵器制限交渉]]（第１次{{ＳＡＬＴ|ソルト}}）
  :::lead
  アメリカ大統領[[ニクソン]]と第一書記[[ブレジネフ]]がおこなった、戦略ミサイルの数量の制限を目指した交渉
  :::

### アメリカの動揺
#### ケネディ政権期
::top
1950年代から、アメリカでは[[キング牧師]]が公民権運動を指導しました。
::
::middle
アメリカ大統領ケネディは、運動に理解を示し、公民権法を提案しました。
::
::last
しかし1963年、[[ケネディ ]]は南部遊説中に暗殺されました。
::
#### ジョンソン政権期
::top
1963年、[[ケネディ ]]が暗殺され、後継として[[ジョンソン  ]]が大統領に就任しました。
::
::last
ジョンソンは、アメリカから差別と貧困をなくす「偉大な社会」計画を提唱しました。
::
::top
1964年、[[公民権法 ]]成立
:::lead
、[[公民権法 ]]成立人種・宗教・性・出身国などによる差別を禁止する法
::
::top
1965年、==北ベトナムへの爆撃==とベトナム戦争の開戦
::
::top
1968年、キング牧師が暗殺され、人種暴動が発生
::
### 東側陣営の停滞
::top
1964年、ソ連の首相兼第一書記フルシチョフが解任されました。
::
::middle
コスイギンがソ連首相に、[[ブレジネフ ]]が第一書記に就任し、国内の自由化の進展を抑えました。
::
::last
一方、他の社会主義国のなかでは、自由化の動きが進み始めました。
::
---arrow---
::top
1968年、[[プラハの春 ]]
:::lead
、[[プラハの春 ]] [[チェコスロヴァキア  ]]で民主化を求める運動が起こり、共産党書記長[[ドプチェク  ]]の下で自由化路線が進行
::
---arrow---
::top
以降、社会主義国では改革が阻まれ、政治や経済が停滞しました。
::
## ヨーロッパでの緊張緩和
### 米ソの緊張緩和の影響
::top
キューバ危機後、米ソの緊張が緩和すると、その影響がヨーロッパに現れました。
::
---arrow---
::top
1975年、全欧安全保障協力会議（ＣＳＣＥ）
:::lead
、全欧安全保障協力会議（ＣＳＣＥ）アルバニアを除く全ヨーロッパ諸国、アメリカ、カナダの首脳がヘルシンキに集まり、主権尊重などをうたったヘルシンキ宣言を採択
::
### 西ドイツの東方外交
::top
1969年、西ドイツで社会民主党を中心とする連立政権が成立しました。
::
::last
西ドイツ首相[[ブラント ]]は、ソ連や東欧諸国との関係改善を図る[[東方外交  ]]を推進しました。
::
### 東西ドイツの接近
::top
1972年末、西ドイツと東ドイツが相互承認し、1973年、両国の国際連合加盟が実現しました。
::
### 軍事政権・独裁体制の崩壊
::top
1970年代、南ヨーロッパでは軍事政権や独裁体制が打倒されました。
::
#### ポルトガル
::top
ポルトガルは、植民地での独立運動をうけ、体制が動揺しました。
::
::last
1974年、ポルトガルの独裁体制が崩壊しました。
::
#### スペイン
::top
1975年、スペインのフランコが死亡し、ファン＝カルロス１世が後継に指名されました。
::
::last
1978年、スペインは新憲法により民主的君主制に移行しました。
::
#### ギリシア
::top
1967年以来、ギリシアは軍事政権になりました。
::
::last
1975年、ギリシアは民主政に復帰しました。
::
