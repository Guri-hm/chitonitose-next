---overview---
明代の後期は、農業・商業・手工業・学術など種々の面で、目覚ましい進展の見られた時代でした。ヨーロッパから訪れた宣教師が、キリスト教の布教活動の一環として暦法・数学なども紹介し、士大夫たちに受容されていきました。
---
## 冊封体制の崩壊
### 大航海時代の到来
::top
16世紀、大航海時代で世界的に商業が発達しました。
::
::middle
東南アジアでは、新興国家やヨーロッパ諸国が、胡椒など香辛料をめぐって争いました。
::
::last
これらの国家は自ら軍事力を強化し、明の国力・権威を頼る冊封体制に入りませんでした。
::
### 貿易統制の動揺
::top
16世紀、交易を求める人々が、侵入・略奪という行為で明の統制打破を目指しました。
::
::middle
16世紀の明は、北方のモンゴルの侵入や南（東）方の倭寇の略奪という外患[[{{北虜南倭|ほくりょなんわ}} ]]に苦しみました。
::
::last
北虜南倭に苦しんだ明は、北方の長城の大規模な改修をおこなったりしました。
::
#### 北方のモンゴル－北虜
::top
16世紀、モンゴル諸部族（韃靼、タタール）が、[[アルタン＝ハン ]]のもとで強盛となりました。
::
::last
アルタン＝ハンはフフホトなどの中国風の城郭都市を建設し、長城を越えて明への侵入を繰り返しました。
::
::sup
アルタン＝ハン
:::lead
アルタン＝ハン
:::
ハン都市フフホトを本拠とし、チベット仏教の一派[[黄帽派 ]]に帰依
::
#### 東シナ海の倭寇－南倭
::top
[[倭寇]]は南倭とも呼ばれ、中国・朝鮮半島沿岸で密貿易や略奪をおこないました。
::
::last
倭寇の代表的人物として中国出身の王直が知られ、日本の五島列島などを拠点に活動しました。
::
::sup
日本人中心の14~15世紀の倭寇を前期倭寇、中国人中心の16世紀の倭寇を後期倭寇と区別
::
### 海外貿易の許可と銀の流入
::top
明は貿易統制を止め、モンゴルとの交易場を設けたり、民間人の海外貿易を許したりしました。
::
::last
結果、明代後期には==スペイン人がもたらしたメキシコ銀や、ポルトガル人がもたらした日本銀が流通し、貨幣経済が活発化しました==。
::
## 明後期の社会
### 穀倉地帯
::top
==明代 ==、穀倉地帯が長江==中流域  ==の湖広（現在の湖北・湖南省）に広がり、「湖広熟すれば天下足る」と言われました。
::
### 陶磁器
::top
江西省の[[{{景徳鎮|けいとくちん}} ]]では、陶磁器が盛んに生産されていました。
::
### 商業・商人
::top
明代に、商品の全国的な流通網が成立し、[[{{山西|さんせい}}商人 ]]や[[{{徽州|きしゅう}}（新安）商人  ]]が活躍しました。
::
::last
そして、各地の都市に同郷・同業者の互助施設[[会館 ]]・[[公所  ]]が置かれました。
::
### 地方の有力者
::top
明代、引退した地主官僚は、[[{{郷紳|きょうしん}} ]]と呼ばれる地方有力者として成長しました。
::
### 税制度の改革
::top
税の納入が銀でおこなわれるようになりました。
::
::last
16世紀、各種税や徭役を銀で納入する[[一条鞭法 ]]が、実施されました。
::

### 儒学の新しい展開
::top
16世紀、[[王陽明]]（王守仁）が、人は本来心のなかに真正の道徳をもつという[[心即理 ]]を主張しました。
::
::middle
また王陽明は、行動よりも知識や修養を重視し過ぎる朱子学の傾向を批判しました。
::
::last
やがて王陽明は、ありのままの心を発揮し（致良知）、心のままに行動を起こす[[知行合一 ]]を説き、[[陽明学  ]]を興しました。
::
---arrow---
::top
陽明学は、学者のみならず庶民からも広く支持されました。
::
