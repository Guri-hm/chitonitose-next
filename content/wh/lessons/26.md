---overview---
隋・唐の時代には、中央集権退勢が強化され、貴族層は魏晋南北朝時代に比べると大きく勢力を後退させたが、しかし、なお大きな政治・社会的勢力を保持していました。そうした意味で、隋・唐の時代は貴族政治の末期・終焉期としてもとらえることができます。また、隋・唐の隆盛は日本を含む東アジア周辺諸国に大きな影響を与え、諸国の国家形成に大きな役割を果たしました。
---

::top

### 唐の文化
#### 首都－長安
::top
唐の首都[[長安]]は、広大な計画都市として造営されました。
::
::last
特筆すべきことは、次の通りです。
::

- 中央の大通りを軸とする==東西対称の==都市として建設
- アジア各地の==首都建設のモデル==（日本の平城京・平安京、渤海の上京竜泉府の都城などに影響）
- アジア各地から==留学生や留学僧、長江使節が訪問==（日本からは遣唐使が訪問）
- 仏教寺院・道教寺院のほか、==景教（ネストリウス派キリスト教）の寺院やゾロアスター教の寺院を築造==

## 唐の動揺・滅亡
### 唯一の女性皇帝
::top
７世紀末、高宗の皇后[[則天武后|そくてんぶこう]]が帝位につきました。
::
::middle
則天武后は唐の国号を[[周]]と改めました。
::
::last
科挙で積極的に官吏を登用する一方で、その権力で政治の混乱を招きました。
::
::sup
周

class="lead">705年、再び国号を唐へと改号
::
### 晩節を汚した皇帝
::top
８世紀初め、皇帝[[玄宗|げんそう]]は、則天武后以来混乱していた唐の再興に努めました。
::
::last
玄宗の政治は、貞観の治と並び称され、「[[開元の治]]」と呼ばれます。
::
---arrow---
::top
この時代には貧富の差が開き、また、均田制・租庸調制・府兵制が崩れていました。
::
::last
玄宗は、傭兵をもちいる[[募兵制]]を採用し、その指揮官[[節度使]]に辺境の防備をまかせました。
::
#### 傾国の美女
::top
治世の後半、玄宗は[[楊貴妃|ようきひ]]を寵愛し、政治を考えなくなりました。
::
::last
楊貴妃の一族が実権を握り、政治が混乱しました。
::
