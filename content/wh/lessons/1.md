---overview---
洪水と異民族の侵入を頻繁に受けたメソポタミアでは、王国の興亡が繰り返し、複雑な歴史が展開した。メソポタミアで最初に活躍し始めたのは、謎の多いシュメール人である。シュメール人は、楔形文字や法律など優れた文明を創り上げた。この文明は他民族にも継承され、例えばアムル人の『ハンムラビ法典』が有名である。
---
## 古代オリエント
### オリエントの風土
::top
ヨーロッパから見た「太陽の昇るところ（東方）」、今日の中東にあたるエジプト・西アジアは[[オリエント]]と呼ばれます。
::
::middle
オリエントは気温が高く、砂漠・岩山の地域が多く広がります。
::
::last
そのため、河川やオアシスで麦やナツメヤシを栽培します。
::
---arrow---
::top
大河では早くから=={{灌漑|かんがい}}農業==・定住化が進みました。
::
::last
次の①②の地域の大河流域には、前3000年頃から高い文明が発達しました。

## メソポタミアの統一
### アッカド王国―最初の統一国家
::top
前3000年頃、アラビア方面からセム語系の[[アッカド]]人がメソポタミア中部に移動・定着しました。
::
::middle
前24世紀、王[[サルゴン１世]]がメソポタミアを統一しました。
::
::last
他地域に遠征を繰り返しましたが、まもなく王国は滅びました。
::
---arrow---
::top
前22世紀、シュメール人の国ウル第３王朝がメソポタミアを支配しました。
::
::gazo
::
::last
前18世紀、[[ハンムラビ王]]の治世下でメソポタミアが統一されました。
::
---arrow---
::top
ハンムラビ王は治水・灌漑を進め、また、かつてのシュメール法を集成して『[[ハンムラビ法典]]』を制定しました。
::
::last
『ハンムラビ法典』は、「目には目を、歯には歯を」という[[{{復讐|ふくしゅう}}法]]の原則や、身分による刑罰差の原則を特徴とします。
::
::sup
ハンムラビ法典
:::lead
ハンムラビ法典
:::
法典[[楔形文字]]で記録された石碑が、ペルシアの古都スサで発見
::
::gazo
::
---arrow---
::top
17世紀、インド＝ヨーロッパ語族の[[ヒッタイト]]人は、アナトリア半島（小アジア）に王国を建国しました。
::
::last
メソポタミアに遠征して古バビロニア王国を滅ぼしました。
::
::sup

::
