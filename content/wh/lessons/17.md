---overview---
東南アジアは、インドシナ半島などの大陸部と、インドネシアやフィリピンを含む島々からなる諸島部（<ruby>島嶼<rt>とうしょ</rt></ruby>部）で構成されます。これらの地域は、香辛料をはじめとする資源の豊かさゆえに、外の世界と交易でつながり、インドや中国、イスラームの影響をうけつつ独自の文明を築きました。
---
## 大陸部の国の形成
### 現在のベトナム周辺
#### 前４世紀～
::top
前４世紀、ベトナム北部を中心に独特の青銅器を生み出した[[ドンソン]]文化が発展しました。
::
::last
この文化を象徴する青銅器[[銅鼓|どうこ]]は、中国南部から東南アジアにわたる各地で発見されています。
::

銅鼓

::last
この国はインド文化の影響を受けました。
::
::gazo
![チャンパー](img/17/champa.svg){.border.border-1}
チャンパー
::
::last
しかし、同地域で少しずつ独立の動きが強まりました。
::
---arrow---
::top
11世紀、{{李公蘊|りこううん}}が[[大越国]]（李朝）を建国しました。
::
::middle

::
::last
陳朝では、ベトナム語を書くために漢字を利用した[[字喃]]（[[チュノム]]）と呼ばれる文字が作られました。
::
::sup

::
::last
この国は、港[[オケオ]]を中心にした港市国家で、海の道の中継地として繁栄しました。

### 現在のタイ周辺
#### ７世紀～11世紀
::top
７世紀、[[モン]]人がチャオプラヤ川下流に[[ドヴァーラヴァティー王国]]を建国しました。
::
#### 13世紀
::top
13世紀、タイ人がチャオプラヤ川流域に、タイ人最古の王朝である[[スコータイ朝]]を建国しました。
::
::last
歴代の王の信仰もあり、[[上座部（小乗）]]仏教が広まりました。
::
## 諸島部（島嶼部）の国の形成
### スマトラ島
#### ７世紀
::top
７世紀、シュリーヴィジャヤ王国（スリウィジャヤ王国）が成立しました。
::
::middle
この王国では、仏教が栄えました。
::
::middle
７世紀、[[義浄]]は、インドへの往復の途中滞在し、仏教が盛んな様子を記しています。
::
::last
また、この王国では海上交易も栄え、[[唐]]に朝貢使節を派遣しました。
::
::double
義浄－『南海寄帰内法伝』より
咸亨かんこう
２年（671年）11月、広州（広東）から船に乗り、帆を挙げて南海に向かい、諸国を歴訪して西に向かった。咸亨４年（673年）２月８日、ようやく東インドの海口（ガンジス河口）に到着した。翌年の５月には、再び西に向かいナーランダに到着した。そして、仏教の聖跡すべてを歴訪した。
