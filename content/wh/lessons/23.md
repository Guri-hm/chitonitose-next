---overview---
所謂「三国志」の時代を経て、晋が成立しましたが、その滅亡後は非漢民族と漢民族の国々が中国に分立していきました。これらは五胡十六国と総称されます。後に華北は北魏によって統一されましたが、華南の王朝との並立は589年まで解消されませんでした。
---
## 三国時代から五胡十六国時代へ
### 三国時代
::top
３世紀初頭に中国で黄巾の乱が起きて以降、各地に軍事集団が現れ、混乱が生じていました。
::
::middle
220年、{{曹操|そうそう}}の子{{曹丕|そうひ}}が後漢の皇帝から位を譲られ、華北に[[魏|ぎ]]を建国しました。
::
::middle
これをうけ、孫権が==長江下流域に==[[呉]]を、{{劉備|りゅうび}}が==内陸の四川に==[[蜀]]を建国しました。
::
::last
３世紀前半の中国は、魏・呉・蜀が三分する[[三国時代]]に入りました。
::
::gazo
![三国時代](img/23/three-kingdoms-period.svg){.border.border-1}
三国時代
::

- 貧民を集めて国が管理する土地を耕作させる[[{{屯田|とんでん}}制]]を導入
- ==倭の邪馬台国の卑弥呼が魏に朝貢し、これに「親魏倭王」の称号を授与==

### 分立の終わり
::top
魏が最も優勢になり、[[蜀]]を滅ぼしました。
::
::last
しかし、魏では[[司馬炎|しばえん]]が国を奪い、265年、魏に替わる[[晋]]（[[西晋]]）を建てました。
::
::sup
西晋
:::lead
復興後の晋と区別するための呼称
::
::gazo
![晋](img/23/shin.svg){.border.border-1}
晋
::
---arrow---
::top
乱のなかで晋は匈奴に滅ぼされました。
::
::last
しかし、皇帝一族の[[司馬睿|しばえい]]が==現在の南京==にあたる建康で即位し、[[晋]]（[[東晋]]）を復興させました。
::
::sup
東晋
:::lead
復興前の晋と区別するための呼称
::
---arrow---
::top
西晋滅亡・東晋復興の混乱のなか、華北には匈奴など五胡が進出し、国家を建国しました。
::
::last
多くの国家が興亡を繰り返し、それら国家は[[五胡十六国]]と総称されます。
::
## 騎馬遊牧民の動向
### 五胡十六国時代まで
::top
前１世紀、匈奴は漢の[[武帝]]に敗れた後、東西に分裂しました。
::
::middle
さらに１世紀、匈奴は==南北に分裂しました==。
::
::last
匈奴の分裂に乗じて、他の遊牧民の活動も始まりました。
::
---arrow---
::top

::
::top
中国から見て、北方に匈奴の一派である[[羯|けつ]]が、東北に鮮卑が、西方に{{氐|てい}}・{{羌|きょう}}が勢力を形成しました。
::
::middle
匈奴を含めたこの５つの遊牧民は[[五胡|ごこ]]と呼ばれ、たびたび中国に侵入しました。
::
::last
４世紀以降、五胡が華北に進出して国を建てますが長続きせず、諸国が興亡を繰り返す[[五胡十六国時代]]に入ります。
::
### 五胡十六国の統一
::top
５世紀、[[鮮卑]]出身の{{拓跋|たくばつ}}氏が建てた[[北魏]]の{{太武帝|たいぶてい}}が、華北を統一しました。
::
::gazo
![北魏](img/23/hokugi.svg){.border.border-1}
北魏
::

- 村落のなかを、５つの家で「隣」、５隣で「里」、５里で「党」とまとめ、それぞれに長を置く[[三長制]]を開始
- 鮮卑の服装や言語を禁止し、中国文化を取り入れる==漢化政策を推進==
- ③を進めるため、[[洛陽]]に遷都

::last
北魏は漢化政策への反発で内部分裂し、==西魏と東魏に分裂しました==。
::
::sup
均田制
:::lead
後に隋・唐にも継承される制度で、北魏では女性にも土地を分与
::
---arrow---
::top
西魏は北周に倒され、東魏は北斉に倒されました。
::
::middle
北斉は北周に倒されました。
::
::last
北魏から北周までの５つの王朝を[[北朝]]と呼びます。
::
::top
５世紀、東晋の劉裕が実権を握り、東晋に替わる[[宋]]を建国しました。
::
---arrow---
::top
その後、宋に替わって、斉・梁・陳の王朝と順に興亡しました。
::
::last
宋から陳までの４つの王朝を[[南朝]]と呼びます。
::
