---overview---
イスラームが中央アジアに及ぶにつれて、非アラブ人によるイスラーム王朝の成立が目立つようになりました。トルコ系のセルジューク朝やモンゴル系のイル＝ハン国がその代表例です。セルジューク朝のもとでは、後にトルコ系のイスラーム王朝で用いられていく「スルタン」の称号が使用され始めます。カリフとスルタンの関係は、日本史の天皇と将軍の関係と対比すると理解しやすいです。
---
## トルコ人の王朝
### セルジューク朝
#### スルタンの地位
::top
11世紀初め、トルコ人[[トゥグリル＝ベク]]が、トルコ人によるイスラームの王朝[[セルジューク朝]]を建国し、中央アジアから西方に進出しました。
::
::middle
1055年、トゥグリル＝ベクはバグダードに入城し、ブワイフ朝を倒しました。
::
::last
トゥグリル＝ベクは、一方でアッバース朝のカリフを宗教的権威として存続させ、他方で==アッバース王朝の ==カリフからイスラーム法の執行権をもつ支配者[[スルタン  ]]と認められました。
::
::sup
アッバース朝およびそのカリフは、スルタンの権力を認めるためだけに存在
::
::top
セルジューク朝は、同族で奴隷身分出身のトルコ人兵士[[マムルーク]]を用いました。
::
#### ファーティマ朝への対抗
::top
セルジューク朝はイスラームのスンナ派であり、シーア派のファーティマ朝に対抗しました。
::
::middle
対抗の一環として、イスラーム教の法学や神学を学ぶ教育機関[[マドラサ ]]を設けました。
::
::last
セルジューク朝が建てたマドラサの代表例は、[[ニザーミーヤ学院]]です。
::
::sup
ファーティマ朝が建てたマドラサの代表例であるアズハル学院と区別
::
#### 十字軍派遣の原因
::top
セルジューク朝はさらに西方に進出し、ビザンツ帝国を圧迫しました。
::
::last
これは後にキリスト教国が十字軍を派遣する原因となりました。
::
### その他トルコ人の王朝
#### カラハン朝
::top
10世紀、トルコ人による最初のイスラームの王朝カラハン朝が、中央アジアに成立しました。
::
#### ガズナ朝
::top
10世紀、トルコ人によるイスラームの王朝[[ガズナ朝 ]]が、アフガニスタンに成立しました。
::
::last
ガズナ朝は北インドへの侵入を繰り返し、北インドのイスラーム化のきっかけをつくりました。
::
## モンゴル人の国
### イル＝ハン国の建国
::top
1258年、モンゴル軍を率いた[[フラグ]]が西アジアに進出し、アッバース朝の首都[[バグダード]]を陥れました。
::
::last
ここにアッバース朝は滅亡しました。
::
---arrow---
::top
フラグはイラン・イラクを領有し、[[イル＝ハン国 ]]を開きました。
::
### イラン＝イスラーム文化
::top
イル＝ハン国７代目ハンの[[ガザン＝ハン ]]の治世に、イスラームをイル＝ハン国の国教に定めました。
::
::middle
イル＝ハン国の支配のもとでイラン＝イスラーム文化が成熟しました。
::
::last
ガザン＝ハンの命令をうけ、イル＝ハン国の宰相[[ラシード＝アッディーン]]が、ペルシア語でユーラシア世界史『[[集史]]』を著しました。

## イスラームの国家と経済
### 貨幣経済の発達
::top
西アジアの貨幣経済は、イスラームのもとで一段と発達しました。
::
### 俸給の変化
::top
イスラームの王朝では、当初租税を徴収し、官僚・軍人に現金の俸給を支払っていました。
::
::last
９世紀後半、カリフの権力が弱まると、この体制を維持することが難しくなりました。
::
---arrow---
::top
946年にバグダードを占領した[[ブワイフ朝 ]]は、==軍人に土地の徴税権を与え  ==、彼らに農民から徴税させることにしました。
::
::middle
ブワイフ朝が初めて実施したこの制度を[[イクター制 ]]と言います。
::
::last
イクター制は、西アジアのセルジューク朝やイル＝ハン国、後のエジプトで成立したアイユーブ朝や[[マムルーク朝 ]]で採用されました。
