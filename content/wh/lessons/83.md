## アフリカの分割
### 植民地化の原則の確認
::top
19世紀後半、[[リヴィングストン ]]・[[スタンリー  ]]がアフリカ大陸の内陸部を探検しました。
::
::middle
探検を支援したベルギーは、報告を受けてコンゴの領有を宣言しました。
::
::last
西欧諸国はベルギーの宣言に反発し、対立が生じました。
::
---arrow---
::top
1884～85年、[[ベルリン会議 ]]（ベルリン＝コンゴ会議）
:::lead
85年、[[ベルリン会議 ]]（ベルリン＝コンゴ会議） [[ベルギー  ]]のコンゴ領有宣言への反発を調停するために、ドイツの[[ビスマルク]]の提唱が開催した、欧米諸国によるアフリカ分割の会議
::

- コンゴをベルギー領と確定
- 先に占領した国が領有できるという原則（占有権）を確認

---arrow---
::top
ベルリン会議後、欧米諸国によるアフリカ分割が急速に進展しました。
::

### フランスのアフリカ進出
#### 縦横の衝突
::top
フランスは、1830年に植民地化した[[アルジェリア ]]、1881年に保護国化した[[チュニジア  ]]を拠点に、アフリカでの植民地の拡大を進めました。
::
::middle
さらにサハラ砂漠周辺をおさえ、フランス領でアフリカを横断しようとしました。
::
::last
フランスは、３Ｃ政策でアフリカを縦断するイギリスと対立しました。
::
---arrow---
::top
1898年、[[ファショダ事件 ]]
:::lead
、[[ファショダ事件 ]] 縦断政策をとる[[イギリス  ]]と横断政策をとるフランスが、アフリカ大陸のスーダンのファショダで衝突した事件
::

::top
20世紀初頭、アフリカはエチオピア帝国とリベリア共和国を除き、西欧諸国の支配下に置かれました。
::
::last

::
