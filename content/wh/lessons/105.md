---overview---
19世紀以降のアフリカ分割で、アフリカ大陸はエチオピア・リベリアを除き西洋諸国の植民地になりました。第二次世界大戦後、アジア・アフリカで独立運動が盛んになると、1957年にガーナが独立を果たしました。1960年には17ヵ国が独立を果たし、この年は「アフリカの年」と呼ばれます。しかし、アフリカの国境線のほとんどは、西洋諸国が人為的に引いたものであり、民族が分断されて紛争の一因となっています。また、植民地時代のような一次産品の輸出が続いていたりします。
---
## 第三世界
### 第三世界の呼称
::top
アジア・アフリカの新興国は、冷戦に巻き込まれることを避けるために、東西陣営のどちらにも属さない第３の勢力を形成しようとしました。
::
::last
この勢力に属すアジア・アフリカ・ラテンアメリカの発展途上国を、第三世界と呼ぶようになりました。
::

## ラテンアメリカ諸国とキューバ革命
### アメリカ主導のラテンアメリカ諸国の結束
::top
第二次世界大戦後、ラテンアメリカ諸国はアメリカの強い影響下にありました。
::
---arrow---
::top
1947年、[[パン＝アメリカ会議 ]]
:::lead
、[[パン＝アメリカ会議 ]] アメリカが、==自国の勢力拡大に利用するために  ==（ラテンアメリカ諸国への影響力を強めるために）、ラテンアメリカ諸国と開催した会議
::
::sup
当初、会議は中南米諸国の連帯強化を意図して開催されたが、19世紀末以降に目的が変質
::
---arrow---
::top
1948年、[[米州機構 ]]（[[ＯＡＳ]]）結成
:::lead
、[[米州機構 ]]（[[ＯＡＳ]]）結成アメリカとラテンアメリカ諸国で構成した協力組織
::
### ラテンアメリカ諸国の反発
::top
アメリカが影響力を強めるなかで、干渉に反発するラテンアメリカ諸国も現れました。
::
#### アルゼンチン
::top
1946年、アルゼンチンの大統領に就任したペロンは、反アメリカ的な民主主義を掲げて社会革命をおこないました。
::
#### グアテマラ
::top
1951年、グアテマラで左翼政権が成立し、土地改革に着手しました。
::
::last
1954年、左翼政権はアメリカに支援された軍部のクーデタに敗れました。
::
#### キューバ
::top
第二次世界大戦後、[[キューバ ]]では親米的なバティスタの独裁政権が続いていました。
::
---arrow---
::top
1959年、[[キューバ革命 ]]
:::lead
、[[キューバ革命 ]] 親米的な独裁者[[バティスタ  ]]に対して、[[カストロ  ]]が指導して起こした革命
::
::top
キューバ革命の結果成立したカストロ政権は、米国資本の砂糖企業を接収しました。
::
::last
1961年、アメリカはキューバと断交し、カストロ政権との対立を強めました。
::
---arrow---
::top
カストロ政権は==社会主義を宣言し ==、==南北アメリカを通じて最初の社会主義国が成立しました  ==。
::
::middle
1961年、アメリカはキューバを除く中南米諸国と「進歩のための同盟」を結びました。
::
::last
さらに1964年、米州機構加盟国はキューバと断交しました。
::
---arrow---
::top
上記の経済封鎖に苦しみながらも、キューバのカストロ政権は、政権を維持しました。
::
## アフリカ諸国の独立
### アフリカの植民との独立
::top
1950年代以降、アフリカの植民地が独立していきました。
::

- 1956年、[[モロッコ]]が[[フランス]]から独立
- 1956年、[[チュニジア]]が[[フランス]]から独立
- 1957年、[[ガーナ]]が[[イギリス]]から独立
  :::lead
  イギリス連邦内の自治領となり、独立を指導した[[エンクルマ]]が初代大統領に就任
  :::

::last
特にアフリカで多くの国々が独立した[[1960 ]]年は、「[[アフリカの年  ]]」と呼ばれました。
::

エンクルマ
:::lead
急進的な政策を進めたが失敗し、軍部のクーデタで政権から追放
::

::top

、[[アフリカ統一機構 ]]（[[ＯＡＵ]]）結成エチオピアのアジスアベバで開催されたアフリカ独立諸国首脳会談で結成された、アフリカ諸国の連帯や旧宗主国の干渉克服を目指す組織
::
::sup

:::lead
植民地を支配する国のこと

### 残り続ける支配
#### 独立できなかった植民地
::top
植民地が次々と独立するなか、ポルトガル植民地は残り続けました。
::
#### 旧宗主国の干渉
::top
1960～65年、コンゴ動乱
:::lead
65年、コンゴ動乱[[ベルギー ]]からの独立直後の[[コンゴ  ]]で生じた内戦で、旧宗主国ベルギーが重要な鉱物資源の産出地域を分離して開始
::
#### 白人支配の強化
::top
[[南アフリカ ]]では、白人支配を維持するために、黒人を差別する人種隔離政策[[アパルトヘイト  ]]が採用されました。
::
### 新興独立国の問題
::top
アフリカの新興独立国は、旧宗主国の方針で単一作物栽培が多く、経済的基盤が貧弱でした。
::
::middle
また、社会的インフラストラクチャーや、教育・医療などの社会制度が未整備でした。
::
::last
加えて、政治・経済の不安定が続き、内戦やクーデタが繰り返されて軍事独裁政権が成立することも度々ありました。
::
