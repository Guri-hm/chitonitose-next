---overview---
20世紀初頭、清朝は西洋諸国による領土分割、過激な排外運動を防げず、その内政は混乱を極めました。1911年、孫文の中国同盟会が武昌で反乱をおこすと、各地で清朝から独立する動きがみられました。1912年に孫文を臨時大統領とする中華民国が成立しましたが、袁世凱が孫文にとってかわって専制政治をおこなったため、革命の理想は失われました。
---
## 辛亥革命
### 義和団事件後の改革
::top
1900～01年の義和団事件後、清は次の改革をおこないました。
::

- 1905年、[[科挙]]の廃止
- 1908年、[[憲法大綱]]の発表
  :::lead
  日本の大日本帝国憲法を模範とした、清の憲法の大綱
  :::
- 1908年、国会開設の公約

---arrow---
::top
上記の改革は、増税と中央集権的な性格のために、民衆の反発を招きました。
::
### 革命運動の始まり
::top
19世紀末～20世紀初頭、華僑や留学した漢人は、満州人の清の打倒を目指す運動を起こしました。
::
::middle
1894年、漢人の[[孫文]]は、華僑を中心とした政治的結社[[興中会 ]]を組織しました。
::
::last
1905年、[[孫文 ]]は革命を目指す諸団体の結集を図り、[[中国同盟会  ]]を==東京で  ==組織しました。
::
---arrow---
::top
[[孫文 ]]は、次の３つを内容とする[[三民主義  ]]を掲げ、革命運動を指導しました。
::

- 民族
  :::lead
  満州人の満州王朝である清の打倒
  :::
- 民権
  :::lead
  共和国の建設
  :::
- 民生
  :::lead
  貧富の差の抑制
  :::

### 鉄道国有化と革命の勃発
::top
1911年、清は民間鉄道を国有化し、この鉄道を西欧諸国からの借款の担保にしようとしました。
::
::last
資本家や地方有力者は、一方的な鉄道国有令に反対し、[[四川 ]]で暴動が発生しました。
::
---arrow---

:::lead
1911～12年、[[辛亥革命 ]]
:::
12年、[[辛亥革命 ]] 四川の暴動の鎮圧を命じられた軍隊が、革命派側に立って[[武昌  ]]で蜂起
::
::gazo
::
::middle
袁世凱は反発を鎮圧して==正式な大総統となり ==、やがて==帝政復活（皇位への即位）を試みました  ==。
::
::last
しかし、試みは失敗し、1916年に袁世凱は病死しました。
::

中国国民党

::

中華民国の統制は失われ、政府の実権を争奪する内戦状態になりました。
::

1911～1912年の辛亥革命は、中国周辺部の独立の気運を高めました。
::

## 韓国併合
### 保護権の確立
::top
20世紀初頭、日本の産業は原料輸入・製品輸出で大陸と強く結びつきました。
::
::last
日露戦争後、日本は大陸への進出を目指すようになりました。
::
---arrow---
::top
1904～07年、第１～３次[[日韓協約]]
:::lead
1904～07年、第１～３次[[日韓協約]]
:::
07年、第１～３次[[日韓協約]]日本の韓国（大韓帝国）に対する保護権確立のため、日本と韓国との間で締結された協約
::
### 保護国化への抵抗
::top
日本による保護権確立に対して、韓国では次の抵抗がありました。
::

- 1907年、ハーグ密使事件
  :::lead
  オランダの[[ハーグ]]で開催された万国平和会議に、[[韓国]]の皇帝高宗が密使を送り、日本による保護国化の不当性を西欧諸国に訴えようとした事件
  :::
- 1907年～、義兵運動
  :::lead
  軍隊解散命令に反対する一部の軍人を中心とした武装闘争
  :::

::last
いずれの抵抗も西欧諸国から黙殺され、日本に抑えられました。
::
