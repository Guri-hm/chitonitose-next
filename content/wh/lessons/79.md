---overview---
19世紀、産業革命を達成したイギリスをはじめとする欧米諸国が、アヘン戦争を契機に中国への進出を開始しました。強大な経済力と軍事力を武器にする欧米諸国の進出は、政治・経済・文化の面で中国の伝統的社会に激しい衝撃と動揺を与えました。これをTheWestern Impact（西洋の衝撃）と呼びます。中国の長い苦難が始まりました。
---

### アヘン戦争
#### 茶の輸入急増と銀の流出
::top
18世紀後半、イギリスは本国での茶の需要が増えたため、清から茶を輸入しました。
::
::middle
一方で、イギリスが生産した綿製品は、清でなかなか売れませんでした。
::
::last
これにより、イギリスの対清貿易は==輸入超過でした ==。
::
---arrow---
::top
輸入超過の結果、イギリスが決済に用いていた[[銀 ]]の流出が顕著となりました。
::
::gazo
![銀の流出](img/79/trade.jpg)
銀の流出
::

- イギリスがインドに綿製品を輸出
- インドが貿易赤字を補うためにアヘンを生産し、イギリスが輸入
- イギリスがインド産のアヘンを中国に密輸
- 清のアヘン中毒者がアヘンを買い求めるため、イギリスに銀が流入

::last
この貿易を[[三角貿易 ]]と呼びます。
::
::gazo

::

::
::last

::
::gazo

::last
林則徐は、アヘンを没収・廃棄処分し、イギリスにアヘン貿易禁止を求めました。

## 西洋の衝撃－The Western Impact－
### 欧米との不平等条約
::top
1842年、[[南京条約 ]]
:::lead
、[[南京条約 ]] [[アヘン戦争  ]]の講和条約
::
::sup
アヘンの密輸は言及されず、以降も密輸が継続
::
---arrow---
::top
南京条約は、追加条約（五港通商章程・虎門寨追加条約）で次の制度・権利も認めました。
::

- 協定関税制
  :::lead
  清側の関税自主権の欠如を容認
  :::
- [[領事裁判権]]
  :::lead
  滞在中の外国人が自国の領事による裁判を受けられる権利
  :::
- 片務的最恵国待遇
  :::lead
  清がイギリスに認めてない特権を他国に認めた場合、その特権をイギリスにも認めること
  :::

---arrow---
::top
1844年、清はイギリスと同様の権利をアメリカ・フランスに認めました。
::

- アメリカ：[[{{望厦|ぼうか}}条約]]
- フランス：[[{{黄埔|こうほ}}条約]]
