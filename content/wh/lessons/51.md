---overview---
十字軍の遠征以来、ヨーロッパで香辛料の需要が増大しました。ところが、その供給はオスマン帝国の地中海支配が進むにつれて不安定となりました。ヨーロッパに原産地と直接取引しよう、また、オスマン帝国に対抗する味方を東方に見つけようという気運が高まり、人々を海へと駆り立てました。大航海時代の幕開けです。ヨーロッパの経済活動の中心が地中海から大西洋へと移りました。
---
## ヨーロッパ世界の拡大
### 大航海時代の幕開け
::top
次のことを背景に、ヨーロッパで海外進出の気運が生じました。
::

- アジアの富や文化への関心が高まったこと
- [[羅針盤]]の改良、快速帆船の普及で航海技術が向上したこと
- 肉食の普及で香辛料の需要が高まる一方、イタリア商人が貿易を独占し、オスマン帝国の地中海進出後はその貿易も困難になったこと
- ==国土回復運動（レコンキスタ）の過程で、キリスト教世界拡大への情熱が高まったこと==

## 商業革命と価格革命
### 商業革命
::top
大航海時代の到来で、ヨーロッパの商業は地球規模に拡大しました。
::
::last
商品の種類・取引額が増え、ヨーロッパ経済の中心は、地中海から大西洋岸都市に移りました。
::
### 価格革命
::top
商業の地球規模化は、資本主義経済の発達を促しました。
::
::middle
また、アメリカ大陸の銀山から銀が流入し、ヨーロッパで急激な物価上昇[[価格革命 ]]が起こりました。
::
::last
物価の騰貴は、固定値代で生活する領主に打撃を与えました。
::
### 東欧・西欧の役割の違い
::top
商業革命で、西ヨーロッパは商工業が活発になりました。
::
::last
他方、東ヨーロッパは西欧の商工業を穀物輸出で支えました。
::
---arrow---
::top
エルベ川以東の==東ヨーロッパでは ==、[[農場領主制  ]]（[[グーツヘルシャフト]]）で==西欧向けの  ==穀物が生産されました。
::
::last
これにより、東ヨーロッパでは農奴に対する支配が、近世に一層強化されました。
