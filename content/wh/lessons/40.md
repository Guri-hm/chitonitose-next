---overview---
西ヨーロッパの中世文化の担い手は、ラテン語を扱えた聖職者や知識人、そして騎士などでした。キリスト教の権威に根ざした文化であり、そこから派生した学問・芸術が特徴的です。
---
## 中世文化
### 教会と修道院
::top
西ヨーロッパ中世では、日常生活全般にローマ＝カトリック教会の権威が行き渡っていました。
::
::middle
教会のみが、出生・結婚など人生の節目に儀式を授け、魂の救済ができるとされました。
::
::last
この枠組みから外される「破門」は、極めて重い罰でした。
::
#### 修道院
::top
修道院は、厳しい規律の下で信仰修行と労働に励む者が共同生活を営む場所です。
::
::middle
６世紀、[[ベネディクトゥス ]]がイタリアの[[モンテ＝カシノ  ]]に修道院を設立しました。
::
::middle
ベネディクト修道会は、「祈り、働け」をモットーに、清貧・純潔・服従の生活を送ることを求めました。
::
::last
この考えは「労働＝奴隷の仕事」という労働観を大きく変えました。
::
---arrow---
::top
修道院は、時代ごとに様々な活動をおこないました。
::
::middle
10世紀、クリュニー修道院が中心となり、教会の腐敗に対して改革運動を起こしました。
::
::middle
12～13世紀、ベネディクト修道会の思想を継ぐ[[シトー修道会 ]]が、開墾運動の先頭に立ちました。
::
::last
13世紀、修道院の外で布教に努める修道会[[{{托鉢|たくはつ}}修道会 ]]が現れ、[[フランチェスコ修道会  ]]と[[ドミニコ修道会  ]]の２つが知られています。
::
::gazo
![開墾する修道士](img/40/forest_clearing.jpg){.quarter}
12世紀の開墾をおこなう修道士
::
::middle
当時の学者・知識人はローマ＝カトリック教会の聖職者であり、学問にラテン語を用いました。
::
::last
大半の人々はラテン語ができず、==ラテン語で書かれた聖書を読めませんでした ==。
::
### 学問と大学
#### ８～９世紀の文芸復興
::top
８～９世紀、[[カール大帝 ]]は宮廷に[[アルクイン  ]]ら学者を集め、学芸を奨励しました。
::
::last
これにより、[[カロリング＝ルネサンス ]]と呼ばれる文芸復興がおこりました。
::
#### スコラ学
::top
９世紀頃、キリスト教信仰との調和を重んじ、論理的に体系化する学問[[スコラ学 ]]が始まりました。
::
::last
スコラ学の中心的議論は、次の実在論と唯名論の普遍論争でした。
::
::top
実在論
:::lead
神や普遍という個物の原型が存在するという立場
::
::top
唯名論（名目論）
:::lead
目論）あらゆるものに共通する原型は思考実験のなかでのみ存在するという立場
::
---arrow---
::top
ギリシアの==アリストテレス哲学が、スコラ学の発展に大きな影響を与えました ==。
::
::last
[[トマス＝アクィナス ]]は、『[[神学大全  ]]』を著し、スコラ学を大成しました。
::
#### 12世紀の文芸復興
::top
11世紀末以降、十字軍が東方に遠征すると、ビザンツ帝国や==イスラーム世界を通して古代ギリシア哲学書が伝えられました ==。
::
::middle
ギリシア古典や==アラビア語の哲学書や医学書がラテン語に翻訳され、西ヨーロッパの哲学・神学や医学に影響を与えました ==。
::
::last
これを契機に学問が発展する[[12世紀ルネサンス ]]がおこりました。
::
::sup
12～13世紀の専門的学問分野は、==法学・神学・医学 ==
::
---arrow---
::top
==ギリシアのアリストテレス哲学がキリスト教の神学に取り入れられ、これを体系化した書物が著されました ==。
::
#### イスラーム科学の影響
::top
十字軍の遠征で、イスラーム科学の影響も受けました。
::
::last
13世紀、[[ロジャー＝ベーコン ]]が自然科学で==実験を重視する  ==方法論を説きました。
::
#### 大学の誕生
::top
12世紀頃、西ヨーロッパで大学がつくられ始めました。
::
::middle
大学は、==教師や学生のギルド的な自治団体として発展しました ==。
::
::middle
ただし、==教会の権威下におかれ、キリスト教に反するような自由な学問・思想は許されませんでした ==。
::
::last
また、大学では==ラテン語が共通の学術語でした ==。
::
---arrow---
::top
次の大学は特定の学問に優れて有名でした。
::

- [[ボローニャ大学]]
  :::lead
  イタリアに創立された最古の大学の１つで、[[法学]]で有名
  :::
- [[サレルノ大学]]
  :::lead
  イタリアの最古の大学の１つで、アラビア医学の影響を受けた[[医学]]で有名
  :::
- [[パリ大学]]
  :::lead
  フランスに創立された大学で、[[神学]]で有名
  :::
- オクスフォード大学
  :::lead
  パリ大学を模範にイギリスに創設された大学で、[[神学]]研究を中心に発展
  :::
- ケンブリッジ大学
  :::lead
  オクスフォード大学の教師・学生が移動して設立された大学で、独自の学寮（コレッジ）制を基に発展
  :::

### 建築・美術
::top
中世初期、建築はビザンツ様式の模様がおこなわれました。
::
---arrow---
::top
11世紀、厚い石壁に小窓をもつ[[ロマネスク様式 ]]が生み出されました。
::
::last
厚い石壁と小窓は、技術が不足し、建物の強度を上げるために採用されました。
::
---arrow---
::top
12世紀、==多彩なステンドグラス・尖塔・アーチを特徴とする ==[[ゴシック様式  ]]が広まりました。
::
::sup
ゴシック様式
:::lead
式ロマネスク様式と比べ、壁を薄くする技術が進歩して広い窓を採用でき、それらをステンドグラスで装飾
::
