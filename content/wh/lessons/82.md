---overview---
19世紀後半からの第２次産業革命は、産業の中心を軽工業から重工業に転換させました。国の工業が発展した一方で、伝統的経済基盤を破壊され、移民として国を離れることになった人々もいました。また、革命が石油・銅・錫・ゴムなどヨーロッパでは得にくい資源を必要としたため、市場および資源の供給地として植民地獲得が欠かせなくなりました。この侵略的な領土拡張の動きを「帝国主義」と言います。
---
## 第２次産業革命から帝国主義の成立へ
### 第２次産業革命
::top
19世紀後半から第一次世界大戦まで、アメリカ合衆国・ドイツを中心に、重化学工業が発展しました。
::
::last
この発展は、[[石油 ]]・電力をエネルギー源・動力源に使ったため、石炭・蒸気力をエネルギー源・動力源にした最初の産業革命に対して、[[第２次産業革命  ]]と呼ばれます。
::
### 移民
::top
第２次産業革命は、最初の産業革命に増して、人々の生活基盤や環境を変化させました。
::
::middle
伝統的経済基盤を破壊された人々は、移民となって国を離れました。
::
::middle
==19世紀末から20世紀初頭 ==、[[東欧・南欧  ]]からの多くの移民が、アメリカ合衆国へ流入しました。
::
::last
この動きは、後に移民制限運動を招くことになりました。
::
### 帝国主義
::top
第２次産業革命は、最初の産業革命に増して、巨額の資本が必要になりました。
::
::middle
資本獲得の市場および資源の供給地として、植民地がより重要になりました。
::
::last
欧米諸国は、本国と植民地の結びつきを強め、また、新たな植民地の獲得に動きました。
::
---arrow---
::top
大航海時代以降、西欧諸国による植民地獲得は、「野蛮」な地域をキリスト教の布教で「文明」化するという大義の下でおこなわれました。
::
::middle
1880年代以降、欧米諸国が進めた植民地化は、「野蛮」の「文明」化という大義がやや影を潜め、国力に物を言わせた市場・資源が優先されました。
::
::last
このような侵略的な領土拡張の動きは、[[帝国主義 ]]と言われます。
::
### すばらしい時代
::top
19世紀末から第一次世界大戦前、欧米諸国の対立はアジア・アフリカ・バルカン半島などで生じました。
::
::middle
ヨーロッパ中心部を舞台にした戦争は起こらず、好景気の持続もあってヨーロッパ諸国は繁栄期を迎えました。
::
::last
ヨーロッパ文明への自信、科学の進歩への信頼に支えられ、19世紀末から第一次世界大戦前はベルエポック（すばらしい時代）と呼ばれました。
::

## 第２インターナショナルの結成
### 第１インターナショナルの解散後
::top
1864年に結成された第１インターナショナルは、マルクスとバクーニンら無政府主義者の内部対立が深刻化しました。
::
::last
パリ＝コミューン後の1876年、第１インターナショナルは解散しました。
::
---arrow---
::top
第１インターナショナルの解散後、欧米諸国では労働運動や社会主義が勢力を伸ばしました。
::
::last
これら運動を支える社会主義思想として、マルクス主義が主流となりました。
::
### 第２インターナショナルの結成
::top
1889年（第一次世界大戦==前 ==）、社会主義の新たな国際的組織として、[[第２インターナショナル  ]]が[[パリ  ]]で結成されました。
::
::middle
第２インターナショナルでは、ドイツ社会民主党が中心となり、帝国主義・軍国主義への反対運動を組織しました。
::
::last
また、８時間労働などの労働条件の改善を訴えました。
::
---arrow---
::top
しかし、社会主義者の中に植民地や自国の利害を重視する者が現れ、第２インターナショナルの結束は徐々に乱れました。
::
