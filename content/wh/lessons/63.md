---overview---
フランスは、アメリカの独立戦争に際して植民地側を援助しました。これにより政難に陥ったフランスでは、聖職者や貴族への課税が議論されました。しかし、議決方法が特権身分に有利であったため、会議は紛糾しました。第三身分（平民）が平等な議決方法を定めた憲法制定を主張し、フランス革命が迎えられました。
---
## フランス革命の前史と開始
### 第三身分の不満
::top
フランス革命以前のフランスにおいて、国民は次のように区別されました。
::

- 第一身分：聖職者
- 第二身分：貴族
- [[第三身分]]：平民

::middle
人口の９割以上が第三身分でした。
::
::last
つまり、少数の特権身分（第一身分・第二身分）が土地・官職などを握っていました。
::
---arrow---
::top
第三身分の商工業者は、財力をつけるにつれて、現状の待遇と税の負担に不満を感じました。
::
::last
1789年、[[シェイエス ]]が『[[第三身分とは何か  ]]』を著し、第三身分の権利を主張しました。
::
### 三部会と国民議会
::top
フランスは、度重なる戦争で財政難に陥りました。
::
::middle

::
::last
しかし、特権身分が抵抗したため、[[ルイ16世 ]]は３つの身分の代表者が議論する[[三部会  ]]を召集しました。

### 国民議会の施策
#### 封建的特権の廃止
::top
1789年８月、[[国民議会 ]]は次の==封建的特権の廃止を決定しました==。
::

- 領主裁判権
- 教会への十分の一税
- 封建地代（領主が農民の所有地から徴収する年貢など）

::last
ただし、封建地代は==有償廃止とされ ==、実際には継続しました。
::

- ==ギルドの廃止==

#### 国王の信頼失墜
::top

::
### 憲法発布と国民議会の解散
::top
1791年、憲法が発布され、==立憲君主政 ==、一定額以上の納税での選挙権が定められました。
::
::last
目的を果たした国民議会は、解散となりました。
::
## 共和政の実現
### 立憲君主政か共和政か
::top
1791年の制限選挙の結果、立法議会が成立しました。
::
::last
立法議会で、次の２派が対立しました。
::

- 立憲君主派
  :::lead
  革命のこれ以上の進行を望まない派閥
  :::
- ジロンド派
  :::lead
  大商人の利害を代表して共和政を主張する派閥
  :::

---arrow---
::top
1792年、[[ジロンド派]]が立憲議会で優勢となり、実権を握りました。
::
### 反革命への戦争
::top
1791年、ピルニッツ宣言
:::lead
、ピルニッツ宣言革命の波及を恐れ反対するオーストリア・プロイセンが、ルイ16世の救援を各国君主に呼びかけたこと
::
---arrow---
::top
1792年、[[ジロンド ]]派が実権を握った[[立法議会  ]]は、オーストリアへの宣戦を可決しました。
::
::middle
しかし、フランスの軍隊には国王を支持する王党派が多く、戦意に欠けました。
::
::last
フランスは、オーストリア・プロイセンの連合軍に国内に侵入されました。
::
---arrow---
::top
この危機に際して、フランス各地から義勇軍が集まりました。
::
::last

:::lead

::

::

::
::last
[[国民公会 ]]は、[[王政の廃止  ]]と共和政の樹立を宣言しました。
::
### 王の処刑と内外の危機
::top
国民公会では、急進的なジャコバン派（山岳派）が勢力を増しました。
::
::last
1791年、共和政が強調され、ルイ16世が処刑されました。
::
---arrow---
::top
君主政のイギリスは、王を打倒する革命が波及することを恐れました。
::
::middle
[[イギリス ]]の首相[[ピット  ]]は、[[ルイ16世  ]]の処刑をうけ、[[第１回対仏大同盟  ]]を提唱・結成しました。
::
::last
結果、フランスは全ヨーロッパと敵対し、国内では王党派と結びついた農民反乱を抱えました。
::
### 国民公会の政策
#### ジロンド派の追放と支持確保の政策
::top
内外の危機に際して、ジャコバン派（山岳派）はジロンド派を[[国民公会 ]]から追放しました。
::
::last
国民公会の主導権を握ったジャコバン派は、次の政策を採用しました。
::

- 1793年憲法制定
  :::lead
  男性普通選挙の採用
  :::
- 封建地代の無償廃止
- ==物価の統制==
- ==徴兵制の導入==
- [[革命暦]]（[[共和暦]]）制定
  :::lead
  グレゴリウス暦の否定、革命祭日として「サンキュロットの日」の設定
  :::
- [[メートル法]]制定

#### 高まる不満とジャコバン派の没落
::top
ジャコバン派の指導者ロベスピエールは、国民公会内の[[公安委員会 ]]に権力を集中させ、独裁・恐怖政治をおこないました。
::
::last
有力農民・市民は、次第にジャコバン派、そしてロベスピエールに不満を高めました。
::
---arrow---
::top
1794年７月、[[テルミドール９日のクーデタ ]]
:::lead
７月、[[テルミドール９日のクーデタ ]] ジャコバン派の指導者[[ロベスピエール  ]]が逮捕・処刑された事件
::
### 新政府の樹立
::top
ジャコバン派の没落後、穏健共和派が有力となりました。
::
::last
1795年憲法に基づく、ブルジョワ勢力中心の政府[[総裁政府 ]]が樹立しました。
