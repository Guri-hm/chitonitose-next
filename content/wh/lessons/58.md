---overview---
啓蒙思想の影響を受けた君主は、国家体制の改善や産業の奨励を君主の絶対的な権力で実現しようとしました。このような君主を啓蒙専制君主と呼び、プロイセンのフリードリヒ２世、オーストリアのマリア＝テレジア、ロシアのエカチェリーナ２世が代表者にあげられます。
---
## プロイセン
### プロイセン王国の成立
::top
17世紀初め、[[ホーエンツォレルン ]]家の[[ブランデンブルク  ]]選帝侯国がプロイセン公国を併合しました。
::
::middle
1701年、プロイセン公国が王国に昇格しました。
::
::last
以降、公国を併合していたブランデンブルク選帝侯国を含め、プロイセン王国と呼ぶようになりました。
::
### フリードリヒ＝ヴィルヘルム１世の治世
::top
プロイセン２代目の王[[フリードリヒ＝ヴィルヘルム１世 ]]は、軍備を増強し，絶対王政の基礎を築きました。
::
### フリードリヒ２世の治世（啓蒙専制君主）
::top
1740年、オーストリアのマリア＝テレジアが、ハプスブルク家の全領土を継承しました。
::
::middle
各国君主は彼女を侮り、その継承を否定しました。
::
::last
特にプロイセンの[[フリードリヒ２世 ]]は、その領土内のシュレジエンの権利を主張して占領しました。
::
---arrow---
::top
1740～48年、[[オーストリア継承戦争]]
:::lead
48年、[[オーストリア継承戦争]][[フランス  ]]と結んだプロイセンが、イギリスに支援されたオーストリアと戦争
::
::top
オーストリア継承戦争後、ハプスブルク家のマリア＝テレジアは、[[シュレジエン ]]の奪回を目指しました。
::
::last
マリア＝テレジアは、長く敵対関係にあった[[フランス ]]と手を結ぶ[[外交革命  ]]を実現しました。
::
---arrow---
::top
1756～63年、[[七年戦争 ]]
:::lead
63年、[[七年戦争 ]] イギリスの支援を得たプロイセンが、[[フランス  ]]と同盟したオーストリアと戦争
::
#### 啓蒙専制主義
::top
君主主導で自国の近代化・強国化を進める体制を啓蒙専制主義と呼びます。
::
---arrow---
::top
フリードリヒ２世は、「君主は国家第一の僕」を自称し、==ヴォルテールなど啓蒙思想家に影響を受けた啓蒙専制君主でした ==。
::
::last
また、啓蒙専制君主として知られるフリードリヒ２世は、[[バッハ ]]を宮廷に招きました。
::
::sup
啓蒙専制君主
:::lead
主ほかにヨーゼフ２世・エカチェリーナ２世が代表的
::
---arrow---
::top
統治にはユンカーの力を頼ったため、農民の地位の改善は進みませんでした。
::

### マリア＝テレジアの治世
::top
1740年、オーストリアのマリア＝テレジアが、ハプスブルク家の全領土を継承しました。
::

- 敵対していたフランスと同盟（外交革命）

::top

::
::last
ヨーゼフ２世の画一的改革は、支配地域の民族感情の反発を招きました。
::

## ポーランド
### ポーランド分割
::top
ポーランドは、16世紀後半の[[ヤゲウォ ]]朝断絶後、[[選挙王制  ]]をとりました。
::
::last
この制度は貴族の専横と外国の干渉を招きました。
::
::sup
選挙王制
:::lead
貴族による国王選出の制度で、外国の王族出身者からも選出可
::
---arrow---
::top
18世紀後半、ポーランドは、プロイセン・オーストリア（第２回除く）・[[ロシア ]]（[[エカチェリーナ２世  ]]の治世）による３度の[[ポーランド分割  ]]に苦しみました。
::

- 1772年、第１回ポーランド分割
- 1793年、第２回ポーランド分割
- 1795年、第３回ポーランド分割

---arrow---
::top
ポーランド分割の結果、ポーランドという国家は消滅しました。
::
::sup
ポーランドの復興は第一次世界大戦後
