---overview---
冷戦が続くなかで、国ごとに変化が生じ、冷戦当初とは異なる状況がみられるようになりました。西側陣営のフランスは、アメリカに追随しない独自の外交を始め、北大西洋条約機構から脱退しました。また、東側陣営の中心であるソ連は、「雪解け」と呼ばれる東西間の緊張緩和を目指すようになりました。
---
## 冷戦下のアメリカ社会
### 軍備と規制の強化
::top
ソ連との対立激化で、アメリカは次のように軍備と共産主義規制を強化しました。
::

- 1947年、国家安全保障法
  :::lead
  国家安全保障会議、国家軍事機構（のちの国防総省）、中央情報局（ＣＩＡ）を設置
  :::
- 1947年、タフト・ハートレー法
  :::lead
  労働組合運動を制約する法律
  :::
- 1950年代～、マッカーシズム（赤狩り）
  :::lead
  共和党上院議員マッカーシーに端を発した、共産主義社の追放を扇動的な運動
  :::

### フランス

## ソ連の「雪どけ」
### ソ連
#### 雪どけ
::top
1953年のスターリン死去後、ソ連のフルシチョフは「[[雪どけ]]」と呼ばれる、次のような平和共存の提唱と東西間の緊張緩和をおこないました。
::

- 1955年、ユーゴスラヴィアとの和解
- 1955年、西ドイツを承認し国交回復
- 1956年、[[スターリン批判]]
  :::lead
  第一書記[[フルシチョフ]]が、ソ連共産党第20回大会でおこなった、スターリン体制下の個人崇拝や反対派の大量処刑などの批判
  :::
- 1955年、[[ジュネーヴ４巨頭会談]]
  :::lead
  アメリカ・イギリス・フランス・ソ連４か国の首脳会談
  :::
- コミンフォルムの解散

#### 米ソ首脳会談の実現と再度の緊張
::top
1958年、ソ連第一書記フルシチョフは首相を兼任しました。
::
::last
1959年、フルシチョフは、次のことでソ連がアメリカと対等な力をつけたとして、アメリカ大統領アイゼンハワー（アイゼンハウアー）との会談に臨みました。
::

- 大陸間弾道ミサイルの開発
- 1957年、世界初の人工衛星[[スプートニク]]の打ち上げ成功
