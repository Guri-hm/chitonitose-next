---overview---
大国スウェーデンは、北方戦争でバルト海の覇権を失い、また、ナポレオン戦争後にフィンランドをロシアに割譲しました。1814年以降、ノルウェーはスウェーデン領とされましたが、独自の憲法を保持し続け、後に平和的に独立を果たしました。デンマークはプロイセン・オーストリアの連合軍と戦い、北ドイツの領土を失いました。
---
## 北欧の動向
### スウェーデン
::top
1700～21年、[[北方戦争 ]]
:::lead
21年、[[北方戦争 ]] ロシアのピョートル１世が[[スウェーデン  ]]に勝利し、[[バルト  ]]海への出口を確保した戦争
::
---arrow---
::top
ナポレオン戦争中、スウェーデンは北ドイツの領土を失いました。
::
::last
1809年、スウェーデン領であったフィンランドがロシアに割譲されました。
::
### ノルウェー
::top
ウィーン会議の結果、ノルウェーは[[スウェーデン ]]領になりました。
::
::last
しかし、ノルウェー独自の憲法は保持されました。
::
---arrow---
::top

## 国際的運動の動向
::top
19世紀後半、工業の発展徒とともに労働運動が国際的に展開しました。
::
::middle
1864年、[[マルクス ]]やバクーニンらは、[[第１インターナショナル  ]]（[[国際労働者協会]]）を[[ロンドン  ]]で結成しました。
::
::last
第１インターナショナルは、社会主義者の国際的協力を促しました。
::
---arrow---
::top
第１インターナショナル内部で、マルクスとバクーニンら無政府主義者の対立が深刻化しました。
::
::last
パリ＝コミューン後の1876年、第１インターナショナルは解散しました。
::
