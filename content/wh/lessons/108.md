::top

::last

::

### ベトナム戦争の影響
#### 沖縄返還
::top
日本の沖縄県の米軍基地は、ベトナム戦争に利用されました。
::
::middle
戦火に巻き込まれることへの批判が高まり、沖縄返還の要求が強まりました。
::
::last
結果として1972年、沖縄返還が実現しましたが、米軍基地は残されました。
::
#### 米中接近
::top
アメリカは台湾の中華民国を「正式な」中国とし、中華人民共和国を否定してきました。
::
---arrow---
::top
アメリカは、ベトナム戦争の和平交渉を進めるため、北ベトナムを支援する中華人民共和国との国交正常化に乗り出しました。
::
::last
1972年、アメリカ大統領[[ニクソン ]]が中華人民共和国を訪問し、毛沢東と今後の関係回復を合意しました。
::
::sup
1978年、[[日中平和友好条約 ]]
:::lead
、[[日中平和友好条約 ]] ニクソンの訪中後の1972年に、日本の首相が訪中して事実上==日中国交正常化し  ==、後に締結された条約
::
---arrow---
::top
アメリカの方針転換により、1971年、中国の国連代表権が中華民国から[[中華人民共和国 ]]に移りました。
::
## 第三世界の動向
### 開発独裁
::top
1960年代、第三世界では、強権的支配の下で政治・社会運動を抑圧しながら近代化を強行する、開発独裁と呼ばれる体制が登場しました。
::
### 大韓民国
::top
大韓民国の大統領李承晩は、抑圧的な反共体制をとりました。
::
::middle
1960年、李承晩は民主化と経済発展を求める運動で失脚しました。
::
::last
1963年、軍人出身の[[朴正熙 ]]が==クーデタで政権を掌握し  ==、大統領に就任して==民主化運動を弾圧しました  ==。
::
---arrow---
::top
1965年、[[日韓基本条約 ]]
:::lead
、[[日韓基本条約 ]] 大韓民国の[[朴正熙  ]]政権と日本が調印した条約で、日韓の国交を樹立
::

::top
