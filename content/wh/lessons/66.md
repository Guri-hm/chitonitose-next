---overview---
ウィーン体制下、大陸諸国で抗争が繰り広げられました。一方、体制から距離をとったイギリスでは労働・選挙・貿易などで自由主義的政策が採られました。しかし、労働者の立場はまだ弱く、富の公平な分配を目指す社会主義思想が生まれたり、選挙法改正に反発するチャーティスト運動が起こったりしました。
---
## イギリスの諸改革
### 自由主義的政策
::top
1820年代、イギリスは次の自由主義的政策を実施しました。
::

- 1824年、団結禁止法廃止
  :::lead
  労働組合の結成を許可
  :::
- 1828年、[[審査法]]廃止
- 1829年、[[カトリック教徒解放法]]成立
  :::lead
  [[オコンネル]]の活動で成立し、カトリック教徒の公職への就任を許可
  :::

### 選挙制度の改革
::top
1832年、[[第１回選挙法改正 ]]
:::lead
1832年、[[第１回選挙法改正 ]]
:::
、[[第１回選挙法改正 ]] 有権者の減った選挙区（腐敗選挙区）を廃止し、産業資本家などの都市中産階級に選挙権を許可
::
::sup
1918年の第４回で21歳以上の男性、30歳以上の女性に選挙権を許可
::
::sup
1928年の第５回で21歳以上の男女に選挙権を許可
::
---arrow---
::top
1830年代後半、改正内容に不満を抱く労働者が、男性普通選挙などを求める人民憲章を掲げて、政治運動[[チャーティスト運動 ]]を起こしました。
::
::last
運動は、1839、42、48年に大規模な請願をおこないましたが、すぐに成果をあげられませんでした。
::
::sup
チャーチスト運動
:::lead
チャーチスト運動
:::
ト運動人民憲章（People's Charter）に由来
::
### 貿易政策
::top
産業革命後、イギリスでは自由貿易の要求によって次の政策が実施されました。
::

- 1846年、[[穀物法]]廃止
  :::lead
  1815年に制定され、地主の利益保護のために==安価な外国産穀物の輸入を制限（高関税を設定）==
  :::
- 1834年、東インド会社の中国貿易独占権の廃止
- 1849年、[[航海法]]廃止

### 独立運動と列強の干渉
::top
19世紀、オスマン帝国支配下の諸民族が、帝国の衰退に乗じて独立運動を起こしました。
::
::middle
列強は、この運動に干渉しながら勢力拡張を目指しました。
::
::last
ここから生じる国際的諸問題を、西欧列強は「東方問題」と呼びました。
::

## 社会主義思想
### 労働者の待遇改善
::top
産業革命後、イギリスの労働者の生活は悲惨な状況にありました。
::
::last
伝統的な手工業者が機械制工場に生活を脅かされ、1810年代に、[[ラダイト運動 ]]（[[機械うちこわし運動  ]]）を起こしました。
::
---arrow---
::top
1833年、[[工場法]]制定
:::lead
1833年、[[工場法]]制定
:::
、[[工場法]]制定児童労働に雇用年齢・時間などの制限を設けた一連の法律
::
---arrow---
::top
以降、労働条件は次第に改善されました。
::

工場や土地などの生産手段を社会の共有にして資本主義の弊害を除き、平等な社会の建設を説く人々が現れました。

#### マルクスの登場
::top
1848年、[[マルクス]]と友人[[エンゲルス]]が、『[[共産党宣言 ]]』を発表しました。
::
::middle
マルクスらは、[[唯物史観 ]]、経済史観（資本主義体制の没落の歴史的必然性）を提唱し、労働者階級の政権獲得を主張しました。
::
::last
また、マルクスらは、ほかの社会主義者の考えを「[[空想的社会主義者 ]]」と批判しました。
