---overview---
古代ギリシアのポリスであるアテネでは、前６世紀に参政権をもつ市民が直接的に運用する民主政治が実現しました。今日の民主政治とはいくつか違いがありますが、その世界史的な意義は非常に大きいものです。アテネはペルシア戦争を経てその民主政治を完成させ、また、デロス同盟の盟主として勢力を強めました。
---

### スパルタ
#### スパルタの概要
::top
系統：ドーリア人
::
::top
政治：貴族政（市民に貴族・平民の区別がないので厳密には「市民政」）
::
::top
奴隷：市民共有、市民人口の10倍、被征服民（非ドーリア人）がほとんど
::
#### 奴隷
::top
スパルタの奴隷は[[ヘイロータイ]]と呼ばれ、==農業に==従事させられました。
::
::last
他には、[[ペリオイコイ]]と呼ばれる奴隷と市民の中間的な存在がいて、==奴隷ではない==が、参政権をもたず、農業・商工業に従事させられました。
::
---arrow---
::top
ヘイロータイは、数で市民を圧倒的に上回り、しばしば反乱を起こしました。
::
::last
立法者[[リュクルゴス]]の指針で、市民は貨幣使用や商工業活動を禁止され、ヘイロータイの反乱を防ぐために軍事訓練に専念しました。
::
::gazo
::
---arrow---
::gazo
::
::middle
アテネは、強大な海軍力を背景に他の同盟国に対する支配力を強めました。
::
::last
アテネでは、軍艦の漕ぎ手を務めるような財産の少ない市民が参政権を得て、前５世紀半ば、将軍[[ペリクレス]]の指導の下、民主政が確立しました。
::
::sup
デロス同盟
:::lead
デロス同盟
:::
同盟の金庫が、当初デロス島に置かれたことに由来
::
---arrow---
::top
ペリクレスの提案に従い、両親ともにアテネの生まれの18歳以上の男性は、その貧富に関わらず、平等に参政権を得ました。
::
::middle
青年男性市民が参加する集会[[民会]]がアテネの政治の最高機関であり、将軍などを除く役人は一般市民からの抽選で選ばれました。
::
::last
役人の責任は、弾劾裁判などで厳しく追及されました。
::
---arrow---

アテネの民主政は、デロス同盟の諸ポリスに広まりました。
::
