---overview---
エーゲ海では、オリエントからの影響のもとに、ヨーロッパではじめて青銅器時代に入りました。前1200年頃、この文明が突如破壊されると、以降ギリシア史は暗黒時代に入り、鉄器時代に移行していきました。混乱から逃れようとギリシア人は移住を続けましたが、前８世紀に各地にポリスが登場すると定住を進め、暗黒時代が終わりました。
---

## ポリスの成立
### ポリスの成立と暗黒時代の終焉
::top
前８世紀、各地の有力者の指導の下、ギリシア人の定住が促されました。
::
::middle
城山[[アクロポリス]]を中心に集住（シノイキスモス）する都市が現れました。
::
::last
これら[[ポリス]]と呼ばれる都市が現れたことで社会が安定し、暗黒時代は終わりました。
::
::gazo
::
---arrow---
::top
社会の安定は人口増加・土地不足を招きました。
::
::middle
前８世紀半ば、ギリシア人は従属させた地域の開発を始め、地中海と黒海の沿岸各地に植民市を建設し、交易を活発化させました。
::
::last
商業活動では、フェニキア文字を基につくったアルファベットを用いました。
::

### 独立と同胞意識
::top
各ポリスは独立していて、古代のギリシア人は統一国家をもちませんでした。
::
::middle
しかし、彼らは中部ギリシアの聖地デルフォイでの[[アポロン]]神の神託、４年に１度開かれるオリンピアの祭典を通じて同一民族の意識をもち続けました。
::
::last
彼らは自分たちをヘレネス（ギリシア人）と呼び、また==異民族を==[[バルバロイ]]（訳の分からない言葉を話すもの）と呼んで区別しました。
::
::gazo
::
### ポリスの住民
::top
ポリスの住民は、自由な市民とこれに隷属する奴隷からなりました。
::
::middle
市民には、さらに貴族と平民の区別がありました。
::
::middle
貴族は、高価な武器を身につけ、騎馬で移動する戦士で、国防の主力を担いました。
::
::last
前７世紀までに、少数の貴族が政治を独占する貴族政が一般的になりました。
::
### ポリスの構造と役割

#### ポリスの役割
::top
アリストテレスは、ポリスにおいて仕事を分業して生きる動物が人間であり、その幸福や徳もポリスの生活でのみ実現されると考えました。
::
::last
そのため、「人間はポリス的動物である」と定義しました。
