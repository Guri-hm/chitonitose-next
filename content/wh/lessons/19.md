---overview---
前221年、秦が広大な中国を初めて統一しました。その後わずか15年で滅びましたが、後の中国で使われていく「皇帝」の称号を使い始めるなど、その影響は大変大きいものです。今日中国をChinaと言うのも、<ruby>秦<rt>シン</rt></ruby>から来ているとされます。
---
## 春秋・戦国時代
### 周の衰退
::top
前８世紀、周は周辺民族に首都を攻略され、都を東方の{{洛邑|らくゆう}}に移しました。
::
::middle
これにより、以前の周を西周、以後の周を東周と区別します。
::
::last
以後、東周の勢力は衰えていきます。
::
---arrow---
::top
周が東周となった前８世紀から前５世紀までを[[春秋時代]]、以後の前３世紀までを[[戦国時代]]と呼びます。
::
::middle
春秋時代には、各地の諸侯は東周の王の権威を利用し、他の諸侯より優位に立とうと争いました。
::
::last
戦国時代には、各地の諸侯は自らを王と自称し互いに争いました。
::
### 春秋・戦国時代
#### 春秋時代
::top
前８世紀から前５世紀、東周の諸侯のなかに、王の権威を利用して他の諸侯より優位に立とうとする者が現れました。
::
::last
このような有力諸侯を[[覇者]]と呼びます。
::
#### 戦国時代
::top
前５世紀、諸侯は東周の王を無視し、自らを王と称して国を建てていきました。
::
::last
小国は次々と併合され、やがて７つの強国[[戦国の{{七雄|しちゆう}}]]がならび立ちました。
::
---arrow---
::top
七雄の中でも、西方の[[秦|しん]]は[[商鞅|しょうおう]]を登用し、彼による改革[[商鞅の変法（改革）]]で富国強兵を進めました。
::
::gazo
::

- [[鉄製農具]]が使用され、農業生産力が向上
- 商工業が盛んになり、[[青銅の貨幣]]が流通
- 実力本位・下剋上の傾向で、有能な人材の登用
- [[諸子百家]]と総称される思想家たちが現れ、新しい社会秩序の在り方を模索

## 初の中国統一
### 秦の台頭
::top
前４世紀、戦国の七雄の１つ[[秦]]は、[[商鞅]]を登用して改革を進め、国力をつけました。
::
::last
前221年、秦は中国を統一しました。
::
---arrow---
::top
秦の王は、「王」の上位である「皇帝」という称号を考案し、[[始皇帝]]と称しました。
::
::gazo
::
#### 中央集権
::top
[[封建制]]を廃止し、中央から派遣した役人に各地の統治をまかせる[[郡県制]]を採用しました。
::
#### ものの統一
::top
各地で異なっていた文字・{{度量衡|どりょうこう}}（長さや重さなどの単位）・貨幣を統一しました。
::
#### 思想の統一
::top
儒家などの書物を禁書として廃棄したり、始皇帝に批判的な学者（主に儒家の学者）を生き埋めにしたりする[[焚書|ふんしょ]]・[[坑儒|こうじゅ]]を強行しました。
::
#### 騎馬遊牧民への対抗
::top
北方の騎馬遊牧民[[匈奴|きょうど]]の侵入に対抗し、戦国時代以来の[[長城]]を修築しました。
::
::gazo
::
::last
始皇帝の死後、農民の反乱[[陳勝・呉広の乱]]が起き、秦は統一後15年で滅びました。
::
