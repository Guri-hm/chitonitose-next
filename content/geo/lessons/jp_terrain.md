#### 地震の発生
::top
海溝付近ではプレートの歪みの力がたまります。
::
::last
そのため、海溝付近では==震源が深い==地震が起きやすいです。
::
#### 列島の形成
::top
海洋プレートと大陸プレートの境には、日本列島のような[[弧状列島]]（[[島弧]]）が形成されます。
::
::top
海洋プレートは、ある程度まで沈みこむとマグマを発生させます。
::
::middle
このマグマが浮力で上昇し、地表付近に火山を形成します。
::
::middle
つまり、火山は==海溝から少し離れたあたりに形成==されることになります。
::
::last
海溝側から大陸側に向かって、火山が噴出しはじめる線のことを[[火山前線]]（[[火山フロント]]）と呼びます。
::
::gazo
::
::top
例えば、東北地方の場合は、==火山前線の西側に火山が分布し、太平洋側には火山が分布しない==ということです。
::
::gazo
::
::top
火山前線が走る日本列島には多くの火山が分布しますが、==四国には存在しません==。
::
::last
火山は火山活動によって降灰による農業への被害をもたらしたりします。
::

::top

#### 糸魚川・静岡構造体
::top
東北日本と西南日本は糸魚川・静岡構造体で二分されています。

#### 中央構造体
::top
西南日本は[[中央構造体]]と呼ばれる断層帯で二分されています。
::
::last

::gazo

::gazo

::last
[[新]]期造山帯のため、急峻な山脈・山地が多く、山麓には扇状地が多数発達しています。
::

::top
日本は山と雨が多いため、土砂が下流へ運ばれて、[[沖積平野]]（谷底平野・扇状地・三角州）が形成されます。

## 日本列島の気候
### 気候区分
::top
国土の大半は[[Cfa]]に属し、北海道や東北地方の一部が[[Df]]です。
::
### 四季
::top
夏と冬で変化する[[季節]]風や付近の気団の影響をうけます。
::
::gazo
::
::middle
低気圧が日本海側にあるとき、太平洋側の高気圧から風が吹き、北アルプスを越えて日本海側に[[フェーン]]現象をもたらします。
::
::last
この現象で==雪解けの洪水や地滑り、そして大火事が発生==したりします。
::
#### ６～７月
::top
北の寒冷・多湿な気団（オホーツク気団）と南の高温・多湿な気団（小笠原気団）が接触します。
::
::last
接触面には[[梅雨前線]]（停滞前線の一種）ができ、接触面の西側（西日本地域）では集中豪雨が生じます。
::
::top
オホーツク気団からは寒冷で湿った風である[[やませ]]が東北地方に吹きます。
::
::last
これが長く吹くと冷害をもたらします。
::
::gazo
::
::last
この時期から秋にかけ、==台風==（熱帯低気圧）が小笠原気団の西側の縁に沿って襲来します。
::
#### 秋
::top
東日本を中心に秋の長雨があります。
::
#### 冬
::top
[[シベリア]]高気圧から[[北西]]季節風が吹きます。
::
::middle
寒冷なこの季節風は、日本海の海上で水を多く取り込み、日本海側に豪雪をもたらします。
::
::middle
そして、風は山脈を越え、乾いた風となって太平洋側に吹きます。
::
::last
これらのことから、冬に==日本海側は湿潤==、==太平洋側は乾燥==します。
::
## 日本列島の災害
### 災害と対策

- 火山
- 豪雨
- 地震
- 地震による津波、埋立地の==液状化==
- 低気圧や台風による高潮
- 都市化によるヒートアイランド

---arrow---
::top
これら災害に対しては、自治体ごとに==ハザードマップ==を作成し、防災・減災の徹底を図っています。
::
