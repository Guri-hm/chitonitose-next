## 地形
::top
ヨーロッパの主な地形は次の図の通りです。
::
::gazo
::
::last
南部には新期造山帯に属する山脈が横断し、北部には古期造山帯に属する山脈が分布します。
::
::top
古期造山帯：スカンディナヴィア山脈・ぺニン山脈など
::
::top
新期造山帯：ピレネー山脈・アルプス山脈など
::
### アイスランド
::top
アイスランドは、[[広がる]]境界上に位置します。
::
::last
そのため、アイスランドの島々には温泉・火山が分布し、海底には[[大西洋中央海嶺]]が形成されています。
::
### ヨーロッパ北部～中部
#### 山脈
::top
スカンディナヴィア半島には、[[古期]]造山帯のスカンディナヴィア山脈が分布します。
::
#### 大陸氷河
::top
ヨーロッパ北部～中部は、かつて氷河に広く覆われました。
::
::last
そのため次の場所には氷河地形が見られます。
::
::top
ノルウェー西岸部：[[フィヨルド]]
::
::top
フィンランド：[[氷河湖]]
::
::top
北ドイツ平原：[[モレーン]]
::
::gazo
::
::top
氷食を受けた地域は、肥沃な土壌が侵食・運搬されたためにやせ地になります。
::
::last
やせ地は畑作に適さないため、酪農や林業が営まれます。
::
::sup
北ドイツ平原の南縁は、モレーンから飛来した肥沃なレス堆積して畑作が活発
::
### ヨーロッパ中部
#### 河口
::top
次の２つの理由から、河口に[[エスチュアリー]]が発達しています。
::

- 河川の勾配が緩い
- 土砂の供給量が少ない

::sup
例：テムズ川・エルベ川・セーヌ川・エヌル川
::
---arrow---
::top
アルプス山脈から大量の土砂が河口に運ばれるため、ライン川にはエスチュアリーが形成されていません。
::
::gazo
::
#### パリ盆地
::top
ケスタが発達しています。
::
::last
適度な傾斜で日照が均等に届き、水はけもよいため、ぶどう栽培に適しています。
::
::gazo
::

### ヨーロッパ西岸部～中部
::top
ヨーロッパの西側は、[[暖]]流の北大西洋海流が流れ、その上を[[偏西]]風が吹きます。
::
::last
この海流と風の影響を受ける地域では、高緯度でありながら冬場も温暖で、気温の年較差が小さくなります。
::
---arrow---
::top
ヨーロッパは標高200m未満の低地が広がり、南部には東西にアルプス・ヒマラヤ造山帯の山脈が連なります。
::
::last
偏西風が遮られず、山脈に沿うように内陸まで到達します。
::
---arrow---
::top
年間を通して湿潤で、寒すぎず暑すぎない（最寒月平均気温がー３℃を下回らない、最暖月平均気温が22℃を上回らない）、[[西岸海洋性]]気候（[[Cfb]]）が広範囲に分布します。
::
::gazo
::
::last
北アメリカや南アメリカは太平洋に沿って南北に山脈が走るため、偏西風が遮られてしまうからです。
::
::gazo
::
::middle
山脈の風上には南北に[[西岸海洋性]]気候（[[Cfb]]）が、風下には[[亜寒帯湿潤気候]]気候（[[Df]]）が分布します。
::
::last
また、西岸海洋性気候が分布する西部に比べて、気温の年較差が大きくなります。
::
::gazo
::
::middle
結果、ヨーロッパ東部では[[亜寒帯湿潤気候]]気候（[[Df]]）が分布します。
::
::last
この地域では、西岸海洋性気候が分布する西部に比べて、気温の年較差が大きくなります。
::
### ヨーロッパ南部
::top
ヨーロッパ南部は、夏に北上した中緯度高圧帯（亜熱帯高圧帯）の影響で少雨になります。
::

### ベルギーの言語
::top

::
