## 離水海岸
::top
浅い海底が隆起または海面の低下した海岸を離水海岸と呼びます。
::
::last
離水海岸のうち、平野の場所を海岸平野、段丘の場所を[[海岸段丘]]と区別します。
::
::gazo
::
### 海岸平野
::top
代表的な海岸平野は、千葉県東岸の九十九里浜です。
::
::gazo
::
### 海岸段丘
::top
海岸段丘の台地面では水が得にくいため、==畑や果樹園に利用==されます。
::
::middle
代表的な海岸段丘は、高知県の{{室戸岬|むろとみさき}}です。
::
::last
なお、海岸段丘は洪積台地の１つです。
::
::gazo
::
::gazo
::
## 沈水海岸
::top
陸地の沈降や海面の上昇した海岸を沈水海岸と呼びます。
::
::gazo
::
::top
沈水海岸には、主に次の３つがあります。
::

- [[リアス海岸]]
  :::lead
  [[V]]字谷が沈水したのこぎりの歯状の海岸
  :::
- [[フィヨルド]]
  :::lead
  氷河によって侵食された[[U]]字谷が沈水した、両側が絶壁の細長い入り江
  :::
- [[エスチュアリー]]（[[三角江]]）
  :::lead
  河口の沈水によるラッパ状の入り江
  :::

### リアス海岸
::top
河川の侵食で形成された谷を[[V字谷]]、氷河による侵食で形成された谷を[[U字谷]]と呼びます。
::
::gazo
::
::top
このうちで、[[V]]字谷が沈水したのこぎりの歯状の海岸をリアス海岸と呼びます。
::
::middle
語源は、この海岸が見られるスペイン北西の海岸「リアスバハス（リアスバイシャス）」です。
::
::last
リアス海岸は、日本国内でも見られ、三陸海岸（岩手・宮城）、志摩半島（三重）、若狭湾（京都・福井）が代表的です。
::
::gazo
::
::top
リアス海岸は、波が遮られて入り江が穏やかになるため、==良港がよく立地します==。
::
::middle
しかし、==地震による津波がひとたび入り江に進入すると、両岸が狭いため、波は一層高くなります==。
::
::last
時には、10階建てのビルも越える高さになります。
::
::gazo
::
::gazo
::
::gazo
::
### フィヨルド
::top
河川の侵食で形成された谷を[[Ｖ字谷]]、氷河による侵食で形成された谷を[[Ｕ字谷]]と呼びます。
::
::gazo
::
::top
このうちで、[[Ｕ]]字谷に海水が流入して形成された地形をフィヨルドと呼びます。
::
::middle
ヨーロッパの[[ノルウェー西岸]]、ラテンアメリカのチリ南部、オセアニアの[[ニュージーランド南西]]のフィヨルドが代表的です。
::
::last
上記以外の場所でも、基本的には==かつて氷河があった高緯度の地域==で見られます。
::
::gazo
::
::gazo
::
::gazo
::
### エスチュアリー
::gazo

::
::top
このような河口が沈水すると、河口に漏斗やトランペットのような地形が形成されます。
::
::last
この地形を[[エスチュアリー]]（三角江）と呼びます。
::
::gazo

::
#### エスチュアリーの分布
::top
エスチュアリーとして有名なのは、アルゼンチンとウルグアイの間を流れるラプラタ川、五大湖からカナダ東部を流れる[[セントローレンス]]川です。
::
::last
ラプラタ川は、大航海時代にマゼラン（生没：1480年頃～1521年）が大きい河口を海峡と誤ったほどです。
::
---arrow---
::top
他のエスチュアリーは、安定陸塊が多く占める==ヨーロッパでよく見られます==。
::
::middle
この理由は、土砂を多く供給する急峻な山地が安定陸塊では少ないためです。
::
::last
代表例は、エルベ川・テムズ川・ジロンド川・セーヌ川などです。
::
::gazo
::
::top
ただし、エスチュアリーが見られない例外的なヨーロッパの川が、[[ライン]]川です。
::
::middle
それは源流が[[新]]期造山帯に属すアルプス山脈だからです。
::
::last
急峻な山地であるため、土砂の供給が多くなります。
::
::gazo
::
#### エスチュアリーと産業
::top
エスリュアリーは産業に強く関わります。
::
::last
エスチュアリーは水深が深く、入り江に面して大平野が広がるため、==貿易港や大都市、工業地域が発達します==。
::
## 海岸の微地形
### 微地形の種類
::top
海岸には、沿岸流による堆積で次のような小規模な地形が見られます。
::

- [[砂嘴|さし]]
  :::lead
  沿岸流が運んだ砂が鳥のくちばし（嘴）状に海へ突き出した地形
  :::
- [[砂州|さす]]
  :::lead
  砂嘴が発達して、湾をふさいでいる地形
  :::
- [[陸繋島|りくけいとう]]
  :::lead
  砂州によって繋がった島
  :::
- [[潟湖（ラグーン）]]
  :::lead
  砂州によって外洋から切り離された湖
  :::

## サンゴ礁
### サンゴ礁の分布
::middle
熱帯から亜熱帯の、水温が==温暖（25～30℃）で==きれいな海の海岸には、サンゴ礁が多く分布します。
::
::middle
そのため==サンゴ礁は寒流の流れる地域に分布しにくいです==。
::
::last
日本では==北緯30度以南==のトカラ列島が北限で、奄美・沖縄と小笠原の島々に限られています。
::
::gazo
::
::gazo
::
### サンゴ礁の発達
::top
サンゴ礁の形態は、島との関係で次の順に変化していきます。
::

- [[裾礁|きょしょう]]
  :::lead
  島の海岸に接して発達したサンゴ礁
  :::
- [[堡礁|ほしょう]]
  :::lead
  島の沈降あるいは海面の上昇で、防波堤のように島を囲んだサンゴ礁
  :::
- [[環礁|かんしょう]]
  :::lead
  島の沈降あるいは海面の上昇で、サンゴ礁のみが環状に残っているもの
  :::

::gazo
src="../share/img/loading.svg" />
::
### サンゴ礁の分布
::top
世界のサンゴ礁の分布を見ると、下図のA・Bの海域にはサンゴ礁の分布が見られません。
::
::last
これはA・Bの海域にはそれぞれ暖流/寒流の[[ペルー海流]]と[[ベンゲラ海流]]が流れているからです。
::
::gazo
::
