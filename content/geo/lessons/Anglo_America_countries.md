## アメリカ合衆国
### 基礎データ
::gazo
::
::middle
1776年、植民地の人々はイギリス本国からの独立を勝ち取り、13州からなる連邦国家としてアメリカ合衆国が成立しました。
::
::last
独立後、アメリカ合衆国は他国の領土を買収や戦争による割譲で獲得し、その国土は太平洋に至りました。
::

#### ヒスパニック
::top
ヒスパニックは、スペイン語を母国語とする中・南米出身者です。
::
::middle
メキシコ付近やフロリダ州に多く居住しています。

::
#### アフリカ系黒人
::top
17～19世紀、多くの黒人が、南部の綿花プランテーションで働く奴隷として、アフリカから連れてこられました。
::
::middle
彼らの子孫であるアフリカ系黒人は、歴史的背景からアメリカ合衆国南部に多く居住しています。
::
::last
ヒスパニック同様に==低賃金労働者が多いです==。
::
#### アジア系
::top
19世紀のゴールドラッシュでアジアから移民し、太平洋側に多く居住しています。
::
::middle
当初は移民に寛容だったアメリカ合衆国ですが、後に移民法が制定され、移民は制限・禁止されました。
::
::middle
1960年代、移民法が改正されると、アジア系・ヒスパニック系の移民が急増しました。
::
::last
特にアジア系移民の流入人口は、ヨーロッパ系移民の流入人口を==上回る==傾向がみられます。
::
---arrow---
::top
アジア系は、アメリカ合衆国の人種・民族のなかで==高学歴者の割合が極めて高いです。==
::
#### 先住民
::top
先住民であるアメリカンインディアンは、ヨーロッパ系の人々の入植の過程で、土地を奪われ、保留地に封じ込められました。
::
::middle
アメリカ合衆国の国土が西へ拡大するのにあわせ（西漸運動）、西へ西へ追いやられたのです。
::
::last
このような歴史的背景から、西部にはアメリカンインディアンが居住しています。
::

### 歴史
::top
16～17世紀、イギリスとフランスが北米で植民地を拡大しました。
::
::middle
1754～63年の英仏戦争の結果、カナダ全土のフランス植民地がイギリスに割譲されました。
::
::middle
1867年、イギリス内のカナダ自治領になりました。
::
::last
1931年、ウェストミンスター条例で外交権を獲得し、法的に独立国家として認められました。
::
