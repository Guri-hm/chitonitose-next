## 主要な工業地域
### 重工業
::top
ヨーロッパの重工業の地域は、炭田付近に古くから立地しています。
::
::last
採掘される石炭を利用した鉄鋼業が盛んです。
::
---arrow---
::top
近年、国内の鉄鉱石が枯渇し、鉄鉱石を輸入に頼るようになりました。
::
::last
鉄鋼業は、==臨海指向型の立地に変化しています==。
::
::gazo
::
#### ルール地方
::top
ドイツのルール地方は、[[ルール炭田]]を基礎として発展したヨーロッパ最大の工業地域です。
::
::middle
炭田で採掘できる[[石炭]]と[[ライン川]]の水運がこの地域を支えています。
::
::last
石炭に依存した鉄鋼業などの重工業（素材型重工業）で栄えてきました。
::
---arrow---
::top
ドイツのルール炭田、フランスの北フランス炭田・ロレーヌ鉄山と結んだ地域は、重工業三角地帯と呼ばれ、第二次世界大戦後の経済発展に貢献してきました。
::
::last
しかし、石油危機以降の資源価格の高騰、国内資源の枯渇などによって、近年この地帯は==停滞・衰退しています==。
::
#### ミッドランド地方
::top
イギリスのミッドランド地方（バーミンガム周辺）は、石炭や鉄鉱石が産出します。
::
::middle
産業革命後にイギリスが「世界の工場」と呼ばれていた頃、この地方は世界最大の工業地域でした。
::
::last
ルール地方同様、今日では停滞・衰退しています。
::
### 先端技術産業
::top
イギリスの首都[[ロンドン]]、フランスの首都[[パリ]]は、大量の労働力と大規模な市場をもっています。
::
::last
市場指向型の工業をはじめ、多種類の工業が発達しています。
::
---arrow---
::top
今日、研究開発施設、良質な労働力を必要とする==先端技術産業も発達してきました==。
::
::gazo
::
::last
鉄鋼や石油化学などの重化学工業は、==臨海指向型になります==。
::
::gazo
::
#### オランダの「EUの玄関」
::top
オランダの[[ロッテルダム]]は、「EUの玄関」と呼ばれる貿易港[[ユーロポート]]を有します。
::
::last
石油化学コンビナートが立地し、精製した石油をパイプラインで内陸に運びます。
::
::gazo
::
---arrow---
::top
フランスやドイツなどEU諸国は、航空機の共同開発で対抗しています。
::
::middle
例えば、エアバスという企業が有名です。
::
::last
この共同開発での航空機の組立は、フランスの南西部の都市[[トゥールーズ]]でおこなわれています。
::
::gazo
