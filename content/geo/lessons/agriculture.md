## 農業の成立条件
### ２つの条件
::top
農業は、地域によって栽培される農作物、飼育される家畜、農業経営の仕方に様々な違いがあります。
::
::last
この違いは、次の２つの条件に大きく成因すると考えられます。
::

- 自然条件
  :::lead
  気温・降水量・地形・土壌など
  :::
- 社会条件
  :::lead
  伝統や生活様式・市場への距離・経営の形態・農業政策など
  :::

### 自然条件
#### 気温
::top
作物の栽培には、作物の種類に適した気温が必要です。
::
::middle
寒い地域に行けば行くほど栽培できる作物の種類は減少し、==最暖月平均気温==[[10]]℃を下回ると、栽培が原則的に困難になります。
::
::middle
このように、その寒さを下回ると作物の栽培が困難になる数値を[[寒冷]]限界と言います。
::
::last
なお、寒冷限界に該当する==最暖月平均気温10℃==未満の地域は、[[寒]]帯（Ｅ）に重なります。
::
::gazo
::
::middle
一方で、垂直方向、つまり高山に登っていくと気温が低下して寒冷限界にぶつかります。
::
::middle
このような垂直方向の寒冷限界を[[高距（高度）]]限界と言います。
::
#### 降水量
::top
気温のほかに、降水量も作物の栽培に大きく関わっています。
::
::middle
そして、農業の種類ごとに適切な年間降水量があります。
::
::last
農業と年間降水量の関係は次の図のように表せます。
::
::gazo
::
::top
稲作にとっての1000mm、畑作にとっての500mm、牧畜にとっての250mmは、それぞれの[[乾燥]]限界を示しています。
::
::last
それぞれ対応する年間降水量を下回ると、経営が困難になります。
::
::top
乾燥地域でも==カナート==・==フォガラ==などを利用した==灌漑==や、わずかな降水の蒸発を抑えて栽培に利用する乾燥農法（ドライファーミング）といった方法で、乾燥限界を克服していたりします。
::
#### 地形
::top
傾斜地は平坦な土地の確保が難しく、また、雨・風による土壌侵食も懸念されます。
::
::last
しかし、制約を克服したり、特徴を活かした農業もおこなわれています。
::
::top
傾斜地の克服例は、日本やインドネシアなどで見られる==棚田==・==段々畑==です。
::
::last
棚田は斜面での平坦な土地の確保を実現し、段々畑は斜面の水はけのよさや日当たりのよさを利点に変えています。
::
::gazo
::
::last
この耕作は、等高線に沿って帯状に畑をつくるもので、雨によって土壌が下へ流されるのを防ぎます。
::
::gazo
::
::gazo
::
::last
例えば、インドのデカン高原に分布する[[レグール]]は==綿花==栽培に適し、ブラジル高原に分布する[[テラローシャ]]は==コーヒー==栽培に適しています。
::
::gazo
::
### 社会条件
#### 伝統・宗教
::top
伝統や宗教によって食生活に違いが生じ、そこから農業にも違いが生じます。
::
#### 市場との距離
::top
市場の規模や市場との距離によって農業形態が決まります。
::
::middle
チューネン著『孤立国』によれば、==市場に近い場所では==[[集約]]的になり、[[園芸農業]]や酪農が営まれます。
::
::last
一方で、==市場に遠い場所では==[[粗放]]的になり、穀物農業や放牧が営まれます。
::
::sup
集約的
:::lead
集約的
:::
iv class="lead">単位面積あたりの労働力の投下量が大
::
::sup
粗放的
:::lead
粗放的
:::
iv class="lead">単位面積あたりの労働力の投下量が小
::
#### 経営の形態
::top
経済発展の差などで、経営の形態が異なります。
::
::last
その形態は次の３つに分かれます。
::

- [[自給的]]農業
  :::lead
  販売を目的とせず、自家消費のためにおこなわれている形態
  :::
- [[商業的]]農業
  :::lead
  販売を目的とした農産物の生産をおこなう形態
  :::
- [[企業的]]農業
  :::lead
  多額の資本投下をおこない、世界市場を対象とした大規模な形態（商業的農業の規模が大きくなったもの）
  :::

::sup
==土地生産性==
:::lead
==土地生産性==
:::
class="marker">土地生産性単位面積あたりの生産量の多さで、肥料・労働力の投下で上昇
::
::sup
==労働生産性==
:::lead
==労働生産性==
:::
class="marker">労働生産性１人あたりの労働によって得られる生産量の多さで、機械化・栽培技術の進歩で上昇
::
#### 技術の進歩
::top
技術の進歩によって耕地が拡大したり、耐寒性・耐乾性の品種が開発されたりします。
::
::last
また、[[冷凍船]]の発明によって、腐りやすい野菜・果実・肉類を遠方に運べるようになりました。
::
::gazo
::
### 三大穀物
#### 米
::top
原産地はインド東部から長江中下流域原とされています。
::
::middle
高温多雨で年降水量[[1000]]mm以上が栽培の条件になります。
::
::last
９割以上がアジアで生産・消費されます。
::
::gazo
::
::last
年間降水量[[500]]mm前後の半乾燥地域が適し、肥沃な土壌であるロシア・==ウクライナ==の[[チェルノーゼム]]、==アメリカ==の[[プレーリー土]]でよく栽培されます。
::
::gazo
::
::middle
温暖な地域に比べると、寒冷な地域では小麦の収穫が遅くなります。
::
::middle
また、北半球と南半球では収穫期が半年ずれるので、一年を通じて世界のどこかで小麦が収穫されています。
::
::last
北半球で小麦が収穫できない時期（{{端境期|はざかいき}}）には、南半球から小麦が輸出されます。
::
---arrow---
::top
受験では北半球と南半球の区別、温暖な地域と寒冷な地域の区別をしましょう。
::
::gazo
::
::middle
労働力も米や小麦に比べて少なくてすみ、食用のほか飼料用・油脂用にも利用されます。
::
::last
近年では、==バイオエタノール==の原料としても注目されています。
::
::gazo
::
::gazo
::
### 主な家畜
#### 牛
::top
肉牛・乳牛に分かれます。
::
::last
また、ネパールやチベットでは[[ヤク]]と呼ばれる牛が育てられています。
::
::gazo
::
::top
乳牛は冷涼な気候を好み、年中牧草が生育する程度に湿潤な地域（年間降水量500mm以上）であれば飼育できます。
::
::middle
そのため、米や小麦などの主要作物の栽培に適さない地域で飼育されます。
::
::last
また、製品の牛乳が傷みやすいため、酪農は大都市近郊が有利になります。
::
::top
ヒンドゥー教では神聖な動物とされ、食べることが禁忌とされています。
::
#### 豚
::top
最も食肉生産量の多い家畜です。
::
::last
世界の豚（豚肉）のの半分は中国で生産されます。
::
::top
イスラームでは不浄な動物とされ、食べることが禁忌とされています。
::
::gazo
::
::last
羊毛種である[[メリノ]]種が有名です。
::
::gazo
