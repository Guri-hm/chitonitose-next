## 気候と気候要素
### 気候とは
::top
気候とは、長期にわたって毎年繰り返される大気の総合的な状態のことです。
::
::middle
「暑くて夏に雨が多く降る」など、ある地域の性格とも言えます。
::
::last
そして、その性格を説明する際に用いる「暑いのか（気温）」「雨が多く降るのか（降水）」「風がどのように吹くのか（風）」という要素を、気候要素と言います。
::
::gazo
気候要素
::
### 気候要素
::top
主な気候要素は次の３つです。
::

- 気温
- 降水
- 風

---arrow---
::top
これら気候要素は、複数の要因に左右されます。
::
::middle
例えば、気候要素の「気温」は、「緯度」「海抜高度」などで「暑い」か「寒い」に分かれます。
::
::last
このような「緯度」「海抜高度」などにあたるものを、気候因子と言います。
::
## 気温
::top
気温の気候因子の１つ目は、「緯度」です。
::
::middle
気温の等しい地点を結んだ線を等温線と呼びます。
::
::last
次の図は上から順に「１月の等温線」「７月の等温線」です。
::
::gazo
１月の等温線
::
::gazo
７月の等温線
::
::top
等温線は、原則==緯線にほぼ平行になります==。
::
::last
高緯度ほど低い値の等温線が通るため、気温は緯度が高くなるほど高く/低くなります。
::
---arrow---
::top
加えて、高緯度ほど１月・７月で等温線の値の差が大きくなっています。
::
::middle
1年のうちの最高気温と最低気温の差を「気温の年較差」と言い、気温の年較差は緯度が高くなるほど大きい/小さくなります。
::
::last
この理由は、地軸のずれによって高緯度ほど１月・７月の入射角の差が大きくなるためです（入射角が大きいほど日光の量が大きい）。
::
::gazo
::
### 大陸と海洋
#### 比熱
::top
気温の気候因子の２つ目は、「大陸と海洋の違い」です。
::
::middle
物質1g当たりの温度を1℃上げるのに必要な熱量を「比熱」と言います。
::
::last
比熱が大きいほど物質は温まりにくく、反対に比熱が小さいほど物質は温まりやすいのです。
::
---arrow---
::top
大陸と海洋のうち、比熱が大きいのは大陸/海洋です。
::
::middle
つまり==海洋は年中ほぼ一定の温度を保ち（気温の年較差が小さい）、大陸は夏に暑く冬に寒くなります（気温の年較差が大きい）==。
::
::last
なお、このような年較差の特徴は、気温の日較差（１日の最低気温と最高気温の差）にも当てはまります。
::
---arrow---
::top
１月・７月の等温線を比べると、等温線は大陸部分で大きい差があります。
::
::last
比熱の違いがここに表れています。
::

#### 乾燥地域
::top
７月の等温線を見てみると、色で強調した大陸部分は、同緯度でも気温が違います。
::
::middle
一口に大陸と言っても、==乾燥地域（サハラ砂漠など）ではより夏に暑く、冬に寒くなります==。
::
::last
これは乾燥地域が比熱の大きい水の影響を受けにくいからです。
::
::gazo
::
::middle
海から離れるほど、隔海度が「大きい」と表現します。
::
::last
隔海度は、内陸に近づくほど大きくなります。
::
---arrow---
::top
海から離れれば、比熱の大きい水の影響を受けにくくなります。
::
::middle
つまり、==隔海度の大きい内陸ほど、夏に暑く冬に寒くなり、１日の最低気温と最高気温の差も大きくなります==。
::
::last
言い換えれば、気温の年較差・日較差は内陸側になるほど大きい/小さくなります。
::
### 東岸と西岸
::top
気温の気候因子の３つ目は、「ユーラシア大陸の東岸と西岸の違い」です。
::
::middle
ユーラシア大陸の西岸は、[[暖]]流である北大西洋海流と、年中南西の風向きである[[偏西風]]の２つの影響を受け、「==夏は冷涼で、冬は温暖==」になります。
::
::last
そのため気温の年較差は東岸に比べて大きく/小さくなります。
::
::gazo
::
::middle
==高度100mにつき、気温は0.65℃増減します==。
::
::middle
これを「気温の逓減率」と言います。
::
::last
==同緯度であっても、土地の標高に応じて気温に差異が生じます。==
::
::sup
0.65℃とあるが、様々な条件の下で変動
::
---arrow---
::top
下図の等温線の地点X付近では、低緯度側に向かって線が蛇行しています。
::
::last
これは、地点X付近にアンデス山脈が位置し、気温が低くなっているためです。
::
::gazo
::
---arrow---
::top
==地表が熱せられると、地表付近の空気が膨張し軽くなって上昇します==。
::
::middle
この上昇する空気の流れを上昇気流と言います。
::
::last
==上昇した空気は、含んでいる水蒸気で雲を形成し、雨を降らせます。==
::
::sup
上昇気流の発生要因は他にもあり
::
::sup
==寒流で空気が冷やされると上昇気流が発生しにくい==
::
::gazo
::
::middle
この下降する空気のながれを下降気流と言います。
::
::last
==下降気流では雲が形成されにくく、雨が降りません==。
::
::gazo
::
::last
下図の地球の模式図を見てください。
::
::gazo
::
::middle
上昇気流が生じて気圧が低くなる赤道付近一帯を[[赤道低圧帯]]（[[熱帯収束帯]]）と言います。
::
::middle
また、上昇した空気は、==緯度30°あたりで下降します==。
::
::last
下降気流が生じて地表面の気圧が高くなる緯度30°付近を[[中緯度高圧帯]]（[[亜熱帯高圧帯]]）と言います。
::
::gazo
::
::top
赤道に対して、==北極・南極では空気が冷え、下降気流が生じます==。
::
::last
下降気流が生じて地表面の気圧が高くなる北極・南極付近を[[極高圧帯]]と言います。
::
::gazo
::
::last
上昇気流が生じて気圧が低くなる緯度60°付近を[[高緯度低圧帯]]（[[亜寒帯低圧帯]]）と言います。
::
::gazo
::
::last
このような３つの流れを「大気の大循環」と言います。
::
::gazo
::
::middle
上昇気流がよく生じる地域では湿潤、下降気流がよく生じる地域では乾燥となります。
::
::last
そのため、次のように整理できます。
::

- 赤道低圧帯：[[湿潤]]
- 中緯度高圧帯：[[乾燥]]
- 高緯度低圧帯：[[湿潤]]
- 極高圧帯：[[乾燥]]

---arrow---
::top
ここまで学習してきた大気の大循環は、地軸の傾きによって南北に移動します。
::
::middle
７月、太陽の見かけの通り道は北寄りになり、赤道低圧帯などが北に移動します。
::
::last
１月、太陽の見かけの通り道は南寄りになり、赤道低圧帯などが南に移動します。
::
::gazo
::
::last
恒常風には、次の３つがあります。
::

- [[貿易風]]
  :::lead
  中緯度高圧帯から赤道低圧帯へ吹く、北半球では[[北東]]より、南半球では[[南東]]よりの風
  :::
- [[偏西風]]
  :::lead
  中緯度高圧帯から高緯度低圧帯へ吹く、北半球では南西より、南半球では北西よりの風
  :::
- [[極東風]]（[[極偏東風]]）
  :::lead
  極高圧帯から高緯度低圧帯へ吹く東よりの風
  :::

---arrow---
::top
これらの風は、「地球の自転」と「各地点の自転速度の差異（低緯度＞高緯度）」によって生じる「コリオリの力」で西より東よりの風に分かれます。
::
::gazo
::
::gazo
::
::last
高緯度から低緯度に吹く風は東よりの風に、低緯度から高緯度に吹く風は西よりの風になります。
::
::gazo
::
::last
夏は、比熱の小さい大陸が温められて上昇気流が発生し、低圧になった部分に海洋から==湿った==風が吹き込みます。
::
::gazo
::
::last
高圧になった大陸から低圧になった海洋へ==乾いた==風が吹き込みます。
::
::gazo
::
::middle
夏は海洋から大陸へ==湿った==風が、冬は大陸から海洋へ==乾いた==風が吹きます。
::
::last
同じ季節でも、==東アジアと東南アジア・南アジアは季節風の風向が異なる==ことに注意しましょう。
::
::gazo
::
::top
日本列島付近では、夏に太平洋から大陸へ向かって南東よりの季節風が吹きます。
::
::middle
この季節風は、太平洋から吹き込むため、初めのうちは湿った風です。
::
::middle
日本列島に吹き込み、==山脈を越える途中で太平洋側に雨を降らし、乾いた風となって山脈を吹き下りていきます==。
::
::last
乾いた風は、日本海で再び水蒸気を補給し、湿った風となって大陸へ吹き込みます。
::
::gazo
::
::top
次に冬の季節風です。
::
::middle
日本列島付近で、冬の季節風は大陸から太平洋へ向かって北西よりに吹きます。
::
::middle
この季節風は、大陸から吹き込むため、当初乾いた風です。
::
::last
日本海で水蒸気を補給し、湿った風となって日本列島に吹き込み、==山脈を越える途中で日本海側に雨を降らし、乾いた風となって山脈を吹き下りていきます==。
::
::gazo

### 熱帯低気圧
::last
熱帯の海洋で水蒸気を多量に空気が、太陽に温められて上昇し、雲を形成しながら渦を巻き、さらに周りの空気を集めて発達します。
::
---arrow---
::top
熱帯低気圧は、地域によって次のように呼称が異なります。
::

- 東アジア：[[台風]]
- 北インド洋：[[サイクロン]]
- 米国南部・メキシコ・西インド諸島：[[ハリケーン]]

::gazo
::
::last
代表的な局地風には、次のものがあります。
::

- [[やませ]]
  :::lead
  親潮やオホーツク海気団の影響で、初夏に北日本の太平洋/日本海側に吹く冷涼で湿った北東風
  :::
- [[シロッコ]]
  :::lead
  サハラ砂漠から地中海に吹く、砂塵やほこりを伴う蒸し暑い風
  :::
- [[ボラ]]
  :::lead
  アルプス山脈（ディナルアルプス山脈）からアドリア海へ吹き下りる、冷たく乾いた風
  :::
- [[フェーン]]
  :::lead
  アルプス山脈北側に吹き下りる、==暑く==乾いた南風
  :::

::gazo
::
::top
風が山脈の斜面を上り反対側に吹き下ろす時、上る側（風上）と吹き下ろす側（風下）で気温が大きく異なる現象を[[フェーン現象]]と言います。
::
::last
アルプス山脈南側から山脈北側に吹き下ろす局地風に由来します。
::
---arrow---
::top
気温の低減率は、100mにつき0.65℃増減しますが、厳密には空気の湿り気で少し変動します。
::
::middle
例えば、山脈を越えた乾いた風が吹き下ろす時、気温の低減率は100mにつき1.0℃増減するようになります。
::
::last
そのため下図のように風上と風下で気温が大きく異なる現象「フェーン現象」が生じるのです。
::
::gazo
::

- 対流性降雨
  :::lead
  強い日射で地表の空気が温められ、上昇気流が生じてもたらされる雨
  :::
- 地形性降雨
  :::lead
  湿った空気が山脈の斜面を上る際にもたらされる雨
  :::
- 前線性降雨
  :::lead
  寒気の上に暖気が乗り上げ、上昇気流が生じてもたらされる雨
  :::
- 低気圧性降雨
  :::lead
  低気圧に周囲から風が吹き込み、上昇気流となってもたらされる雨
  :::
