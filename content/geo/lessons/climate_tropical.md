## 熱帯
### ３つの気候区
::top
気候帯が寒帯（Ｅ）・乾燥帯（Ｂ）のどちらでもない場合、==最寒月平均気温==が[[18]]℃以上なら熱帯（Ａ）になります。
::
---arrow---
::top
熱帯（Ａ）は、ここから３つの気候区に分かれます。
::
::last
降水型がｆならAf（ただし最少雨月降水量60mm未満ならAm）、降水型がｗならAwです。
::

- Af
  :::lead
  [[熱帯雨林]]気候
  :::
- Am
  :::lead
  [[弱い乾季のある熱帯雨林]]気候
  :::
- Aw
  :::lead
  [[サバナ]]気候
  :::

## Af（熱帯雨林気候）
::gazo
::
::middle
また、太陽光が当たる日中は気温が著しく高くなるため、日中の気温の日較差が大きくなります。
::
::last
そして、[[赤道低圧]]帯の影響で年中多雨になります。
::
::last
上昇気流が発生しやすく、[[スコール]]という対流性降雨の豪雨が午後に見られることも特色です。
::
::gazo
::
### 植生
::top
多種類の常緑広葉樹による密林を形成します。
::
::last
南アメリカのアマゾン川流域の密林を[[セルバ]]、これ以外の地域の密林を[[ジャングル]]と呼びます。
::
::gazo
::
### 土壌
::top
土壌は、雨に栄養分が流されて鉄とアルミニウムの水酸化物が多く残されます。
::
::middle
このような==赤く酸性の==土壌を[[ラトソル]]と言います。
::
::last
この土壌は==農業に適していません==。
::
::gazo
::
::last
特に熱帯雨林気候のプランテーション農業は、==天然ゴム==・==カカオ==・==油やし==・==サトウキビ==・==バナナ==をよく栽培します。
::
---arrow---
::top
高温多湿な環境のため、建物を高床することで通気性をよくしています。
::
::gazo
::

## Am（熱帯モンスーン気候）
::gazo
::
::last
Afのうち、==最少雨月降水量が60mm未満の月がひと月でもあれば==、弱い乾季がある熱帯雨林気候、いわゆる熱帯モンスーン（Am）と判別できます。
::
::sup
Awは冬に乾燥する気候区
::
### 植生
::top
Af（熱帯雨林気候）と同様ですが、落葉樹林が混じります。
::
### 土壌
::top
[[ラトソル]]と呼ばれる、==赤く酸性で農業に適さない==土壌です。
::
### 生活
::top
==焼畑農業==や==プランテーション農業==が営まれます。
::
::last
特にAm（熱帯モンスーン気候）のプランテーション農業は、==天然ゴム==・==カカオ==・==油やし==・==サトウキビ==・==バナナ==をよく栽培します。
::
---arrow---
::top
高温多湿な環境のため、建物を高床することで通気性をよくしています。

### 特色・成因
::top
==Af（熱帯雨林気候）の周辺部に発達します==。
::
---arrow---
::top
夏は[[赤道低圧]]帯に入り雨季に、冬は[[中緯度高圧]]（[[亜熱帯高圧]]）帯に入り乾季になります。
::
::last
雨季と乾季の区別は大変はっきりしています。
::
---arrow---
::top
成因は、上記の通り低圧帯・高圧帯の移動です。
::
::last
「気候要素」のところで見た図をもう一度見てみましょう。
::
::gazo
::
::middle
これは、地軸のずれにより、太陽の見かけ上の軌道が赤道から南北に移動するためです。
::
::last
下図を見ると、太陽光が最も当たる部分は、夏至では赤道よりも北に、冬至では赤道よりも南に来ています。
::
::gazo
::
::gazo
::
### 植生
::top
==バオバブ==や==アカシア==など背の低い木からなる密度の低い林（疎林）、そして、長草草原が見られます。
::
::last
これら疎林と長草草原の広がる地帯を==サバナ==と呼びます。
::
::gazo
::

### 土壌
::top
土壌は、Af（熱帯雨林気候）・Am（弱い乾季のある熱帯雨林気候）と同じく[[ラトソル]]と呼ばれる、==赤く酸性で農業に適さない==土壌です。
::
---arrow---
::top
ラトソルが多い地域で乾燥した時期があると、表層近くで鉄やアルミニウムが集積して固結し、ラテライトが形成されます。
::
::middle
サバナ気候はこの条件をよく満たすため、ラテライトが多く形成されます。
::
::last
ラテライトのうち、酸化アルミニウムを多く含む鉱石をボーキサイトと呼び、アルミニウムの原料になります。
::
### 生活
::top
==焼畑農業==や==プランテーション農業==が営まれます。
::
::last
特にAw（サバナ気候）のプランテーション農業は、==綿花==・==コーヒー==をよく栽培します。
::
---arrow---
::top
高温多湿な環境のため、建物を高床することで通気性をよくしています。
::
