## 工業の発達
### 工業とは
::top
工業とは、原材料（農林水産物や鉱山資源など）を加工し、有用な製品をつくる産業です。
::
::last
加工する過程で、技術・労働力・動力などが必要になります。
::
### 発達順
::top
どの国でも次の①～③の順に工業が発展します。
::

- 軽工業
  :::lead
  繊維工業などが該当し、大量の労働力が必要
  :::
- 重化学工業
  :::lead
  鉄鋼業・石油化学工業などが該当し、金・施設（資本）が必要
  :::
- 先端技術産業
  :::lead
  エレクトロニクス産業などが該当し、高度な知識・技術が必要
  :::

### 工業とコスト
::top
人が原材料を加工するのは、付加価値によって利益を生むためです。
::
::middle
そして、利益を多くする単純な方法は、コストを抑えることです。
::
::last
原材料費以外に、次のことにコストがかかります。
::

- 輸送費
  :::lead
  原材料の産地から加工場所までの、あるいは、加工場所から消費地までの運搬費用
  :::
- 人件費
  :::lead
  人の労働で発生する費用のことで、発展途上国の方が先進国に比べて安い
  :::

#### 練習問題
::top
例えば、重い原料があり、その原料は加工後にすごく軽くなるとしましょう。
::
::last
産地と加工品の消費地が下図のような配置であれば、A～Dのどこを加工場所にするべきでしょうか。
::
::gazo
::
::last
輸送費は重量が大きいものほど高くなるので、なるべく軽くして運ぶ方がコストを抑えられます。
::
## 工業の立地
### 立地の種類
::top
「輸送費」「人件費」をどう抑えて利益を出すかで、工業立地、つまり工場をどこに置くかが決定します。
::
::last
おおまかに次の表のようになります。
::
::sup
指向
:::lead
「重視する」と解釈するとよい
::
### 原料指向型
#### 立地の理由
::top
輸送費は、重い荷物ほどかかります。
::
::middle
もし原材料が加工される過程で軽くなるもの(重量減損原料）なら、可能な限り産地近くで加工すべきです。
::
::last
こうした立地が、[[原料]]指向型です。
::
---arrow---
::top
原料指向型の代表的な業種は、金属工業のほかに、石灰石が得られる場所で[[セメント工業]]、水・木材が得られる場所で[[製紙業]]や[[パルプ業]]が立地します。
::
#### セメント工業
::top
セメントの主要原料は、[[石灰石]]です。
::
::middle
石灰石は不純物を含むため、セメントになると軽くなります。
::
::last
産地近くで加工し、輸送した方がコストがかかりません。
::
::sup
石灰石
:::lead
日本で採取できる数少ない資源
::
::gazo
::
::gazo
::
#### 鉄鋼業
::top
鉄鋼の原料の鉄鉱石と燃料の石炭は、鉄鋼よりも重いです。
::
::middle
例えば、昔は鉄鉱石２t、石炭４tから１tの鉄鋼を生産していました。
::
::middle
従って、セメント工業同様に、原料の産地（鉄山や炭田）の近くで加工します。
::
::last
しかし、近年の先進国の鉄鋼業は、国内原料の枯渇や安価な輸入原料への依存から、後述する臨海指向型へ移行しています。
::
::gazo
::
#### パルプ業・製紙業
::top
紙は、木材からパルプという繊維を取りだして作ります。
::
::middle
抽出後は重量が小さくなるため、原料指向型の立地です。
::
::last
日本では、豊富な山林を周辺にもつ四国や苫小牧（北海道）に、工場が立地します。
::
::gazo
::

- 大量の処理用水が必要用水の近くに立地（例：富士）
- 輸入木材を利用木材の輸入港に立地（例：新潟）
- 古紙を原料として利用古紙の供給源である大都市に立地（例：東京）

::gazo
::
### 臨海指向型
#### 立地の理由
::top
輸入原料を使用する場合は、輸入に便利な港の近く（臨海地域）に工場を立地させます。
::
::last
日本は資源のほとんどを海外に依存しているので、鉄鋼業や石油化学工業が原料/臨海指向型です。
::
#### 鉄鋼業
::top
かつて鉄鋼業の多くは、鉄山と炭田が付近に位置する[[原料]]指向型の立地でした。
::
::last
近年の先進国では、国内の資源枯渇などを理由に原料指向型が減り、輸入した鉄鉱石・石炭を臨海で加工する[[臨海]]指向型の立地が多くなりました。
::
::sup
発展途上国の鉄鋼業は原料指向型
::
::gazo
::
#### 石油化学工業
::top
鉄鋼業と同様に、臨海指向型の立地です。
::
::gazo
::
### 臨空港指向型
#### 立地の理由
::top
軽量で小さく、高付加価値である電子部品は、輸送費が高い空輸で素早く組立工場まで運んだ方が利益を出します。
::
::last
このように高い輸送費を度外視できる工業は、臨空港指向型（空港付近や空港に続く高速道路付近の立地）になります。
::
::gazo
::
::top
日本では九州や東北地方が工場の誘致に力を入れており、それぞれシリコンアイランド、シリコンロードという別名がつけられています。
::
::gazo
::
::middle
工場が集まって立地することで、設備の共同利用や輸送コストの削減ができます。
::
::last
このような工場の立地を集積指向型と呼びます。
::
::gazo
::
::top
日本の自動車工業は、各メーカーの創業地と強く結びついています。
::
::gazo
::
::middle
また、情報などは人が多く住む消費地ほど大量に速く入手できます。
::
::last
これらの理由で大都市近郊に立地するのが、市場指向型です。
::
#### ビール・清涼飲料
::top
ビール・清涼飲料の原材料は、水が最大の重量を占めます。
::
::middle
こだわりの水でなければ、水は普遍原料と言えます。
::
::middle
そして、缶・ビンに詰めれば重量が増えるので、消費地近くで製造する方が輸送費を抑えられます。
::
::last
日本では東京・札幌・福岡など==大都市近郊に工場が立地します==。
::
::gazo
::
::top
ただし、飲料全般が市場指向型ではないので、注意が必要です。
::
::last
例えば、ワインは原材料のブドウが腐りやすく、水を使わないため[[原料]]指向型になり、また、日本酒はその土地特有の水と結びつきが強く、用水指向型になります。
::
#### 印刷業・出版業
::top
情報は量・速さが勝負です。
::
::middle
また、これらを印刷して書籍化すると重量がかなり増えます。
::
::last
印刷業・出版業は、情報が得やすく輸送費が抑えられる==大都市近郊に==展開します。
::
#### アパレル業
::top
流行は都市部の人々の方が敏感です。
::
::last
アパレル（服飾）製品の企画やデザインをおこなう事業所は、消費市場の情報を求めて大都市に立地する傾向があります。
::
### 労働力指向型
#### 立地の理由
::top
手作業や比較的やさしい家電の組み立て（ローテク産業）では、大量の労働力が必要になります。
::
::middle
人件費を安く抑えれば、それだけ利益が出せます。
::
::last
安価な人件費を求めた立地が、労働力指向型で、[[繊維]]工業や[[電気機械]]工業に見られます。
::
---arrow---
::top
近年、企業の工場が海外へ移転するのは、労働力指向型だからです。
::
### 電力指向型
#### 立地の理由
::top
アルミニウムはボーキサイトという鉱石から取り出されます。
::
::last
取り出す過程で電気分解という工程があり、大量の電気エネルギーが必要になります。
::
---arrow---
::top
安価に電力を供給できる場所を求めた立地が、電力指向型です。
::
::last
アルミニウム製錬では、特に[[水力]]発電が頼られました。
::
::top
かつて日本は世界でも有数のアルミニウム精錬国でした。
::
::last
しかし、電力費の高騰で徐々に衰退し、水力発電に頼っていた静岡県{{蒲原|かんばら}}工場も2014年３月に閉業し、==現在の日本でアルミニウムの生産はなくなりました==。
::
::sup
リサイクルによる再生あり
::
---arrow---
::top
現在、日本で使われるアルミニウムは[[オーストラリア]]からの輸入に最も頼っています。
