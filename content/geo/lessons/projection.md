## 図法の必要性
### 球体から平面へ
::top
球体である地球を平面である地図として表す時、面積・距離・方位・角度・形などの要素で「ゆがみ」が生じます。
::
::last
ある要素の「ゆがみ」を許し、用途に応じて必要になる特定の要素を正しく表現するため、様々な地図の描き方（図法）が考えられてきました。
::
::gazo
::
::middle
球体であったミカンの皮を剥くと、平面になります。
::
::last
しかし、その形は四方八方が断裂していて、仮にこれが地図であれば、非常に見にくく使いにくいものです。
::
::gazo
::

- 正積図法
  :::lead
  地球上の面積と地図上の面積との比率を正しく表現する図法
  :::
- 正角図法
  :::lead
  地球上の角度の関係を地図上に正しく表現する図法
  :::
- 正方位図法
  :::lead
  地球上のある基準点からの方位を地図上に正しく表現する図法
  :::

### 正積図法
#### 低緯度を正確に－サンソン図法
::gazo
::

- 長所：中・高緯度（特に中緯度）の大陸の形が比較的正確
- 短所：低緯度の大陸の形にゆがみ発生

#### いいとこ取り－グード図法
::gazo
::
::last
両者のいいとこ取りをしたがために、海洋に断裂を入れて大陸の形を整える必要が生じました。
::

- 長所：大陸のゆがみが全体的に少ないため、分布を示すことに最適
- 短所：断裂があるため、人や物の動きを矢印で表現することに不適

#### 地方図に最適－ボンヌ図法
::gazo
::
::gazo
::
::gazo
::
::middle
地点Ａから地点Ｂに向かう時、北に対して取る船の舵の角度（舵角）を、この地図上で２点間を結んだ直線と経線のなす角度にすれば、いずれ目的地である地点Ｂに到達できるからです。
::
::last
==遠回りになるが==確実に目的地に到達できるこの直線を[[等角航路]]と言います。
::
::gazo
::
::last
例えば、下図に示したアルジェリア（赤）とグリーンランド（青）の面積はほぼ等しいのですが、メルカトル図法ではグリーンランドが圧倒的に大きく表現されています。
::
::gazo
::
::last
緯度60度では、赤道上と比べて距離が[[２]]倍、面積が[[４]]倍で表現されます。
::
::gazo
::

- 長所：等角航路を地図上で表現でき、航海用（海図）に利用可
- 短所：高緯度での面積の拡大がひどく、また、最短距離が曲線

### 正方位図法
#### 方位と距離を正しく－正距方位図法
::gazo
::
::middle
また、図の中心と別の地点を結ぶ直線（下図の赤線）は、==２点間の最短距離==を表します。
::
::last
この最短距離は、[[大圏航路]]と言います。
::
::gazo
::
::middle
下図のように北極点中心の図ならば、外周円は南極点です。
::
::last
地球１周は[[40000]]㎞なので、図の中心から外周までの半径（下図の黄線）は[[20000]]㎞になります。
::
::gazo
::
::middle
図の中心からの方位が正しいので、東京から見るとサンティアゴは[[東]]、サンパウロは[[北北東]]にあります。
::
::middle
また、図の中心からの距離が正しいので、サンパウロ・サンフランシスコ・サンティアゴのうち、東京から最も距離が短いのは[[サンフランシスコ]]です。
::
::last
そして、図の中心と別の地点を結ぶ線（下図の緑線）は、その間の最短距離[[大圏航路]]を示します。
::
::gazo
::
::last
下図で等角航路と大圏航路の違いを確認してみましょう。
::
::gazo
::
::gazo
::

- 長所：最短距離、つまり大圏航路を図の中心と別の地点を結ぶ直線で表現でき、航空図に利用可
- 短所：周辺部のゆがみ大
