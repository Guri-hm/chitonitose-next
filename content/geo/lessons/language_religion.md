## 民族・言語
### 人々の区別
#### 人種
::top
肌の色や毛髪など身体的特徴で人を区別したものが人種です。
::
::last
人種は次の３種類に大別されます。
::

- モンゴロイド（黄色人種）
- コーカソイド（白色人種）
- ネグロイド（黒色人種）

#### 民族
::top
言語・宗教などの各文化の大部分や「私たちは～人」という意識を共有する集団を民族と言います。
::
::last
肌の色や毛髪など身体的特徴で人々を区別する人種と異なり、文化的特徴で区別するものです。
::
---arrow---
::top
民族を区別する１つの指標に言語があります。
::
::last
民族を区別する際に、特に言語に注目し、文法の特徴などで比べ分けたものを[[語族]]と言います。
::
::gazo
::
::gazo
::

## 宗教
### 世界宗教と民族宗教
#### 世界宗教
::top
世界宗教とは、人種や民族の枠を超えて信仰されている宗教を指します。
::
::last
例えば、キリスト教・イスラーム・仏教が該当します。
::
#### 民族宗教
::top
民族宗教とは、特定の民族のみに信仰されている宗教を指します。
::
::last
ヒンドゥー教・ユダヤ教・神道・道教が該当します。
::
### キリスト教
::top
キリスト教はユダヤ教から派生し、イエスが創始した宗教です。
::
#### 基礎知識
::top
聖典：旧約聖書（ユダヤ教の聖典）・新約聖書
::
::top
聖地：エルサレム
::
#### ３つの宗派
::top
キリスト教は次の３つの宗派に大きく分かれます。
::

- カトリック：南ヨーロッパや中・南アメリカ
- プロテスタント：北・西ヨーロッパ
- 東方正教：東ヨーロッパやロシア

::last
アメリカに移民したヒスパニック系にはカトリックが多く、多産傾向となります。
::
### イスラーム
::top
ムハンマドが創始した宗教です。
::
#### 基礎知識
::top
聖典：コーラン
::
::top
聖地：メッカ・メディナ・エルサレム
::
::top
禁忌飲食（豚・酒）、人前での女性の肌の露出
::
::top
暦太陰暦（月の満ち欠けが周期）
::
::top
イスラーム教徒（ムスリム）が多い国では、豚の飼育が少なくなります。
::
::top
イスラーム教徒が多い国では女性の地位が低く、社会進出が遅れています。
::
#### ２つの宗派
::top
イスラームは次の２つの宗派に大きく分かれます。
::

- スンニ派：多数派で、西アジアや北アフリカで多い
- シーア派：少数派で、イランやアゼルバイジャンで多い

### 仏教
::top
釈迦（ブッダ）が創始した宗教です。
::
::last
仏教では、悟りを開き、「仏」になることが目指されます。
::
#### ２つの宗派
::top
仏教は悟りを開く方法で次の宗派にわかれます。
::

- 大乗仏教：教えを広め、多くの人を救済していくなかで悟りを開くことが目的
