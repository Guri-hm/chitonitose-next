## 熱帯林破壊
### 要因

- [[焼畑]]農業の拡大
- プランテーション農園や牧場の開発
- [[薪炭]]材（燃料用木材）の過伐採
- [[マングローブ]]林の伐採（エビ養殖場の開発）

::sup
マングローブ
:::lead
マングローブ
:::
ブ熱帯・亜熱帯の海岸・河口の汽水域となる地域に広く分布する植物
::
::gazo
::
### 影響

- 熱帯の多雨で土壌侵食や土壌流出
- 森林の保水効果焼失で土砂崩れや洪水
- 生態系の破壊
- 森林による[[二酸化炭素]]の吸収力減退地球規模の問題である[[地球温暖化]]にも関係

### 対策
::top
林業（果実のなる樹木などを多様に植樹）と農業（樹木の間で作物を栽培）の組合せである、アグロフォレストリーが取り組まれています。
::
::gazo
::
::middle
なかには人為的な要因が大きい砂漠もあり、乾燥地域周辺のステップ（短草草原）地帯で生じています。
::
::middle
この地帯では、緑があった土地が徐々に砂漠になっていく砂漠化が進み、砂漠が拡大しています。
::
::last
特に==サハラ砂漠==南縁の[[サヘル]]では、深刻な砂漠化が生じています。
::
::gazo
![](img/desertification07.jpg)
サヘル（http://www.jccca.org/)」
::
### 要因

- 気候変動による降雨の現象
- 過耕作
- 過放牧
- 薪炭材の過伐採
- 過灌漑による塩害（地表面に塩類の集積）

## 大気汚染・酸性雨
### 大気汚染
::top
近年、PM2.5と呼ばれる直径2.5μm以下の微粒子状の大気汚染物質が注目されています。
::
::gazo
::
### 要因
::top
工場、自動車、[[火力]]発電所などで使用した化石燃料が主な原因です。
::
::middle
燃焼の際に発生した硫黄酸化物（SOx）や窒素酸化物（NOx）は、空を曇らせたり、大気中で亜硫酸や硝酸に変化して酸性雨になります。
::
::last
ほかにも、硫黄酸化物や窒素酸化物は、紫外線と反応して光化学スモッグを発生させます。
::
::gazo
::
::middle
また、酸性雨は森林の枯死、湖の酸性化による魚の死滅、歴史的建造物や石像の腐食をもたらします。
::
::last
大気汚染のうち、森林の枯死では、ドイツ南西側の[[シュヴァルツバルト]]の被害が有名です。
::
::gazo
::
::gazo
::
::middle
近年は中国・インドなど発展途上国でも深刻化しています。
::
::middle
大気汚染で厄介なことは、[[偏西]]風の影響で東側の地域にも被害をもたらすことです。
::
::last
これを越境汚染と言い、大気汚染は国内被害で留まりません。
::
### 対策

- 湖や森林に中和のための[[石灰]]を散布
- 硫黄酸化物や窒素酸化物の排ガス処理装置の設置
- ハイブリッドカー、電気自動車の利用
- 1979年、ジュネーヴ（長距離越境大気汚染）条約の締結

## オゾン層破壊
### オゾン層
::top
オゾン層は、地表から10～50㎞の上空にある気体の層で、太陽からの有害な紫外線を吸収します。
::
::last
南極の上空には、オゾン層が薄くなり、穴（ホール）のように見える場所があります。
::
::last
近年、このオゾンホールの拡大が観測されています（オゾン層の破壊）。
::
::gazo
::
### 要因
::top
スプレーの噴射剤、冷蔵庫、エアコンで使用される[[フロン]]が、オゾン層を破壊します。
::
### 影響
::top
有害な[[紫外線]]の量が増加し、皮膚ガンや白内障など健康被害に繋がります。
::
### 対策

- 1985年、ウィーン条約
  :::lead
  オゾン層保護の国際的取り組みを採択
  :::
- 1987年、モントリオール議定書
  :::lead
  特定フロン（フロンのうちオゾン層破壊に繋がるフロン）などの規制
  :::

## 地球温暖化
### 地球温暖化
::top
長い歴史のなかで地球の平均気温は変動を続けてきました。
::
::last
1980年以降は地球の平均気温が急激に上昇しており（地球温暖化）、21世紀末には最高６℃以上上昇すると予測されています。
::

::
::gazo
::

::

::
### 対策

- 1992年、国連環境開発会議（別称：地球サミット）での[[気候変動枠組]]条約採択
- 1997年、[[京都議定書]]採択
  :::lead
  先進国全体で温室効果ガスの排出を削減
  :::
- 2015年、==パリ協定==採択
  :::lead
  ==途上国を含む全ての参加国と地域に==温室効果ガスの排出削減の目標を設定
  :::
- [[環境税]]の導入…二酸化炭素などの排出に対する課税（オランダ、スウェーデンでは既に導入）

::

::gazo

### その他の問題
#### 産業廃棄物の問題
::top
産業廃棄物の処分が有料化したことで不法投棄問題が発生しました。
::
::middle
処分を減らし問題をなくすため、循環型社会の実現が目指されています。
::
::last
そのためのキーワードとなるのが3Rと総称される「Reduca（削減）・Reuse（再利用）・Recycle（再生利用）」です。
::
#### 環境ホルモン（内分泌攪乱化学物質）の問題
::top
工業製品、農薬、殺虫剤、プラスチック製品などに含まれる化学物質で、特に体内の正常な働きをするホルモンの働きを破壊し、例えば生殖器の異常などを引き起こすものを環境ホルモンと言います。
::
#### ダイオキシン汚染
::top
ダイオキシンは、農薬の製造や塩化プラスチック系の物質の燃焼で発生する化合物です。
::
::last
発癌性などの毒性が高く、環境汚染による健康や生態系への被害が懸念されています。
::
