## 水産業
### 好漁場の条件
::top
漁場が{{好漁場|こうぎょじょう}}（たくさん獲れる場所）になるためには、いくつかの条件が必要です。
::
#### 自然条件
::top
好漁場の自然条件は、豊富な栄養分と日光があることです。
::
::last
「栄養分→植物性プランクトン→動物性プランクトン→魚類」という食物連鎖を考えれば、植物性プランクトンが必要とする栄養分と光合成のための太陽光が重要だとよく分かります。
::
---arrow---
::top
海中の栄養分は重いため、海底に沈んでいます。
::
::middle
栄養分が、海底から上昇する海水の流れ[[湧昇流]]によって、太陽光の届く浅い水域まで運ばれれば、好漁場が成立します。
::
::last
湧昇流は、次の場所で生じます。
::

- [[大陸棚]]
  :::lead
  大陸の周りを縁取り、水深200ｍ程度までの浅い海域
  :::
- [[バンク]]（[[浅堆|せんたい]]）
  :::lead
  大陸棚上で特に水深の浅い部分
  :::
- [[潮境|しおざかい]]（[[潮目|しおめ]]）
  :::lead
  暖流と寒流が出会う海域に生じるもの
  :::

::gazo
::
#### 社会条件
::top
好漁場は、自然条件だけでは成立しません。
::
::last
消費地の存在や食文化などの社会条件も影響しています。
::

## 世界の水産業
### 世界の水産業の動向
::top
世界各国で水産資源の需要が高まるなか、近年、==漁獲量に占める養殖業の割合が増加しました==。
::
::top
かつて日本の漁業・養殖業生産量は世界一でした。
::
::last
現在漁船漁業では[[中国]]が１位です。
::
### 中国
::top
中国の漁獲量は、1990年代以降に==著しい伸びを示しました==。
::
::middle
今では世界最大の漁獲高になっています。
::
::last
この伸びの背景には、次の３つが挙げられます。
::

- 日本などへの輸出が増加
- 中国の経済発展によって食生活の水準が向上し、国内需要が増加
- 湖沼や池などの内水面で営む、コイ科の魚種を中心とした==養殖業==を政府が援助し、多種の淡水魚を生産

---arrow---
::top
余談になりますが、中国の漁獲統計の数値は過大に見積もられている疑いがあります。
::
::last
1990年代と2000年代の接合がいったんの減少と増加という不思議なもので、FAOとの見直しで修正作業が入ったものと思われます。
::
::gazo
::
::middle
変動幅が大きいのは、漁獲量の大部分を占める[[アンチョビー]]（カタクチイワシ科）の豊漁不漁によって起こされています。
::
::middle
１種の魚類に依存するため、漁獲量に大きく影響します。
::
::last
このような依存は、1960年代から始まりました。
::
---arrow---
::top
アンチョビーの漁獲量は、えさとなるプランクトンの量を左右する湧昇流にかかっています。
::
::last
1970～1980年代は、湧昇流の発生を弱める[[エルニーニョ]]現象に見舞われ、アンチョビーの不漁となりました。
::
::gazo
::
::gazo
::
::middle
これは日本人が魚類を食べなくなったわけではありません。
::
::last
むしろ、日本の水産物の消費量は、1960年代より増加しています。
::
---arrow---
::top
日本の漁獲高の減少は、海面漁業（遠洋・沖合・沿岸）の衰退や輸入の増加に理由があります。
::
#### 漁業の衰退
::top
日本の水産業は、==養殖を除いて全て漁獲量が減少しています==。
::
::top
[[遠洋]]漁業は、1973年を最後に激減しています。
::
::last
これは、次の２つを背景に操業可能な漁場が縮小したからです。
::

- 第四次中東戦争に端を発した[[石油危機]]による燃料費の高騰
- 世界各国による[[排他的経済水域]]（[[200海里経済水域]]）の設定

::gazo
::
::last
しかし、次の２つを背景にこの漁業も減少しました。
::

- 乱獲や海洋環境の変化によるマイワシなどの不漁
- 国民の嗜好が高級魚に変化

::gazo
::
#### 輸入量の増加
::top
漁業の衰退の一方で、日本の水産物の消費量は増加しています。
::
::last
現在は==水産物の半数以上を輸入に頼り、アメリカ合衆国と並ぶ世界的な輸入国になっています。==
::
