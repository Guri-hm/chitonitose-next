## 温帯
### 気候区
::top
気候帯が寒帯（Ｅ）・乾燥帯（Ｂ）のどちらでもない場合、==最寒月平均気温==が[[-3]]℃以上[[18]]℃未満なら温帯（Ｃ）になります。
::
::middle
温帯（Ｃ）は、ここから４つの気候区に分かれます。
::
::last
降水型がｓならCs、ｗならCw、ｆで==最暖月平均気温==が[[22]]℃未満ならCfb、ｆで==最暖月平均気温==が[[22]]℃以上ならCfaです。
::
---arrow---
::top
４つの気候区と表記をまとめると、次の通りです。
::

- Cs
  :::lead
  [[地中海性]]気候
  :::
- Cfb
  :::lead
  [[西岸海洋性]]気候
  :::
- Cfa
  :::lead
  [[温暖湿潤]]気候
  :::
- Cw
  :::lead
  [[温暖冬季少雨]]気候
  :::

::gazo
::
::last
夏は北上した[[中緯度高圧]]（[[亜熱帯高圧]]）帯の影響で乾燥し、冬は海洋から吹き込む[[偏西風]]と高緯度低圧（亜寒帯低圧）帯の影響で少し降雨が見られ、比較的温暖です。
::
::gazo
::
### 植生
::top
[[オリーブ]]や==コルクがし==など、乾燥に強い硬葉樹が多く分布します。
::
::gazo
::
### 土壌
::top
石灰岩の風化による==赤い土壌==[[テラロッサ]]が、地中海周辺に分布しています。
::
::last
この土壌はオリーブの栽培に適しています。
::
### 生活
#### 農業
::top
夏は乾燥に強い==ブドウ==や==オリーブ==などを栽培し、冬は降水を利用して==小麦==を栽培します。
::
::last
このような農業を[[地中海式]]農業と言います。
::
#### 住居
::top
住居は、森林に比べて豊富な石を利用します。
::
::middle
地中海周辺には石造りの白い家が多く見られます。
::
::last
この白は石灰で、遮熱効果を期待したものです。
::
::gazo
::

## Cw（温暖冬季少雨気候）
::gazo
::
::middle
サバナ気候付近の地域では、夏は[[赤道低圧]]帯の影響で高温多湿、冬は[[中緯度高圧]]（[[亜熱帯高圧]]）帯の影響で比較的温暖で乾燥します。
::
::last
大陸東岸の地域では、季節風の影響で夏に高温多湿、冬に温暖で乾燥します。
::
::gazo
::
### 植生
::top
シイ・カシ・クスなど照葉樹が多く分布します。
::
::last
ただし、落葉広葉樹・針葉樹も見られます。
::
::gazo
::
::gazo
::
::last
特に、中国==華南==では温暖な気候を利用して、米の二期作がおこなわれます。
::
### 分布・都市
::gazo
::
::gazo
::
### 特色・成因
::top
主に==緯度40～60度の大陸西岸==で見られます。
::
::middle
高緯度のわりには、同緯度の他地域に比べて、夏は冷涼、冬は比較的温暖です。
::
::middle
そのため==気温の年較差は比較的小さく==なります。
::
::last
また、[[高緯度低圧]]（[[亜寒帯低圧]]）帯の影響で==年中降雨があります==。
::
---arrow---
::top
西ヨーロッパにおいて、この気候になる理由は、[[偏西風]]と[[暖]]流の[[北大西洋]]海流の影響です。
::
### 植生
::top
ブナなどの落葉広葉樹が多く、針葉樹も見られます。
::
::gazo
::
### 生活
::top
次のような様々な農業形態が見られます。
::

- [[混合]]農業
  :::lead
  牧畜と穀物の栽培を組み合わせた農業
  :::
- [[酪農]]
  :::lead
  乳牛を飼育して牛乳および乳製品の生産をおこなう農業
  :::
- [[園芸]]
  :::lead
  花卉や野菜を栽培する農業
  :::

---arrow---
::top
住みやすく人口が多く分布し、商工業が発達しています。

### 特色・成因
::top
大陸東岸に発達し、夏は高温、冬は寒冷になります。
::
::middle
これにより、気温の年較差は大きくなります。
::
::middle
==四季==の変化がはっきりし、[[季節風]]の影響で年間を通して降水が見られます。
::
::last
また、台風・ハリケーンなど==熱帯低気圧==が降雨をもたらします。
::
::gazo
::
::gazo

::top

::
::last

::
---arrow---
::top
