## 貿易の概説
### 貿易とは
::top
国は、自国が得意な生産物を他国に輸出し、得意でない生産物を輸入に頼ります。
::
::last
このように国同士が生産を分担することを国際分業と言い、そして、生産物を取引することを貿易と言います。
::
---arrow---
::top
今日、第３次産業の情報・通信などを取引するサービス貿易も増えてきました。
::
### 先進国と途上国の貿易
::top
貿易は、国同士が取引する産品の組み合わせで、次の２つに分けられます。
::

- 水平貿易
  :::lead
  工業製品・サービスなどを輸出入し合う貿易
  :::
- 垂直貿易
  :::lead
  工業製品・サービスなどと一次産品を輸出入し合う貿易別名は南北貿易
  :::

::sup
一次産品
:::lead
工業製品として加工が加えられていない商品
::
#### 水平貿易
::top
水平貿易は、先進国間で多く、で工業製品・サービスなどを輸出入し合う貿易です。
::
::last
水平貿易の額は大きく、世界貿易の半分を占めています。
::
::gazo
::
::middle
垂直貿易では、付加価値の高い工業製品を輸出する先進国が多くの利益を取得でき、一次産品を輸出する途上国では国の発展が見込めませんでした。
::
::last
このことは、先進国の多い北半球と途上国の多い南半球の経済格差「南北問題」を引き起こしました。
::
---arrow---
::top
近年、NIEsやASEANに属す途上国では、輸出品のなかで工業製品が占める割合が増えています。
::
::gazo
::
#### 中国
::top
世界最大の輸出額は、2018年時点で[[中国]]です。
::
::sup
以前までアメリカが世界最大の輸出額
::
::top
アメリカは、お金をもつ人が多くて購買力が強く、かなりの輸入超過（輸入額>輸出額）で==世界最大の貿易赤字国です==。
::
#### EU
::top
貿易額の大きい中国やアメリカも、EU（ヨーロッパ連合）には敵いません。
::
::middle
構成する国々それぞれの貿易額が大きいので、自然とEUの貿易額も大きくなります。
::
::last
例えばNAFTAなど、EU以外の国々も経済協力を図って貿易額を上げています。
::
### 日本
#### 輸出超過から輸入超過へ
::top
次の日本のグラフを用いて解説します。
::
::top
日本やドイツなどの==工業製品を輸出する国は、輸出超過（輸出額>輸入額）で貿易黒字になりやすいです==。
::
::middle
1980年代から日本の輸出超過は顕著になりました。
::
::last
この過程で、日本とアメリカの間には貿易摩擦が生じました。
::
---arrow---
::top
ところが日本の貿易は、==2011年の原発停止以降に、輸入超過（輸入額>輸出額）に転じました==。
::
::last
これは原発停止による化石燃料の輸入増などが原因です。
::
#### 輸入超過の内訳
::top
下の表を見ると、日本が輸入超過国から原油・石炭・天然ガスを輸入していることが分かります。
::
::middle
==日本が資源を多く輸入する国は日本の輸入超過国になります==。
::
::last
他にもイタリアからは高級服飾・高級ブランド品を輸入したりと、意外な輸入超過の背景があります。
::
#### 最大の輸出国・輸入国
::top
戦後、日本の最大の輸出国・輸入国は[[アメリカ]]でした。
::
::last
近年、最大の輸出国・輸入国は[[中国]]になりました。
::
## 貿易依存度
::top
貿易依存度とは、国内総生産（GDP）に対する輸出額・輸入額の割合です。
::
::last
人口が少なく、国内市場が小さい国は、==貿易依存度が高くなります==。
::
## 世界貿易の変遷
### GATTとWTO
#### 貿易の自由化の推進
::top
第二次世界大戦の要因の１つは、過度な保護貿易と考えられています。
::
::last
保護貿易を解消し、自由貿易を推進して政治問題を解決する体制づくりが目指されました。
::
::gazo
::
::top
1948年、[[GATT]]（関税および貿易に関する一般協定）発効
:::lead
、[[GATT]]（関税および貿易に関する一般協定）発効関税の引き下げなど貿易の障壁をなくすことを目指す協定
::
---arrow---
::top
1986～94年、GATTウルグアイラウンド
:::lead
94年、GATTウルグアイラウンドGATTが「モノ」のみを対象にしていたので、「サービス貿易」「知的財産権」に関する協定を作成
::
#### GATTの改組
::top
1995年、[[WTO]]発足
:::lead
、[[WTO]]発足GATTウルグアイラウンドの結果、GATTを改組して発足した国際機関
::
::gazo
::
::middle
WTOの意思決定は加盟国の全会一致を原則としたため、加盟国が増えるにつれて機能不全に陥りました。
::
::last
このため、特定の国・地域間で例外的に協定を結び、自由貿易を推進するようになりました。
::
::gazo
::
::top
1994年、[[NAFTA]]（[[北米貿易協定]]）発効
:::lead
、[[NAFTA]]（[[北米貿易協定]]）発効アメリカ合衆国・カナダ・メキシコが締結した自由貿易協定
::
::top
1995年、[[MERCOSUR]]（[[南米南部共同市場]]）発効
:::lead
、[[MERCOSUR]]（[[南米南部共同市場]]）発効アルゼンチン・ウルグアイ・パラグアイ・ブラジルが締結した、域内の関税を緩和・撤廃する同盟
::
::sup
USMCA
:::lead
United StatesｰMexicoｰCanada Agreement
::
#### EPA
::top
[[EPA]]
:::lead
onclick="chg(this)" class="all">EPA特定の国・地域間で、関税などの貿易の障壁を低減・撤廃することに加えて、投資・人的交流の円滑化や知的財産の保護を目指す協定
::
### 公正な貿易を目指して
::top
発展途上国では、受け取る賃金が不当に安くされ、労働者の権利が守られないことが多くあります。
::
::last
先進国が発展途上国の農産加工品などを適正な価格で直接購入する貿易[[フェアトレード  ]]があり、現地の人々の生活水準の向上が目指されています。
::
### 日本・アメリカの例
::top
戦後、日本はアメリカに対する自動車や電気機器の輸出を伸ばしていきました（貿易黒字）。
::
::middle
安価な日本産に押され、アメリカの日本への輸出は少なくなりました（貿易赤字）。
::
::last
1980年代、日米間の貿易収支の不均衡は、国家間の対立[[貿易摩擦]]を起こしました。
::
::gazo
::
---arrow---
::top
アメリカは、日本に対して輸入の自由化を迫り、貿易収支の不均衡を解消しようとします。
::
::middle
一方、日本は「日本の輸出＝相手国の生産圧迫」という貿易摩擦の根本的問題を回避するため、アメリカに工場を建設して現地生産するようにしました。
::
::last
貿易摩擦は、日本の対策のように対外直接投資で解消しようとされます。
::
::gazo
::
::last
この解消方法では、自国の[[産業の空洞化]]が生じ、自国の雇用が減るなどの問題を起こします。
::
::gazo
