## 鉱山資源
### ボーキサイト
::top
ボーキサイトは、アルミニウムの原料で、乾季のある熱帯の赤土ラトソルに含まれるアルミニウム分が凝縮したものです。
::
::last
つまり、==オーストラリ北部のサバナ気候には世界的な産地が分布します==。
::
---arrow---
::top
オーストラリアに地理的に近い==インドネシアやマレーシアでは、若干の産出が あるのみです。==
::
::last
このことは、ボーキサイトと鉄鉱石を区別する目安になります。
::
### 鉄鉱石
::top
鉄鉱石は、主に安定陸塊に分布します。
::
::last
オーストラリアはピルバラ地区を有し、世界の産出量の３分の１を占める有力な産出国です。
::
---arrow---
::top
==オーストラリア以外のオセアニアや東南アジアには目立った産出国は存在しません。==
::
::last
このことは、ボーキサイトと鉄鉱石を区別する目安になります。
::
