## 地図の大別
### ２種類
::top
地図は大きく２種類に分けられます。
::

- [[一般図]]
  :::lead
  特定のテーマを持たず、地形や地名、河川や交通路、集落や行政界などの情報を描いた平均的な地図
  :::
- [[主題図]]
  :::lead
  利用目的に応じて、ある特定のテーマの下に重要要素を強調した地図
  :::

## 一般図
### 実測図
::top
一般図のうち、現地での測量や空中写真測量などをもとに作製した地図を実測図と言います。
::
::gazo
::
::last
２万５千分の１地形図４面分を１面にまとめたものが、下図の５万分の１地形図です。
::
::gazo
::
## 主題図
### 土地利用図
::top
土地利用（田か畑かなど）ごとに色分けした地図です。
::
::gazo
::
::gazo
::
::gazo
::
::last
地理で扱う統計数値は、大きく次の２つに分かれ、示すのに適した統計地図が異なります。
::

- 絶対的数値
  :::lead
  人口や降水量などの数値
  :::
- 相対的数値
  :::lead
  単位面積あたりの数値（人口密度）や人口１人あたりの数値（所得など）の数値
  :::

#### ドットマップ
::top
点で一定の数量や分布を表現した地図です。
::
::sup
点は示すものが存在する位置の真上
::
::gazo
::
::gazo

::gazo
::
::gazo
::
::last
==相対的数値（人口密度や人口１人あたりの所得など）を示すのに適しますが、絶対的数値（人口や生産量）のように国や地域の面積の大小が統計数値の大小に影響するものでは、実際の分布とは異なる印象を与えるため、この統計地図は不適切です==。
