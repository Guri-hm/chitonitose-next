## H（高山気候）
::gazo
::
::middle
ケッペンが気温・降水のみを指標に気候区を区分したため、標高を指標に加えた気候は、本来のケッペンの気候区分に存在しません。
::
::middle
後に追加する形で、標高2000ｍ以上の地域がある程度の広がりをもつ地域を==高山気候==と分類し、記号「==Ｈ==」で表記するようにしました。
::
::last
なお、高山気候は、本来のケッペンの気候区分では別の気候区に該当します。
::
### 特色・成因
#### 気温
::top
気温の日較差は大きく、年較差は==小さく==なります。
::
::middle
気温は、気温の低減率に従って海抜高度100mにつき0.65℃増減します。
::
::middle
そのため==高山気候では同緯度の周辺と比べて気温が低くなります==。
::
::middle
本来低緯度の地域は、年中気温が高くなりますが、同緯度でも標高の高い地域では、気温の逓減率に従い、==年中春ごろの気温になります（{{常春|とこはる}}）==。
::
---arrow---
::top
シンガポールとキトの雨温図を比べてみましょう。
::
::last
キトはエクアドルの首都で、赤道直下にある平均標高2,850mの都市です。
::
::gazo
::
#### 降水量
::top
降水量は1500～2000㎜で極大となり、以降高度が増すと減少します。
::
### 植生・土壌
::top
低地から高地にかけ、熱帯植物から寒帯植物へ垂直に変化します。
::
::gazo
::
::last
気候の面では涼しいので低地よりも暮らしやすく、昔から都市が発達しました。
::
::last
古代インカ帝国のマチュピチュは大変有名です。
::
#### 農業
::top
標高4000ｍ以上は作物が育たないので、アンデス山脈では==リャマ==や==アルパカ==を、チベットでは==ヤク==を飼育します。
::
::middle
また、アンデス山脈の西側斜面では、低地の気温が高い地域では熱帯性の作物が、高地の冷涼な地域ではじゃがいもなど寒冷な気候でも生育できる作物が栽培されます。
::
::gazo
::
::gazo
::
#### 食
::top
アンデスでは、標高4000ｍよりもやや低い土地で==じゃがいも==を育てます。
::
::gazo
::
::top
