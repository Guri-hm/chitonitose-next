## 乾燥帯
### 気候区
::top
乾燥帯（Ｂ）には、次の２つがあります。
::

- BS
  :::lead
  [[ステップ気候]]
  :::
- BW
  :::lead
  [[砂漠気候]]
  :::

---arrow---
::top
気候帯が寒帯（Ｅ）でない場合、次に乾燥帯（Ｂ）かどうかを疑いましょう。
::
::last
乾燥帯（Ｂ）の判定には、まず降水型を判定します。
::
#### BS・BWの目安
::top
BS・BWの判定は数値の計算が必要ですが、入試問題ではおおよその目安さえ把握しておけば十分です。
::
::last
年間降水量が[[500]]mm～[[250]]mm程度なら[[ステップ]]気候（BS）、[[250]]mm未満なら[[砂漠]]気候（BW）です。
::
---arrow---
::top
判定時の細かな計算方法を知りたい方は、以下の内容を参考にしてください。
::
#### 降水型の判定
::top
夏に最少雨月があるなら、①「夏の最少雨月降水量✕[[３]]」と②「冬の最多雨月降水量」を比較します。
::
::last
夏の方が冬より少なければ（①＜②）、降水型はｓ（夏に乾燥）、そうでなければｆ（年中湿潤）となります。
::
---arrow---
::top
冬に最少雨月があるなら、①'「冬の最少雨月降水量✕[[10]]」と②'「夏の最多雨月降水量」を比較します。
::
::last
冬の方が夏より少なければ（①'＜②'）、降水型はｗ（冬に乾燥）、そうでなければｆ（年中湿潤）となります。
::
#### 降水型に応じた計算
::top
年間降水量をＲ、年間平均気温をｔ、乾燥限界値をｒとおきます。
::
---arrow---
::top
降水型ｓならｒ＝20ｔ
::
::top
降水型ｆならｒ＝20（ｔ＋[[７]]）
::
::top
降水型ｗならｒ＝20（ｔ＋[[14]]）
::
---arrow---
::top
求められたｒをＲと比較します。
::
::last
ｒ≦ＲならＢではない（Ａ・Ｃ・Ｄのいずれか）
::
::last
1/2ｒ≦Ｒ＜ｒならBS（[[ステップ]]気候）
::
::last
Ｒ＜1/2ｒならBW（[[砂漠]]気候）
::
## BS（ステップ気候）
::gazo
::
---arrow---
::top
３ヶ月程度の雨季があるものの、全体として降水量が少なく、比熱の大きい水の影響が小さい地域です。
::
::middle
つまり、大陸の中でも特に熱しやすく冷めやすい地域となります。
::
::last
従って、==気温の日較差・年較差は大きくなります==。
::

::top
乾燥に耐えられる短草草原が広がっています。
::
::last
このような草原を==ステップ==と言います。
::
::gazo
::
::last
==ウクライナ==～西シベリア南部の[[チェルノーゼム]]、北アメリカの[[プレーリー土]]が該当します。
::
::gazo
::
### 生活
::top
中国やモンゴルでは、ヤギや羊を飼育する==遊牧==が営まれます。
::
::last
アメリカ大陸では、大規模な農業や牧畜業が営まれます。

### 生活
#### 水資源
::top
農業は、==オアシス==・==外来河川==・地下水路（イランの[[カナート]]、北アフリカの[[フォガラ]]）を利用し、==小麦==・==ぶどう==・==なつめやし==を栽培します。
::
---arrow---
::top
なお、カナート・フォガラなどの地下水路は、地域によって呼称が変わります。
::
::middle
地下の帯水層に届く最初の井戸（母井戸）を掘り、次に母井戸から横に伸びる地下水路を造り、水源から離れた場所まで水を流します。
::
::middle
水路は地中深くにあり、太陽光による水の蒸発を防ぎます。
::
::last
また、地下水路上には工事・掃除用の縦穴が無数にあります。
::
::gazo
::
::gazo
::
::last
外気の熱や砂塵を防ぐため、壁の開口部が==小さく==なっています。
::
::gazo
