## 地形
::top
メキシコ以南の地域（中央アメリカおよび南アメリカ大陸）は、スペインやポルトガル語などラテン語派の国々に支配され、言語や文化、宗教などに影響を受けました。
::
::last
このような背景から、中央アメリカおよび南アメリカ大陸はラテンアメリカと呼ばれています。
::
::gazo
::
::last
境界に並行して海底に海溝が分布

### 河川
#### アマゾン川
::top
世界で最も流域面積が広い河川です。
::
::last
流域に熱帯雨林（セルバ）が広まりますが、近年の開発で減少しています。
::
::sup
アマゾン川の河口が赤道付近と押さえておくと、白地図での北半球・南半球の判断に役立つ
::
::gazo
::
::last

::
::gazo
::
::gazo

::gazo

::top
