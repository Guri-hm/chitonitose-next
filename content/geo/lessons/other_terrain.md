### 氷河
#### 氷河とは？
::top
氷河は、溶けずに残った積雪（万年雪）が自重で圧縮されてできた氷の塊で、岩盤を削りながらゆっくり動きます。
::
::sup
1年に数メートル程度の速度で低位へ移動
::
::gazo
::

- 大陸氷河
  :::lead
  大陸の全体を広く覆って発達した氷河で、今日は高緯度の南極大陸、デンマーク領グリーンランドのみに分布
  :::
- 山岳氷河
  :::lead
  緯度関係なく、標高の高い山に発達した氷河
  :::

::gazo
::
#### 今日の大陸氷河
::top
今日、大陸氷河は南極大陸とグリーンランドの２カ所のみです。
::
::middle
約２万年前（最終氷期）には、全陸地の約３分の１、特に高緯度地域が大陸氷河に覆われていました。
::
::last
なお、==カスピ海周辺や冬季に降水量が少ないシベリアは、氷河に覆われていません==。
::
::gazo
::
### 氷河地形
::top
氷河が移動・融解する時、氷の下や側面の岩盤を削り、また、削ったものを運搬します。
::
::middle
このような働きで形成された地形を、氷河地形と総称します。
::
::last
氷河地形には、主に次の５つがあります。
::

- [[ホルン]]（[[ホーン]]）
  :::lead
  周囲を氷河によって削られた岩峰
  :::
- [[カール]]
  :::lead
  氷河の侵食によって形成された半円状のくぼ地
  :::
- [[モレーン]]
  :::lead
  氷河によって削られた砂礫が氷河の運搬作用によって周縁部に堆積したもの
  :::
- [[氷河湖]]
  :::lead
  氷河が削ったくぼ地やモレーンによって形成された湖
  :::
- [[U]]字谷
  :::lead
  氷河がつくった谷で、谷底は幅が広く平坦
  :::

### カルスト地形とは
::top
石灰岩を含む地層が、二酸化炭素を含む雨水・地下水などに溶食されると、[[カルスト地形]]と総称される地形ができます。
::
::last
この「カルスト」という名は、この地形が発達したスロベニアのカルスト地方に由来します。
::
::gazo
::

- [[鍾乳洞]]
  :::lead
  地中の石灰岩が溶食されてできた地下の空洞
  :::
- [[ドリーネ]]
  :::lead
  溶食や鍾乳洞の落盤によってできたくぼ地で、特に最小規模のもの
  :::
- ウバーレ
  :::lead
  ドリーネが拡大・連続してできた長さ１㎞程度のくぼ地
  :::
- [[ポリエ]]
  :::lead
  ドリーネやウバーレが拡大・連続してできた大規模なくぼ地
  :::

::gazo
