## 侵食平野
### 侵食平野とは
::top
地層は、水や風などの外的営力によって削られていきます。
::
::middle
このような働きを侵食と呼びます。
::
::last
そして、侵食によって平坦になった地形を侵食平野と呼びます。
::
### 準平原
::middle
侵食過程の最終段階であり、起伏がなくなった地形を[[準平原]]と呼ます。
::
::middle
また、準平原のなかで侵食から取り残された部分を[[残丘]]（[[モナドノック]]）と呼びます。
::
::last
準平原は、長い年月侵食された==安定陸塊によく分布し==、そこでは大昔の地層（先カンブリア時代の地層）が露出しています。
::
::gazo
::
#### 楯状地
::top
準平原のなかには、中世の騎士の楯を伏した形に見えることから、形に注目して[[楯状地]]と呼ばれるものもあります。
::
::sup
受験においては、準平原≒楯状地と考えてよい
::
::gazo
::
::middle
後に隆起すると、水や風の侵食が進み、硬い地層を地表に残して、平坦な地形が形成されます。
::
::last
地表の様子は準平原と同じに見えますが、平坦な地形の下に地層が重なっています。
::
---arrow---
::top
このようにほぼ水平な地層が広範囲に分布する場所を[[構造平野]]と呼び、東ヨーロッパ平原などで見られます。
::
::last
この平野の中で、侵食から取り残された部分は、小規模なものを[[ビュート]]、大規模なものを[[メサ]]と呼びます。
::
::gazo
::
#### 卓状地
::top
先カンブリア時代の地層を土台とした構造平野は、形に注目して[[卓状地]]と呼ばれます。
::
::sup
受験においては、構造平野≒卓状地と考えてよい
::
::gazo
::
::last
このようにして形成される==非対称の断面をもつ==丘陵を[[ケスタ]]と呼びます。
::
::gazo
::
::last
崖部分の水はけや日照に対して適度な傾斜を利用して、[[ブドウ]]を栽培しています。
::
::gazo
::
### 準平原と構造平野の図解
::gazo
::
::middle
このような働きを{{堆積|たいせき}}と呼びます。
::
::last
堆積が進行中の場所を特に[[{{沖積|ちゅうせき}}平野]]と呼び、堆積後に隆起し台地状になった場所を[[洪積台地]]と呼びます。
::
::gazo
::
::last
川は、山地から海へと流れる間に、大量の土砂を運び、また、その土砂を流域に堆積させていきます。
::
---arrow---
::top
山地から海にかけて川がつくる地形は、下図のようになります。
::
::gazo
::

- [[{{谷底|こくてい}}平野]]
  :::lead
  河川の侵食で広がった谷底に土砂が堆積した平野
  :::
- [[扇状地]]
  :::lead
  谷口で河川の流速が衰え、=={{砂礫|されき}}==が扇状に堆積した地形
  :::
- [[氾濫原|はんらんげん]]
  :::lead
  洪水時に河川からあふれた水が土砂を堆積させてできた低平な土地
  :::
- [[三角州]]
  :::lead
  河川最下流の河口付近で土砂や粘土が堆積した低湿地
  :::

---arrow---
::top
河川が上流で削り取った土砂は、粒が小さいものほど河口まで運ばれていきます。
::
::last
つまり、河川の上流から河口にかけて、礫（小石）・砂・泥が順に分布します。
::
::gazo
::
::gazo
::
#### 河川の中流
::top
中流部では上流から運ばれた土砂が堆積し、[[谷底平野]]をつくります。
::
::gazo
::
::top
土地隆起または海面低下によって河川の流速が増え、川底を削る力が強まって、谷底平野に階段状の[[河岸段丘]]をつくります。
::
::sup
河岸段丘は洪積台地だが、便宜的にここで解説
::
::gazo
::
::gazo
::
::top
谷の出口では流水の運搬力が弱まり、砂礫が扇状に堆積して[[扇状地]]がつくられます。
::
::last
扇状地は、谷に近い方から順に、次の３つに細分化されます。
::

- [[扇頂]]
  :::lead
  砂礫（小石）が多く堆積し、河川の水が地下にもぐりこむ（伏流）
  :::
- [[扇央]]
  :::lead
  河川が伏流して=={{水無川|みずなしがわ}}==になるため、水が得にくい
  :::
- [[扇端]]
  :::lead
  伏流していた河川が地表に出るため（{{湧水|ゆうすい}}）、==集落や水田が立地==
  :::

::gazo
::
::top
扇状地の扇央・扇端や氾濫原にある、土砂の供給量が多い河川では、河道を堤防によって固定すると、堤防内に土砂が堆積して川底が高くなります。
::
::middle
川底が高くなると洪水の危険性が高まるため、堤防をかさ上げします。
::
::last
すると川底が次第に両側の平地を上回り、[[天井川]]が形成されます。
::
::gazo
::
::gazo
::
::gazo
::
::top
扇頂・扇央・扇端の土地利用を地形図で確認しましょう。
::
::middle
扇頂に水田、扇央に畑・果樹園、扇端に水田や集落が立地しています。
::
::last
なお、水無川は地形図において破線で示されます。
::
::gazo
::
::top
平野を流れる川は、S字状に蛇行し、洪水時には土砂を堆積させていきます。
::
::last
このように川の氾濫を受ける低平地を[[氾濫原|はんらんげん]]と総称します。
::
::top
氾濫原には、次のような地形が含まれます。
::

- [[自然堤防]]
  :::lead
  氾濫原を流れる河川の両側に土砂が堆積してできた微高地
  :::
- [[後背湿地]]
  :::lead
  氾濫時に自然堤防を越えてあふれた水が、長期間停滞している湿地
  :::
- [[三日月湖（河跡湖）]]
  :::lead
  蛇行の激しい河川で、流路の変化により一部が取り残されて湖沼となったもの
  :::
- 旧河道
  :::lead
  河川の以前の流路跡
  :::

::gazo
::
::last
後背湿地は水持ちがよく、水田などに利用されます。
::
::gazo
::
::top
洪水や人工的な工事で河川の流路が変わり、昔の河川の流路跡がカーブ描くように残っている地形を旧河道と呼びます。
::
::middle
水田がカーブを描くようにつくられていれば、そこが旧河道であったと推定されます。
::
::last
さらに、旧河道に水が残り、湖となっているものを三日月湖と呼びます。
::
::gazo
::
::middle
三角州は水はけが悪く（水持ちがよく）、==水田によく利用==されます。
::
::last
また、三角州は土砂供給量や沿岸流の侵食によって形状が変わります。
::

- 円弧状三角州
  :::lead
  土砂の量が多く、海の波がおだやかで水量の少ない場所に形成
  :::
- {{鳥趾状|ちょうしじょう}}三角州
  :::lead
  海底の勾配が緩やかで、海岸の波や流れに対して河川の堆積が相対的に大きい場合に形成
  :::
- カスプ状三角州
  :::lead
  海の侵食力が強く、波や潮流で運ばれた土砂が堆積して形成
  :::

::gazo
::
::gazo
::
### 洪積台地
::top
堆積後に平野の一部が隆起し、台地になることもあります。
::
#### 河岸段丘
::top
洪積台地のうち、平野の一部隆起や海面低下による河川の下方侵食増大（流速増大）で、階段状になった場所を[[河岸段丘]]と呼びます。
::
::last
河岸段丘の多くは、谷底平野をもとに形成されています。
::
::gazo
::
::top
段丘の台地面では地下水面が低いため、==畑や果樹園に利用==されます。
::
::middle
ただし、地下水面より上に局部的に存在する地下水があり、水を得られることがあります。
::
::last
この地下水を宙水と呼びます。
::
