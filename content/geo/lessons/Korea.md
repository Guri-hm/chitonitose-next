### 南西部の海岸
::top
朝鮮半島南西部は、氷河期に海面が低くなり、河川が流れ込むことで深い谷が形成されました。
::
::middle
氷河期が終わって海面が上昇し、谷に海水が入り込みました。
::
::last
そのため、朝鮮半島南西部の海岸は、[[リアス]]海岸になっています。
::
::gazo
::
### 地域ごとの気候区分
#### 朝鮮半島北部
::top
緯度が高いため最寒月平均気温が低く、冬に北西季節風の影響で少雨になります。
::
::last
[[亜寒帯冬季少雨]]気候（[[Dw]]）が分布します。
::
#### 朝鮮半島南部
::top
北部と同様に冬に北西季節風の影響で少雨になりますが、緯度が低いため最寒月平均気温が高いです。
::
::last
[[温暖冬季少雨]]気候（[[Cw]]）が分布します。
::
---arrow---
::top
対馬海峡付近は、暖流の対馬海流の影響で暖かく、１年を通して降水があります。
::
::last
[[温暖湿潤]]気候（[[Cfa]]）が分布します。

## 南北分断
### 歴史
::top
第二次世界大戦後、朝鮮半島の南北に次の国が成立しました。
::
::top
北部：社会主義の朝鮮民主主義人民共和国（北朝鮮）
::
::top
南部：資本主義の大韓民国（韓国）
::
::middle
1950～53年、南北間の朝鮮戦争が起きましたが、休戦協定が結ばれ戦闘は停止しました。

::last
戦争は休戦状態で現在も続いています。

## 韓国
### 基礎データ
### 人口
#### 人口密度
::top
韓国の人口密度は、世界有数の高さです。
::
#### 都市人口
::top
2018年時点での都市人口率は81.5%です（日本は91.6%）。
::
::middle
ソウルの人口が約1,000万人で、国民５人に１人が首都に住んでいます（首都圏では２人に１人）。
::
::last
このようにソウルは人口が一極集中する都市であり、プライメートシティの一つに数えられます。
::
#### 教育・出生率
::top
韓国の高等教育就学率は非常に高いです。
::
::middle
韓国では子どもの教育にかなりの費用がかかります。
::
::last
このことが背景となり、近年出生率は深刻な低下を示しています。
::
::gazo
::
::middle
1970年代、韓国は日本の援助を得て、鉄鋼・造船・自動車などの重化学工業を発展させました。
::
::last
