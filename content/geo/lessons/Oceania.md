### 地域ごとの気候区分
#### 南回帰線付近
::top
一年を通して[[中緯度高圧]]（[[亜熱帯高圧]]）帯の影響を受け、乾燥気候（BW）・ステップ気候（BS）が分布
::
::sup
南回帰線はオーストラリア大陸のほぼ中央を横断
::
#### オーストラリア大陸北部
::top
冬季に北上した[[中緯度高圧]]（[[亜熱帯高圧]]）帯の影響を受け、サバナ気候（AW）が分布
::
#### オーストラリア大陸南西部
::top
夏季に南下した[[中緯度高圧]]（[[亜熱帯高圧]]）帯の影響を受け、地中海性気候（Cs）が分布
::
#### オーストラリア大陸東部とニュージーランド
::top
東部沿岸を流れる[[暖]]流の東オーストラリア海流の影響を受け、温暖湿潤気候（Cfa）・西岸海洋性気候（Cfb）が分布
::
::last
特にニュージーランドは、暖流に加えて偏西風の影響も大きく、年間を通じて湿潤になり、西岸海洋性気候（Cfb）が分布
::
::gazo
::
