## 集落
### 集落とは
::top
人が集まり居住するところを集落と言います。
::
::last
集落のうち、第１次産業が経済基盤のものを村落、第２・３次産業が経済基盤のものを都市と呼びます。
::
::sup
厳密な区別は国によって異なり、日本では人口規模で区別
::
### 集落の立地
::top
集落の立地には、次の２つの条件があります。
::

- 自然条件
  :::lead
  飲料や農業用の==水が得やすいこと==
  :::
- 社会条件
  :::lead
  外敵から防御しやすいこと
  :::

#### 自然条件
::top
人は水が得やすいところに集まり居住します。
::
::last
特に扇状地では、谷からの水が地下に伏流する前に得やすい[[扇頂]]と、伏流した水が湧出する[[扇端]]に集落が立地します。
::
::gazo
::
::top
ただし、扇状地も含め沖積平野では水害が心配されます。
::
::last
集落は、氾濫原の[[自然堤防]]のような微高地、河岸段丘の段丘面に立地します。
::
::gazo
::
::gazo
::
#### 社会条件
::top
人は、その歴史・経済発展のなかで、外敵から防御しやすいところや物資の交換しやすいところに居住しました。
::
::last
山地と平地との物資の交換のために、河川が山地から平地に出る谷口に発達した集落などがあります。
::
