## 気候帯
### Ａ～Ｅの分類
::top
ドイツの気候学者ケッペンは、植生とそれに深く関わる==気温==・==降水==に注目し、気候の分類をおこないました。
::
::gazo
::
::top
ケッペンは、樹林の有無などの景観に基づき、赤道から極地（北極あるいは南極）にかけて気候が似た地域を大きく５つに分類できると考えました。
::
::middle
赤道に近い方から順に、「[[熱帯]]・[[乾燥帯]]・[[温帯]]・[[亜寒帯]]（[[冷帯]]）・[[寒帯]]と分類したのです。
::
::last
これらは気候帯と呼ばれ、それぞれアルファベット１文字で「[[Ａ]]・[[Ｂ]]・[[Ｃ]]・[[Ｄ]]・[[Ｅ]]」と表記されます。
::
---arrow---
::top
これら５つの気候帯のうち、==Ａ・Ｃ・Ｄは樹木が生育できる==ため樹林気候と呼ばれ、==Ｂ・Ｅは樹木が生育できない==ため無樹林気候と呼ばれます。
::
::gazo
::
::middle
12の気候区は、気候帯の記号（Ａ～Ｅ）に、ｆやａなどの小文字あるいはＳやＴなどの大文字を併せ、記号で表すことができます。
::
::last
なお、これらｆやＳなど２文字目以降のアルファベットは、ある意味をもつドイツ語の頭文字です。
::
#### ２文字目のアルファベット意味
::top
ｆ…年中降雨（feucht）
::
::top
ｓ…夏に乾燥（sommer trocken）
::
::top
ｗ…冬に乾燥（winter trocken）
::
::top
ｍ…中間（mittelform）
::
::top
Ｓ…ステップ（Steppe、平らな土地の意）
::
::top
Ｗ…砂漠（Wüste）
::
::top
Ｔ…ツンドラ（Tundre、木がない土地の意）
::
::top
Ｆ…氷雪（Froste）
::
#### ３文字目のアルファベット意味
::top
ａ…最暖月平均気温22℃以上
::
::top
ｂ…最暖月平均気温22℃未満
::
### 気候区の分布
::top
気候区の分布は次の通りです。
::
::gazo
::
::middle
つまり、北半球に広く分布すると言えます。
::
::last
反対に、北半球で最も分布が狭いのは[[地中海性]]気候です。
::
---arrow---
::top
ステップ気候・砂漠気候は、[[オーストラリア大陸]]、次いで[[ アフリカ大陸 ]]に広く分布します。
::
::last
==亜寒帯冬季少雨気候がユーラシア大陸にしかないこと==、==オーストラリア大陸・アフリカ大陸には亜寒帯湿潤気候・亜寒帯冬季少雨気候・ツンドラ気候・氷雪気候が存在しないこと==も覚えておきましょう。
::
### 緯度や東岸・西岸との関係性
::top
気候区の分布は、緯度や東岸・西岸に影響されるためある程度の法則があり、それをわかりやすくしたのが大陸の模式図です。
::
::middle
地球の緯度ごとの陸地と海洋の割合に基づき、地球の大陸を１つとして考えると次のような図で表せます。
::
::middle
色塗り部分が陸地で、これを仮想大陸と言います。
::
::last
==南緯60°あたりは陸地がない==ため、色塗り部分がなくなっています。
::
::gazo
::
::middle
実際の世界の気候区の分布と比べてみましょう。
::
::last
概ねこの模式図のように分布しているのではないでしょうか。
::
::gazo
::
::middle
12の気候区は、気温・降水に基づき分類されていると既に学習しました。
::
::middle
ある都市の気温・降水に関する次のような表を見れば、その都市がどの気候区に該当するかを判定できます。
::
::last
判定は、「Ｅか否か」→「Ｂか否か」→「ＡかＣかＤか」と順に進めていきます。
::
#### まず「Ｅか否か」
::top
各月の平均気温のうち、最も高いものを==最暖月平均気温==と言います。
::
---arrow---
::top
最暖月平均気温が[[10]]℃未満なら[[寒]]帯（Ｅ）と判定されます。
::
::middle
もしＥであれば、さらにETかEFかを判定しましょう。
::
::last
最暖月平均気温が10℃未満で[[０]]℃以上なら[[ツンドラ]]気候（ET）、最暖月平均気温が[[０]]℃未満なら[[氷雪]]気候（EF）です。
::
---arrow---
::top
Ｅでなければ、次の判定に移りましょう。
::
#### 次に「Ｂか否か」
::top
Ｂかどうかの判定は大変煩雑です。
::
::top
各月の降水量のうち、最も少ないものを最少雨月降水量、反対に最も多いものを最多雨月降水量と言います。
::
::middle
また、==１年間の各月の気温が高い時期を「夏」、低い時期を「冬」と言います==。
::
::last
==北半球では「６～８月」に夏が、南半球では「12～２月」が夏です==。
::
---arrow---
::top
夏に最少雨月があるなら、次の①②を比較します。
::

- 夏の最少雨月降水量✕[[３]]
- 冬の最多雨月降水量

---arrow---
::top
冬に最少雨月があるなら、次の①'②'を比較します。
::

- '冬の最少雨月降水量✕[[10]]
- '夏の最多雨月降水量

::last
冬の方が夏より少なければ（①'＜②'）、降水型はｗ（冬に乾燥）、そうでなければｆ（年中湿潤）となります。
::
::top
ここでは、年間降水量をＲ、年間平均気温をｔ、乾燥限界値をｒとおきます。
::
---arrow---
::top
降水型ｓならｒ＝20ｔ
::
::top
降水型ｆならｒ＝20（ｔ＋[[７]]）
::
::top
降水型ｗならｒ＝20（ｔ＋[[14]]）
::
---arrow---
::top
求められたｒをＲと比較します。
::
::middle
ｒ≦ＲならＢではない（Ａ・Ｃ・Ｄのいずれか）
::
::middle
1/2ｒ≦Ｒ＜ｒならＢＳ（ステップ気候）
::
::last
Ｒ＜1/2ｒならＢＷ（砂漠気候）
::
---arrow---
::top
次の都市の気候区を試しに判定してみましょう。
::
---arrow---
::top
月平均気温が最も高い月（最暖月）は５月であり、この時期あたりの夏に最多雨月があります。
::
::middle
その半年後の冬に最少雨月があります。
::
::middle
冬の最少雨月降水量✕10＝0（mm）、これは夏の最多雨月降水量161.2mmを下回ります。
::
::last
すなわち降水型はｗ（冬に乾燥）です。
::
---arrow---
::top
降水型ｗならｒ＝20（ｔ＋14）なので、ｒ＝20（29.6＋14）＝872となります。
::
::last
1/2ｒ≦R＜ｒ、具体的には508.5＜872となるので、この都市の気候区はステップ気候と判定できます。
::
::top
共通テスト・センター試験の問題では、細かく計算しなくても問題が解けます。
::
::last
計算が煩雑で苦手という人、覚える時間的余裕がない人は、次のようにひとまず押さえておきましょう。
::
---arrow---
::top
年間降水量が500mm未満であれば、おおよそ[[乾燥]]帯（Ｂ）と判定されます。
::
::last
年間降水量が500mm未満～[[250]]mm程度なら[[ステップ]]気候（ＢＳ）、[[250]]mm未満なら[[砂漠]]気候（ＢＷ）です。
::
::sup
必ずしもこの通りではないので注意
::
#### 最後に「ＡかＣかＤか」
::top
Ｅ・Ｂのどちらでもなければ、最後に「ＡかＣかＤか」を判定します。
::
::last
各月の平均気温のうち、最も低いものを==最寒月平均気温==と言います。
::
---arrow---
::top
最寒月平均気温が[[18]]℃以上なら[[熱]]帯（Ａ）、[[18]]℃未満で[[-3]]℃以上なら[[温]]帯（Ｃ）、[[-3]]℃未満なら[[亜寒（冷）]]帯（Ｄ）です。
::
---arrow---
::top
Ａ・Ｃ・Ｄのどれか判定できたら、「Ｂか否か」のところで判定した降水型を思い出しましょう。
::
::last
Ｄで降水型がｆなら亜寒帯湿潤気候（Df）、降水型がｗなら亜寒帯冬季少雨気候（Dw）です。
::
::top
Ａで降水型がｆなら熱帯雨林気候（Af）ですが、==最少雨月降水量が60mm未満の月がひと月でもあれば==、弱い乾季がある熱帯雨林気候、いわゆる熱帯モンスーン（Am）となります。
::
::middle
また、Ｃで降水型がｆのものは、温暖湿潤気候（Cfa）と西岸海洋性気候（Cfb）の２つがあります。
::
::last
最暖月平均気温が[[22]]℃以上なら温暖湿潤気候（Cfa）、そうでなければ西岸海洋性気候（Cfb）となります。
::
## 気候区に関わるデータの提供方法
### 表
::top
気候区の判定は、気温・降水量に基づいておこないます。
::
::middle
その気温・降水量のデータは様々な形で提供されます。
::
::last
その一例が、次の「表」です。
::
### 雨温図・ハイサーグラフ
::top
「表」のほかに、気温・降水量を示すものに、「雨温図」「ハイサーグラフ」があります。
::
::gazo
::
#### 雨温図
::top
雨温図は、気温を折れ線グラフで、降水量を棒グラフで示したものです。
::
::middle
折れ線の凹凸が大きいほど気温の年較差が大きく、水平に近いほど小さいことになります。
::
::last
また、==棒グラフに数ヶ月続く凹または凸があると、乾季・雨季が存在する==ことになります。
::
---arrow---
::top
雨温図では、==７・８月頃の気温が他の月よりも高めの場合は北半球==であることを、反対に==１・２月頃の気温が高めの場合は南半球==であることを示しています。
::
#### ハイサーグラフ
::top
ハイサーグラフは、月ごとに気温と降水量の相関値を点で示し、１月から順に線で結んだものです。
::
::last
グラフの縦幅は気温の年較差の大小を示し、横幅は乾季・雨季の降水量の差があることを示している。
::
---arrow---
::top
ハイサーグラフでは、縦軸で示された==７・８月頃の気温が他の月よりも高めの場合は北半球==であることを、反対に==１・２月頃の気温が高めの場合は南半球==であることを示しています。
::
