## 寒帯
### 気候区
::top
==最暖月平均気温==が[[10]]℃未満なら寒帯（Ｅ）になります。
::
::middle
==最暖月平均気温==が[[10]]℃未満[[０]]℃以上ならET、==最暖月平均気温==が[[０]]℃未満ならEFです。
::
::last
寒帯（Ｅ）は、ここから２つの気候区に分かれます。
::

- ET
  :::lead
  [[ツンドラ]]気候
  :::
- EF
  :::lead
  [[氷雪]]気候
  :::

## ET（ツンドラ気候）
::gazo
::
::last
極圏（66度33分以上）では、夏に一日中太陽が沈まない白夜に、冬は太陽が昇らない極夜になります。
::
### 植生
::top
夏は永久凍土が溶けて地表面近くが湿地になり、蘚苔類・地衣類が生育します。
::
::last
灌木（低木）が生育するところもわずかにあり、このような荒原を[[ツンドラ]]と言います。
::
::gazo
::
::gazo
::
::middle
また、地表面の蘚苔類・地衣類など有機物は低温のため分解が十分に進まず、泥炭化して蓄積されます。
::
::last
蓄積した泥炭を含む酸性で灰褐色の土壌を==ツンドラ土==と言い、地表に広がっています。
::
### 生活
#### 民族
::top
遊牧民や狩猟民族など少数民族が暮らしています。
::
::last
北アメリカの[[イヌイット]]はアザラシなどの狩猟を、スカンディナヴィア北部の[[サーミ]]はトナカイの遊牧を営んでいます。
::
::gazo
::

### 特色・成因
::top
==最暖月平均気温==が==０℃未満==で、年中寒冷で氷雪に覆われた地域です。
::
::middle
==ブリザード==といわれる突風が、月に数回吹き荒れます。
::
::last
降水は、ほとんど見られません。
::
### 植生
::top
大陸氷河が発達し、植物の生育はありません。
::
::gazo
::
### 生活
::top
学術調査や地下資源調査に従事する人が一時的に住むのみで、人の居住は原則ありません。
::
::middle
このように人の居住が困難な地域を、地理学では[[アネクメーネ]]と言います。
::
::last
なお、反対に居住できる地域を==エクメーネ==と言います。
::
