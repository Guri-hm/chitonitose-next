## 賃金
### 国民総所得（GNI）
::top
「１人当たりの国民総所得（GNI）」≒「１人当たりの年収」≒「各国の経済水準」と考えます。
::
::last
また、入試においては、他に国民総生産（GNP）、国内総生産（GDP）という表現も出てきますが、国民総所得（GNI）とほぼ同じ意味と考えるとよいです。
::
---arrow---
::top
産業別に国民総所得（GNI）に占める割合を見ると、基本的には==第３次産業＞第２次産業＞第１次産業==となります。
::
::middle
また、経済の発達に伴って第１次産業人口が減少し、第２次・第３次産業人口が増加します。
::
::last
つまり、==先進国ほど国民総所得（GNI）が高くなります。==
::
### 各国のGNI
#### 北欧
::top
人口１人当たりのGNIが高い先進国では、社会保障が充実しています。
::
::last
社会保障が充実する北欧諸国は、やはり表にあるように人口１人当たりのGNIが高いです。
::
#### スイス
::top
スイスは、チューリヒなどに金融・保険業が集積し、金融サービス部門に特化して経済水準が極めて高いです。
::
::last
==人口１人当たりのGNIは、先進国の中でもトップクラスに高いです==。
::
#### 日本
::top
日本の人口１人当たりのGNIは、先進国としてのイメージから高いと思われますが、==スイスの半分以下に過ぎません。==
::
::sup
==入試ではスイスと日本の値のどちらかを覚えておくと応用可==
