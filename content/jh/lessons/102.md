---overview---
日本の「開国」は、国交を認めた「和親条約」と通商を認めた「通商条約」の２段階を踏んでおこなわれました。幕府は通商を最後まで渋りました。アジアの国際環境を根本的に変化させたアヘン戦争が、貿易を開戦のきっかけとしたためです。幕府は、西洋諸国の通商要求の背後に、侵攻の可能性を意識せざるを得ませんでした。
---
## 国交の拡大と開港
### アヘン戦争の衝撃
::top
1842年、清はイギリスに[[アヘン戦争]]で敗れ、南京条約を締結しました。
::
::last
この条約で、清は国交・通商の制限を緩めた、つまり「開国」しました。
::
::gazo
::
::top
清の敗北や西洋の脅威は、オランダ風説書で日本に伝えられました。
::
::middle
1825年以来の[[異国船打払令]]（[[無二念打払令]]）は、西洋の報復を招く恐れがありました。
::
::last
1842年、老中水野忠邦は異国船打払令を撤回して[[薪水給与令]]を出し、異国船には帰国を説得し、必要であれば燃料・食糧を給与することにしました。
::
---arrow---
::top
1844年、**オランダ国王**は国書にて、日本の「開国」を説きました。
::
::last
幕府はこの忠告を謝絶し、「鎖国」維持の意向を伝えました。
::
### 米露の「開国」要求
::top
19世紀、アメリカは{{捕鯨|ほげい}}船や清との貿易船の{{寄港|きこう}}地を日本に求めました。
::
::middle
1846年、アメリカ東インド艦隊司令官[[ビッドル]]が、[[浦賀]]に来航して国交・通商を求めました。
::
::last
幕府はこれを拒絶し、ビッドルも積極的関心をもっていなかったため退去しました。
::
::gazo
::
::top
1848年、アメリカがカリフォルニアを獲得し、領土を太平洋岸に到達させました。
::
::last
アメリカは、以前にも増して寄港地を日本に求めるようになりました。
::
---arrow---
::top
1853年６月、アメリカ東インド艦隊司令官[[ペリー]]が、[[浦賀]]に来航し、フィルモア大統領の国書を提出して国交・通商を求めました。
::
::last
幕府は国書を受け取り、翌年回答することを約束してペリーを退去させました。
::
::gazo
::
---arrow---
::top
1853年７月、ロシア使節[[プチャーチン]]が**長崎**に来航し、国交・通商と国境画定を求め、交渉中に退去しました。
::
::last
幕府との交渉が長引くなか、プチャーチンは中断して退去しました。
::
::gazo
::
### 異国への日本一丸での対抗
::top
ペリー来航直後、13代将軍[[徳川家定]]のもとで老中[[阿部正弘]]は、日本一丸で異国へ対抗しようとしました。
::
::middle
先例を破り、==朝廷に国書について報告し、藩主らに回答について意見させました==。
::
::middle
また、海防に意欲的な前水戸藩主[[徳川{{斉昭|なりあき}}]]を幕政に登用しました。
::
::last
幕政に関わらせたことは、朝廷や雄藩の藩主の幕政介入を招く転機となりました。
::

::
::last
