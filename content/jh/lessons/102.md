## 国交の拡大と開港

### アヘン戦争の衝撃

:::top
1842年、清はイギリスに[[アヘン戦争]]で敗れ、南京条約を締結しました。
:::

:::last
この条約で、清は国交・通商の制限を緩めた、つまり「開国」しました。
:::

:::gazo
![](/images/jh/img/102/1.jpg)

アヘン戦争
:::

::arrow

:::top
清の敗北や西洋の脅威は、オランダ風説書で日本に伝えられました。
:::

:::middle
1825年以来の[[異国船打払令]]（[[無二念打払令]]）は、西洋の報復を招く恐れがありました。
:::

:::last
1842年、老中水野忠邦は異国船打払令を撤回して[[薪水給与令]]を出し、異国船には帰国を説得し、必要であれば燃料・食糧を給与することにしました。
:::

::arrow

:::top
1844年、**オランダ国王**は国書にて、日本の「開国」を説きました。
:::

:::last
幕府はこの忠告を謝絶し、「鎖国」維持の意向を伝えました。
:::

## 安政の改革開始

### 開港後の海防新方針

:::top
開港後、阿部正弘は安政の改革と称される改革で海防強化を図りました。
:::

:::last
この改革の柱は、武器・軍事施設や西洋技術の導入でした。
:::

:::gazo{size="half"}
![](/images/jh/img/102/7.jpg)

阿部正弘
:::

#### 武器・軍事施設の導入

- 大船建造の禁を解除
- 軍艦の修理用機械を設備した長崎製鉄所を建設
- 大砲を据え付ける砲台台場を江戸湾に建設
- 溶鉱炉の一種反射炉はんしゃろを築造し、それによる大砲の製造

#### 西洋技術の導入

- 天文方てんもんかたの洋書翻訳の機関<ruby>蛮書和解御用
				<rt>ばんしょわげごよう</rt></ruby>を源流とする、洋書翻訳と西洋技術教授の機関<ruby>蕃書調所
				<rt>ばんしょしらべしょ</rt></ruby>を設置
- 西洋砲術を含む武術全般の訓練機関講武所を江戸に設置
- 海軍伝習所を長崎に設置し、オランダ人が造船・航海術を伝習

:::sup
1860年、日米修好通商条約批准のため、海軍伝習所で学んだ[[勝海舟]]が軍艦[[<ruby>咸臨丸<rt>かんりんまる</rt></ruby>]]で太平洋を横断
