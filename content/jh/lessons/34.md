---overview---
軍事部門を主に担う貴族を「兵（軍事貴族）」と呼びます。平安時代の地方（平安京の門から一歩でも出た場所）は不便で粗野だが、実力次第でどこまでも活躍できる場が開けていました。摂関家の支配で、都での出世を望めない軍事貴族に、地方は大変魅力に富んでいました。まさに田舎も「住めば都」でした。
---
## 戦う貴族と仕える貴族
### 軍事部門を担う貴族
::top
中・下級貴族のうち、軍事部門を担う者は兵（軍事貴族）と呼ばれました。
::
::last
都での出世が望めない彼らには、次の２つの道が開かれていました。
::

- 朝廷との接触を減らしながら、辺境の地方に土着
- 都やその周辺に根拠地を置き、宮中警備・国司を務めて朝廷との関係を維持

### 天皇の血筋を引く貴族
::top
親王（天皇の子弟）の生活は、朝廷の収入でまかなわれていました。
::
::last
時には親王に氏姓（～氏）を与えて臣下とし、朝廷の負担を軽減してきました。
::
::sup
天皇や親王など皇族は氏姓なし
::

### 地方に土着する貴族
::top
９世紀末～10世紀、軍事貴族は賊の鎮圧に国司として地方へ派遣されました。
::
::last
国司は行政官のため、軍事的な統率・指揮権を帯びる[[押領使|おうりょうし]]や[[追捕使|ついぶし]]を別に兼任する形で==賊の鎮圧==にあたりました。
::
::sup
押領使が鎮圧できない場合、別に追捕使を派遣
::
---arrow---
::top
任期後に土着し、在庁官人として働きながら田地経営する軍事貴族もいました。
::
::middle
彼らは受領と対立すると、田地を守るために武力を活かして抵抗しました。
::
::last
受領に悩む有力農民は、軍事貴族の武力を頼って従者となりました。
::
---arrow---
::top
地方に土着した軍事貴族として、次の２人が有名です。
::

- {{上総|かずさ}}国国司として盗賊鎮圧にあたった桓武平氏の祖{{高望王|たかもちおう}}（平高望）
- {{伊予|いよ}}国国司として海賊鎮圧にあたった藤原{{純友|すみとも}}

::last
高望王の子孫、そして藤原純友はやがて大きな反乱を起こしました。

## 大乱と社会変化
### 東と西の大乱
#### 935～940年、[[平将門の乱]]
::top
一族内で争った[[平将門|たいらのまさかど]]（高望王の孫）は、やがて東国一帯を舞台に争乱を引き起こしました。
::
::middle
果てには将門は[[新皇]]と称するに至りました。
::
::last
平[[貞盛|さだもり]]と押領使の藤原[[秀郷|ひでさと]]が将門を討ち、争いは鎮圧されました。
