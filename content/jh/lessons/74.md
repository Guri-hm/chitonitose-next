---overview---
江戸幕府は、大名・朝廷・宗教など様々な方面に対する統制をとりました。「禁中並公家諸法度」で天皇・公家の行動を制限し、これに反目した紫衣事件を契機に、さらに厳格な態度で朝廷に臨みました。また、豊臣政権で不徹底に終わったキリスト教の取締りを強化し、寺院に民衆全員を所属させることで信教面を管理しました。
---

## 宗教への統制
### 仏教への統制
::top
17世紀前半、幕府は宗派ごとに[[寺院法度]]を出し、その宗派の末寺を管理させました。
::
::last
各宗派の本山・本寺の傘下に末寺全てを置く制度を本末制度と呼びます。
::
---arrow---
::top
1665年、幕府は諸宗寺院法度を出し、宗派を越えた共通の仏教統制を始めました。
::
::sup
17世紀半ば、明僧[[隠元隆琦|いんげんりゅうき]]の伝えた[[{{黄檗|おうばく}}宗]]が新しい宗派として登場
::
### 神道への統制
::top
1665年、幕府は**{{諸社禰宜神主法度|しょしゃねぎかんぬしはっと}}**を出し、神社・神職の統制を始めました。
::
::last
公家の吉田家を筆頭にして統制を任せました。
::
### キリスト教への統制
::top
1612年幕領に、翌年全国に[[禁教令]]を出して、キリスト教の信仰を禁じました。
::
::last
以降、幕府や諸藩は宣教師や信徒に対して、次の①②のような迫害を加えました。
::

- 1614年、大名[[高山{{右近|うこん}}]]を[[マニラ]]へ追放（他300人ほど国外追放）
- 1622年、**{{元和|げんな}}の{{大殉教|だいじゅんきょう}}**（長崎の宣教師・信徒の処刑）
