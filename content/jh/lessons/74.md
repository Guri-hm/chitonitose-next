## 大名への統制

### 幕府による統制

:::top
大名は幕府から次のような統制をうけました。
:::

- 参勤交代
  :::lead
  <ruby>国元<rt>くにもと</rt></ruby>と==江戸==とを１年交代で往復
  :::
- 手伝普請
  :::lead
  河川の改修などの負担
  :::
- 軍役
  :::lead
  軍事力や兵糧の提供
  :::

### 藩の支配体制

:::top
江戸時代の大名の領地とその支配機構を、今日では[[藩]]と呼びます（大名＝藩主）。
:::

::arrow

:::top
江戸時代初期、大名は領地を分割して有力武士に与え、土地・人民支配を認めました。
:::

:::last
このような藩の体制を[[<ruby>地方知行制<rt>じかたちぎょうせい</rt></ruby>]]と呼びます。
:::

地方知行制
:::

::arrow

:::

:::middle
藩内の<ruby>支城<rt>しじょう</rt></ruby>を拠点にする有力武士が弱体化し、大名に対抗できなくなりました。
:::

:::last
地方知行制が廃止され、大名の領内一円支配が17世紀半ばまでに一般化しました。

### 藩士の収入

:::top
地方知行制では、有力武士は支配する土地からの年貢を収入としました。
:::

:::last
領内一円支配で有力武士が藩士となると、彼らの収入は消滅しました。
:::

::arrow

:::top
藩は蔵入地（直轄領）からの年貢米を、藩士に[[俸禄]]として支給しました。
:::

:::last
この支給制度を俸禄制度と呼びます。
:::

:::sup
俸禄
:::lead
蔵米と別称され、その支給にちなんで藩士を[[<ruby>蔵米取<rt>くらまいどり</rt></ruby>]]とも呼称
:::

## 宗教への統制

### 仏教への統制

:::top
17世紀前半、幕府は宗派ごとに[[寺院法度]]を出し、その宗派の末寺を管理させました。
:::

:::last
各宗派の本山・本寺の傘下に末寺全てを置く制度を本末制度と呼びます。
:::

::arrow

:::top
1665年、幕府は諸宗寺院法度を出し、宗派を越えた共通の仏教統制を始めました。
:::

:::sup
17世紀半ば、明僧[[<ruby>隠元隆琦<rt>いんげんりゅうき</rt></ruby>]]の伝えた[[<ruby>黄檗<rt>おうばく</rt></ruby>宗]]が新しい宗派として登場
:::

:::last

## 史料

### 禁中並公家諸法度

#### 原文

:::top
一天子諸芸能の事、第一御**学問**也。……
:::

:::top
一摂家為りと雖も、其器用無きは、三公摂関に任ぜらるべからず。況んや其外をや。
:::

:::top
一武家の官位は、公家当官の外為るべき事。
:::

:::top
一改元、漢朝の年号の内、吉例を以て相定むべし。……
:::

:::top
一**紫衣**
		の寺住持職、先規希有の事也。近年猥りに勅許の事、……甚だ然るべからず、向後に於ては、其器用を撰び、……申し沙汰有るべき事。
:::

#### 現代語訳

:::top
一、天皇が修めなければならない諸芸能の第一は学問である。……
:::

:::top
一、五摂家出身の人物であっても、能力のない者は三公・摂関に任命してはならない。ましてやそれより下の家柄の者はいうまでもないことである。
:::

:::top
一、武家に与える官位は、公家の在官者とは別に扱うこととする。
:::

:::top
一、年号を改めることについては、中国の年号から、縁起の良いものを定めよ。……
:::

:::top
一、紫衣を許される寺の住職は、以前はきわめて少なかった。しかし、近頃はやたらに勅許されている。……はなはだけしからぬことである。以後は、その能力をよく吟味して……任命すべきである。
:::
