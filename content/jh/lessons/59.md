## 惣村の形成と土一揆の頻発

### 自立的な村の登場｜鎌倉時代後期～

:::top
鎌倉時代後期から、近畿地方やその周辺部の荘園・公領に、農民たちの自立的・自治的な村[[惣（惣村）]]が形成されました。
:::

:::last
惣村の村民は有力農民の名主や小農民から構成され、惣百姓と呼ばれました。
:::

### 一揆の頻発と代表例

:::top
惣百姓は、時に荘官の免職や年貢の減免を求める団結状態[[一揆]]を結びました。
:::

:::middle
一揆を結んだ後、荘園領主のもとに大勢で押しかける[[強訴]]、耕作を放棄して全員で他領へ逃げ込む[[逃散]]をおこないました。
:::

:::middle
惣村同士が荘園などの枠を超えて蜂起する[[土一揆]]もおこないました。
:::

:::last
免税などの恩恵を施す政治「徳政」を求め土一揆は、[[徳政一揆]]と呼称されました。
:::

:::sup
代始めの徳政
:::lead
支配者が替わると、人間・金銭関係が一新されるという社会通念あり
:::

:::sup
<ruby>分一<rt>ぶいち</rt></ruby>徳政令
:::lead
[[分一銭]]（債務額の一部）を幕府に納めることで発布される徳政令
:::

#### 代表的な一揆

- 1428年、正長の徳政一揆
  :::lead
  [[近江]]国の運送業者[[馬借]]が「徳政」を要求して蜂起し、次いで京都近郊の者も蜂起して土倉・酒屋を襲った土一揆
  :::
  :::lead
  ==幕府による徳政令はなく==、民衆の実力で徳政を強行（私徳政）
  :::
  :::lead
  ６代将軍[[足利義教]]の就任時に起きた「代始めの徳政」
  :::
- 1429年、播磨の土一揆
  :::lead
  守護**赤松<ruby>満祐<rt>みつすけ</rt></ruby>**の家臣の国外退去を要求した土一揆
  :::
- 1441年、<ruby>嘉吉<rt>かきつ</rt></ruby>の徳政一揆
  :::lead
  [[嘉吉の変]]を機に、「徳政」を要求して京都で起こった土一揆
  :::
  :::lead
  ７代将軍**足利<ruby>義勝<rt>よしかつ</rt></ruby>**の就任時に起きた「代始めの徳政」
  :::
