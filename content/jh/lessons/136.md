### 新派劇

:::top
オッペケペー節で知られた[[川上音二郎]]は、素人演劇「壮士芝居」に進出し、日清戦争を題材にして成功を収め、[[新派劇]]と呼ばれる潮流を形成しました。
:::

### 新劇

:::top
日露戦争後、<ruby>坪内逍遙<rt>つぼうちしょうよう</rt></ruby>の文芸協会や[[<ruby>小山内薫
				
			<rt>おさないかおる</rt></ruby>]]の演劇団体[[自由劇場]]が、西洋の近代劇を翻訳・上演し、歌舞伎や新派劇に対して[[新劇]]と呼ばれました。
:::

## 音楽

### 西洋音楽

:::top
西洋音楽は、まず軍楽隊で取り入れられました。
:::

:::last
伊沢修二の努力で、小学校教育に西洋の歌謡を模倣した[[<ruby>唱歌
			<rt>しょうか</rt></ruby>]]が採用されました。
:::

::arrow

:::top
1887年、[[東京音楽学校]]が設立され、本格的な音楽教育が始まりました。
:::

:::last
東京音楽学校は、『<ruby>荒城<rt>こうじょう</rt></ruby>の月』『花』の作曲家[[<ruby>滝廉太郎
				
			<rt>たきれんたろう</rt></ruby>]]を輩出しました。

### 近代美術の概観

:::top
1876年、[[工部美術学校]]が設立され、西洋美術を教授しました。
:::

:::middle
[[岡倉天心]]とアメリカ人[[フェノロサ]]が、伝統美術復興の気運を起こしました。
:::

:::middle
政府は、この気運や欧米での日本画の高い評価を見て、工部美術学校を閉鎖しました。
:::

:::last
1887年、岡倉天心・フェノロサの尽力で、西洋美術を除外した[[東京美術学校]]が設立されました。
:::

:::sup
岡倉天心・フェノロサ
:::lead
法隆寺
			<ruby>夢殿<rt>ゆめどの</rt></ruby><ruby>救世観音<rt>くぜかんのん</rt></ruby>像を検査
:::

::arrow

:::top
西洋画も次第に盛んになり、1896年、東京美術学校に西洋画科が新設されました。
:::

:::last
1907年、第１次西園寺公望内閣の文相牧野<ruby>伸顕<rt>のぶあき</rt></ruby>が、伝統美術と西洋美術の共栄を図り、日本画・洋画・彫刻の展覧の場[[文部省美術展覧会]]（[[文展]]）を開催しました。
:::

### 西洋画

:::top
西洋画は、高橋<ruby>由一<rt>ゆいち</rt></ruby>に開拓されたが、伝統美術復興の気運で一時衰退しました。
:::

::arrow

:::top
浅井<ruby>忠<rt>ちゅう</rt></ruby>が日本初の西洋美術団体の明治美術会を結成し、また、[[フランス]]で学んだ[[黒田<ruby>清輝
			<rt>せいき</rt></ruby>]]の帰国で、西洋画は次第に盛んになりました。
:::

:::top
1896年、黒田清輝は東京美術学校に西洋画科を新設しました。
:::

:::last
同年、黒田清輝らが洋画団体[[白馬会]]を創立し、<ruby>画壇
		<rt>がだん</rt></ruby>の主流を形成しました。
:::
