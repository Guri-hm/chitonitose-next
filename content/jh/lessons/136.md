---overview---
明治時代の芸術の世界は、欧化主義の影響によって、洋風が当初栄えました。しかし、国粋主義の台頭と相まって、伝統美術復興の気運が起こり、伝統芸術への回帰が進みました。伝統美術と西洋美術は、どちらを優位とするかでその後も対立していましたが、1907年の文部省美術展覧会の設置を契機に、両者共栄の道へ向かいました。
---

::top
幕末から明治初期にかけて、[[河竹{{黙阿弥|もくあみ}}]]が{{歌舞伎|かぶき}}で活躍しました。
::
---arrow---
::top

- ５代目尾上菊五郎

### 新劇
::top
日露戦争後、{{坪内逍遙|つぼうちしょうよう}}の文芸協会や[[小山内薫おさないかおる ]]の演劇団体[[自由劇場]]が、西洋の近代劇を翻訳・上演し、歌舞伎や新派劇に対して[[新劇]]と呼ばれました。
::
## 音楽
### 西洋音楽
::top
西洋音楽は、まず軍楽隊で取り入れられました。
::
::last
伊沢修二の努力で、小学校教育に西洋の歌謡を模倣した[[唱歌|しょうか]]が採用されました。
::
---arrow---
::top
1887年、[[東京音楽学校]]が設立され、本格的な音楽教育が始まりました。
::
::last
東京音楽学校は、『{{荒城|こうじょう}}の月』『花』の作曲家[[滝廉太郎たきれんたろう   ]]を輩出しました。
::
::gazo
::
::middle
[[岡倉天心]]とアメリカ人[[フェノロサ]]が、伝統美術復興の気運を起こしました。
::
::middle
政府は、この気運や欧米での日本画の高い評価を見て、工部美術学校を閉鎖しました。
::
::last
1887年、岡倉天心・フェノロサの尽力で、西洋美術を除外した[[東京美術学校]]が設立されました。
::
::sup
岡倉天心・フェノロサ
:::lead
岡倉天心・フェノロサ

::
---arrow---
::top
西洋画も次第に盛んになり、1896年、東京美術学校に西洋画科が新設されました。
::
::last
1907年、第１次西園寺公望内閣の文相牧野{{伸顕|のぶあき}}が、伝統美術と西洋美術の共栄を図り、日本画・洋画・彫刻の展覧の場[[文部省美術展覧会]]（[[文展]]）を開催しました。
::
::double
1906年、岡倉天心は『茶の本』で次の言葉を記しました。「西洋人は、日本が平和のおだやかな技芸に耽ふけ
っていたとき、野蛮国とみなしていたものである。だが、日本が満州の戦場で大殺戮だいさつりく
を犯しはじめて以来、文明国と呼んでいる。〔中略〕もしもわが国が文明国となるために、身の毛のよだつ戦争の光栄に拠よ
らなければならないとしたら、われわれは喜んで野蛮人でいよう。われわれの技芸と理想にふさわしい尊敬がはらわれる時まで喜んで待とう」。これまでの学習を振り返り、天心のこの言葉をどう受けとめるか。

### 西洋画
::top
西洋画は、高橋{{由一|ゆいち}}に開拓されたが、伝統美術復興の気運で一時衰退しました。
::
---arrow---
::top
浅井{{忠|ちゅう}}が日本初の西洋美術団体の明治美術会を結成し、また、[[フランス]]で学んだ[[黒田{{清輝|せいき}}]]の帰国で、西洋画は次第に盛んになりました。
::
::top
1896年、黒田清輝は東京美術学校に西洋画科を新設しました。
::
::last
同年、黒田清輝らが洋画団体[[白馬会]]を創立し、{{画壇|がだん}}の主流を形成しました。
::
