## 対露開戦の準備

### 強まるロシアの脅威

:::top
ロシアの南下が朝鮮半島に及べば、日本の国防にも影響を与えます。
:::

:::last
日清戦争後、ロシアは次のことで着実に脅威を増しました。
:::

:::list{class="en"}
- 朝鮮に親露政権が樹立し、国号を大韓帝国と改称
- 遼東りょうとう半島の<ruby>旅順
				<rt>りょじゅん</rt></ruby>・<ruby>大連<rt>だいれん</rt></ruby>を租借
- 東清とうしん鉄道が旅順・大連まで延長
- 北清事変を契機に、満州全域を事実上占領
:::

### ２つの対露政策

:::top
対ロシアの政策として、日本では次の２つの考え方に分かれました。
:::

- 日露協商論
  :::lead
  「ロシアの満州支配」「韓国での日本の優越権」の日露での相互承認、いわゆる満韓交換を交渉する考え方
  :::
- 日英同盟論
  :::lead
  ロシアの南下を警戒する[[イギリス]]に接近する考え方
  :::

::arrow

:::top
1902年、日英同盟協約締結
:::lead
1923年まで続いた、日英の軍事同盟を約束する協約
:::

:::sup
同盟
:::lead

## 日露戦争

### 日本とロシアの武力衝突

:::lead
日露の交渉決裂後、日本軍の<ruby>旅順<rt>りょじゅん</rt></ruby>・<ruby>仁川<rt>インチョン</rt></ruby>攻撃で始まった軍事衝突
:::

:::lead
日本は国力、ロシアは革命運動が問題となり、戦争継続が途中で困難化
:::

:::lead
局地戦①奉天会戦：１週間の激戦の末、ロシア軍を後退させた戦い
:::

:::lead
局地戦②日本海海戦：1905年５月、<ruby>三笠<rt>みかさ</rt></ruby>を<ruby>旗艦
		<rt>きかん</rt></ruby>とする日本連合艦隊が、ロシアのバルチック艦隊をほぼ全滅させた海戦
:::

:::lead

### 講和条約

:::lead
アメリカ大統領[[セオドア＝ローズヴェルト]]の仲介を受け、日本全権[[小村寿太郎]]とロシア全権[[ウィッテ]]が結んだ日露戦争の講和条約
:::

:::list{class="en"}
- 大韓帝国に対する日本の指導・保護・監督権を承認
- 清からの旅順・大連の租借そしゃく権を日本に譲渡
- 東清鉄道の長春ちょうしゅん以南とその付属の利権を日本に譲渡
- <ruby>樺太<rt>
						からふと</rt></ruby>の北緯50度以南とその付属の諸島を日本に譲渡
- 沿海州えんかいしゅうとカムチャッカの日本の漁業権を承認
:::
