## 加速する国風化の背景

### 公的な交流の断絶―背景１

:::top
894年、[[菅原道真]]は次の２点を理由に遣唐使中止を建議しました。
:::

:::list{class="en"}
- 唐の衰退
- 渡航の危険性（８世紀初めからの航路：南路）
:::

:::last
唐との公的な交流は断たれました。
:::

:::sup
①②の他に、唐の商人との貿易を確立でき、公的な交流が不要になったから
:::

::arrow

:::top
907年、唐が滅亡し、その余波で周辺の国々に興亡が生じました。
:::

:::middle
中国は[[宋]]に再統一され、中国東北部の<ruby>渤海<rt>ぼっかい</rt></ruby>は[[契丹]]（遼）に滅ぼされました。
:::

:::middle
また、朝鮮半島の新羅は[[高麗]]に滅ぼされました。
:::

:::last
公的な交流はありませんでしたが、宋・高麗の商人が九州[[博多]]によく来航しました。
:::

:::sup
僧以外の日本人の渡航は律で禁止

## 国文学と国風美術の発達

### 文字の発達

:::top
漢字をもとにした[[かな文字]]（平がな：草書体の簡略化、片かな：字の一部分）が発達しました。
:::

:::last
公式な文書でのかな文字の使用は避けられましたが、物語や日記では広く使われました。
:::

## 信仰の発達

### 浄土教の布教

:::top
釈迦の死没後2000年が経つと（1052年以降）、釈迦の教え（解脱の方法）は効果を失い、また、世の中は乱れるという考え[[<ruby>末法<rt>まっぽう</rt></ruby>思想]]が一部の者のなかにありました。
:::

:::middle
これらの者は、死後に阿弥陀仏（如来）の慈悲で仏の世界（極楽浄土）へ<ruby>往<rt>い</rt></ruby>き（往生）、そこで仏の直接指導のもと解脱を遂げればよいとする教え[[浄土教]]を人々に説きました。
:::

:::last
貴族や庶民に浄土教は広がり、信仰の対象である阿弥陀仏の美術品が作成されました。
:::

:::gazo{size="half"}

### 浄土教以外の信仰

:::top
経塚
:::lead
写経した経典を筒に入れて土中に埋納し、その上に築いた塚
:::

:::top
<ruby>本地垂迹<rt>ほんじすいじゃく</rt></ruby>説
:::lead
仏教信仰に<ruby>神祇<rt>じんぎ</rt></ruby>信仰の要素が融合する神仏習合のなか、神と仏の並存状態を説いた理論人の救済のため、仏（<ruby>本地<rt>ほんじ</rt></ruby>）が神という仮の姿（<ruby>垂迹<rt>すいじゃく</rt></ruby>）で現れたと納得・説明
:::

:::top
御霊会
:::lead
災いを起こす怨霊（例：菅原道真）や疫神を慰め、祟りを逃れようとする鎮魂の祭礼
:::
