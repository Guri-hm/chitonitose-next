## 化政文化

### 化政文化とは

:::top
18世紀末、11代将軍[[徳川家斉]]のもと、老中[[松平定信]]が、諸改革[[寛政の改革]]のなかで倹約令や出版統制令を発令しました。
:::

:::last
文化の発展は、これらの制限により一時停滞しました。
:::

::arrow

:::top
1793～1841年の文化～<ruby>天保<rt>てんぽう</rt></ruby>期（大御所時代）、文化は再び発展し始めました。
:::

:::middle
文化・文政の年号の１字をとり、この時期の文化を[[化政]]文化と呼びます。
:::

:::last
この文化は次の①②の特色をもちます。
:::

:::list{class="en"}
- 幕藩体制の動揺・矛盾に対する、現実的・批判的な精神の展開
- 多種多様な内容で、江戸を中心に国民的な広がり
:::

## 蘭学と科学技術

### 幕府の方針決定

:::top
18世紀末、海岸防備の必要性が高まり、地理・天文に対する蘭学の有益性が評価されました。
:::

:::middle
一方で、蘭学は異国への憧れを育て、幕政の批判へと人を誘惑しがちと警戒されました。
:::

:::last
幕府は、民間に委ねずに幕府の手で保護・研究し、その成果も機密にすべきとしました。
:::

#### 地理学

:::top
[[伊能忠敬]]は、還暦を司る役職[[<ruby>天文方<rt>てんもんかた</rt></ruby>]]の**高橋<ruby>至時<rt>よしとき</rt></ruby>**
		に測地・暦法を学びました。
:::

:::last
伊能忠敬は幕府の命令を受け、1800～16年にかけて全国の沿岸を測量し、『[[大日本沿海<ruby>輿地<rt>よち</rt></ruby>全図]]』の作成にあたりました。
:::

:::sup
『大日本沿海輿地全図』
:::lead
日本初の実測による全国海岸線の地図で、伊能忠敬の死後３年の1821年に完成
:::

#### 天文学

:::top
幕府は<ruby>天文方<rt>てんもんかた</rt></ruby>の**高橋<ruby>至時<rt>よしとき</rt></ruby>**
		に命じて、西洋流の暦**寛政暦**を作成させました。
:::

::arrow

:::top
1811年、幕府は至時の子で天文方の[[高橋景保]]の提唱をうけ、==洋書翻訳のための機関==[[<ruby>蛮書和解御用<rt>ばんしょわげごよう</rt></ruby>]]を新設し、景保らが翻訳に当たった。
:::

:::sup
蛮書和解御用
:::lead
後には洋書で得た知識の教授も担い、[[<ruby>蕃書調所<rt>ばんしょしらべしょ</rt></ruby>]]と改称
:::

:::sup
元オランダ通詞[[志筑忠雄]]は、『[[暦象新書]]』で万有引力説・地動説を紹介し、また、[[ケンペル]]の『[[日本誌]]』の一部を邦訳し、『[[鎖国論]]』と命名

### 方針違反への対応

:::top
幕府は、蘭学に対する方針（成果の機密・民間研究の禁止）違反に次の弾圧をしました。
:::

- 1828年、シーボルト事件
  :::lead
  [[シーボルト]]の帰国時、「大日本沿海<ruby>輿地<rt>よち</rt></ruby>全図」などの国外持出が発覚した事件
  :::
  :::lead
  文献を得るためにシーボルトに地図を贈った天文方の[[高橋景保]]は死罪
  :::
  :::sup
  シーボルト…国外追放後、『日本』を著して西洋に日本を紹介
  :::
- 1839年、蛮社の獄
  :::lead
  1837年の[[モリソン号事件]]や幕府の対外政策への批判を理由に、蘭学を私的に研究していた[[渡辺崋山]]・[[高野長英]]を処罰した事件
  :::
  :::lead
  見せしめが目的で、蘭学を学んで幕政批判に傾かないように警告
  :::
