---overview---
貨幣価値の安定を前提として、1886年頃から鉄道・紡績を中心に会社設立ブームが起こり、1890年の恐慌で中断されましたが、日清戦争後には再びブームを迎えました。工場生産が目覚ましく発展し、日本の産業革命が本格的に始まりました。それは、工場をもつ資本家が労働者を雇い、富を形成する資本主義の本格的確立も意味しました。
---
## 産業革命
### 最初の企業勃興
::top
1880年代の[[松方]]財政は、紙幣価値を下落させる大量の不換紙幣を処分し、紙幣価値の上昇、つまり物価を下落させるデフレ政策をとりました。
::
::last
米・生糸・{{繭|まゆ}}などの価格が暴落し、行き過ぎたデフレは問題となりました。
::
---arrow---
::top
[[銀]]本位と、それに基づく{{兌換|だかん}}制度の銀本位制が確立されると、物価は安定し、資金賃借料（金利）が低下しました。
::
::last
企業が資金を借りやすくなり、産業界は活気づきました。
::
::gazo
::
::top
==1886～89年==、鉄道や{{紡績|ぼうせき}}を中心に会社設立ブーム[[企業{{勃興|ぼっこう}}]]が起こり、機械技術を本格的に用いる産業革命が日本でも始まりました。
::
::last
しかし、企業勃興は金融機関の資金不足による「1890年の恐慌」で挫折しました。

==日清戦争の賠償金をもとに==、日本政府は次のことに取り組みました。
::

- 軍備拡張の推進
- 1897年、**貨幣法**制定
  :::lead
  [[金]]本位と、それに基づく{{兌換|だかん}}制度の金本位制を確立させた法令
  :::
- 政府監督下で、特定の分野に資金を供給する特殊銀行を設置
  :::lead
  代表的な特殊銀行は、貿易金融を目的とする**横浜正金銀行**、台湾の開発・近代化を目的とする[[台湾銀行]]
  :::

::gazo
::
::top
鉄道や紡績で、企業設立ブーム[[企業勃興]]が再び起こりました。
::
::middle
繊維産業を中心に、生産手段をもつ資本家が労働者を雇い、商品生産をおこなう経済体制資本主義が本格的に成立しました。
::
::last
しかし、企業勃興は過剰生産による恐慌で再び挫折しました。
::
#### 金本位制にした理由
::top
当時、金に対する銀の価値が低下し、銀本位制は次の効果をもたらしました。
::

- 金本位制の国への輸出を増加（＝利点）
- 金本位制の国からの輸入を減少（資本の輸入で不利）
- 金と銀の相場の変動で貿易が不安定（＝不利）
