## 公武合体と幕政の大改革

### 政局の混乱と安定の模索

:::top
1860年、大老[[井伊直弼]]が[[桜田門外の変]]で暗殺されました。
:::

:::last
朝廷の権威増大や<ruby>薩摩<rt>さつま</rt></ruby>藩など<ruby>雄藩<rt>ゆうはん</rt></ruby>の幕政介入で政局が混乱する中、老中[[安藤<ruby>信正<rt>のぶまさ</rt></ruby>]]が幕政の中心となり、次の３つに取り組みました。
:::

- 貿易による国内の品不足と物価高騰への対処
- 貿易に反感をもち通商条約の早急な破却を目指す運動攘夷運動の阻止
- 朝廷（公）と幕府（武）の融和公武合体で政局の安定化

![](/images/jh/img/104/1.jpg)

安藤信正
:::

#### 品不足と物価高騰

:::top
貿易が日本の輸出超過のため、国内は品不足と物価高騰に陥りました。
:::

:::middle
1860年、安藤信正は物価抑制を目的に[[五品江戸廻送令]]を発令しました。
:::

:::last
西洋諸国や輸出に関わる商人の反対で、五品江戸廻送令は効果があがりませんでした。
:::

:::gazo
![](/images/jh/img/103/4.svg)

五品江戸廻送令
:::

:::top
貿易に対する反感から攘夷運動が展開し、次の出来事が起こりました。

:::

:::last

## 攘夷運動と反幕府の激化

### 攘夷運動と英国の対日政策硬化

:::top
折からの攘夷運動に加え、次の出来事でイギリスは対日政策を硬化させました。
:::

- 1862年、生麦事件
  :::lead
  島津久光一行の江戸からの帰途、横浜近郊でイギリス人が行列を横切り、久光の従者に殺傷された事件
  :::
  :::lead
  1863年、事件の報復として薩摩藩・イギリス間に[[薩英戦争]]が勃発
  :::
- 1862年、イギリス公使館焼打ち事件
  :::lead
  建築中のイギリス公使館が、長州藩士高杉<ruby>晋作<rt>しんさく</rt></ruby>らに襲撃された事件
  :::

:::gazo
![](/images/jh/img/104/6.jpg)
