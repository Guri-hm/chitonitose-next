## 憲法制定の準備

### 憲法の方針

:::top
1881年、政府は国会の開設・在り方と君主権について、次の内紛を生じさせていました。
:::

- 急進論（主張者：大隈重信）
  :::lead
  国会を早期に開設し、イギリス流の議院内閣制で君主権を弱くするべきと主張
  :::
- 漸進論（主張者：伊藤博文・右大臣岩倉具視ともみ）
  :::lead
  国会をゆっくり開設し、ドイツ流の<ruby>欽定<rt>きんてい</rt></ruby>憲法で君主権を強くするべきと主張
  :::

:::last
大隈重信が[[明治十四年の政変]]で追放されると、君主権を強くする方針が決まりました。
:::

::arrow

:::top
1882年、政府は伊藤博文をヨーロッパに派遣して、憲法調査に当たらせました。
:::

:::last
伊藤博文は、ベルリン大学の[[グナイスト]]、ウィーン大学の[[シュタイン]]から、==ドイツ流の憲法理論を学び==、帰国後に国会開設・憲法制定の準備を進めました。

### 貴族院の土台作り

:::top
1884年、華族令制定
:::lead
華族を「<ruby>公<rt>こう</rt></ruby>・<ruby>侯<rt>こう</rt></ruby>・<ruby>伯<rt>はく</rt></ruby>・<ruby>子<rt>し</rt></ruby>・<ruby>男<rt>だん</rt></ruby>」の５<ruby>爵<rt>しゃく</rt></ruby>にわけ、==明治維新の功労者も華族にした==法令国会開設に備えて、公選によらない議院「[[貴族院]]」の議員の候補者確保
:::

### 内閣制度の制定

:::top
1885年、太政官制廃止・内閣制度制定
:::lead
天皇の指名する総理大臣（首相）が、各省長官（国務大臣）を率いて、国家の最高行政機関「内閣」を組織し、内閣は会議「閣議」で意思決定<ruby>宮内<rt>くない</rt></ruby>省は宮中事務を司り、宮中と政府の区別のために内閣から分離==内閣に属さない者が==、[[内大臣]]となって宮中に置かれ、<ruby>御璽<rt>ぎょじ</rt></ruby>・<ruby>国璽<rt>こくじ</rt></ruby>（天皇・日本の印）を保管し、天皇を常侍<ruby>輔弼<rt>ほひつ</rt></ruby>（天皇への助言）[[伊藤博文]]が初代総理大臣に就任し、また、例外的に宮内大臣を兼任
:::

### 強い政府統制での地方制度

:::top
ドイツ人顧問[[モッセ]]の助言を得て、地方制度が整備されました。
:::

- 1888年、市制・町村制
  :::lead
  1878年の郡区町村編制法で定めた行政単位に、「市」を加えて再整備
  :::
  :::lead
  市長は内務大臣が選任、無給の名誉職である町村長は町村会で公選
  :::
- 1890年、府県制・郡制
  :::lead
  郡長・郡参事会を行政機関、町村会議員と大地主の互選による郡会を議決機関
  :::
  :::lead
  府県知事は官選（政府の任命）、府県会議員は市会・郡会の間接選挙
  :::

### 政府による憲法草案

:::top
1886年頃から、政府は国民に極秘で憲法草案を作成し、私擬憲法に対抗しました。
:::

:::middle
草案作成は、ドイツ人顧問[[ロエスレル]]らの助言を得て、**伊藤博文**を中心に**井上<ruby>毅<rt>こわし</rt></ruby>**・伊東<ruby>巳代治<rt>みよじ</rt></ruby>・金子<ruby>堅太郎<rt>けんたろう</rt></ruby>らが当たりました。
:::

:::last
草案の審議は、1888年設置の[[<ruby>枢密院<rt>すうみついん</rt></ruby>]]で天皇臨席のもとで重ねられました。
:::

:::sup
1887年の保安条例で私擬憲法の作成を禁じ、憲法制定は政府主導で進行
:::

## 大日本帝国憲法

### 憲法の発布

:::top
1889年２月11日（紀元節）、大日本帝国憲法発布
:::lead
[[黒田清隆]]内閣の時に発布され、日本を近代的立憲制国家とした憲法天皇（君主）が自ら制定して国民に与えた憲法[[<ruby>欽定<rt>きんてい</rt></ruby>憲法]]という体裁

#### 強い君主権

:::top
神聖不可侵とされた天皇は、統治権の全てを握る<ruby>総攬<rt>そうらん</rt></ruby>者として、国会も関与できない[[天皇大権]]と総称される次の強い権限をもちました。
:::

:::list{class="en"}
- 文官・武官の任免権、宣戦・講和や条約の締結などの外交権
- 非常事態時、軍隊に治安権限を与える戒厳令かいげんれいの発令権
- 議会の閉会中などの緊急事態時、天皇の命令が法律に代わる緊急勅令の制定権
- 陸海軍の作戦・用兵などの指揮統率をおこなう<ruby>統帥権<rt>とうすいけん</rt></ruby>
:::

:::sup
1886年の帝国大学令で大学を官吏養成機関とし、1887年に文官高等試験の制を公布
:::

:::sup
統帥権
:::lead
内閣も関与できず（陸・海軍省から分離）、[[統帥権の独立]]と呼称
:::

:::sup
帷幄いあく上奏
:::lead
参謀総長、軍令部総長、陸・海軍大臣が、天皇に直接意見を伝えること
:::

#### 二院制の国会

:::top
大日本帝国憲法下の最高立法機関の国会は、[[帝国議会]]と呼ばれ、対等の権限を２つの議院から成る二院制でした。
:::

:::list{class="en"}
- 貴族院：皇族議員、世襲・互選による華族の議員、天皇が任命する勅任議員、以上の議員から成る国会の機関
- 衆議院：公選による議員から成る国会の機関
:::

#### 憲法下の日本国民

:::top
大日本帝国憲法下の日本国民は、天皇の支配を受ける者[[<ruby>臣民<rt>しんみん</rt></ruby>]]と呼称され、法律の範囲内でのみ、信教の自由、言論・出版・集会・結社の自由を許可されました。
:::
