## 諸法典の整備

### 皇室関係の法典

:::top
1889年、<ruby>皇室典範<rt>こうしつてんぱん</rt></ruby>制定
:::lead
大日本帝国憲法と同時に制定された皇室関係の基本法典皇位の継承、天皇・皇族に対する犯罪、内乱罪の厳罰化などを規定
:::

:::sup
皇室典範
:::lead
民衆が干渉するものではないと公布なし（通常、政府機関紙『官報』で公布）
:::

### フランス流の諸法典

:::top

:::

:::last
しかし、これらのうち[[民法]]は、公布以前から一部の法学者に批判されていました。

#### 民法典論争

:::top
ボアソナードの民法は、個人主義を重んじ、日本の伝統的な倫理に適しませんでした。
:::

:::last
民法の公布後、[[民法典論争]]と呼ばれる論争が本格化し、次の法学者が戦いました。
:::

- 反対：<ruby>穂積八束<rt>ほづみやつか</rt></ruby>が「民法<ruby>出<rt>い</rt></ruby>デゝ忠孝<ruby>亡<rt>ほろ</rt></ruby>ブ」の論文を発表
- 賛成：梅謙次郎うめけんじろう

:::gazo{size="half"}
![](/images/jh/img/120/2.jpg)

穂積八束
:::

:::gazo{size="half"}
![](/images/jh/img/120/3.jpg)

梅謙次郎
:::

::arrow

:::top
民法は、施行までに数年の周知期間を設けてあり、論争を受けて、1892年の第三議会で、商法とともに修正前提の施行延期が決まりました。
:::

:::last
新民法が、1896年・1898年の２回に分けて公布され、1898年に施行されました。
:::

::arrow

:::top
以上の結果、新民法では、次のような日本の伝統的な倫理が維持されました。
:::

- <ruby>戸主<rt>こしゅ</rt></ruby>権
  :::lead
  家族に対する[[<ruby>戸主
  					<rt>こしゅ</rt></ruby>]]の強い権限（婚姻の同意や居所指定など）
  :::
- 家督かとく相続制度
  :::lead
  家督相続とは、戸主の地位・財産の継承を指し、普通長男が相続
  :::

## 条約改正交渉の決着

### 条約改正④

:::top
井上馨の後任の外相[[大隈重信]]は、条約改正に好意的な国から個別に交渉しました。
:::

:::middle
アメリカ・ドイツ・ロシアとの間に、改正条約を調印しました。
:::

:::last
しかし、最高裁判所に相当する[[<ruby>大審院
			<rt>だいしんいん</rt></ruby>]]に限り、外国人判事の任用を認めたため、激しい反対運動が起こりました。
:::

:::gazo{size="half"}
![](/images/jh/img/120/4.jpg)

大隈重信
:::

::arrow

:::top
1889年、大隈重信が対外硬派の団体<ruby>玄洋社<rt>げんようしゃ</rt></ruby>の青年<ruby>来島恒喜<rt>くるしまつねき</rt></ruby>に負傷させられました。
:::

:::last
大隈重信の負傷で、条約改正交渉は中断されました。
:::

### ロシアの南下政策と日英の対応

:::top
19世紀後半、ロシアは<ruby>不凍港<rt>ふとうこう</rt></ruby>を求め、東アジアでの南下を進めました。
:::

:::middle
1891年、ロシアがシベリア鉄道敷設に着工すると、イギリスはロシアへの警戒を一気に強めました。
:::

:::last
ロシアがさらに不凍港を求め、[[朝鮮]]半島に南下を進めると、日本もいずれ危機に陥ると懸念されました。

#### イギリスの対応―日本への接近

:::top
イギリスは、日本に好意を示し、ロシアへ抵抗する味方にしようとしました。
:::

:::last
条約改正に最も反対するイギリスが方針を変え、交渉に好機が到来しました。
:::

#### 日本の対応―朝鮮への期待

:::top
日本は、朝鮮の文明化とそれによる同国・自国の安全を望みました。
:::

:::last
しかし、朝鮮は[[清]]を宗主国とし続け、旧態依然としていました。
:::

### 条約改正⑥と領事裁判権の撤廃

:::top
1894年、外相[[<ruby>陸奥<rt>むつ</rt></ruby><ruby>宗光
			<rt>むねみつ</rt></ruby>]]は、条約の一部改正という条件で交渉しました。
:::

:::middle
対外硬派連合は、条約の一部のみの改正を許さず、全面改正を主張しました。
:::

:::last
陸奥宗光は、自由党の支持によって反対派の声を抑えました。
:::

:::gazo{size="half"}
![](/images/jh/img/120/8.jpg)

陸奥宗光

::arrow

			相互対等の[[最恵国待遇]]

:::

:::last
