## 国大名の政策

### 戦国大名の分国支配

:::top
戦国大名は、分国（領国）支配のために[[分国法]]を制定しました。
:::

#### 分国法の特徴

:::list{class="en"}
- 御成敗式目の影響
- 新しい権力の性格を示す条文の登場
:::

:::sup
②としてが[[<ruby>喧嘩<rt>けんか</rt></ruby><ruby>両成敗
			<rt>りょうせいばい</rt></ruby>]]が有名
:::

:::sup
喧嘩両成敗
:::lead
理由を問わず、争いをした者を、双方とも処罰すること武士の自力救済原理を否定
:::

## 中世都市の形成

### 新しい都市

:::top
政治都市（京都・奈良・鎌倉）を除けば、14～16世紀の都市は、次の３つのタイプに大別できます。
:::

#### タイプ①：門前町・寺内町

:::top
寺社の門前に[[<ruby>門前町<rt>もんぜんまち</rt></ruby>]]や、==浄土真宗（一向宗）の==寺院の周囲に[[<ruby>寺内町
			<rt>じないまち</rt></ruby>]]が発達しました。
:::

:::last
門前町としては、伊勢神宮の宇治・山田、信濃国[[善光寺]]の長野、延暦寺の**坂本**が有名です。
:::

:::last
寺内町としては、越前国の**吉崎**、摂津国の[[石山]]、河内国の[[<ruby>富田林<rt>とんだばやし</rt></ruby>]]が有名です。
:::

:::gazo
![](/images/jh/img/65/2.svg)
:::

#### タイプ②：港町・宿場町

:::top
遠隔地商業の発展で港町・宿場町が成立しました。
:::

:::last
港町としては、伊勢国の[[桑名]]、薩摩国の**<ruby>坊津
				
			<rt>ぼうのつ</rt></ruby>**が挙げられます。
:::

:::sup
<ruby>草戸千軒<rt>くさどせんげん</rt></ruby>
:::lead
備後国にあった港町で、洪水のためか廃絶
:::

::arrow

:::top
日明貿易で栄えた[[堺]]や[[博多]]などの港町は、豪商の組織に運営され、自治都市の性格をもちました。
:::

:::last
堺では[[<ruby>会合衆
			<rt>えごうしゅう</rt></ruby>]]と呼ばれる36人の豪商が、博多では[[年行司]]やと呼ばれる12人の豪商が運営しました。
:::

:::sup

		が『<ruby>耶蘇会士<rt>やそかいし</rt></ruby>日本通信』で報告

### 伝統的な都市―応仁の乱後

:::top
古い政治都市の京都でも、人々は町という組織単位でまとまり、町に属する裕福な商工業者[[町衆]]を中心に自治をおこない始めました。
:::

:::last
町ごとに[[町法]]という独自の掟を定めました。
:::

:::
