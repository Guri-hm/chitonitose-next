## 地方支配の改革

### 財源確保の失敗

:::top
朝廷の財源の要「調・庸」は、次の２つの理由で徴収しにくくなりました。
:::

- 寺院・豪族が初期荘園経営に、逃亡・浮浪中の農民を扶養・使役
- 男を女と戸籍に記載する偽籍の増加（租以外は男のみに課税）

:::last
朝廷の財源が減少し、皇族・貴族への給与も困難になりました。
:::

:::gazo{size="half"}
![](/images/jh/img/32/1.svg)

税の行方
:::

::arrow

:::top
田地を収集・直営し、稲穂による財源確保が進められました。
:::

:::middle
==大宰府に==直営の[[公営田]]、==畿内に==中央政府直営の[[官田]]を設置しました。
:::

:::middle
<ruby>院宮王臣家<rt>いんぐうおうしんけ</rt></ruby>も自分の私財確保のために、田地を過度に収集・直営しました。
:::

:::last
農民自身の農事とは別に、彼らを直営に使役したので混乱を招きました。
:::

::arrow

902年、[[醍醐]]天皇は[[延喜の荘園整理令]]を出し、以降の勅旨田の設置と院宮王臣家による過度な収集・直営を禁止しました。
:::

914年、**<ruby>三善清行<rt>みよしきよゆき</rt></ruby>**が、醍醐に「[[意見封事十二箇条]]」を提出し、地方の混乱と財源の窮乏を報告

### 守の権限拡大と財源確保

:::top
10世紀以降、<ruby>守<rt>かみ</rt></ruby>（国司）に朝廷への一定額の税納入をさせ、代わりに一国の統治（自由な支配）を許しました。
:::

:::middle
守は==この統治権を前任者から引き継ぐことから==、10世紀以降[[<ruby>受領<rt>ずりょう</rt></ruby>]]とも呼ばれました。
:::

:::last
徴税など受領の役割は拡大し、郡司の役割は消滅していきました。
:::

:::sup
一国に国司は６人派遣され、最上位の国司が守（四等官制）

  :::lead

### 名の経営

:::top
有力農民は、受領から名ごとの経営権を貸与され、[[<ruby>田堵
			<rt>たと</rt></ruby>]]として経営を請け負いました。
:::

:::last
田堵は、下級農民を使役して名を耕作し、官物・臨時雑役を納税しました。
:::

:::sup
田堵
:::lead
名を請け負うことから負名とも呼称
:::

:::sup
多数の下級農民を使役し、多数の名を経営した田堵を、特に大名田堵と呼称
:::

### 尾張国郡司百姓等解

#### 原文

:::top
尾張国郡司百姓等解し申す官裁を請ふの事
:::

:::top
裁断せられんことを請ふ、当国の<ruby>守<rt>かみ</rt></ruby>藤原朝臣元命、三箇年の内に責取る非法の官物ならびに濫行横法三十一箇条の愁状
:::

:::top
一裁断せられんことを請ふ、例挙の外、三箇年の内に収納せる加徴の正税冊三万千二百四十八束の息利十二万九千三百七十四束四把一分の事…
:::

:::top
一裁断せられむことを請ふ、元命朝臣が子弟郎等、郡司百姓の手より雑物等を乞ひ取るの事…
:::

:::top
一裁断せられむことを請ふ。守元命朝臣、京より下向する度毎に、有官・散位の従類、同じき不善の<ruby>輩<rt>
				ともがら</rt></ruby>を引率するの事……
:::

:::top
…仍て具さに三十一箇条の事状を勒し、謹みて解す。
:::

:::top
永延二年十一月八日郡司百姓等
:::

#### 現代語訳

:::top
尾張国の郡司と百姓が太政官の裁決を申請すること
:::

:::top
当国の守藤原元命がこの三ヵ年の間に行った非法な徴税と不法行為に関する三一ヵ条（の訴え状）について裁決をお願いします。
:::

:::top
一、定例の出挙のほかに、三年間に正税四三万一二四八束の利息として一二万九三七四束四把一分を徴収したことについて裁断して下さい。
:::

:::top
一、国守元命の子弟や郎等が、郡司や百姓からさまざまな物を奪い取ることについて裁断して下さい。
:::

:::top
一、国守元命が、京から下向する度に有官・散位の従者やよからぬ者たちを引きつれてくることについて裁断して下さい。
:::

:::top
…こうした理由で詳細に三一ヵ条にわたって記し、謹んで申し上げます。
:::

:::top
永延二（九八八）年十一月八日郡司百姓等
:::

#### 解説

:::top
988年、尾張国の郡司と百姓が尾張国国司の藤原元命の非法を訴えました。10世紀以降、国司は任国の統治権をもち、朝廷に納める以外の税収入を自分のものとすることができました。そのため、国司は一族や武装した兵を伴って任国に赴き、厳しい税の取り立てなどをおこないました。原文では「不善の輩」と表現されています。元命が訴えられた背景には、このような国司の増長があったのです。

		余談ではありますが、元命のその後についてです。訴状をうけて元命は尾張国の国司を解任されますが、その後も他国の国司に任命されています。藤原氏の権力や「成功」「重任」がうかがえます。
:::
