### 動乱の収束

:::top
南北朝の動乱は、足利尊氏の孫で３代将軍[[足利義満]]の頃に収束に向かいました。
:::

:::last
1392年、南朝の[[後亀山]]天皇が皇位を放棄し、天皇が北朝の[[後小松]]天皇１人となったことで、南北朝の合体が実現しました。
:::

### 動乱が長引いた背景

:::top
動乱が長引いた背景には、次の２つが挙げられます。
:::

- 全国的に対立が増加
- 相続のあり方が変化（分割相続から単独相続へ）

:::last
一方が北朝側につけば、他方は南朝側につくという有様でした。
:::

## 動乱による社会変動

### 守護の権限拡大

:::top
[[守護]]は、鎌倉幕府が各国に置いた治安維持担当の役職です。
:::

:::last
南北朝の動乱で各地の武士が力を伸ばすと、武士を統括・動員するために、従来①のみであった守護の権限に、次の②～④が追加されました。
:::

#### 守護の権限

- 大犯三ヵ条（京都大番役の催促・謀叛むほん人の逮捕・殺害人の逮捕をする権限）
- <ruby>刈田狼藉<rt>かりたろうぜき</rt></ruby>を取り締まる権限
- <ruby>使節<rt>しせつ</rt></ruby><ruby>遵行
				<rt>じゅんぎょう</rt></ruby>（幕府の裁判の判決を強制執行する権限）
- <ruby>半済令<rt>はんぜいれい</rt></ruby>で認められた、国内の荘園・公領の年貢を半分徴発する権限

:::sup
刈田狼藉

:::lead
田地をめぐる紛争時、所有権を主張して稲を一方的に刈り取る実力行為
:::
:::

:::sup
半済令

:::lead
1352年の発布時点では、近江国・美濃国・尾張国の３国のみ有効で、期限も１年限りであったが、==後に全国的に、そして期限なしに変化==
:::
:::

::arrow

:::top
鎌倉時代以降、地頭が荘園・公領の年貢納入を怠り始めていました。
:::

:::last
守護の力の増大にあわせ、年貢徴収を守護に請け負わせる[[守護請]]が始まりました（地頭の消滅）。
:::

::arrow

:::top
室町時代の守護は、受領の権限を奪って国全体の支配権を握るなど、鎌倉時代と力が大きく異なりました。
:::

:::last
室町時代の守護を守護大名とも呼びます。
:::
