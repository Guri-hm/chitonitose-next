---overview---
足利尊氏が持明院統の光明天皇を即位させると、朝廷は２つに分裂して、南北朝の動乱と呼ばれる長い対立が始まりました。やがて室町幕府が内紛に荒れると、南北朝の対立はより複雑に、そして激しくなっていきました。対立は３代将軍足利義満の時に解消しましたが、対立が諸国の守護・地頭に与えた影響は大きいものでした。
---

### 動乱の収束
::top
南北朝の動乱は、足利尊氏の孫で３代将軍[[足利義満]]の頃に収束に向かいました。
::
::last
1392年、南朝の[[後亀山]]天皇が皇位を放棄し、天皇が北朝の[[後小松]]天皇１人となったことで、南北朝の合体が実現しました。
::
### 動乱が長引いた背景
::top
動乱が長引いた背景には、次の２つが挙げられます。
::

- 全国的に対立が増加
- 相続のあり方が変化（[[分割]]相続から[[単独]]相続へ）

::last
一方が北朝側につけば、他方は南朝側につくという有様でした。

## 動乱による社会変動
### 守護の権限拡大
::top
[[守護]]は、鎌倉幕府が各国に置いた治安維持担当の役職です。
::
::last
南北朝の動乱で各地の武士が力を伸ばすと、武士を統括・動員するために、従来①のみであった守護の権限に、次の②～④が追加されました。
::
#### 守護の権限

- [[大犯三ヵ条]]（京都大番役の催促・{{謀叛|むほん}}人の逮捕・殺害人の逮捕をする権限）
- [[刈田狼藉|かりたろうぜき]]を==取り締まる==権限
- [[使節|しせつ遵行じゅんぎょう]]（幕府の裁判の判決を強制執行する権限）
- [[半済令|はんぜいれい]]で認められた、国内の荘園・公領の年貢を半分徴発する権限
