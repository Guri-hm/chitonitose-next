## 言論に訴える反抗

### 自由民権運動の始まり

:::top
1874年、板垣退助・後藤象二郎らが、政治的な集まり**愛国公党**を結成しました。
:::

::arrow

:::top
1874年、民撰議院設立建白書提出
:::lead
板垣・後藤らが、新政府の少数有力者による政治「<ruby>有司専制
			<rt>ゆうしせんせい</rt></ruby>」を批判し、世間の人々の意見「公論」に基づく政治実現のために、国会の開設を求めた文書主張の根拠は、五箇条の<ruby>誓文<rt>せいもん</rt></ruby>の第１条「<ruby>万機公論<rt>ばんきこうろん</rt></ruby>ニ<ruby>決
			<rt>けっ</rt></ruby>スヘシ」
:::

:::middle
愛国公党は、建白書提出という目的を遂げて解散しました。
:::

:::last
英人ブラックが創刊した新聞『[[<ruby>日新真事誌
			<rt>にっしんしんじし</rt></ruby>]]』に、民撰議院設立建白書が掲載された後、民衆参加の政治を新政府に求める[[自由民権運動]]が盛んになりました。
:::

:::gazo
![](/images/jh/img/115/4.jpg)

:::

:::gazo
![](/images/jh/img/115/5.jpg)

『日新真事誌』１月18日の記事

＊枠内に愛国公党の連名
:::

::arrow

:::top
1874年、[[板垣退助]]は郷里の==土佐で==
		**<ruby>片岡健吉<rt>かたおかけんきち</rt></ruby>**ら同志を集め、自由民権運動を進める政治結社（政社）[[立志社]]を組織しました。
:::

:::last
1875年、立志社を母体に、自由民権運動を全国的に進める政治結社[[愛国社]]が==大阪で==組織されました。
:::

### 立憲政治導入の約束

:::top
1875年、大阪会議
:::lead
新政府の指導者[[大久保利通]]、台湾出兵に反対して辞職した[[木戸孝允]]、そして[[板垣退助]]、以上３人が大阪でおこなった秘密の会談時間をかけた国会開設が約束され、結果、木戸孝允・板垣退助の２人が新政府に復帰
:::

::arrow

:::top
1875年、<ruby>漸次立憲<rt>ぜんじりっけん</rt></ruby>政体樹立の<ruby>詔<rt>みことのり
				</rt></ruby>
:::lead
「漸次ニ国家立憲ノ政体ヲ立テ」ることを示した詔立法諮問機関[[元老院]]と最高裁判所に相当する機関[[大審院]]を設置、府知事・県令からなる会議[[地方官会議]]（第１回）を開催
:::

::arrow

:::top
1876年、元老院が憲法草案の起草に着手し始めました。
:::

:::last
数次の案を経て、1880年に「日本国憲<ruby>按<rt>あん</rt></ruby>」が完成しましたが、岩倉具視らの反対で廃案となりました。
:::

### 政府批判の封殺

:::top
1875年、<ruby>讒謗律<rt>ざんぼうりつ</rt></ruby>・新聞紙条例制定
:::lead
==政府批判の抑止を目的に==、政府を誹謗する発言や批判の新聞掲載を禁じる法規
:::
