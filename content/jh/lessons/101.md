#### 歌舞伎

:::top
歌舞伎では、人気役者とともに次の作者が活躍しました。
:::

- <ruby>鶴屋南北<rt>つるやなんぼく</rt></ruby>
  :::lead
  代表作は毒殺された亡霊による復讐怪談『[[東海道<ruby>四谷<rt>よつや</rt></ruby>怪談]]』
  :::
- 河竹<ruby>黙阿弥<rt>もくあみ</rt></ruby>
  :::lead
  鶴屋南北の門下で、代表作は３人の盗賊が義兄弟となる物語
  :::

::arrow

:::top
これら歌舞伎は、<ruby>錦絵<rt>にしきえ</rt></ruby>や出版物などで全国各地に伝えられました。
:::

:::middle
村の若者が中心となって演じた歌舞伎「村芝居」が、神社の境内で開かれました。
:::

:::last
村芝居を通して、歌舞伎の衣服・化粧・道具・言葉遣いが村に伝わりました。
:::

### 信仰と結びつく娯楽

#### 寺社の催し

:::top
寺社は、修繕費や経営費を得るために、次の催しをおこないました。
:::

- 縁日
  :::lead
  特定の神仏の縁とそのご利益にあずかれる日
  :::
- 開帳
  :::lead
  寺社の秘仏・秘宝を一定期間公開すること
  :::
- 富突とみつき
  :::lead
  主催者が札を販売し、後日抽選して賞金を支払うもの

#### 行事・集まり

:::top
五節句・<ruby>彼岸会<rt>ひがんえ</rt></ruby>・<ruby>盂蘭盆<rt>うらぼん</rt></ruby>会などの行事と、次のような庶民の集まりがありました。
:::

- 日待ひまち
  :::lead
  前夜から心身を清めて寝ずに神仏を拝み、飲食をともにして日の出を待つ集まり
  :::
- <ruby>月待<rt>つきまち</rt></ruby>
  :::lead
  十三夜・十五夜など特定の月齢日に、飲食をともにして月の出を待つ集まり
  :::
- <ruby>庚申<rt>こうしん</rt></ruby>講
  :::lead
  <ruby>十干<rt>じっかん</rt></ruby>十二支で庚申に当たる日の夜、人間の体内にいる<ruby>三尸虫<rt>さんしちゅう</rt></ruby>が、人間が寝ている間に抜け出して<ruby>天帝<rt>てんてい</rt></ruby>に罪悪を報告するので、その夜は仲間と徹夜するという集まり
  :::

:::sup
庚申塔（塚）
:::lead
三尸虫を押さえ込むと信じられた青面(しょうめん)金剛などが彫られた石造物
:::
