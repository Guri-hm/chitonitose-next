### 国学

:::top
元禄期、『<ruby>万葉代匠記<rt>まんようだいしょうき</rt></ruby>』の著者<ruby>契沖<rt>けいちゅう</rt></ruby>が、日本古典の考証的研究を進めました。
:::

::arrow

:::top
18世紀、古典の研究が進み、日本古来の精神を説く学問[[国学]]が成立しました。
:::

:::last
国学は、儒教・仏教（儒仏）を外来思想として排除する傾向が強く、批判的精神も強い学問でした。
:::

## 思想の発展

### 尊王論

:::top
幕藩体制の中の天皇を王者として尊ぶ思想を[[<ruby>尊王<rt>そんのう</rt></ruby>論]]と呼びます。
:::

:::middle
尊王論は元来「朝廷を尊ぶことで、天皇の委任で政権を預かる将軍・幕府の権威を守ろう」という思想だが、後に「“王者”の天皇が“覇者（武力での支配者）”の将軍に勝る」という尊王斥覇に変質し、討幕に繋がりました。
:::

:::last
水戸藩の『[[大日本史]]』編纂事業を通して興った学問[[水戸学]]は尊王論が強く、朱子学を軸に国学・神道を総合して天皇尊崇と秩序の確立を説き、尊王斥覇の考えが主流でした。
:::

:::sup
尊王論…主張者は、諸国を遊説した高山<ruby>彦九郎<rt>ひこくろう</rt></ruby>、陵墓荒廃を嘆いた<ruby>蒲生君平<rt>がもうくんぺい</rt></ruby>、『**日本外史**』を著した<ruby>頼山陽<rt>らいさんよう</rt></ruby>
:::

:::sup
朝廷が実際に政権委任を強調したのは、1863年の14代将軍徳川<ruby>家茂<rt>いえもち</rt></ruby>の時
:::

:::

### 生活に密接な思想

#### 庶民的生活倫理

:::top
18世紀初め、京都の町人[[石田<ruby>梅岩<rt>ばいがん</rt></ruby>]]が儒学に仏教・神道を取り入れ、庶民の生き方を説く学問[[心学]]を創唱しました。
:::

:::last
また、石田梅岩は『**<ruby>都鄙<rt>とひ</rt></ruby>問答**
		』を著し、四民の人間的平等、商業の必要と利潤の正当性を訴えました。
:::

::arrow

:::top
石田梅岩の教えは、**<ruby>手島堵庵<rt>てじまとあん</rt></ruby>**・中沢<ruby>道二<rt>どうに</rt></ruby>
		よって全国に広められました。
:::

#### 社会への根本的批判

:::top
18世紀半ば、「生産労働者＝被支配者」という構造を基盤とする社会を、根本から批判し、それを改めようとする意見が現れました。
:::

:::last
例えば、<ruby>八戸<rt>はちのへ</rt></ruby>の医者[[安藤<ruby>昌益<rt>しょうえき</rt></ruby>]]は『[[自然<ruby>真営道<rt>しんえいどう</rt></ruby>]]』を著して、武士が農民から搾取せず、万人が耕作して生活する世の中を理想と主張しました。
