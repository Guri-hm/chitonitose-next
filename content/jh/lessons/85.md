## 貨幣の流通

### 貨幣の供給

:::top
同じ規格・品質で全国に通用した初めての金・銀貨幣は、1600年頃に徳川家康が発行し、[[慶長金銀]]と総称されました。
:::

:::middle
近世の初めには、輸入貨幣や<ruby>私鋳銭<rt>しちゅうせん</rt></ruby>・われ銭など悪質な銭貨も混用されましたが、17世紀前半、銭貨[[寛永通宝]]が大量に<ruby>鋳造<rt>ちゅうぞう</rt></ruby>されました。
:::

:::last
金貨・銀貨・銭貨を[[三貨]]と総称し、17世紀中頃までに全国に普及しました。
:::

::arrow

:::top
しかし、各貨幣は鋳造所・通用の仕方・主たる流通圏が異なりました。

#### 三貨

| 項目 | 金貨 | 銀貨 | 銭貨 |
| :---: | :--- | :--- | :--- |
| 鋳造所 | [[金座]] | [[銀座]] | [[銭座]] |
| 場所 | 江戸・京都 | <ruby>駿府<rt>すんぷ</rt></ruby>・伏見 ＊のちに江戸・京都 | 江戸・近江坂本 ＊のちに全国各地 |
| 統括者 | 後藤庄三郎 |  |  |
| 通用の仕方 | [[計数]]貨幣 ＊枚数で価値決定 | [[<ruby>秤量<rt>しょうりょう</rt></ruby>]]貨幣 ＊重量で価値決定 | 計数貨幣 ＊枚数で価値決定 |
| 種類 | 小判・一分金など | 丁銀・豆板銀など | 寛永通宝など |
| 流通圏 | 主に==東日本== （金遣い） | 主に==西日本== （銀遣い） | 全国で流通 |
| 単位 | 両・<ruby>分<rt>ぶ</rt></ruby>・<ruby>朱<rt>しゅ</rt></ruby> | <ruby>貫<rt>かん</rt></ruby>・<ruby>匁<rt>もんめ</rt></ruby>・<ruby>分<rt>ぶん</rt></ruby>・<ruby>厘<rt>りん</rt></ruby>・<ruby>毛<rt>もう</rt></ruby> | 貫・<ruby>文<rt>もん</rt></ruby> |

### 貨幣流通の促進

:::top
貨幣流通は、三貨間の両替や<ruby>秤量<rt>しょうりょう</rt></ruby>をおこなう商人[[両替商]]に促されました。
:::

:::last
大坂や江戸の信用・資力の大きい両替商は特に本両替と呼ばれ、幕府・藩の公金の<ruby>出納<rt>すいとう</rt></ruby>や<ruby>貸付<rt>かしつけ</rt></ruby>なども担いました。
:::

:::sup

## 三都の発達

### 三都

:::top
全国への商品流通を担う<ruby>市場<rt>しじょう</rt></ruby>を特に全国市場といいます。
:::

:::last
重要な全国市場の「江戸・大坂・京都」を三都と総称し、17世紀後半、三都は世界有数の大都市へと成長しました。
:::

#### 江戸

:::list{class="en"}
- 別称は「将軍のお膝元」
- 総人口100万を有する日本最大の消費都市
- 幕府の諸施設、大名の屋敷（藩邸はんてい）、旗本、御家人の屋敷が存在
- 町人地には商人・職人や日用（日雇ひやとい）など50万人の町人が居住
:::

:::sup
[[<ruby>札差<rt>ふださし</rt></ruby>]]という商人が、旗本･御家人の蔵米の販売代行や金貸し
:::

:::gazo
![](/images/jh/img/85/3.svg)
:::

#### 大坂

:::list{class="en"}
- 別称は「天下の台所」
- 全国の物資の集散地として栄えた商業都市
- 幕府は西日本や大坂の支配の要かなめに大坂城代・大坂町奉行を設置
- 諸藩の大名は領内の年貢米・特産物の倉庫兼取引所である蔵屋敷を置き、蔵物と総称されるこれら年貢米・特産物を販売して貨幣を獲得
  :::last
  [[蔵元]]と呼ばれる商人が蔵物の販売・保管をおこない、[[掛屋]]と呼ばれる商人がその販売代金の<ruby>出納<rt>すいとう</rt></ruby>・藩への送金を担当
  :::
- 商人が全国から大坂に集めた商品を<ruby>納屋物<rt>なやもの</rt></ruby>と呼び、江戸や全国に出荷
:::

#### 京都

:::list{class="en"}
- 天皇・公家が居住し、寺院の本寺・本山や大神社が存在
- 幕府は朝廷の権威の利用や全国の寺社の支配に京都を重視
- 幕府は朝廷・公家・寺社との関係保持や畿内一帯の支配のために、京都所司代・京都町奉行を設置
- 呉服屋・両替商など大商人の本拠地が多く存在
  :::last
  [[西陣]]織や<ruby>京染<rt>きょうぞめ</rt></ruby>・<ruby>京焼<rt>きょうやき</rt></ruby>などの手工業が発達
