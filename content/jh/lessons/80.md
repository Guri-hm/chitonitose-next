## 農林水産業

### 農業

#### 近世の農業の特徴｜16～19世紀）

:::top
近世の農業は次の特徴をもち、社会を支える基盤と考えられました。
:::

- 低い労働生産性
  :::lead
  狭い土地に対して、集中的な労働力
  :::
- 高い土地生産性
  :::lead
  狭い土地に対して、多い収穫量
  :::

#### 農業の推進と年貢米の増収

:::top
17世紀初頭以降、幕府・大名は次の施策で農業の推進を図りました。
:::

- 灌漑用水の整備
  :::lead
  [[<ruby>箱根<rt>はこね</rt></ruby>用水]]（水源：<ruby>芦<rt>あし</rt></ruby>ノ湖）・<ruby>見沼代<rt>みぬまだい</rt></ruby>用水（分水：<ruby>利根<rt>とね</rt></ruby>川）
  :::
- 新田開発
  :::lead
  <ruby>有明<rt>ありあけ</rt></ruby>海（<ruby>干潟<rt>ひがた</rt></ruby>干拓）・<ruby>下総<rt>しもうさ</rt></ruby><ruby>椿<rt>つばき</rt></ruby><ruby>海<rt>のうみ</rt></ruby>（湖沼干拓）
  :::

:::last
全国の耕地が増加し、併せて村も形成されて年貢米の増収をもたらしました。
:::

:::sup
新田開発
:::lead
17世紀末から、有力商人が開発に出資する[[町人請負新田]]が増加
:::

#### 農具

:::top
<ruby>鋤<rt>すき</rt></ruby>・<ruby>鍬<rt>くわ</rt></ruby>・鎌に加え、牛馬などの<ruby>畜力<rt>ちくりょく</rt></ruby>を利用する<ruby>犂<rt>すき</rt></ruby>なども普及しました。
:::

:::last
農具は鉄製で、城下町の職人が村を回って生産・修理しました。
:::

#### 肥料と生産物

:::top
肥料として、<ruby>刈敷<rt>かりしき</rt></ruby>・草木灰・<ruby>下肥<rt>しもごえ</rt></ruby>が基本的に==使用されました==。
:::

::arrow

:::top

:::

:::last

### 漁業

:::top
漁業は、食料供給と<ruby>魚肥<rt>ぎょひ</rt></ruby>（魚類を原料とする肥料）利用を目的に多様に発達しました。
:::

:::last

:::

## その他の産業

### 手工業

:::top
職人は町や村に住み、幕府や大名に国役として無償で技術労働を奉仕しました。
:::

:::last
職人は、百姓の負担や町人の負担（町人足役）を免除されました。
:::

::arrow

:::top
17世紀半ばの手工業の発達に併せ、職人は[[仲間]]（組合）をつくりました。
:::

:::last
都市部の職人の人口が増加し、ほとんどが借家として定着しました。
:::

::arrow

:::top
村では、零細な家内手工業が、製紙・織物・<ruby>紙漉<rt>かみすき</rt></ruby>・酒造などの分野で、早期から見られました。
:::

:::sup
紙漉による和紙の生産は、<ruby>流漉<rt>ながしすき</rt></ruby>という技術とともに全国に拡大
:::

:::gazo{size="half"}

### 鉱山業

:::top
戦国時代に鉱山開発が進み、鉱山町が形成されました。
:::

:::middle
鉱山には、佐渡金山・石見大森銀山・**住友**家が経営する別子銅山などがありました。
:::

:::middle
鉱物の中で、[[銀]]は世界でも有数の生産量を誇り、主要な輸出品でした。
:::

:::last
17世紀後半に==金銀の産出量は減り==、かわって[[銅]]が最大の輸出品となりました。
:::

:::sup
銀
:::lead
**<ruby>灰吹法<rt>はいふきほう</rt></ruby>**と呼ばれる製錬技術で生産
:::

:::sup
鉄
:::lead
足踏みふいごで空気を送るたたら精錬が、中国地方を中心に展開
:::

:::top
17世紀後半になると、次の三者による商品流通の機構が整備されました。
:::

- 問屋
  :::lead
  生産者あるいは仲立業者から商品を買い入れ、直接消費者に売る者あるいは別の仲立業者に売り渡す業者
  :::
- <ruby>仲買<rt>なかがい</rt></ruby>
  :::lead
  生産者と問屋、問屋と直接消費者に売る者の仲立業者
  :::
- 小売商人
  :::lead
  問屋・仲買から商品を買い入れ、消費者に売る者
