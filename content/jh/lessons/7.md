### 農耕に関する祭祀

:::top
弥生時代同様、古墳時代でも農耕に関する祭祀が大切でした。
:::

:::last
なかでも、春の初めに豊作を祈る[[<ruby>祈年祭
			<rt>としごいのまつり</rt></ruby>]]と、秋に収穫を祝い感謝する[[<ruby>新嘗祭
			<rt>にいなめのまつり</rt></ruby>]]が重要でした。
:::

::arrow

:::top
これらの祭祀は、「神」と呼ばれる存在に向けておこなわれました。
:::

### 神という存在

:::top
人々は[[アニミズム]]という考えをもち、あらゆるものに霊威（魂）が存在すると信じていました。
:::

:::gazo{size="twice"}
![](/images/jh/img/7/6.svg)

アニミズム
:::

::arrow

:::top
なかでも、次の２つは信仰の対象となりました。
:::

:::list{class="en"}
- 凄さを感じさせるもの（整った形の山・巨木・絶海の孤島など）
- 恐怖させるもの（日照り・氾濫する川・落雷・深い森など）
:::

:::last
信仰の対象となった霊威が「神」です。
:::

:::sup
信仰の対象となった、氏族の重要な祖先（死者）が**氏神**
:::

:::sup
目に見えない霊威が「神」になるので、当然「神」の姿はなし
:::

:::top
祭祀は基本的に、怒った神（例：稲を枯らす日照り）に祈りを捧げ、普段の神（例：耕作に適した日差し）に戻す儀式です。
:::

:::last
神は本来怒り、人々を苦しめる存在です。
:::
