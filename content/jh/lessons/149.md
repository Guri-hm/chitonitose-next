## 満州事変と満州国建国

### 満州事変の開始

#### 第２次若槻礼次郎内閣｜1931年４月～1931年12月

:::top
1931年、立憲民政党の第２次[[<ruby>若槻礼次郎<rt>わかつきれいじろう</rt></ruby>]]内閣が組織されました。
:::

:::gazo{size="half"}
![](/images/jh/img/146/1.jpg)

若槻礼次郎
:::

::arrow

:::lead
[[関東軍]]の参謀**石原<ruby>莞爾<rt>かんじ</rt></ruby>**らが、<ruby>奉天<rt>ほうてん</rt></ruby>郊外の[[<ruby>柳条<rt>りゅうじょう</rt></ruby>湖]]で、満鉄の線路を爆破し、これを国民政府（中国軍）の行為とした事件
:::

:::lead
事件を起こした理由は、国民政府が日本の権益を侵害していたからであり、事件をきっかけに満州全域を占領し、権益の強化を画策
:::

:::sup
石原莞爾
:::lead
将来の日米間の戦争を想定し、満州を占領する必要があると主張
:::

:::gazo
![](/images/jh/img/149/1.svg)

満州
:::

:::gazo{size="half"}
![](/images/jh/img/149/2.jpg)

石原莞爾
:::

::arrow

:::top
柳条湖事件後、関東軍は国民政府の軍隊と武力衝突を重ねていき、ここに<ruby>所謂<rt>いわゆる</rt></ruby>[[満州事変]]が始まりました。
:::

:::middle
満州事変に対し、若槻礼次郎内閣は戦線の不拡大を呼びかけましたが、戦線は世論・マスコミの支持を受けて拡大を続けました。
:::

:::last
若槻礼次郎内閣は、事態の収拾に自信を失って総辞職しました。
:::

### 満州国の建国

#### 犬養毅内閣｜1931年12月～1932年５月

:::top
1931年、[[立憲政友]]会の[[犬養毅]]内閣が組織されました。
:::

:::gazo{size="half"}
![](/images/jh/img/149/1.jpg)

犬養毅
:::

::arrow

:::top
1932年、第一次上海事変
:::lead
満州事変で排日運動が激化した結果、上海で日本人僧侶が殺害され、日本軍と国民政府の軍の衝突に発展した事件
:::

::arrow

:::top
1932年、満州の主要地域を占領した関東軍は、清の最後の皇帝[[<ruby>溥儀<rt>ふぎ</rt></ruby>]]を執政とし、[[満州国]]の建国を宣言させました。
:::

:::middle
アメリカは、日本の一連の動きに不承認宣言をおこないました。
:::

:::last
国際連盟理事会は、国民政府の訴えと日本の提案をうけ、リットンを団長とする調査団[[リットン調査団]]に事実調査をさせました。

## 軍部の台頭

### 国家改造運動の活発化

:::top
1930年代初め、昭和恐慌のような日本の行き詰まりが、目立ち始めました。
:::

:::last
陸海軍の青年将校や右翼は、この原因を財閥・政党などの無能と腐敗に求め、その打倒と新しい国家体制づくりを目指す運動国家改造運動を活発化させました。
:::

::arrow

:::top
青年将校や右翼は、政権樹立や要人殺害を次のように試みました。
:::

- 1931年３月、三月事件
  :::lead
  橋本<ruby>欣五郎<rt>きんごろう</rt></ruby>を中心とする陸軍青年将校の秘密結社<ruby>桜会<rt>さくらかい</rt></ruby>が、右翼指導者大川周明の協力を得て、軍部政権樹立を試み、失敗した事件
  :::
- 1931年10月、十月事件
  :::lead
  桜会と右翼が協力して、内閣の構成員殺害と軍部政権樹立を試み、失敗した事件
  :::
- 1932年２～３月、血盟団事件
  :::lead
  [[井上日召]]を中心とする右翼団体[[血盟団]]が、前蔵相[[井上準之助]]・三井合名会社理事長[[<ruby>団琢磨<rt>だんたくま</rt></ruby>]]を殺害した事件
  :::
- ④1932年５月、五・一五事件
  :::lead
  海軍青年将校が[[<ruby>犬養毅<rt>いぬかいつよし</rt></ruby>]]首相を射殺し、政党政治・政党内閣を終わらせた事件

### 政党内閣の崩壊

#### 斎藤実内閣｜1932年５月～1934年７月

:::top
財閥・政党が、青年将校や右翼の活動に脅かされていることをうけ、元老の西園寺公望は、穏健派の海軍大将[[斎藤<ruby>実<rt>まこと</rt></ruby>]]を次の首相に推薦しました。
:::

:::last
ここに、加藤高明内閣からの「憲政の常道」に終止符が打たれ、政党政治・政党内閣は終戦まで復活しませんでした。
:::

:::gazo{size="half"}
![](/images/jh/img/149/6.jpg)

斎藤実
:::

::arrow

:::top
1932年、日満議定書調印
:::lead
満州国は、同国における日本の権益の尊重と日本軍の駐屯の許可を、日本は、独立国として満州国を承認することを約した議定書
:::

:::gazo
![](/images/jh/img/149/7.jpg)

日満議定書の調印
:::

::arrow

:::top
1933年、国際連盟は、臨時総会で[[リットン調査団]]の報告書を審議し、満州国が日本の<ruby>傀儡<rt>かいらい</rt></ruby>国家であり、日本は満州国の承認を撤回すべきと勧告しました。
:::

:::middle
勧告が可決される中、日本全権代表**松岡<ruby>洋右<rt>ようすけ</rt></ruby>**は総会から退場しました。
:::

:::last
1933年３月、日本は正式に[[国際連盟]]からの脱退を通告しました。

### 満州事変の終息

#### 斎藤実内閣｜1932年５月～1934年７月

:::top
1933年、<ruby>塘沽<rt>タンクー</rt></ruby>停戦協定（日中軍事停戦協定）
:::lead
満州事変を終息させた、日本軍と国民政府間の停戦協定この協定で、国民政府は日本の満州と<ruby>熱河<rt>ねっか</rt></ruby>省の支配を事実上黙認
:::

::arrow

:::top
1934年、満州国は溥儀を皇帝とする帝政に移行しました。
