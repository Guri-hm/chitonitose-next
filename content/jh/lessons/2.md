## 自然環境の変化

### 更新世から完新世への過渡期

:::top
更新世から[[完新世]]（約１万年前～現在）に至る過渡期に、地球の気候は次第に温暖になりました。
:::

:::last
日本列島では次の３つの変化が生じました。
:::

- 動物の変化
  :::lead
  大型動物が絶滅し、==中・小型動物が多く生息==
  :::

- 植生の変化
  :::lead
  寒冷な気候に適した==針葉樹林の分布が狭まり==、東日本に落葉広葉樹林、西日本に照葉樹林が分布
  :::

- 海面の変化
  :::lead
  ==海面が上昇して==、沿岸部に入り江が増加
  :::

## 縄文時代の展開

### 時代区分の変化

:::top
地球が約１万年前に更新世から完新世に移った頃、２つの時代区分も変化しました。
:::

:::top
石器による区分

:::lead
旧石器時代から[[新石器時代]]（約１万2000年前～）
:::

:::top
土器による区分

:::lead
先土器時代から[[縄文時代]]（約１万2000年前～）
:::

### 食料獲得と生活の変化

:::top
縄文時代の食料獲得の手段には、次の３つがありました。
:::

:::top
[[狩猟]]
:::lead
変化①により、Ⓐ中・小型動物（イノシシ・ニホンシカ）が狩猟対象
:::

:::top
[[採取]]
:::lead
変化②により、Ⓑ豊富な木の実（クリ・クルミ・カシ）が採取可能
:::

:::top
[[漁労]]
:::lead
変化③により、Ⓒ食料獲得の手段に新たに追加
:::

:::last
これらは、旧石器時代と比べて食料獲得手段の多様化を意味します。
:::

### 道具の変化

:::top
打製石器とともに、[[磨製]]石器を様々な場面で利用しました。
:::

#### 狩猟

:::top
Ⓐにより、素早い中・小型動物を捕らえるために、**弓矢**・**落とし穴**を利用しました。
:::

:::last
石器としては、==矢の先に取り付ける==[[石鏃]]や、==動物の皮を剥ぐための==[[石匙]]などが利用されました。
:::

:::gazo{size="half"}
![石鏃](/images/jh/img/2/6.gif)
:::

:::gazo{size="half"}
![石匙](/images/jh/img/2/7.gif)
:::

#### 採取

:::top
Ⓑにより、採取した木の実を[[石皿・すり石]]ですり潰したり、土器で煮て<ruby>灰汁<rt>あく</rt></ruby>抜きしたりしました。
:::

:::last
土器は縄目文様をもつことから[[縄文土器]]と呼ばれ、低温（600℃程度）で焼成されたために、==厚手で黒褐色==です。
:::

![石皿・すり石](/images/jh/img/2/8.gif)
:::

:::gazo{size="half"}

:::

::arrow

:::top
縄文時代は、土器の特徴の差異から６つの時期に区分されます。
:::

:::last
草創期・早期・前期・[[中期]]・後期・[[晩期]]の６つです。
:::

:::sup
中期
:::lead
縄文土器の最盛期
:::

:::sup
晩期
:::lead
亀ヶ岡式土器に代表される精巧な土器がつくられた時期
:::

#### 漁労
