---overview---
11世紀末、上皇が直接的な父権に基づき、天皇（上皇の子や孫）の政務を代行する院政が始まりました。摂関政治は母方の祖父という立場に活かし、対する院政は実父という立場に活かすものでした。院政の開始は、後三条天皇の治世下で摂関家が力を失ったことと、そして、子孫で皇位継承を続けたいという白河上皇の熱意によるものです。
---
## 院政の成立
### 子孫の皇位継承の維持
::top
後三条天皇の死後、子の[[白河]]天皇が即位しました。
::
::last
白河は、自分の弟ではなく、自分の子孫で皇位継承を続けたいと考えました。
::

::top
1086年、白河は自ら譲位し、幼少の子[[堀河]]天皇を即位させました。
::
::last
白河上皇は、天皇の実父（あるいは実祖父）として、天皇の政務を代行する[[院政]]を開始し、子孫の皇位継承を守り続けていきました。
::
::sup
上皇の住居を院と呼び、やがて上皇自身のことも院と呼称
::
::sup
藤原氏は堀河の摂政に就いたが、既に摂関政治をする意志も力も喪失
::
::sup
薬子の変後、発言力の優劣は「上皇＜天皇」と決まったが、実の親子同士の話になれば「実父（上皇）＞実子（天皇）」に逆転
::
::sup
大江{{匡房|まさふさ}}は日記に「今の世のことは、まず上皇の御気色を仰ぐべきか」と記載
::

### 上皇と仏教
::top
院政をおこなう上皇は、世俗の制約を避けるために出家し、[[法皇]]と呼ばれました。
::
::last
白河法皇が建立した[[法勝寺]]をはじめ、「勝」の字をもつ６寺院を[[六勝寺]]と総称します。
::
::sup
[[熊野]]は、浄土につながる場所として信仰をあつめ、上皇もたびたび参詣
::

#### 現代語役
::top
白河院。…古来の儀式を復活させて嵯峨野行幸などもなされた。また白河に法勝寺を建立し、九重塔なども昔建てられた御願寺の規模をこえたものになされ、比類ないほどに壮麗におつくりになった。天下を治められること十四年。太子に譲位して太上天皇となり、政治をはじめて院でおとりになられた。のちに出家なされても、そのまま崩御まで政治をおとりになった。…白河上皇の時代には（上皇が）院で政務をおとりになっていたので、摂関はただその職についたというだけで実権の伴わないものになってしまった。しかしこの時代から政治の古来の姿（摂関政治）が一変したといってよいのではないだろうか。摂関が政治を担当したとはいっても、国家の行政は天皇の宣旨や太政官符を下して行っていたのだが、白河上皇の時代から院宣や院庁下文を重んじたことによって、天皇までもがただ位につかれたにすぎない名ばかりのものになってしまった。世も末の姿というべきではないだろうか。
::
