## 恐慌からの脱出

### 金輸出再禁止と円安利用

#### 犬養毅内閣｜1931年12月～1932年５月

:::top
1931年に成立した[[犬養毅]]内閣の蔵相[[高橋<ruby>是清<rt>これきよ</rt></ruby>]]は、再度の[[金輸出禁止]]と円の金<ruby>兌換<rt>だかん</rt></ruby>停止を断行しました。
:::

:::last
結果、日本経済は金本位制ではなく、政府が通貨発行額を管理・統制する[[管理通貨]]制度に移行しました。
:::

### ブロック経済と日本非難

:::top
世界恐慌からの脱出のため、欧米は排他的なブロック経済圏を形成しました。
:::

:::last
例えばイギリスは、本国と植民地以外からの輸入に高い関税をかけ、商品の生産・消費を本国と植民地で完結させる政策をとりました。
:::

::arrow

:::top
円安のために、日本は欧米のブロック経済圏にも輸出を拡大できました。
:::

:::last
欧米は、これを国ぐるみの投げ売り（ソーシャル＝ダンピング）と非難しました。
:::

### 恐慌脱出と産業構造の変化

:::top
日本は、輸出の躍進や財政の膨張に支えられ、1933年、工業全体の生産額が、世界恐慌以前の生産水準に戻りました。
:::

::arrow

:::top
1933年、金属・機械・化学工業合計の生産額が、繊維工業を上回りました。
:::

:::last
1938年、それらの工業は、工業全体の生産額の50％を超え、産業構造が軽工業中心から重化学工業中心へと変化しました。
:::

### 農村の救済

#### 斎藤実内閣｜1932年５月～1934年７月

:::top
[[昭和]]恐慌による農村の困窮を背景に、農村救済請願運動が高まりました。
:::

:::middle
**斎藤実**内閣は、1932年から**<ruby>時局匡救<rt>じきょくきょうきゅう</rt></ruby>事業**
		と称して公共土木事業をおこない、農民を日雇い労働に雇用して現金収入を得させました。
:::

:::last
さらに同内閣は、農村の窮乏を農村自身の力で救済するため、[[農山漁村経済更生運動]]を進め、産業組合の拡充して農民の結束を図りました。
:::

## 思想の収束

### 転向

:::top
満州事変を契機に、日本国内の世論・マスコミは戦争熱に浮かされました。
:::

:::last
弾圧と相まって、社会主義・共産主義者がその思想を捨てる[[転向]]が発生しました。
:::

#### 社会主義から

:::top
1932年、赤松<ruby>克麿<rt>かつまろ</rt></ruby>が中心となって日本国家社会党を結成しました。
:::

:::middle
この党は、資本主義の問題を国家政策（戦争など）で解決する国家社会主義を掲げました。
:::

:::last
従来の社会主義勢力は、社会大衆党を結成したが、やがて国家社会主義に傾きました。
:::

:::sup
国家社会主義
:::lead
従来の社会主義と違い、現国家体制（天皇制）の否定しない立場
:::

:::gazo
![](/images/jh/img/150/5.svg)

社会主義の分裂
:::

::arrow

:::top
1937年、社会主義を守り続けた鈴木茂三郎らの日本無産党も、活動を停止しました。
:::

#### 共産主義から

:::top
1933年、獄中の日本共産党幹部の佐野学・鍋山貞親が、現国家体制を支持する転向声明書を発表しました。
:::

:::last
この声明を契機に、獄中の日本共産党党員の大半が転向しました。
:::

:::gazo
![](/images/jh/img/150/3.jpg)

転向を伝える新聞

:::top
社会主義・共産主義だけでなく、自由主義・民主主義的な学問への弾圧も起こりました。
:::
