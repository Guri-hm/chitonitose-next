## 商業の発達

### 近世初期の商業と交通整備

:::top
16世紀末～17世紀前半には、交通整備が未発達でした。
:::

:::last
豪商と呼ばれる堺・京都・博多・長崎・<ruby>敦賀<rt>つるが</rt></ruby>などの一部の商人が、地域格差・私有の船や蔵・朱印船貿易を利用して巨大な富を得ました。
:::

::arrow

:::top
江戸幕府が交通を整備し、また、貿易を制限すると、三都や城下町の商人・<ruby>問屋<rt>といや</rt></ruby>が商業・流通の中心となりました。
:::

:::middle
問屋は、生産地の<ruby>仲買<rt>なかがい</rt></ruby>から商品を引き受け、都市の仲買に<ruby>卸<rt>おろし</rt></ruby><ruby>売
		<rt>う</rt></ruby>りしました。
:::

:::last
一部の問屋は、産地に資金・原料を貸与して生産させる[[問屋制家内工業]]を始めました。
:::

### 商業上の組織と営業税

:::top
問屋は、「<ruby>海損<rt>かいそん</rt></ruby>の共同保障・荷物運送の安全・流通の独占」を目的に、多様な職種からなる仲間（組合とも）という連合組織をつくりました。
:::

:::middle
例えば、江戸の[[<ruby>十組問屋<rt>とくみといや</rt></ruby>]]や大坂の[[<ruby>二十四組問屋<rt>にじゅうよくみといや</rt></ruby>]]が有名です。
:::

:::middle
仲間は仲間掟を定めて営業の独占を図りました。
:::

:::last
幕府は仲間を公認するかわりに、営業税[[<ruby>運上
			<rt>うんじょう</rt></ruby>]]・[[<ruby>冥加<rt>みょうが</rt></ruby>]]を課しました。
:::

:::sup
株
:::lead
幕府に公認された営業の独占権
:::

:::sup
株仲間
:::lead
株をもつ問屋・商人らの仲間
:::

::arrow

:::top
幕府は運上・冥加の増収で財源を支えようとしました。
:::

:::last
18世紀以降、特に田沼時代は問屋以外の商人・職人らの仲間が広く公認されました。
:::
