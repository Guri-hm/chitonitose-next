## 歴史書と地誌

### ２つの歴史書

#### 歴史書①

:::top
６世紀、「帝紀」（大王の系譜）・「旧辞」（朝廷の伝承）が編纂されたと考えられます。
:::

:::last
天武天皇は、豪族のもつ「帝紀」「旧辞」の写本に誤りが多く、また、不必要な部分を削って一書にまとめるべきとし、訂正・集成に臨みました。
:::

:::sup
写本の過程での誤りに加え、「帝紀」「旧辞」は難解で誤読も多発
:::

::arrow

:::top
712年、『古事記』
:::lead

				<rt>ひえだのあれ</rt></ruby>]]が「帝紀」「旧辞」の内容を正しく<ruby>誦習<rt>しょうしゅう</rt></ruby>し、それを[[<ruby>太安万侶<rt>おおのやすまろ</rt></ruby>]]が整理しながら漢字で筆録
		物語的要素が強く、神話・伝承も含んだ、天皇家による国土統治の歴史書

### 地誌

:::top
713年、諸国の国司に、国の地誌『[[<ruby>風土記
			<rt>ふどき</rt></ruby>]]』の編纂が命じられました。
:::

:::last
国司は、国内の地名とその由来・動植物・伝承をまとめました。
:::

:::sup
まとめる際に、国・郡・里の名称は、縁起の良い２字に改名（例：泉→<ruby>和泉<rt>いずみ</rt></ruby>）
:::

::arrow

:::top
現在、５カ国（<ruby>常陸<rt>ひたち</rt></ruby>・<ruby>出雲<rt>いずも</rt></ruby>・<ruby>播磨<rt>はりま</rt></ruby>・<ruby>豊後
			
		<rt>ぶんご</rt></ruby>・<ruby>肥前<rt>ひぜん</rt></ruby>の『風土記』の写本が残っています。
:::

:::last
この中で、欠損部分がない（完本である）のは、『出雲国風土記』です。
:::

### 教育制度

:::top
教育機関は律令に従って整えられ、次の２種類に大別できます。
:::

- 大学
  :::lead
  **式部省**の管轄下で、==中央（都）に==置かれた教育機関
  :::
  :::lead
  一定以上の位階の者の子弟が優先して入学
  :::
- 国学
  :::lead
  国司の管轄下で、==地方（国）に==置かれた教育機関
  :::
  :::lead
  **郡司**の子弟が優先して入学
  :::

::arrow

:::top
教科には、例えば次の３種類がありました。
:::

- 明経道
  :::lead
  五経や『論語』などの儒教の経典を学ぶ教科
  :::
- 明法道
  :::lead
  律令などの法律を学ぶ教科
  :::
- 紀伝道
  :::lead
  漢詩・歴史を学ぶ教科（９世紀に重視）
  :::

:::last
これらの教科の修了後には位階がもらえました。
:::
