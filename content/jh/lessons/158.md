---overview---
絶対国防圏の崩壊後、サイパン島から飛来する米軍機の本土空襲が激化しました。衣食・その他の物資が不足していたところに、本土空襲が加わり、人々の厭戦気分が徐々に醸成されました。連合国は、戦争の裏で対日処理を討議し、ポツダム宣言という形で結実させました。原爆の投下を経て、日本は宣言を受諾し、敗戦を認めました。
---
## 国民生活の崩壊
### 総力戦
::top
開戦後、日本は民需生産の工場を軍需工場へ転用し、軍需生産を優先しました。
::
::middle
また、国民に対しては生活の切り詰めと兵力・労働力の動員をおこないました。
::
::last
1943年、学生を動員する次の２策が実施され始めました。
::

- [[学徒出陣]]
  :::lead
  在学中の徴兵猶予制を廃止し、文科系学生を軍に徴集
  :::
- [[勤労動員]]
  :::lead
  学生・生徒や[[女子{{挺身隊|ていしんたい}}]]の女性を軍需工場で動員
  :::

1942年、物資不足のため、衣料の[[切符制]]（総合切符制）も始まりました。

::

::
### 空襲の激化
::top
1944年の[[サイパン島]]陥落以降、同島からの米軍機で、==本土空襲が本格化しました==。
::
::middle
空襲は、軍需工場の破壊から戦意喪失のための都市爆撃へと次第に変わりました。
::
::last
都市で、防空壕の掘削、工場の移転、生徒の集団疎開[[学童疎開]]が始まりました。
::
::gazo
::
