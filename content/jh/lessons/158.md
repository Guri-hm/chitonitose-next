## 国民生活の崩壊

### 総力戦

:::top
開戦後、日本は民需生産の工場を軍需工場へ転用し、軍需生産を優先しました。
:::

:::middle
また、国民に対しては生活の切り詰めと兵力・労働力の動員をおこないました。
:::

:::last
1943年、学生を動員する次の２策が実施され始めました。
:::

- 学徒出陣
  :::lead
  在学中の徴兵猶予制を廃止し、文科系学生を軍に徴集
  :::
- 勤労動員
  :::lead
  学生・生徒や[[女子<ruby>挺身隊
  					<rt>ていしんたい</rt></ruby>]]の女性を軍需工場で動員
  :::

:::sup
女子挺身隊
:::lead
労働力不足を補うため、25歳以下の==未婚の==女性で組織した勤労団体

### 品不足

:::top
制海・制空権の喪失で、軍需生産に必要な鉄鉱石・石炭・石油も欠乏しました。
:::

:::gazo
![](/images/jh/img/158/4.jpg)

代用燃料車

＊木炭・薪を燃料に利用
:::

::arrow

:::top
1942年、物資不足のため、衣料の[[切符制]]（総合切符制）も始まりました。
:::

:::last
また、米の配給量が減り、イモ・小麦などの代用品の割合が増えました。
:::

:::sup
総合切符制
:::lead
切符制と違い、点数の載る切符を足し合わせて、物資と交換する制度
:::

:::sup
1942年、米穀確保を目的に[[食糧管理法]]が制定され、戦後米作保護に機能
:::

### 空襲の激化

:::top
1944年の[[サイパン島]]陥落以降、同島からの米軍機で、==本土空襲が本格化しました==。
:::

:::middle
空襲は、軍需工場の破壊から戦意喪失のための都市爆撃へと次第に変わりました。
:::

:::last
都市で、防空壕の掘削、工場の移転、生徒の集団疎開[[学童疎開]]が始まりました。
:::

### 敗戦

#### 鈴木貫太郎内閣｜1945年４月～1945年８月

:::top

			<rt>かんたろう</rt></ruby>]]が内閣を組織しました。
:::

:::gazo{size="half"}
![](/images/jh/img/158/15.jpg)

鈴木貫太郎
:::

::arrow

:::top
1945年５月、ドイツが無条件降伏しました（イタリアの降伏は1943年９月）。
:::

:::last
日本の軍部は本土決戦を叫び、鈴木貫太郎内閣はソ連に和平交渉の仲介を計画しました。
:::

::arrow

:::top
アメリカは、７月のポツダム宣言に対する日本の反応を「拒否」と捉えました。
:::

:::last
1945年８月、米国は、<ruby>広島
			
		<rt>８月６日</rt></ruby>・<ruby>長崎
			
		<rt>８月９日</rt></ruby>に[[原子爆弾]]を投下しました。
:::

:::sup
同月、ソ連が日ソ中立条約を無視して日本に宣戦し、満州・朝鮮・南樺太に侵入

:::

:::last
８月15日、天皇のラジオ放送「玉音放送」で戦争終結が全国民に発表されました。
