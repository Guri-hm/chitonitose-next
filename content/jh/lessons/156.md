## 開戦回避の模索

### 日米交渉の開始

#### 第２次近衛文麿内閣｜1940年７月～1941年７月

:::top
1940年の日独伊三国同盟調印は、アメリカの対日姿勢をより硬化させました。
:::

:::last
第２次[[<ruby>近衛文麿
			<rt>このえふみまろ</rt></ruby>]]内閣は、日米衝突の回避に、次の２策を講じました。
:::

- 1941年４～11月、日米交渉
  :::lead
  駐米大使**野村吉三郎**と米国務長官ハルとの間の協議で、日米衝突の回避を目指す国交調整
  :::
- 1941年４月、日ソ中立条約調印（交渉：外相松岡<ruby>洋右<rt>ようすけ</rt></ruby>）
  :::lead
  日本とソ連が相互の中立友好と不可侵を約した条約
  :::
  :::lead
  独・伊のほかに、ソ連を味方につけ、アメリカの対日態度の軟化に期待
  :::
  :::lead
  また、南進政策の一層の推進のため、北方の安全を確保
  :::

## 対米戦の開戦

### ９月６日の御前会議決定

#### 第３次近衛文麿内閣｜1941年７月～1941年10月

:::top
1941年９月６日、天皇臨席の下の最高会議である御前会議が開かれました。
:::

:::last
帝国国策遂行要領が作成され、日米交渉の期限を10月上旬とすること、交渉失敗時に対米（およびイギリス・オランダ）開戦に踏み切ることが決まりました。
:::

::arrow

:::top
日米交渉は、米が日本軍の中国からの全面撤退などを要求したため、妥協点を見い出せないまま10月半ばを迎えました。
:::

:::last
妥協を主張する近衛文麿首相と、開戦を主張する陸相東条<ruby>英機<rt>ひでき</rt></ruby>が対立し、1941年10月、近衛内閣は総辞職した。
:::

### 交渉の決裂と開戦

#### 東条英機内閣｜1941年10月～1944年７月

:::top
内大臣<ruby>木戸幸一<rt>きどこういち</rt></ruby>は、９月６日の御前会議決定を白紙に戻すこと、つまり日米交渉の継続を条件として、[[東条<ruby>英機<rt>ひでき</rt></ruby>]]を首相に推挙しました。
:::

:::last
首相が陸相・内相を兼任する形で、東条英機が内閣を組織しました。
:::

:::sup
最後の元老である西園寺公望の死後、首相選定は木戸<ruby>孝允<rt>たかよし</rt></ruby>の孫の幸一を中心に、首相経験者で構成される重臣会議の合議で決定
:::

:::gazo
![](/images/jh/img/156/6.jpg)

東条英機内閣
:::

::arrow

:::top
1941年11月26日、米からの交渉文書（通称：[[ハル＝ノート]]）が、日本に提示されました。
:::

:::middle
ハル＝ノートは、中国・仏印からの全面的無条件撤退、満州国・<ruby>汪兆銘
		<rt>おうちょうめい</rt></ruby>政権の否認など、日本に満州事変以前の状態への復帰を要求する厳しい内容でした。
:::

:::last
ハル＝ノートは、これまでの妥協案を全く認めず、日本に交渉失敗を確信させました。
:::

::arrow

:::top
1941年12月１日、天皇臨席の下の最高会議である御前会議が開かれ、日米交渉の失敗の判断と米・英への開戦が決まりました。
:::

:::middle
1941年12月８日、日本陸軍が英領**マレー半島**に奇襲上陸し、日本海軍がハワイの[[真珠湾]]を奇襲攻撃して、日本は米・英に宣戦布告しました。
:::

:::last
日本は、日中戦争を含めた戦争の呼称を「[[大東亜戦争]]」と決めました。
:::
