## ヤマト政権の体制

### 大王の支配体制（５世紀後半、雄略天皇の代～）

:::top
５世紀頃よりヤマト政権の首長（＝大和地方のクニの首長）は、[[大王]]と呼ばれました。
:::

:::middle
ヤマト政権は連合であり、大王と各地のクニの首長（豪族）の間に、支配・被支配の関係はそれほどありませんでした。
:::

:::last
５世紀後半～６世紀、大王は氏姓制度をつくり上げ、大王が頂点となった支配体制を確立していきました。
:::

:::gazo
![](/images/jh/img/8/1.svg)

大王の立ち位置
:::

#### 氏（ウジ）

:::top
豪族を血縁関係ごとに[[<ruby>氏<rt>ウジ</rt></ruby>]]と呼ばれる集団に編成しました。
:::

:::middle
例えば、蘇我氏・大伴氏・<ruby>物部<rt>もののべ</rt></ruby>氏・<ruby>筑紫<rt>つくし</rt></ruby>氏・<ruby>東漢
		<rt>やまとのあや</rt></ruby>氏などです。
:::

:::last
氏を与えてやった大王は、与えてもらえた豪族より地位が高くなります。
:::

:::sup
氏の命名は職・地名に由来
:::

:::sup
氏の代表者を**<ruby>氏上<rt>うじのかみ</rt></ruby>**と呼称
:::

:::gazo
![](/images/jh/img/8/2.svg)

氏による支配関係
:::

#### 姓（カバネ）

:::top
氏ごとの性格・勢力に応じて[[<ruby>姓
			<rt>カバネ</rt></ruby>]]を与え、氏を序列化しました。
:::

:::last
姓には、例えば[[臣]]・[[連]]・[[君]]・[[直]]などがありました。
:::

:::gazo
![](/images/jh/img/8/3.svg)

姓による序列化
:::

::arrow

:::top
氏姓の組合せは、例えば蘇我臣、大伴連・物部連、筑紫君などです。
:::

### ヤマト政権の政治体制（６世紀初め～）

#### 中央

:::top
臣・連の姓をもつ氏から[[大臣]]・[[大連]]が選ばれて、ヤマト政権の中央の政務を担当しました。
:::

:::last
大臣・大連の下に[[伴造]]を置き、==伴や部（総称：品部）を統率させて==、軍事・財政・祭祀などを分担させました。
:::

:::sup
伴
:::lead
一般役人のような職
:::

:::sup
大王の一族に直属して奉仕する人々を[[名代]]・[[子代]]と呼称
:::

:::gazo{size="half"}
![](/images/jh/img/8/4.svg)

中央の組織
:::

#### 地方

:::top
==現地の==豪族を[[<ruby>国造
			<rt>くにのみやつこ</rt></ruby>]]に任じ、クニの支配権を保障する代わりにヤマト政権への奉仕を求めました。
:::

:::sup
豪族は私有民[[<ruby>部曲<rt>かきべ</rt></ruby>]]を、支配拠点の田地[[<ruby>田荘<rt>たどころ</rt></ruby>]]の耕作に使役
:::

:::sup
氏の家々で奴隷として奉仕する人々を[[<ruby>奴婢<rt>ぬひ</rt></ruby>]]と呼称
:::

:::sup
国造の下には[[<ruby>県主<rt>あがたぬし</rt></ruby>]]が置かれ、小範囲を支配
:::

## ヤマト政権の国際関係

:::top

### 仏教公伝（６世紀中頃、欽明天皇の代）

:::top
６世紀中頃、百済・新羅による加耶諸国の分割が著しく進み、[[大伴金村]]は一連の責任を糾弾されて失脚しました。
:::

:::last
中央の政務担当は、大連の**物部尾輿**と、渡来人と結んで台頭した大臣の[[<ruby>蘇我稲目<rt>そがのいなめ</rt></ruby>]]の２強になりました。
:::

:::sup
加耶諸国は562年に滅亡
:::

:::sup
蘇我氏はヤマト政権の財政面を担い、財を収める三蔵（斎蔵・内蔵・大蔵）も管理
:::

::arrow

:::top
538（552）年、[[仏教]]が百済の[[聖王]]（[[聖明王]]）によって公伝しました。
:::

:::last
大王（[[欽明]]天皇）は、大臣・大連たちに仏教を受容するかどうかを意見させました。
:::

:::

:::last
対立は物部尾輿の子[[物部守屋]]と蘇我稲目の子[[蘇我馬子]]の間でも継続しました。
