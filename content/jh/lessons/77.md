## 学問と文芸

### 朱子学の受容

:::top
鎌倉時代、儒学の一体系で<ruby>朱熹<rt>しゅき</rt></ruby>（朱子）に構築された朱子学が、日本へ伝来しました。
:::

:::middle
朱子学は<ruby>君臣<rt>くんしん</rt></ruby>・<ruby>父子<rt>ふし</rt></ruby>の別をわきまえ、礼節・上下秩序を重んじる学問です。
:::

:::last
教えのなかで、特に[[大義名分論]]は後醍醐天皇に影響を与えました。
:::

:::sup
大義名分論
:::lead
君臣の間には不変の秩序があり、これは守られるべきとする考え
:::

::arrow

:::top
室町時代、[[桂庵玄樹]]は肥後の菊池氏・薩摩の島津氏に招かれて朱子学を講じ、[[南村梅軒]]は土佐で朱子学を講じて[[谷時中]]が確立した南学の基礎を築きました。
:::

:::sup
桂庵玄樹
:::lead
[[薩南学派]]の祖で、薩摩で<ruby>朱熹<rt>しゅき
				</rt></ruby>の『大学章句』を刊行
:::

::arrow

:::top
安土・桃山時代から江戸時代初頭、禅僧であった[[藤原<ruby>惺窩
			<rt>せいか</rt></ruby>]]が、朱子学の教育に努めました。
:::

:::middle
江戸幕府・諸藩は、現状の秩序維持を目的に朱子学の受容を決めました。
:::

:::middle
藤原惺窩に推挙されたことで、その門人[[林<ruby>羅山
			<rt>らざん</rt></ruby>]]は徳川家康に用いられ、秀忠・家光・家綱と４代の<ruby>侍講<rt>じこう</rt></ruby>を務めました。
:::

:::middle
林羅山は幕府の命で、子の<ruby>鵞峰<rt>がほう</rt></ruby>とともに歴史書『[[本朝<ruby>通鑑
				
			<rt>つがん</rt></ruby>]]』を完成させました。
:::

:::last
林羅山の子孫は[[林家]]と呼ばれ、彼の子孫は代々儒者として幕府に仕えました。

### 朱子学の受容の問題

:::top
朱子学は上下秩序を重んじる学問です。
:::

:::middle
大義名分論に厳密に従えば、幕府が朝廷を差し置いて実権を握る状態は本来否定されます。
:::

:::last
江戸幕府はこの問題を棚に上げて朱子学を受容しました。
:::

::arrow

:::top
江戸時代を通して朱子学の研究が深まると、この問題が表出しました。
:::

:::last
幕末、朱子学を基盤に天皇家を尊ぶ思想「<ruby>尊王<rt>そんのう</rt></ruby>論」が強まり、倒幕運動へ発展しました。
:::

:::last
権現造の代表例は、徳川家康を祀る霊廟建築[[日光東照宮]]です。

:::top

:::last
