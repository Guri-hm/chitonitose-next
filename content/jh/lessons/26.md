---overview---
701年、大宝律令の完成で日本は律令国家として再編されました。当初律令は上手く機能していました。国司はやる気に満ち、勧農に励んで口分田からの収穫を増やそうとしました。しかし、月日を経るなかで徐々に国司は不正を働き、律令が社会の変化に対応しきれなくなりました。朝廷は対応に追われるが、次第に修正不可能になっていきました。
---
## 律令の問題と不正の増加
### 有力農民の登場
::top
律令制では、班田収授法に基づき口分田が配給されました。
::
::middle
当然口分田ごとの土地の質は異なります。
::
::last
良質の口分田を得た農民は、豊かな収穫によって有力農民へと成長しました。
::
::gazo
![](img/26/1.svg){.half}
有力農民の登場
::
::last
国府は[[出挙|すいこ]]という税に頼り、財源を増加させていきました。
::
::sup
出挙
:::lead
国府の倉で保存する稲穂（米の種がついた状態）を、春に貸付け、秋に利息と一緒に徴収
::
::gazo
![](img/26/2.svg)
出挙の仕組み
::
::top
一部の国司は上限を超えて稲穂を貸付け、利息を着服しました。
::
::middle
時に国司は上限超えの分を回収できず、国府の倉の出納に誤差が生じました。
::
::last
国司は上限超えの不正や出納の誤差を誤魔化せず、任期６年を終えて新任の国司と交代する時に揉めました。
::
---arrow---
::top
国司は交代時に新任の国司から{{解由状|げゆじょう}}をもらい、仕事に不備のなかったことを証明する必要がありました。
::
::last
不正・誤差の増加で、解由状のやり取りが滞ってしまいました。
::
::gazo
![](img/26/3.svg)
新任の国司と前任の国司
::
::top
国司の交代を円滑に進めさせるため、[[桓武]]天皇は[[勘解由使|かげゆし]]を設けました。
::
::last
勘解由使は第三者として前任と新任の揉め事を解決したり、新任が前任の不正を見逃していないか調査したりしました。
::
### 戸籍の崩壊
::top
農民は調・庸・雑徭の負担を少なくし、また、口分田を長く手元に残したいと考えました。
::
::last
国司は農民に調・庸の一部を貢がせ、見返りに次のような[[偽籍]]（虚偽の申告）を許しました。
::

- 男子を女子と偽る…女子には調・庸・雑徭の負担なし
- 死者を高年齢の生者と偽る…65歳超えに調・庸・雑徭の負担なし、口分田の収公延期

::sup
調・庸は、中央（朝廷）財源として納められる税
::
---arrow---
::top
偽籍による調・庸などの未納で、中央財源は減っていきました。
::
::gazo
![](img/26/4.svg){.half}
税の流れ
::
#### 負担の縮小
::top
桓武天皇は農民の偽籍を減らすために、次の３点に取り組みました。
::

- 班田収授を６年１回から==12年１回に変更==
- 出挙の利息を５割から３割に変更
- 雑徭の期間を年間60日から30日に半減

#### 田地の直営・収集
::top
稲穂による財源確保のため、田地を収集・直営する動きが見られました。
::

- 823年、==大宰府に==直営の[[公営田]]を設置
- 879年、==畿内に==中央政府直営の[[官田]]を畿内に設置
- 天皇が直営の勅旨田を所有
- 院宮王臣家が田地を収集

::sup
院宮王臣家
:::lead
天皇と親近な皇族や貴族の呼称
::

## 律令の再整備
### 律令の補完
::top
基本的に律令の条文には手が加えられず、必要に応じて次のような法令で補完しました。
::

- [[格|きゃく]]
  :::lead
  律令の規定を補足・修正する法令
  :::
- [[式]]
  :::lead
  律令の或る規定をさらに細かく指示する法令
  :::

---arrow---
::top
嵯峨天皇は格・式をまとめた[[{{弘仁|こうにん}}格式]]を編纂させました。
::
::middle
９～10世紀、{{清和|せいわ}}天皇が[[{{貞観|じょうがん}}格式]]、{{醍醐|だいご}}天皇が[[{{延喜|えんぎ }}格式]]を編纂させました。
::
::last
弘仁格式・貞観格式・延喜格式は[[三代格式]]と総称されました。
::
::sup
語呂
:::lead
嵯峨は盛（清和）大（醍醐）に、唐風を向（弘仁）上（貞観）へ（延喜）
::
