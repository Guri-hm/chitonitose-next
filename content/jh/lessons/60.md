## 生産活動の発達

### 農業の発達

:::list{class="en"}
- 農業技術（水車での灌漑・排水施設）の発達二毛作が各地に広がり、畿内では三毛作が開始されました。
- 水稲の品種改良成熟速度の異なる「早稲わせ・中稲なかて・晩稲おくて」が普及しました。
- 外来米の普及鎌倉時代に輸入された多収穫米大唐米が、西国で普及しました。
- 肥料の種類増加従来からの刈敷・草木灰に加えて、人糞尿を肥料にした<ruby>下肥
				<rt>しもごえ</rt></ruby>が広く利用されました。
- 加工原料の栽培藍あい・漆・茶・<ruby>楮
				<rt>こうぞ</rt></ruby>・桑などの栽培が盛んになりました。
:::

### 貨幣経済の発達

:::top
貨幣には宋銭と、新たに日明貿易で流入した[[明銭]]が使用されました。
:::

:::last
明銭には、最も多く使用された[[永楽通宝]]の他に、洪武通宝がありました。
:::

:::gazo{size="half"}
![](/images/jh/img/56/4.jpg)

明銭（永楽通宝）
:::

::arrow

:::top
農村でも年貢・公事・夫役を銭納することが多くなりました。
:::

:::middle
貨幣のさらなる需要増大から、国内の民間業者が鋳造した粗悪な[[<ruby>私鋳銭
			<rt>しちゅうせん</rt></ruby>]]も、市場を流通するようになりました。
:::

:::middle
取引で粗悪な貨幣を嫌い、良質な貨幣を選別する[[<ruby>撰銭
			<rt>えりぜに</rt></ruby>]]がよく行われました。
:::

:::last
選別は大変手間で、時にいざこざを起こして円滑な流通を妨げました。
:::

:::gazo{size="half"}
![](/images/jh/img/60/8.jpg)

私鋳銭
:::

::arrow

:::top
撰銭を規制するため、たびたび[[撰銭令]]が出され、良銭・悪銭の基準や混入比率などが定められました。
:::

### 運輸業の発達

#### 水上運輸

:::top
鎌倉時代の問丸が一般の卸売りも兼ねて発展した[[<ruby>問屋
			<rt>といや</rt></ruby>]]が増加しました。
:::

:::sup
問丸
:::lead

:::

:::top
京都までの輸送では、馬上に荷物を載せて運搬する[[<ruby>馬借
			<rt>ばしゃく</rt></ruby>]]や、荷車を牛馬に引かせる[[<ruby>車借
			<rt>しゃしゃく</rt></ruby>]]などの業者が活躍しました。

馬借・車借
