---overview---
田中角栄内閣以後も、自民党の一党支配は崩れませんでした。1989年１月、昭和天皇が崩御し、元号は平成と改められましたが、この間の日本の政治は、経済・社会の安定に支えられて、大きな変化を伴うことなく、小さな波瀾を繰り返しました。時間的にそれほど隔たりをもたないこの時代の評価は、後世に評価を委ねなければなりません。
---
## 1972～89年の内閣
### 腐敗政治の露呈
#### 第１・２次田中角栄内閣｜1972年７月～1974年12月
::top
1972年、[[田中角栄]]が==「日本列島改造論」を掲げて==内閣を組織した。
::
::gazo
![](img/169/1.jpg){.half}
田中角栄
::
::top
1972年、[[日中共同声明]]調印
:::lead
、[[日中共同声明]]調印日本側代表の田中角栄と中国側代表の**{{周恩来|しゅうおんらい}}**が、北京で日中国交正常化を声明
::
::top
交通網整備などの列島改造政策は、地価の高騰を引き起こしました。
::
::middle
これに[[第１次石油危機]]による原油価格の高踏が重なり、[[狂乱物価]]と呼ばれる激しい物価上昇が発生しました。
::
::last
政府が物価上昇抑制に金融を引き締め、1974年、戦後初のマイナス成長となりました。
::
::top
首相の政治資金調達をめぐる疑惑[[金脈問題]]が明るみに出て、1974年、田中角栄内閣は総辞職しました。
::

## 経済大国「日本」
### バブル経済
::top
第１次石油危機後、日本は省エネ・減量経営を図り、第２次石油危機も乗り切りました。
::
::last
1980年代、日本は欧米と比べると高い成長率を維持し、安定成長を続けました。
::
---arrow---
::top
日本の貿易黒字の拡大は、欧米との間に[[貿易摩擦]]を起こしました。
::
::last
アメリカでは、日本製自動車を叩き壊すなどのジャパン＝バッシングも起こりました。
::
::gazo
![](img/169/8.jpg){.half}
「平成」の発表
::
::top
1985年の[[プラザ合意]]のドル高是正・円高加速で、日本の輸出は減退しました。
::
::middle
しかし、日本は内需の拡大に支えられ、景気を回復しました。
::
::last
内需景気は、地価・株価の暴騰とともに進行し、[[バブル経済]]と呼ばれました。
