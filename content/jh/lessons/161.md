---overview---
ＧＨＱは、「憲法の自由主義化」のために、内閣に憲法の改正を指示しました。しかし、提出された改正試案が、天皇の統治権を認めるものであったため、ＧＨＱはこれを拒否し、マッカーサー草案を急ぎ作成しました。この草案が和訳・修正されて政府原案になり、さらに衆議院・貴族院で修正・可決されて「日本国憲法」になりました。
---
## 憲法の制定
### 政党の復活と結成
#### 幣原喜重郎内閣｜1945年10月～1946年５月
::top
1945年10月、ＧＨＱの指示で出獄した徳田球一を中心に、[[日本共産党]]が合法政党として活動を開始しました。
::
::last
ＧＨＱの民主化政策が進む中、次の各政党も復活ないし結成されました。
::

- **日本社会党**
  :::lead
  旧無産政党を統合した政党
  :::
- [[日本自由党]]
  :::lead
  旧==立憲政友会系で==翼賛選挙の非推薦議員を中心に結成
  :::
- [[日本進歩党]]
  :::lead
  旧==立憲民政党系で==大日本政治会に属した議員を中心に結成
  :::
- **日本協同党**
  :::lead
  労資協調を掲げる政党で、解散・合同を経て国民協同党に発展
  :::

### 憲法の改正
#### 幣原喜重郎内閣｜1945年10月～1946年５月
::top
1945年10月、[[幣原喜重郎|しではらきじゅうろう]]内閣はＧＨＱに憲法改正を指示されました。
::
::middle
幣原喜重郎内閣は、松本{{烝治|じょうじ}}を委員長とする**憲法問題調査委員会**を設置しました。
::
::middle
委員会作成の改正試案が天皇の統治権を認めていたため、ＧＨＱはこれを拒否し、英文の改正草案（マッカーサー草案）を作成しました。
::
::last
1946年２月、内閣は==草案を和訳・修正したものを政府原案として提示しました==。
::
::sup
改正草案：国会が衆議院のみの一院制
::
::sup
政府原案：参議院を加えた二院制
::
::sup
ＧＨＱは、高野岩三郎ら発表の「憲法草案要綱」を執筆に参考
::
#### [[第１次吉田茂]]内閣｜1946年５月～1947年５月
::top
1946年11月３日、[[日本国憲法]]公布（1947年５月３日、施行）
:::lead
11月３日、[[日本国憲法]]公布（1947年５月３日、施行）大日本国憲法を改正する形で、==衆議院・貴族院で==修正可決されて公布
::
::sup
衆議院の修正で、第９条第２項に「前項の目的を達するため」との字句が、{{芦田均|あしだひとし}}の発案で加えられ、自衛のための軍隊保持を許容
::

## 生活の混乱
### 人口増加と物価上昇
::top
空襲で焼け出された人々は、防空壕や焼け跡に建てたバラック小屋で生活しました。
::
---arrow---
::top
将兵の[[復員]]や海外居留民の[[引揚げ]]で、==日本の人口が急増しました==。
::
::middle
食糧不足が深刻化し、米などの配給の不足・遅配・欠配が続きました。
::
::last
民衆は、生活物資を求めて、農村への[[買出し]]や公定価格を無視した市場[[闇市]]の利用を余儀なくされました。
::
---arrow---
::top
物不足に加えて、戦後処理などで貨幣が増発され、激しい物価上昇を招きました。
::
