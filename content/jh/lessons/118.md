## 自由民権運動の失速

### 手を引く者と激化させる者

:::top
1881年からの大蔵卿<ruby>松方正義<rt>まつかたまさよし</rt></ruby>による[[松方財政]]は、==物価を下落させました==。
:::

:::middle
農村の多くの者は、米・<ruby>繭<rt>まゆ</rt></ruby>などの換金で生計を立てるため、物価下落で窮乏しました。
:::

:::last
生活難の者は、自由民権運動から手を引く、あるいは運動を激化させました。
:::

### 運動の過激化と自由党の解散

:::top
自由党員や農民は、政府の不況下の増税や政党弾圧を受け、直接行動に臨みました。
:::

:::last
次の過激な事件と鎮圧の繰り返しで、自由民権運動は次第に衰退していきました。
:::

- 1882年、福島事件
			
			
			三島通庸
			河野広中
  :::lead
  福島県令[[三島<ruby>通庸<rt>みちつね</rt></ruby>]]が、労役で県道をつくろうとし、農民が抵抗した事件
  :::
  :::lead
  自由党は訴訟で支援しただけだが、三島通庸が[[<ruby>河野広中<rt>こうのひろなか</rt></ruby>]]を首謀者として検挙し、この事件を皮切りに、各地で自由党の過激行動が頻発
  :::
- 1884年、<ruby>加波山<rt>かばさん</rt></ruby>事件
  :::lead
  栃木県令に転任した三島通庸の圧政に対し、自由党員が県令暗殺を計画した事件
  :::
  :::lead
  この事件直後、自由党は党内の統率に自信を失って解散
  :::
- 1884年、<ruby>秩父<rt>ちちぶ</rt></ruby>事件
  :::lead
  約3000人の農民が、旧自由党員の指導で[[困民党]]を組織し、==負債の減免を求めて==高利貸・警察・役所を襲撃し、軍隊派遣に発展した事件
  :::
- 1885年、大阪事件
  :::lead
  旧自由党員[[大井憲太郎]]や[[景山英子]]が、独立党の政権樹立を助けて、日本人の政治への関心を喚起しようとし、逮捕された事件
  :::

### 不平等条約への強まる反感

:::top
1886年、ノルマントン号事件
:::lead
神戸に向かう英貨物船が難破し、英人船長ら乗組員は無事に脱出したが、日本人船客25人が船中に残されて水死した事件事件の裁判は、[[領事裁判権]]に従って英領事がおこない、全乗組員が無罪判決（再審で船長のみ禁固３ヶ月）
:::

:::last
この一件を契機に、世論は条約改正による領事裁判権の撤廃を強く求めました。
:::

:::gazo
![](/images/jh/img/118/7.jpg)

ビゴーによるノルマントン号事件の風刺画

＊1887年のメンザレ号の遭難事件にノルマントン号事件を重ね、イギリスの対応の悪さを非難
:::

### 条件付きの改正案

:::top
1887年、欧米諸国は[[領事裁判権]]を撤廃する改正案に、次の条件付きで一応了承しました。
:::

:::list{class="en"}
- 被告が外国人の裁判では、半数以上の外国人判事を採用
- 外国人に日本国内での居住・旅行・営業を許可（内地雑居）
:::

:::last
政府内部にさえも、上の条件は国家主権の侵害であると批判が起こりました。
:::

### 運動の再燃と交渉の中止

:::top
国会開設の年「1890年」が近づくと、自由民権運動は息を吹き返し始めました。
:::

:::last
旧自由党員<ruby>星亨<rt>ほしとおる</rt></ruby>・[[後藤象二郎]]は、思想の小異よりも目的の大同を優先し、立憲改進党と団結して国会開設に備えようと[[大同団結運動]]を推進しました。
:::

:::gazo{size="half"}
![](/images/jh/img/118/8.jpg)

後藤象二郎
:::

::arrow

:::top
1887年、[[星<ruby>亨<rt>とおる</rt></ruby>]]は、井上馨外相の条約改正交渉への反対に、次の３つを主張する[[三大事件建白運動]]を推進しました。
:::

:::list{class="en"}
- 地租の軽減
- 言論・集会の自由
- 外交失策の回復（対等条約の締結）
:::

:::last
井上馨外相の交渉は、政府内部からの批判や民衆からの批判を受けて中止されました。
:::
