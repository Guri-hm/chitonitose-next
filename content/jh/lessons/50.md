## 現世への対応

:::top
人々は今を生きます。この現世でいかに仏の救いを得るかが最大の関心となりました。仏教は現世の問題にどのように対応できるのか。この問いに対して、２人の人物が新たな鎌倉仏教を開きました。
:::

#### 教え

:::top
[[日蓮]]は、『[[<ruby>法華経<rt>ほけきょう</rt></ruby>]]』のみが釈迦の教えを正しく伝え、末法の世でも効力をもつと考えました（下の①②）。
:::

- 題目と呼ぶ「南無妙法蓮華経」の７字を唱える。
- 人々は生きながら（現世で）成仏できる。

::arrow

:::top
日蓮は、『法華経』を<ruby>蔑<rt>ないがし</rt></ruby>ろにする禅・念仏など他宗攻撃を激しくおこないました。
:::

:::middle
邪法である禅・念仏を止めなければ、いつか外敵侵入と内乱に遭うと予言し、『立正安国論』を５代執権北条時頼に提出しました。
:::

:::last
時頼は日蓮を流罪とし、その後蒙古襲来には遭いましたが侵入されませんでした。
:::

:::sup
日蓮は幕府の激しい弾圧を何度もうけたが、教えは==関東の武士や商人に==浸透

## 神の信仰とその他

### 神の信仰―教義の必要性

:::top
仏教受容の一方、神の信仰（<ruby>神祇<rt>じんぎ</rt></ruby>信仰）が受け継がれていました。
:::

:::middle
当初、理論・体系などの教義は必要なく、布教も必要ありませんでした。
:::

:::last
そして、重要な神社は朝廷の経済的な保護をうけました。
:::

::arrow

:::top
律令制が崩壊して朝廷の権威が失墜すると、神社は神々の威光・由来を宣伝し、寄進や参詣人を集めなければならなくなりました。
:::

:::middle
その過程で「神道（神祇信仰＋教義）」が生み出されました。
:::

:::last
動きはやや遅れて伊勢神宮にも起こり、伊勢<ruby>外宮<rt>げくう</rt></ruby>の神官[[<ruby>度会家行<rt>わたらいいえゆき</rt></ruby>]]は、[[伊勢神道]]を生み出しました。
:::

:::sup
度会家行
		<ruby>神本仏迹説<rt>しんぽんぶつじゃくせつ</rt></ruby>（反本地垂迹説）を唱えて、神（<ruby>本地
			<rt>ほんじ</rt></ruby>）が仏という仮の姿（<ruby>垂迹<rt>すいじゃく</rt></ruby>）でも現れたのだと説明・納得
:::lead
『[[<ruby>類聚神祇本源
				<rt>るいじゅうじんぎほんげん</rt></ruby>]]』を著し、<ruby>本地垂迹<rt>ほんじすいじゃく</rt></ruby>説を否定
:::

### その他

- 修験道（日本古来の山岳信仰と密教の融合）が独自の立場を確立
- 朱熹しゅき（朱子）に構築された朱子学（儒学の一派）が日本に伝来

:::last
