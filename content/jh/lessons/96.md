## 将軍家斉の治世―寛政～天保期

### 長期的な将軍在職

:::top
1787年、11代将軍に[[徳川<ruby>家斉<rt>いえなり</rt></ruby>]]が就任しました。
:::

:::middle
家斉の補佐として老中には白河藩主[[松平定信]]が就任し、質素倹約を重視する諸改革[[寛政の改革]]を実施しました。
:::

:::last
1793年、尊号一件などで対立を増やした定信が退陣した後、家斉は1804～30年の文化・文政期、<ruby>天保<rt>てんぽう</rt></ruby>期の一時期もなお将軍に在職しました。
:::

### 文化期と文政期の変化と問題

#### 商業・文化の成長と問題

:::top
文化期（1804～18年）、寛政の改革の質素倹約が維持されました。
:::

:::last
文政期（1818～30年）、幕府は品質の劣る貨幣を大量に流通させ、次の結果を招きました。
:::

- 物価の上昇に伴う幕府財政の潤沢
  :::lead
  将軍や<ruby>大奥<rt>おおおく</rt></ruby>の生活が華美になり、<ruby>賄<rt>わい</rt></ruby><ruby>賂<rt>
  						ろ</rt></ruby>も横行して政治が腐敗
  :::
- 商業の活発化と庶民文化の発達
  :::lead
  豪農・地主が増長する一方、百姓が土地を失って荒廃地域が発生
  :::

:::gazo{size="half"}
![](/images/jh/img/96/2.jpg)

大奥
:::

#### 治安問題と対策

:::top
農村の荒廃が深刻化し、その過程で村落秩序から脱落する者がいました。
:::

:::middle
そのような無宿人や博打を生業とする<ruby>博<rt>ばく</rt></ruby><ruby>徒<rt>と</rt></ruby>が、関東の農村で増えて治安が乱れました。
:::

:::last
幕府は==治安回復のために==次の対策をとりました。
:::

- 1805年、関東取締出役設置
  :::lead
  役人から８名を関東取締出役に選出し、２人１組で<ruby>関八州<rt>かんはっしゅう</rt></ruby>（関東８カ国）を巡回させ、領主の区別なく（例え大名領でも）無宿人や博徒の逮捕・取締りを実行
  :::
- 1827年、寄場組合結成
  :::lead
  領主の違いをこえて、近隣の村々から成る組織「寄場組合」を関東取締出役の下に結成させ、協同して地域の治安回復や風俗の取締りを実行
  :::

:::gazo{size="half"}
![](/images/jh/img/96/1.svg)

関東取締出役
:::

### 天保期の凶作

:::top
天保期の1833～36年、収穫が例年の半分以下になり、全国的な米不足を招いて、[[天保の飢饉]]と呼ばれる全国的な<ruby>飢饉<rt>ききん</rt></ruby>となりました。
:::

:::middle
困窮した人々が農村・都市にあふれ、百姓一揆・打ちこわしが相次ぎました。
:::

:::last
しかし、幕府・諸藩は適切な対策を立てられませんでした。
:::

::arrow

:::top
天保の飢饉は1836年が特に厳しく、極度に達した地域では激しい百姓一揆が起こりました。
:::

:::last
特に[[<ruby>甲斐<rt>かい</rt></ruby>]]国の郡内一揆と三河国の加茂一揆が有名です。
:::

## 大御所家斉の治世―天保期

### 実権の掌握維持

:::top
天保期の1837年、徳川家斉が将軍職を譲り、[[徳川家慶]]が12代将軍に就任しました。
:::

:::middle
しかし、家斉は**大御所**として実権を握り、1841年の死去まで治世を続けました。
:::

:::last
家斉の治世は約50年間におよび、特に文化～天保期を大御所時代とも呼びます。
:::

:::sup
大御所時代
:::lead
この期間の政治は大御所政治と呼ばれ、幕政退廃の代名詞
