---overview---
11代将軍徳川家斉は、約50年間の治世をおこないました。このうち、特に文化～天保期を大御所時代と称し、幕政退廃の代名詞に挙げられる。国内には風紀の退廃や賄賂の横行、治安悪化による無頼漢の跋扈が目立ち、国外には異国の接近を経験しました。寛政の改革で問題となった内憂外患は解決されず、悪化する一方でした。
---
## 将軍家斉の治世―寛政～天保期
### 長期的な将軍在職
::top
1787年、11代将軍に[[徳川{{家斉|いえなり}}]]が就任しました。
::
::middle
家斉の補佐として老中には白河藩主[[松平定信]]が就任し、質素倹約を重視する諸改革[[寛政の改革]]を実施しました。
::
::last
1793年、尊号一件などで対立を増やした定信が退陣した後、家斉は1804～30年の文化・文政期、{{天保|てんぽう}}期の一時期もなお将軍に在職しました。
::
::gazo
![](img/96/1.jpg)
徳川家斉
::
::last
文政期（1818～30年）、幕府は品質の劣る貨幣を大量に流通させ、次の結果を招きました。
::

- 物価の上昇に伴う幕府財政の潤沢
  :::lead
  将軍や{{大奥|おおおく}}の生活が華美になり、{{賄|わい}}{{賂|ろ}}も横行して政治が腐敗
  :::
- 商業の活発化と庶民文化の発達
  :::lead
  豪農・地主が増長する一方、百姓が土地を失って荒廃地域が発生
  :::

::gazo
![](img/96/2.jpg){.half}
大奥
::
::middle
そのような無宿人や博打を生業とする{{博|ばく}}{{徒|と}}が、関東の農村で増えて治安が乱れました。
::
::last
幕府は==治安回復のために==次の対策をとりました。
::

- 1805年、[[関東取締出役]]設置
  :::lead
  役人から８名を関東取締出役に選出し、２人１組で{{関八州|かんはっしゅう}}（関東８カ国）を巡回させ、領主の区別なく（例え大名領でも）無宿人や博徒の逮捕・取締りを実行
  :::
- 1827年、**寄場組合**結成
  :::lead
  領主の違いをこえて、近隣の村々から成る組織「寄場組合」を関東取締出役の下に結成させ、協同して地域の治安回復や風俗の取締りを実行
  :::

::gazo
![](img/96/1.svg){.half}
関東取締出役
::
::middle
困窮した人々が農村・都市にあふれ、百姓一揆・打ちこわしが相次ぎました。
::
::last
しかし、幕府・諸藩は適切な対策を立てられませんでした。
::
---arrow---
::top
天保の飢饉は1836年が特に厳しく、極度に達した地域では激しい百姓一揆が起こりました。
::
::last
特に[[甲斐|かい]]国の郡内一揆と三河国の加茂一揆が有名です。
::
## 大御所家斉の治世―天保期
### 実権の掌握維持
::top
天保期の1837年、徳川家斉が将軍職を譲り、[[徳川家慶]]が12代将軍に就任しました。
::
::middle
しかし、家斉は**大御所**として実権を握り、1841年の死去まで治世を続けました。
::
::last
家斉の治世は約50年間におよび、特に文化～天保期を大御所時代とも呼びます。
::
::sup
大御所時代
:::lead
この期間の政治は大御所政治と呼ばれ、幕政退廃の代名詞
::
::gazo

::middle
一方で==富裕な商人は米を買い占めて暴利を獲得し==、町奉行は窮民救済策をとりませんでした。
::
::last
そして、米不足にも関わらず、米は江戸へ回送されていました。

大塩平八郎は、蜂起の２週間前に蔵書を売り、得た代金で貧民を救済したと歴史教科書で称賛されます。ところが、水戸藩に伝わる史料に、米不足に困る同藩が大塩から大量の米を斡旋してもらったと記されています。当時、同藩主徳川斉昭を大塩が敬慕していたことは有名な話でした。もし実話であれば、この斡旋もまた米不足・米価高騰の一因になり、大塩も庶民を苦しめた元凶の１人ということになります。
