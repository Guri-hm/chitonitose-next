### ヤマト政権と朝鮮半島の諸国

#### 加耶諸国

:::top
ヤマト政権は加耶諸国に鉄素材を求め、両者は早くから密接な関係にありました。
:::

#### 百済

:::top
地理上、百済は高句麗に対して背後を固める必要がありました。
:::

:::last
372年、百済王は倭王に[[<ruby>七支刀
			<rt>しちしとう</rt></ruby>]]を贈り、国交を結ぼうとしました。
:::

:::sup
七支刀
:::lead
奈良県の<ruby>石上<rt>いそのかみ</rt></ruby>神宮所蔵の神宝
:::

## 大陸文化の受容

### 技術を運ぶ渡来人

:::top
４～５世紀、ヤマト政権は朝鮮半島から技術を学びました。
:::

:::last
[[渡来人]]と呼ばれる、大陸から日本列島に渡った人々もいて、彼らによっても様々な技術が伝えられました。
:::

:::gazo
![](/images/jh/img/6/7.svg)

技術を運ぶ渡来人
:::

::arrow

:::top
ヤマト政権は、渡来人を技術ごとの集団に編入しました。
:::

:::top
史部ふひとべ
:::lead
記録・文書を担当した集団
:::

:::top
鞍作部くらつくりべ
:::lead
馬具（鞍など）を製作する集団
:::

:::top
<ruby>韓鍛冶部<rt>からかぬちべ</rt></ruby>
:::lead
鍛冶・銅工などをおこなう集団
:::

:::top
<ruby>陶作部<rt>すえつくりべ</rt></ruby>
:::lead
陶器（<ruby>須恵器<rt>すえき</rt></ruby>）を製作する集団
:::

:::top
<ruby>錦織部<rt>にしごりべ</rt></ruby>
:::lead
錦を織る集団
:::

:::last
上記の集団は品部と総称、<ruby>伴造<rt>とものみやつこ</rt></ruby>に統率されました。
:::
