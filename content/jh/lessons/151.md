## 軍部の政治的台頭

### 二・二六事件と軍の実力

#### 岡田啓介内閣｜1934年７月～1936年３月

:::top
穏健派の海軍大将[[斎藤<ruby>実<rt>まこと</rt></ruby>]]による内閣の後を継ぎ、1934年、穏健派の海軍大将[[岡田<ruby>啓介<rt>けいすけ</rt></ruby>]]が、内閣を組織しました。
:::

:::gazo{size="half"}
![](/images/jh/img/151/1.jpg)

岡田啓介

::arrow

:::top
1932年の五・一五事件後、国内政治に対する政党の影響力が小さくなる一方で、軍部や右翼が台頭し、これに一部の官僚・政党人が同調しました。
:::

:::middle
彼らは、海軍穏健派内閣が２代続いたことに、不満を募らせていました。
:::

:::last
1934年、陸軍省が「**国防の本義と<ruby>其<rt>その</rt></ruby>強化の提唱**
		」と題するパンフレットを発行し、その中で政治に関する軍部の意向を示して大きな反響を呼びました。

:::last
岡田啓介首相は、譲歩に追い込まれ、[[国体明徴声明]]を出して天皇機関説を否定しました。

### 国防方針の変更

#### 岡田啓介内閣｜1934年７月～1936年３月

:::top
1934年、[[岡田<ruby>啓介<rt>けいすけ</rt></ruby>]]内閣は、次の海軍軍縮条約の廃棄を余儀なくされました。
:::

- ワシントン海軍軍縮条約
  :::lead
  主力艦（戦艦・航空母艦）の保有数制限
  :::
- ロンドン海軍軍縮条約
  :::lead
  補助艦（巡洋艦・駆逐艦・潜水艦）の保有数制限
  :::

#### 広田弘毅内閣｜1936年３月～1937年１月

:::top
1936年、軍の要望実現を約すことで、[[広田<ruby>弘毅<rt>こうき</rt></ruby>]]内閣がかろうじて成立しました。
:::

:::gazo{size="half"}
![](/images/jh/img/151/7.jpg)

広田弘毅
:::

::arrow

:::top
1936年、軍部大臣現役武官制復活
:::lead
現役の武官である大将・中将から、陸・海軍大臣を任用する制度復活とは、1913年に予備・後備役まで拡大した任用資格を戻したこと
:::

::arrow

:::top
同年、ワシントン海軍軍縮条約・ロンドン海軍軍縮条約が廃棄から２年の有効期間を経て==失効するため==、内閣は、陸海軍の帝国国防方針の改定に基づき、「国策の基準」を決めました。
:::

:::last
外交ではドイツと提携を強めてソ連に対抗し、国内では大建艦計画を推進しました。
:::

:::sup

1937年１月、穏健派の陸軍大将**<ruby>宇垣一成<rt>うがきかずしげ</rt></ruby>**が、次の首相に推薦されました。
:::

:::last

### 首相への熱狂的期待

#### 第１次近衛文麿内閣｜1937年６月～1939年１月

:::top
1937年６月、貴族院議長[[<ruby>近衛文麿<rt>このえふみまろ</rt></ruby>]]が、内閣を組織しました。
:::

:::last
近衛文麿は、若さ・家柄や軍人出でも官僚上りでも政党人でもない点で、未知の魅力にあふれた人物と映り、軍部から一般民衆までの広い期待を集めました。
:::
