## 報道活動

### 日刊の新聞

:::top
1880～90年代、自由民権運動・条約改正などは、世論に賛否が問われました。
:::

:::middle
結果、==政治評論中心の==新聞[[大新聞]]が相次いで創刊されました。
:::

:::last
大新聞は、政治思想を生み、また、近代文学の育成にも貢献しました。
:::

:::sup
日本最初の日本語の日刊新聞は、1871年に創刊された『[[横浜毎日新聞]]』
:::

:::gazo{size="half"}
![](/images/jh/img/135/1.jpg)

『横浜毎日新聞』
:::

:::

## 文学

### 戯作文学と政治小説

:::top
江戸時代の<ruby>洒落<rt>しゃれ</rt></ruby>本や<ruby>滑稽<rt>こっけい</rt></ruby>本の伝統を継ぐ勧善懲悪中心の[[<ruby>戯作<rt>げさく</rt></ruby>文学]]と、政治思想を盛り込んだ[[政治小説]]が書かれました。
:::

#### 代表作家

:::top
<ruby>仮名垣魯文<rt>かながきろぶん</rt></ruby>
:::lead
文明開化の風俗を滑稽に描写し、代表作は『[[<ruby>安愚楽鍋<rt>あぐらなべ</rt></ruby>]]』
:::

:::top
<ruby>矢野龍渓<rt>やのりゅうけい</rt></ruby>
:::lead

:::

:::top
東海散士さんし
:::lead
政治小説『<ruby>佳人之<rt>かじんの</rt></ruby>奇遇』で、各国の独立運動と憂国の情を描写

### 写実主義文学

:::top
1885年、[[<ruby>坪内逍遙<rt>つぼうちしょうよう</rt></ruby>]]が『[[小説<ruby>神髄<rt>しんずい</rt></ruby>]]』で西洋の文学理論を紹介して、日本の勧善懲悪的な文学観や政治小説を批判しました。
:::

:::last
人情や世相をあるがままに描く作風[[写実]]主義が流行し、近代文学が始まりました。
:::

#### 代表作家

:::lead
『[[<ruby>浮雲<rt>うきぐも</rt></ruby>]]』を著し、話し言葉に近い文章言文一致体を先駆

### 自然主義文学

:::top
日露戦争前後、フランス・ドイツの影響で、人間が隠す黒い欲望や暗い現実を、あるがままに描く作風[[自然]]主義が盛んになりました。
:::

#### 代表作家

:::top

:::lead

:::lead

:::lead

:::

:::top
石川<ruby>啄木<rt>たくぼく</rt></ruby>
:::lead
