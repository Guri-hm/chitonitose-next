## 条約改正―不平等性の是正

### 諸条約の不平等性

:::top
幕府が開国に際して結んだ諸条約には、次の不平等性が含まれていました。
:::

- 日本のみが義務を負う一方的な最恵国待遇さいけいこくたいぐうを許可
- 領事裁判権（治外法権）を許可
- 日本側の関税自主権の欠如を容認

:::last
新政府は、幕府から諸条約を引き継ぎ、改正による不平等性の解消を目指しました。
:::

### 琉球の帰属問題

:::top
江戸時代以来、[[琉球王国]]は、事実上[[<ruby>薩摩<rt>さつま</rt></ruby>]]藩に支配されました。
:::

:::last
しかし、琉球王国は名目上清を宗主国（一部権限を握り、上に立つ国）としていました。
:::

::arrow

:::top
新政府は琉球王国を日本領とする方針をとり、1872年、[[琉球藩]]を置きました。
:::

:::last
琉球王国の王[[<ruby>尚泰
			<rt>しょうたい</rt></ruby>]]を藩王としましたが、清はこの措置を認めませんでした。
:::

:::sup
琉球藩
:::lead
1871年の==廃藩置県後==であるが、清や現地の反発を恐れ、王の統治を維持

### 台湾出兵と琉球処分

:::top
琉球藩の設置後も、琉球王国が清と日本のどちらに属すか未解決でした。
:::

::arrow

:::top
1874年、台湾出兵
:::lead
1871年、琉球漂流民殺害事件が[[台湾]]で起き、事件を無視した清に対して、新政府は「琉球民は日本国民」と主張し、台湾に軍を出兵した出来事清は、イギリスの調停もあって、日本の出兵を正当な行動と容認
:::

:::last
この一件で、新政府は「琉球王国が日本に属すと示す事実」ができたと確信しました。
:::

::arrow

:::top
1879年、琉球処分
:::lead

### 朝鮮との外交

:::top
新政府は、朝鮮に日本の新政府発足を通告し、新たな国交を求めました。
:::

:::middle
朝鮮は、宗氏を窓口にしてのみ交流を許す、江戸時代同様の恩恵的な国交を望みました。
:::

:::last
そのため、新政府の望む国交は朝鮮から拒否されました。
:::

::arrow

:::top
岩倉使節団派遣中の新政府、いわゆる[[留守政府]]のあいだでは、[[<ruby>西郷隆盛<rt>さいごうたかもり</rt></ruby>]]・==<ruby>板垣退助<rt>
					いたがきたいすけ</rt></ruby>==らが、朝鮮との国交を武力で開く[[<ruby>征韓論
			<rt>せいかんろん</rt></ruby>]]を唱えました。
:::

:::last
征韓論は、1873年に帰国した岩倉使節団の反対で否決されました。
:::

:::sup
征韓論
:::lead
別の目的は、新政府に対する士族らの不満を国外にそらすこと
:::

:::sup
