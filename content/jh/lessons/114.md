---overview---
新政府の最大の懸案は、幕府が結んだ諸条約の改正でした。1871年、岩倉使節団が改正交渉と欧米視察に派遣されました。使節団派遣中の新政府「留守政府」は、国内の改革に着手する一方、急激な改革による国内の不満を外に向けようと、征韓論を唱えました。使節団の帰国後、征韓論をめぐって新政府内に大きな亀裂が生じました。
---
## 条約改正―不平等性の是正
### 諸条約の不平等性
::top
幕府が開国に際して結んだ諸条約には、次の不平等性が含まれていました。
::

- 日本のみが義務を負う一方的な{{最恵国待遇|さいけいこくたいぐう}}を許可
- 領事裁判権（治外法権）を許可
- 日本側の関税自主権の欠如を容認

::last
新政府は、幕府から諸条約を引き継ぎ、改正による不平等性の解消を目指しました。
::

### 琉球の帰属問題
::top
江戸時代以来、[[琉球王国]]は、事実上[[薩摩|さつま]]藩に支配されました。
::
::last
しかし、琉球王国は名目上清を宗主国（一部権限を握り、上に立つ国）としていました。
::
---arrow---
::top
新政府は琉球王国を日本領とする方針をとり、1872年、[[琉球藩]]を置きました。
::
::last
琉球王国の王[[尚泰|しょうたい]]を藩王としましたが、清はこの措置を認めませんでした。
::
::sup
琉球藩
:::lead
1871年の==廃藩置県後==であるが、清や現地の反発を恐れ、王の統治を維持
