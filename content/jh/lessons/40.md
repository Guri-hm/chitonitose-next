## 地方と庶民の文化

### 地方へ普及する文化

:::top
国風文化以降、末法思想と結びついた[[浄土教]]が次第に広がりました。
:::

:::middle
12世紀には、<ruby>聖<rt>ひじり</rt></ruby>や<ruby>上人<rt>しょうにん</rt></ruby>の布教で全国に広がりました。
:::

:::last
奥州藤原氏などの地方豪族も「阿弥陀堂」を建立しました。
:::

:::sup
聖
:::lead
寺院に所属せずに諸国をまわって修行する仏教僧
:::

:::sup
上人
:::lead
寺院に所属する高位の仏教僧への敬称
:::

## 院政期の文学と絵画

### 文学

:::top
地方・武士の動きに関心が高まったことで、合戦を題材にした軍記物語が書かれ、また、時代の転換期を感じて歴史に関心が高まったことで、[[歴史物語]]が書かれました。
:::

#### 軍記物語

:::top
『将門記』
:::lead
平将門の乱（935～40年）を記す、最初の軍記物語
:::

:::top
『<ruby>陸奥話記<rt>むつわき</rt></ruby>』
:::lead
==前九年合戦==（1051～62年）を記す軍記物語
:::

#### 歴史物語

:::top
『大鏡』
:::lead
藤原道長を中心とする摂関政治の栄華を批判的に記述した歴史物語
:::

:::top
『今鏡』
:::lead
大鏡の後の出来事を記した歴史物語
:::

:::top
『栄華物語』
:::lead
藤原道長の栄華の賛美を中心とした歴史物語
:::

:::sup
『大鏡』
:::lead
別称は『世継物語』
:::

:::sup
『大鏡』『今鏡』に鎌倉時代の『水鏡』と南北朝時代の『増鏡』を合わせ、「四鏡」と総称
:::

:::sup
『六国史』のような歴史書が漢文で記述されたのに対し、歴史物語はかな書き和文で記述
:::

#### その他

:::top
『今昔物語集』
:::lead
インド・中国・日本の仏教説話を和漢混淆文で記した説話集
:::

### 絵画

:::top
大和絵の手法による絵に<ruby>詞書<rt>ことばがき</rt></ruby>を織り交ぜて時間進行を表現する絵巻物や、扇形の紙に経文を書いて装飾した写経（装飾経）が作られました。
:::

:::sup
大和絵の手法は、例えば「<ruby>吹抜屋台ふきぬきやたい<rt></rt></ruby>」「<ruby>引目鉤鼻<rt>ひきめかぎばな</rt></ruby>」
