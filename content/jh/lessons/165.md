## 占領期の文化

### 学術の発展

:::top
天皇制に関するタブーも解かれ、また、マルクス主義が急速に復活しました。
:::

:::last
これらを通して、人文・社会科学の研究に新しい分野が開かれました。
:::

:::sup
1949年、あらゆる分野の学者の代表会議[[日本学術会議]]を設置
:::

#### 人文・社会・自然科学

:::top
考古学：[[岩宿遺跡]]・登呂遺跡の発掘
:::

:::top
政治学：丸山<ruby>真男<rt>まさお</rt></ruby>が、西欧近代との比較で日本の後進性を批判
:::

:::top
経済史学：大塚久雄が、いわゆる大塚史学を構築
:::

:::top
法社会学：川島<ruby>武宜<rt>たけよし</rt></ruby>が、近代社会における前近代的制度の意味を考察
:::

:::top
物理学：1949年、[[湯川秀樹]]が日本人初のノーベル賞を受賞、1965年、[[<ruby>朝永<rt>ともなが</rt></ruby>振一郎]]が日本人２人目のノーベル賞を受賞
:::

### 文化勲章

:::top
学問・芸術の奨励に制定された文化勲章が、1946年に復活されました。
:::

### 文学

:::top
常識や既成の観念に挑戦する作品、自身の戦争体験を表現した作品が出ました。
:::

:::last
一方、米軍や原爆に関する表現は、ＧＨＱの[[プレス＝コード]]で統制されました。
:::
