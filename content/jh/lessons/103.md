---overview---
日本は、和親条約締結で「鎖国」を部分的に放棄しましたが、さらに通商条約締結で全面的な「開国」に踏み切りました。同時期、将軍継嗣をめぐって一橋派と南紀派が対立しました。一橋派は幕政の改革派であり、南紀派は保守派の立場でした。この対立と並行して通商条約調印に関する勅許問題が起こり、極めて高度な政治問題に発展しました。
---
## 無勅許調印と継嗣問題
### 通商条約締結
::top
{{下田|しもだ}}駐在のアメリカ人領事[[ハリス]]は、通商条約締結を幕府に求めました。
::
::middle
幕府内で通商を望む声も強まり、ハリスと老中[[堀田{{正睦|まさよし}}]]の交渉は、条約調印（締結の合意書への署名）直前まで進みました。
::
::middle
本来不要だが、調印に対する天皇の{{勅許|ちょっきょ}}を得て、反対派を抑えようとしました。
::
::last
しかし、[[孝明]]天皇は{{攘夷|じょうい}}の考えが強く、勅許は得られませんでした。

## 貿易とその影響
### 貿易開始
::top
貿易は、1859年に箱館・横浜・長崎の３港でまず始まりました。
::
::last
輸出入品の取引は、居留地で銀貨を用いておこなわれました。
::
::top
輸出入額は、上の３港のうち[[横浜]]が圧倒的に多いものでした。
::
::last
西洋諸国は日本の[[生糸]]を最も求め（日本の輸出品の約80％）、また、西洋諸国のうち[[イギリス]]が日本と一番多く取引しました。
::

### 幕府の貿易統制政策
::top
貿易は、日本の輸出超過（輸出量＞輸入量）でした。
::
::middle
日本の主要輸出品は、国内需要が従来高いものでした。
::
::last
突然の輸出により、国内市場は品不足に陥って物価が高騰しました。
::
::gazo
![](img/103/5.jpg){.half}
橋本左内
::
::top
1860年、幕府は物価抑制を目的に[[五品江戸廻送令]]を発令しました。
::
::middle
５品「雑穀・{{水油|みずあぶら}}（菜種油）・{{蠟|ろう}}・呉服・生糸」の産地からの横浜直送を禁じ、江戸の{{問屋|といや}}にいったん送って国内需要を満たすことを義務づけた。
::
::last
西洋諸国や輸出に関わる商人の反対で、五品江戸廻送令は効果があがりませんでした。
::
