## 無勅許調印と継嗣問題

### 通商条約締結

:::top
<ruby>下田<rt>しもだ</rt></ruby>駐在のアメリカ人領事[[ハリス]]は、通商条約締結を幕府に求めました。
:::

:::middle
幕府内で通商を望む声も強まり、ハリスと老中[[堀田<ruby>正睦
			<rt>まさよし</rt></ruby>]]の交渉は、条約調印（締結の合意書への署名）直前まで進みました。
:::

:::middle
本来不要だが、調印に対する天皇の<ruby>勅許<rt>ちょっきょ</rt></ruby>を得て、反対派を抑えようとしました。
:::

:::last
しかし、[[孝明]]天皇は<ruby>攘夷
		<rt>じょうい</rt></ruby>の考えが強く、勅許は得られませんでした。

## 貿易とその影響

### 貿易開始

:::top
貿易は、1859年に箱館・横浜・長崎の３港でまず始まりました。
:::

:::last
輸出入品の取引は、居留地で銀貨を用いておこなわれました。
:::

:::top
輸出入額は、上の３港のうち[[横浜]]が圧倒的に多いものでした。
:::

:::last
西洋諸国は日本の[[生糸]]を最も求め（日本の輸出品の約80％）、また、西洋諸国のうち[[イギリス]]が日本と一番多く取引しました。
:::

:::sup

#### 日本の輸出入品

:::top
輸出
:::lead
ほとんど原料品（生糸・[[茶]]・<ruby>蚕卵紙<rt>さんらんし
				</rt></ruby>、海産物など）
:::

:::top
輸入
:::lead
加工品や軍需品（毛織物・綿織物、鉄砲・<ruby>艦船<rt>かんせん</rt></ruby>など）
:::

### 幕府の貿易統制政策

:::top
貿易は、日本の輸出超過（輸出量＞輸入量）でした。
:::

:::middle
日本の主要輸出品は、国内需要が従来高いものでした。
:::

:::last
突然の輸出により、国内市場は品不足に陥って物価が高騰しました。
:::

:::gazo
![](/images/jh/img/103/3.svg)

横浜港の輸出入別貿易額
:::

::arrow

:::top
1860年、幕府は物価抑制を目的に[[五品江戸廻送令]]を発令しました。
:::

:::middle
５品「雑穀・<ruby>水油<rt>みずあぶら</rt></ruby>（菜種油）・<ruby>蠟
		<rt>ろう</rt></ruby>・呉服・生糸」の産地からの横浜直送を禁じ、江戸の<ruby>問屋<rt>といや</rt></ruby>にいったん送って国内需要を満たすことを義務づけた。
:::

:::last
西洋諸国や輸出に関わる商人の反対で、五品江戸廻送令は効果があがりませんでした。
:::

:::gazo
![](/images/jh/img/103/4.svg)

### 金銀比率の是正

:::top
金銀交換比率は、西洋で金１：銀15、日本で金１：銀５でした。
:::

:::last
西洋諸国は、銀を日本に持ち込めば、金を自国よりも安く入手できました。
:::

::arrow

:::top
1860年以降、幕府は金の含有率を==下げた==[[万延小判]]を鋳造しました。
:::

:::last
しかし、貨幣価値の下落は物価の上昇を意味し、庶民の生活を圧迫しました。
:::
