## 農業生産の進展

### 農業

#### 商品経済の浸透

:::top
幕府や諸藩は、都市での年貢米販売で貨幣収入を得るとともに、[[商品作物]]の生産を奨励して収入の増大を図りました。
:::

:::last
村々で桑・<ruby>楮<rt>こうぞ</rt></ruby>・麻・綿花・菜種などの商品作物が盛んに生産されました。
:::

:::sup
1643年の田畑勝手作りの禁は、ほぼ有名無実化（1871年廃止）
:::

:::sup
四木しぼく三草
:::lead
幕府や諸藩が重視した商品作物の総称四木（茶・桑・楮・漆）三草（麻・<ruby>藍<rt>あい</rt></ruby>・<ruby>紅花<rt>べにばな</rt></ruby>）

#### 農書（17～19世紀）

:::top
次のような農業技術を教える書籍（<ruby>農書<rt>のうしょ</rt></ruby>）が==普及しました==。
:::

:::top
『清良記せいりょうき』
:::lead
栽培技術や農業技術を説く日本最古（17世紀前半）の農書
:::

:::top
『農業全書』
:::lead
17世紀末、[[宮崎安貞]]が著した体系的農書
:::

:::top
『農具便利論』
:::lead
19世紀、[[大蔵永常]]が農具の用法を図解した農書
:::

:::top
『広益国産考』
:::lead
19世紀、大蔵永常が経済発展に合わせた農業経営を説く農書
:::

:::top
耕地の開発が進み、刈敷が不足しました。
:::

:::last

			<rt>きんぴ</rt></ruby>]]が普及しました。
:::

:::sup

:::lead

			<rt>ありた</rt></ruby>]]で生産を始めた磁器は、長崎貿易の主要な輸出品となりました。

:::top

			<rt>なだ</rt></ruby>]]・[[<ruby>伊丹<rt>いたん</rt></ruby>]]の<ruby>銘酒
