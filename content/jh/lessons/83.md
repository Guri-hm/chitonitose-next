---overview---
17世紀後半以降の１世紀の間（江戸時代中期）に、小規模な経営を基礎とする農業やその他諸産業で、生産力が著しく発展しました。この時期に登場した特産品は今なお存在し、例えば野田の醤油や京都の西陣織が著名です。この知識は各地をめぐる際にも役立つため、「地名―特産品」の組合せは覚えておきましょう。
---
## 農業生産の進展
### 農業
#### 商品経済の浸透
::top
幕府や諸藩は、都市での年貢米販売で貨幣収入を得るとともに、[[商品作物]]の生産を奨励して収入の増大を図りました。
::
::last
村々で桑・{{楮|こうぞ}}・麻・綿花・菜種などの商品作物が盛んに生産されました。
::
::sup
1643年の田畑勝手作りの禁は、ほぼ有名無実化（1871年廃止）

#### 肥料
::top
耕地の開発が進み、刈敷が不足しました。
::
::last
都市周辺部では{{下肥|しもごえ}}が、商品作物の一大生産地では[[金肥きんぴ ]]が普及しました。
::
::sup
金肥
:::lead
金で買われる、{{干鰯|ほしか}}・{{〆粕|しめかす}}・{{油粕|あぶらかす}}・{{糠|ぬか}}などの高い栄養効率の肥料の総称
::

### 工芸品
::top
17世紀前半、肥前[[有田|ありた]]で生産を始めた磁器は、長崎貿易の主要な輸出品となりました。
::
::sup
木などのに漆を塗り重ねる工芸品の漆器では、[[会津|あいづ]]塗・{{輪島塗|わじまぬり}}が有名
::
### 醸造業
#### 酒造
::top
[[伏見]]・[[灘|なだ ・伊丹いたん]]の銘酒 めいしゅ が生まれ、各地に酒屋が発達しました。
::
#### 醤油造
::top
{{醤油|しょうゆ}}は西日本で早くから造られましたが、後に全国でも大量に生産されました。
::
::last
関東では、[[野田]]・[[銚子|ちょうし]]の醤油が有名でした。
