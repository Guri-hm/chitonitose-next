## 松平定信の登場

### 謂われのない責任転嫁

:::top
1782年からの大<ruby>飢饉<rt>ききん</rt></ruby>[[天明の飢饉]]は社会不安を増大させました。
:::

:::last
[[田沼意次]]は天明の飢饉や米価上昇の責任を転嫁され、1786年、10代将軍[[徳川家治]]の死後に老中を<ruby>退<rt>しりぞ</rt></ruby>きました。
:::

### 脅威下の政権交代

:::top
1787年、全国的な打ちこわし[[天明の打ちこわし]]が江戸でも生じました。
:::

:::middle
この民衆蜂起の脅威の下、11代将軍[[徳川家斉]]の補佐として、白河藩主[[松平定信]]が老中に就任しました。
:::

:::last
松平定信は、[[寛政の改革]]と総称される改革で、“内憂外患”に対処しました。
:::

:::top
内憂…飢饉で危うい農村の再興、民衆蜂起の防止、武士の不満解消
:::

:::top
外患…日本近海に現れたロシア船・イギリス船などの外国船への対応
:::

:::sup
松平定信
:::lead
８代将軍徳川吉宗の次男田安<ruby>宗武<rt>むねたけ</rt></ruby>の子で、田安家の出身
:::

## 寛政の改革―内憂

### 農村の再興

:::top
松平定信は、荒廃した農村を再興させるために次の政策をおこないました。
:::

:::list{class="en"}
- 人口の少ない東北・北関東において、百姓の他所への出稼ぎを制限
- 耕地復旧のために、全国で公金の貸付かしつけを開始
- 飢饉用米穀べいこくの貯蔵倉社倉・義倉を各地に設置し、そこに米穀を蓄える政策<ruby>囲米
				<rt>かこいまい</rt></ruby>を実施
:::

:::sup
社倉
:::lead
藩主が設置する貯蔵槽
:::

:::sup
義倉
:::lead
富裕者の寄付で設置する貯蔵倉
:::

:::gazo
![](/images/jh/img/94/2.jpg)

社倉・義倉
:::

### 民衆蜂起の防止

:::top
松平定信は、打ちこわしが起きた江戸の次なる蜂起の防止に取り組みました。
:::

:::last
政策実行に、<ruby>両替商<rt>りょうがえしょう</rt></ruby>などを10名からなる[[勘定所御用達]]に登用しました。
:::

#### 農業人口確保と流入民減少

:::top
[[旧里帰農令]]を発令して、出稼ぎ目的での江戸への流入民に資金を与え、故郷へ帰らせようとしました。
:::

#### 蜂起の危険分子の拘禁

:::top
江戸<ruby>石川島<rt>いしかわじま</rt></ruby>に[[人足寄場]]を設置して、危険分子の<ruby>無宿人<rt>むしゅくにん</rt></ruby>を強制収容し、社会復帰のために職業技術を覚えさせてから釈放しました。
:::

:::sup
農村からの流入民は、大半が定職・住居を持たずに秩序を乱す無宿人に転落
:::

:::gazo
![](/images/jh/img/94/3.jpg)

#### 救済用資金の確保

:::top
江戸の各町に町費（町入用）節約を命じ、その節減額の７分（70％）を積み立てさせ、[[七分積金]]と呼ばれる飢饉・災害時の救済用資金に<ruby>充<rt>あ</rt></ruby>てました。
:::

:::last
そして、新設した[[江戸町会所]]に七分積金の管理・運用を任せました。

### 武士の不満解消

#### 困窮する武士の救済

:::top
旗本・御家人の代理として<ruby>蔵米<rt>くらまい</rt></ruby>を売却する[[<ruby>札差<rt>
					ふださし</rt></ruby>]]は、金融業で巨利を得ていました。
:::

:::last
旗本・御家人は札差からの貸金によく頼り、その返済に苦しみました。
:::

::arrow

:::top
松平定信は[[<ruby>棄捐令
			<rt>きえんれい</rt></ruby>]]を発令し、札差に貸金返済の帳消しを命じました。
:::

:::last
旗本・御家人への貸金は、新設した貸金会所に低利でおこなわせました。
:::

#### 武士の不満抑制と解消

:::top
儒学による武士の教育が、出世などに対する不満の抑制に繋がると考えられました。
:::

::arrow

:::top
定信は[[寛政異学の禁]]を発して、儒学のうちで[[朱子学]]を“正学”とし、[[湯島聖堂]]に付属する<ruby>林家
		<rt>りんけ</rt></ruby>の私塾で朱子学以外の学派“異学”の講義を禁じました。
:::

:::middle
1797年、この私塾は切り離され、幕府直轄の学問所[[昌平坂学問所]]となりました。
:::

:::last
学問所では人材登用のための学術試験をおこない、下級武士に出世の機会をつくりました。
:::

:::sup
異学
:::lead
例えば、儒学の一派である古学（古学派）・陽明学など
:::

:::sup
昌平坂学問所
:::lead
岡田<ruby>寒泉<rt>かんせん</rt></ruby>（後に<ruby>古賀精里<rt>こがせいり</rt></ruby>と交代）・[[柴野栗山]]・**<ruby>尾藤二洲<rt>びとうじしゅう</rt></ruby>**
			が、学問所の教官を務め、当時の人は彼らを「[[寛政の三博士]]」と呼称
:::
