## 比肩のための標語

### 旧交通制度と旧商業制度の廃止

:::top
新政府は、江戸時代の次の制度を廃止しました。
:::

- 交通：関所・宿駅・助鄕すけごうの廃止
- 商業：株仲間の廃止

### 近代化産業の展望と指導者

:::top
新政府は、日本を欧米列強と肩を並べる近代国家にするため、経済発展と軍事力強化を目指す標語[[富国強兵]]を掲げました。
:::

:::middle
富国強兵達成のための具体的政策の１つが、産業育成[[殖産興業]]です。
:::

:::last
殖産興業には、外国人教師、<ruby>所謂<rt>いわゆる</rt></ruby>[[お雇い外国人]]の技術指導を仰ぎました。
:::

## 近代化産業の育成

### 工部省の推進

:::top
1870年、[[工部省]]が新設され、==鉄道==・鉱工業・郵便などの推進を司りました。
:::

#### 鉄道敷設

:::top
1872年、鉄道が[[新橋]]・[[横浜]]間に敷設されました。
:::

:::sup
イギリス人**モレル**の指導で敷設
:::

:::gazo
![](/images/jh/img/111/2.jpg)

八ツ山下を走る汽車

＊「八ツ山下」は品川のやや南に位置
:::

#### 鉱山

:::top
佐渡金山・<ruby>生野<rt>いくの</rt></ruby>銀山などの鉱山、高島・三池などの炭鉱を==政府経営（官営）としました==。
:::

#### 造船所

:::top
長崎造船所（元長崎製鉄所）・[[横須賀造船所]]（元横須賀製鉄所）など、==旧幕府の施設を拡充し、官営の造船所としました==。
:::

:::gazo
![](/images/jh/img/111/3.jpg)

横須賀造船所

#### 郵便

:::top
1871年、[[<ruby>前島密<rt>まえじまひそか</rt></ruby>]]の立案で、==飛脚に代わる官営の郵便制度が発足した==。
:::

:::middle
全国均一料金制による書状・荷物の確実な郵送を可能にしました。
:::

:::last
1877年、日本は万国郵便連合条約に加盟し、国際郵便の強化を図りました。
:::

1869年、==東京・横浜間で==[[電信]]事業が開始され、電報が取り扱われました。
:::

:::last
数年間で長崎と北海道まで架設、さらに長崎・上海間の海底電線で欧米に接続されました。

:::

### 内務省の推進

:::top
1873年、[[内務省]]が新設され、官営模範工場経営・農畜産業振興などを司りました。
:::

:::sup
内務省
:::lead
大蔵省・工部省の事務の一部を引き継ぎ、初代長官（内務卿）は[[大久保利通]]==地方行政・警察行政も統轄==
:::

#### 製糸

:::top
新政府は、開国以来の主要輸出品[[生糸]]の品質改良と大量生産を可能にするため、1872年、==フランス人==ブリューナの指導の下、==群馬県に==[[富岡製糸場]]を設立しました。
:::

:::last
富岡製糸場は、民間での技術・機械導入促進に、政府が経営した[[官営模範工場]]で、==士族の子女など（工女・女工）が==技術習得をしました。
