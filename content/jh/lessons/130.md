---overview---
日清戦争の賠償金を基礎にした積極的な金融政策に導かれて、綿糸の生産「紡績」・生糸の生産「製糸」などが顕著な発達を遂げ、貿易のあり方を対欧米・対アジアの二局面構成に変化させていきました。また、財政補助にも助けられて、海運業・鉄道業が伸張し、海陸の交通網を整備していきました。
---
## 紡績業―日本産業革命の中心
### 紡績の勃興
::top
幕末以来、イギリス製綿製品の輸入に圧迫されて、日本における綿花の栽培や綿糸・綿織物の生産は一時衰えました。
::
::middle
しかし、綿織物生産は次の取り組みで次第に上向きました。
::

- 織物の原料に輸入綿糸を使用
- ジョン＝ケイが発明した飛び{{杼|ひ}}を取り入れて{{手織機|ておりばた}}を改良

::last
綿織物生産の上向きが、綿糸の生産（{{紡績|ぼうせき}}）の勃興に繋がりました。
::
::gazo
::
::middle
1882年に[[渋沢栄一]]らが設立した[[大阪紡績会社]]は、[[蒸気]]を動力とした**ミュール**紡績機による大規模経営に成功しました。
::
::last
このような紡績機械を用いた工場が相次いで設立され、生産が拡大しました。
::

#### [[日清]]戦争==前==
::top
1890年、==綿糸の生産量が輸入量を上回り==、中国・朝鮮への輸出が増えました。
::
#### [[日清]]戦争==後==
::top
1897年、==綿糸の輸出量が輸入量を上回りました==（原料の==綿花の輸入も増加==）。
::
::gazo
::
::top
農村：[[豊田{{佐吉|さきち}}]]らが考案した国産の小型力織機の導入で、手織機による問屋制家内工業が機械生産の小工場に転換
::
::last
1909年、綿織物の輸出額が輸入額を超えました。
::
::gazo
::
::last
特に安価な**インド**産の輸入は多く、綿業に関する貿易は輸入超過でした。
::
::gazo
::
::last
生糸の生産（製糸）は欧米向けの輸出産業として急速に発達しました。
::
### 製糸技術の発達
::top
当初、簡単な手動装置による[[座繰|ざぐり]]製糸が普及していました。
::
::middle
輸入器械を改良した[[器械|きかい]]製糸の小工場が、長野県・山梨県などのなどの農村地帯に続々と生まれました。
::
::last
生糸の原料である{{繭|まゆ}}を供給する{{養蚕|ようさん}}農家も増加した。
::
### 輸出増加
#### [[日清]]戦争頃
::top
[[器械]]製糸の生産量が、[[座繰]]製糸の生産量を上回りました。
::
::last
生糸を原料とする絹織物の生産も盛んになり、力織機も導入されました。
::
#### 日露戦争後
::top
[[アメリカ]]向けを中心に、生糸輸出がさらに伸びました。
::
::last
1909年、清を追い越し、日本は世界最大の生糸輸出国となりました。
::
::gazo
::
::middle
貿易では、特殊銀行である横浜正金銀行や、三井物産会社などの商社が重要な役割を果たしました。
::
::last
また、[[日本郵船会社]]が次の海運業奨励の法令に助けられ、インドへのボンベイ航路、欧・米・豪への遠洋航路を開きました。
::

- 1896年、[[航海奨励法]]公布
  :::lead
  1000トン・速力10ノット以上の鋼鉄汽船に奨励金を交付する法令
  :::
