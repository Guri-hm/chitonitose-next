## 紡績業―日本産業革命の中心

### 紡績の勃興

:::top
幕末以来、イギリス製綿製品の輸入に圧迫されて、日本における綿花の栽培や綿糸・綿織物の生産は一時衰えました。
:::

:::middle
しかし、綿織物生産は次の取り組みで次第に上向きました。
:::

:::list{class="en"}
- 織物の原料に輸入綿糸を使用
- ジョン＝ケイが発明した飛び杼ひを取り入れて手織機ておりばたを改良
:::

:::last
綿織物生産の上向きが、綿糸の生産（<ruby>紡績<rt>ぼうせき</rt></ruby>）の勃興に繋がりました。
:::

:::

### 綿糸輸出の増加―日清戦争前後

:::top
<ruby>手紡<rt>てつむぎ</rt></ruby>や[[<ruby>臥雲辰致
			<rt>がうんたっち</rt></ruby>]]が発明した[[ガラ<ruby>紡<rt>ぼう</rt></ruby>]]で綿糸が生産されました。
:::

:::middle
1882年に[[渋沢栄一]]らが設立した[[大阪紡績会社]]は、[[蒸気]]を動力とした**ミュール**紡績機による大規模経営に成功しました。
:::

:::last
このような紡績機械を用いた工場が相次いで設立され、生産が拡大しました。
:::

:::sup
ガラ紡
:::lead

:::

## 製糸業―外貨獲得の中心産業

### 製糸の発達

:::top
幕末以来、[[<ruby>生糸<rt>きいと</rt></ruby>]]は最大の輸出品でした。
:::

:::last
生糸の生産（製糸）は欧米向けの輸出産業として急速に発達しました。
:::

### 製糸技術の発達

:::top
当初、簡単な手動装置による[[<ruby>座繰<rt>ざぐり</rt></ruby>]]製糸が普及していました。
:::

:::middle
輸入器械を改良した[[<ruby>器械
			<rt>きかい</rt></ruby>]]製糸の小工場が、長野県・山梨県などのなどの農村地帯に続々と生まれました。
:::

:::last
生糸の原料である<ruby>繭<rt>まゆ</rt></ruby>を供給する<ruby>養蚕<rt>ようさん</rt></ruby>農家も増加した。
:::

### 輸出増加

#### 日清戦争頃

:::top
[[器械]]製糸の生産量が、[[座繰]]製糸の生産量を上回りました。
:::

:::last
生糸を原料とする絹織物の生産も盛んになり、力織機も導入されました。
:::

#### 日露戦争後

:::top
[[アメリカ]]向けを中心に、生糸輸出がさらに伸びました。
:::

:::last
1909年、清を追い越し、日本は世界最大の生糸輸出国となりました。
:::

![](/images/jh/img/130/3.svg)

## 輸送業

### 海運業

:::top
綿花などの原料品や機械・鉄などの重工業製品の輸入が増加し、日本は大幅な輸入超過（赤字）でした。
:::

:::middle
貿易では、特殊銀行である横浜正金銀行や、三井物産会社などの商社が重要な役割を果たしました。
:::

:::last
また、[[日本郵船会社]]が次の海運業奨励の法令に助けられ、インドへのボンベイ航路、欧・米・豪への遠洋航路を開きました。
:::

- 1896年、造船奨励法公布
  :::lead
  700トン以上の鋼鉄製汽船建造に助成金を給与する法令
  :::
- 1896年、航海奨励法公布
  :::lead
  1000トン・速力10ノット以上の鋼鉄汽船に奨励金を交付する法令
  :::

:::sup
ボンベイ航路
:::lead
インド産の綿花の大量輸入を可能にした日本初の遠洋航路
:::

:::

### 鉄道業

:::top
1881年、[[華族]]の出資で[[日本鉄道会社]]が設立されました。
:::

:::middle
政府の保護を受け、商人や地主による会社設立ブームが起こりました。
:::

:::last

		民営鉄道は営業キロ数で官営を上回っていました。
:::
