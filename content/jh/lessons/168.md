## 経済成長のひずみと終焉

### 高度経済成長のひずみ

:::top
公害問題：公害のうち、特に大きな被害をもたらした次の①～④が発生
:::

:::list{class="en"}
- 熊本県水俣みなまた市の水俣病
- 新潟県阿賀野川流域の新潟水俣病
- 三重県四日市市の四日市ぜんそく
- 富山県神通川じんづうがわ流域のイタイイタイ病
:::

:::middle
1967年に[[公害対策基本法]]制定、1971年に[[環境庁]]設置
:::

:::last
①～④の四大公害の訴訟は、1973年までに==いずれも原告側が勝訴==
:::

:::top
人権問題：差別解消に向け、1946年の[[部落解放全国委員会]]結成以降も、様々な施策が図られたが、今日まで未解決
:::

### ひずみの中での期待

:::top
1967年、日本社会党・日本共産党に推薦された[[<ruby>美濃部亮吉
			<rt>みのべりょうきち</rt></ruby>]]が、東京都知事に当選しました。
:::

:::middle
1970年代初め、京都府・大阪府知事、横浜市市長も同様の推薦者が当選しました。
:::

:::last
このような首長を革新首長と呼び、その自治体を[[革新自治体]]と呼びます。
:::

::arrow

:::top
革新自治体は、公害の規則や老人医療の無料化など、==福祉政策で成果を上げました==。

### 高度経済成長の終焉

#### 田中角栄内閣｜1972年７月～1974年12月

:::top
1973年10月、第４次中東戦争
:::lead
アラブ諸国とイスラエルの間の戦争アラブ諸国で構成されたＯＡＰＥＣ（アラブ石油輸出国機構）は、イスラエルを支持する欧米・日本に石油を売らない戦術を展開
:::

:::gazo
![](/images/jh/img/168/6.jpg)

第４次中東戦争
:::

::arrow

:::top
石油不足とそれに伴う世界的な経済混乱[[第１次石油危機]]が生じました。
:::

:::middle
日本は、安価な原油の安定的供給という経済成長の条件を失いました。
:::

:::last
1974年、戦後初のマイナス成長となり、高度経済成長は終わりました。
