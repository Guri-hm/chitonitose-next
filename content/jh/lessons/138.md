## 第一次世界大戦の概観

### 第一次世界大戦の勃発

:::top
日露戦争を経た20世紀初め、ヨーロッパは次の陣営に分かれました。
:::

- 三国同盟：ドイツ・オーストリア・イタリア
- 三国協商：イギリス・フランス・ロシア

:::last
日本は、イギリスとの[[日英同盟協約]]、ロシアとの[[日露協約]]に基づき、三国協商側に立ちました。
:::

:::

:::top
1914年６月、サライェボ事件
:::lead
ヨーロッパの火薬庫と称されたバルカン半島で、オーストリア帝位継承者がセルビア人青年に暗殺された事件事件は、オーストリアとセルビアの戦争に発展
:::

:::middle
ドイツがオーストリア支持を、ロシアがセルビア支持を表明しました。
:::

:::last
他の国々も、軍事関係に従って参戦を決め、[[第一次世界大戦]]と呼ばれる４年に及ぶ戦争になりました。
:::

新兵器が投入されて多くの死者を出し、兵士は塹壕にたてこもりながら一進一退を繰り返しました。
:::

### 大戦の転機①と終結

:::top
戦争は、ドイツ優勢（三国同盟優勢）で展開しました。
:::

:::last
しかし、ドイツが無制限潜水艦作戦を開始したことで、[[アメリカ]]が三国協商側で参戦すると、形勢は逆転しました。
:::

::arrow

:::top
1918年11月、最後まで抵抗したドイツも降伏し、大戦は終結しました。
:::

### 大戦の転機②と革命の余波

:::top
1917年、ロシアで皇帝と戦争に反対する[[ロシア革命]]が起こりました。
:::

:::last
結果、世界初の社会主義国家（後のソヴィエト連邦）が生まれました。
:::

## 第一次世界大戦中の日本

### 対独の軍事行動と権益強化

#### 第２次大隈重信内閣｜1914年４月～1916年10月

:::top
第２次[[<ruby>大隈重信
			<rt>おおくましげのぶ</rt></ruby>]]内閣は、外相加藤高明の主導で、イギリスとの[[日英同盟協約]]を理由に参戦を決めました。
:::

::arrow

:::top
日本は、ドイツが勢力をもつ次の地域を占領・接収しました。
:::

- 中国山東省のドイツ根拠地<ruby>青島<rt>
						チンタオ</rt></ruby>およびドイツ権益
- 赤道以北のドイツ領南洋諸島

:::gazo
![](/images/jh/img/138/2.svg)

20世紀前半のアジア
:::

::arrow

:::top
1915年、大隈重信内閣は中華民国の大総統[[袁世凱]]の政府に、次の内容から成るいわゆる[[二十一か条の要求]]を突きつけました。
:::

- 山東省のドイツ権益を継承
- 旅順・大連および南満州鉄道の租借権を99年間延長
- 中国の製鉄会社<ruby>漢冶萍<rt>かんやひょう</rt></ruby><ruby>公司
				<rt>コンス</rt></ruby>を日本と共同経営
- 中華民国の顧問として日本人を採用

:::middle
1915年５月、袁世凱政府は要求の一部（④除く）を受諾しました。
:::

:::last
中国では、受諾した５月９日を「[[国恥記念日]]」とし、反日感情を強めました。
