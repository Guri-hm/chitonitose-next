### 戦乱期から40年

#### 社会問題①

:::top
大名と旗本は、跡継ぎを作れないと<ruby>改易<rt>かいえき</rt></ruby>されました。
:::

:::last
跡継ぎを作らず、死に際して養子を急に取ることは[[<ruby>末期養子<rt>まつごようし</rt></ruby>の禁止]]で禁止され、これによる改易で、多くの武士が仕える主君を失い、[[牢人]]となって路頭に迷いました。
:::

#### 社会問題②

:::top
社会秩序が安定すると、規範に収まりきれない血気ある者が突出しました。
:::

:::last
奇抜な姿で勝手に振る舞う[[かぶき者]]となり、頻繁に騒動を起こした。
:::

### 成人した４代将軍の政策

:::top
成人した４代将軍[[徳川家綱]]は、次の①～④の政策をおこないました。
:::

- 1663年、武家諸法度（寛文令）
- 1663年、<ruby>殉死<rt>じゅんし</rt></ruby>の禁止
  :::lead
  主君の死後、家臣があとを追って自殺することを禁じ、跡継ぎに奉公することを義務化
  :::
  :::middle
  これは、家臣が主君個人ではなく主君の家に仕えるということ、
  				つまり、「主君の家－代々主人」「家臣の家―代々家臣」という不動の主従関係を確立
  :::
  :::last
  下の者が上の者にとってかわる<ruby>下剋上<rt>げこくじょう</rt></ruby>の風潮は消滅
  :::
- 1664年、領知宛行状りょうちあてがいじょう（領地の確認文書）の発給
  :::lead
  領地は将軍が与えているものと強調し、その権威を再確認
  :::
- 1664年、幕領の一斉検地
  :::lead
  幕府の財政収入の安定化を意図
  :::

## 同時代の諸藩

### 諸藩の取り組み

:::top
諸藩は、平和による軍役の減少と1641～42年の寛永の<ruby>飢饉<rt>ききん</rt></ruby>の影響で、17世紀半ばから藩政の安定と経済発展を図りました。
:::

:::last
次の①～④の諸藩では、大名が儒学者（朱子学者）を用いて改革を図りました。
:::

- 岡山藩（藩主：池田光政）
  :::lead
  <ruby>郷校<rt>ごうこう</rt></ruby>[[<ruby>閑谷<rt>しずたに</rt></ruby>学校]]の設置
  :::
  :::lead
  ==陽明学==者[[<ruby>熊沢蕃山<rt>くまざわばんざん</rt></ruby>]]による<ruby>花畠<rt>はなばたけ</rt></ruby>教場の設置
  :::
- 加賀藩（藩主：前田<ruby>綱紀<rt>つなのり</rt></ruby>）
  :::lead
  朱子学者[[木下順庵]]の招致
  :::
- 会津藩（藩主：保科正之）
  :::lead
  朱子学者山崎<ruby>闇斎<rt>あんさい</rt></ruby>に師事
  :::
- 水戸藩（藩主：水戸<ruby>光圀<rt>みつくに</rt></ruby>）
  :::lead
  明から亡命した儒学者朱舜水の招致
  :::
  :::lead
  江戸に設置した彰考館で『[[大日本史]]』を<ruby>編纂<rt>へんさん</rt></ruby>開始
  :::

:::sup
1662年、明の亡命政権が消滅し、清が中国統一
:::

:::sup
『大日本史』
		幕末の水戸藩の思想や運動に影響
:::lead
尊王を基本とした朱子学の大義名分論で貫かれる歴史書
:::
