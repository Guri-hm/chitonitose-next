---overview---
幕藩体制は、１～３代将軍が大名らの統制などに取り組むことで確立され、４代将軍徳川家綱の頃には安定期へ入りました。小さなことで揺らがずに政治を運用でき、実際、幼将軍の就任や牢人問題は、それほど大きな影響がありませんでした。明暦の大火でも、従来の方針を踏襲することで人心の安定を図ることを第一義としました。
---
## ４代将軍の治世
### 幼将軍の就任
::top
1651年、３代将軍[[徳川家光]]が死去しました。
::
::middle
徳川家光の子[[徳川家綱]]が11歳で４代将軍に就任しました。
::
::last
徳川家綱は幼将軍のため、{{叔父|おじ}}で会津藩主の[[保科正之|ほしなまさゆき]]に補佐されました。
::
::sup
上記は、将軍の幼弱に関係なく、幕府を運用できる体制ができていたことを意味
::

::last
跡継ぎを作らず、死に際して養子を急に取ることは[[{{末期養子|まつごようし}}の禁止]]で禁止され、これによる改易で、多くの武士が仕える主君を失い、[[牢人]]となって路頭に迷いました。
::
#### 社会問題②
::top
社会秩序が安定すると、規範に収まりきれない血気ある者が突出しました。
::
::last
奇抜な姿で勝手に振る舞う[[かぶき者]]となり、頻繁に騒動を起こした。

::top

::
::last

### 成人した４代将軍の政策
::top
成人した４代将軍[[徳川家綱]]は、次の①～④の政策をおこないました。
::

- 1663年、武家諸法度（寛文令）
- 1663年、[[殉死|じゅんし]]の禁止
  :::lead
  主君の死後、家臣があとを追って自殺することを禁じ、跡継ぎに奉公することを義務化
  :::
- 1664年、{{領知宛行状|りょうちあてがいじょう}}（領地の確認文書）の発給
  :::lead
  領地は将軍が与えているものと強調し、その権威を再確認
  :::
- 1664年、幕領の一斉検地
  :::lead
  幕府の財政収入の安定化を意図
  :::
