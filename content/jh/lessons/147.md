## 中国統一の動き

### 軍閥の割拠

:::top
1916年、中華民国の大総統であった軍人<ruby>袁世凱<rt>えんせいがい</rt></ruby>が死没しました。
:::

:::last
袁世凱の配下の部将たち<ruby>軍閥<rt>ぐんばつ</rt></ruby>が各地で抗争を始め、中国内は分裂しました。
:::

::arrow

:::top
次の２つの団体が組織され、軍閥打倒・中国統一を目指しました。
:::

- 中国国民党
  :::lead
  1919年、孫文に組織された団体
  :::
- 中国共産党
  :::lead
  1921年、ソ連の支援を得て組織された団体
  :::

:::last
1924年、両団体の協力関係第１次国共合作が実現しました。
:::

### 北伐の開始

:::top
孫文の死後、[[<ruby>蔣介石<rt>しょうかいせき</rt></ruby>]]が中国国民党の指導者となりました。
:::

:::last
1926年、蔣介石は中国国民党で編成した軍隊（国民革命軍）をつくり、軍閥打倒・中国統一のために、広州から北上する[[<ruby>北伐<rt>ほくばつ</rt></ruby>]]を始めました。

### 山東半島の日本人保護

:::top
1922年の山東懸案解決条約締結後も、山東省における日本の権益は少し残されました。
:::

:::last
国民革命軍は各国居留民の殺害事件を起こしており、日本人居留民も脅威にさらされました。

#### 田中義一内閣｜1927年４月～1929年７月

:::top
1927～28年、山東出兵
:::lead
田中義一首相が、==日本人居留民保護を目的に==、山東省に３度出兵したことこの間に[[東方会議]]を開き、中国における権益を実力で守る強硬方針を確認２度目の出兵時に、日本軍・国民革命軍が**<ruby>済南<rt>さいなん</rt></ruby>**で武力衝突する事件が発生（**済南事件**）
:::

### 関東軍の計画と失敗

:::top
1919年、関東州（旅順・大連を含む遼東半島南端）の統治機関関東都督府が、行政担当の[[関東庁]]と軍事担当の[[関東軍]]に分離しました。
:::

:::last
関東軍は、関東州・満鉄沿線の守護と居留民の保護を担い、また、満州の軍閥[[<ruby>張作霖<rt>ちょうさくりん</rt></ruby>]]の支援して国民革命軍の進行を防ごうとしました。

#### 田中義一内閣｜1927年４月～1929年７月

:::top
1928年、張作霖爆殺事件
:::lead
関東軍の一部が、国民革命軍に寝返る気配を見せた張作霖を列車ごと爆破し、混乱に乗じた満州の支配を画策した事件田中義一は、真相公表と厳重処分を昭和天皇に誓ったが、閣僚・陸軍の反対を受けて、首謀者<ruby>河本大作<rt>こうもとだいさく</rt></ruby>の停職処分に方針転換
:::

:::sup
張作霖爆殺事件
:::lead
事件の内容を国民に隠すため、満州某重大事件と当初呼称
:::

:::gazo
![](/images/jh/img/147/3.jpg)

張作霖爆殺事件
:::

::arrow

:::top
約束を違えた田中義一は、昭和天皇から叱責され、1929年に総辞職しました。
:::

### 中国統一の達成

:::top
張作霖の子**<ruby>張学良<rt>ちょうがくりょう</rt></ruby>**
		は、事件の真相を知っており、日本の指示を無視して、勢力下の満州を国民政府支配下の土地と認めました。
:::

:::last
ここに北伐は完了し、国民政府による中国統一が達成されました。
:::

:::sup
易幟えきし
:::lead
国民政府との合流に際して、張学良が忠誠を表明するために、国民党の旗「<ruby>青天白日旗<rt>せいてんはくじつき</rt></ruby>」を満州全域に掲げたこと
:::

張学良
:::

:::gazo
![](/images/jh/img/147/5.svg)

:::

::arrow

:::top
この頃の中国では、諸外国に与えた権利などの回収を目指す気運が高まりました。
:::

:::last
1931年、国民政府は不平等条約の無効を一方的に宣言し、また、日本の権益（租借地や満鉄経営など）も回収しようとし始めました。
:::

## 逆風に立つ日本

### 右翼の一源流

:::top
日本は、1919年のパリ講和会議に戦勝国として臨みましだが、権益について諸外国から予想外の批判を浴びました。
:::

:::middle
このような時代背景下、次の２人が日本の進むべき道を模索しました。
:::

- 北一輝
  :::lead
  『[[日本改造法案大綱]]』を著し、日本の国際的孤立を打開するための国家改造を述べた人物
  :::
- 大川周明
  :::lead
  <ruby>猶存社<rt>ゆうぞんしゃ</rt></ruby>を結成し、国家改造を主張した人物
  :::

:::last
彼らに影響された軍や右翼は、後に国家改造運動を進めていきました。
:::

:::sup
右翼
:::lead
天皇制などの日本的な伝統を重視し、特に排外的な傾向を持つ勢力
