:::top

### 仏教の６学派

:::top
仏教は釈迦の教えに従う宗教であるが、釈迦自身は教えを文字にしていません。
:::

:::middle
年月が経ち、弟子たちが釈迦の言葉を「経典」にまとめていきました。
:::

:::last
まとめた内容に差異があり、経典は何百、何千種類にも分かれました。
:::

::arrow

:::top
僧は「この内容が釈迦の正しい教えだ」と思う経典を派閥ごとに研究しました。
:::

:::last
奈良時代、[[南都六宗]]と総称される６つの学派が研究に力を注いだ。

### 唐僧の招来

:::top
当初、日本では「受戒（僧になるための儀礼）」なしで僧になれました。
:::

:::middle
そして、僧になると税を免除されました。
:::

:::last
読経もできずに、勝手に出家を宣言して僧となった**<ruby>私度僧<rt>しどそう</rt></ruby>**が続出した。
:::

::arrow

:::top
「受戒」を義務化したいが、それを執り行える僧が日本にいませんでした。
:::

:::last
唐から僧を招いたところ、753年、[[鑑真]]が渡来して戒律を伝えました。

## 天平の美術

### ２つの技法

:::top
当時の仏像の作り方は、金銅像の他に次の２種類が主流でした。
:::

- <ruby>乾漆像<rt>かんしつぞう</rt></ruby><div class="last">高価な像で、本尊（寺の主要な像）はこの技法で作成</div>
  :::lead
  原型の上に麻布を幾重にも漆で塗り固め、後で原型を抜き取る像
  :::
  :::lead
  材料の漆が貴重で高価
  :::
- <ruby>塑像<rt>そぞう</rt></ruby><div class="last">本尊の周囲に配置される仏像はこの技法で作成</div>
  :::lead
  木を芯として粘土を塗り固めた像
  :::
  :::lead
  材料が粘土のため、乾漆像と比べて安価
  :::
