## 人と身分

### 身分とは

:::top
江戸時代、①大名・旗本②農民・漁民③僧侶・神職など様々な職業がありました。
:::

:::middle
①は「武士」、②は「百姓」、③は「宗教者」と分類され、分類ごとに、例えば「武士－軍事・政治」「百姓―生産」と、果たすべき役割が固定されました。
:::

:::middle
役割以外、例えば「百姓」が軍事に手を出すことは固く禁止されました。
:::

:::last
<ruby>語弊<rt>ごへい</rt></ruby>もありますが、江戸時代の「身分」とは、役割の固定された上記分類を指します。
:::

:::top
身分ごとの役割が固定されているに過ぎず、身分の変更は可能でした。
:::

### 様々な身分と武士

:::top
身分は、武士・百姓・町人・宗教者・医者など<ruby>多岐<rt>たき</rt></ruby>にわたりました。
:::

:::last
大部分の人は、「武士」「百姓・職人・商人」のいずれかの身分にあり、このような江戸時代の社会状況は[[士農工商]]と呼ばれました。
:::

::arrow

:::top
[[武士]]は大名・旗本・御家人など複数の階層から構成される上位の身分で、[[帯刀]]・[[苗字]]・<ruby>切捨御免<rt>
				きりすてごめん</rt></ruby>などの特権をもちました。
:::

:::sup
切捨御免
:::lead
下位の身分から無礼を受けた時、斬殺しても処罰されないこと
:::

### 下位の身分

:::top
様々なある身分のなかで、次の２つは下位の身分とされました。
:::

- かわた（長吏ちょうり）
  :::lead
  <ruby>皮革<rt>ひかく</rt></ruby>の製造・わら細工、時に死牛馬の処理・<ruby>行刑役<rt>
  						ぎょうけいやく</rt></ruby>に従事
  :::
  :::lead
  死・血に携わり<ruby>汚<rt>けが</rt></ruby>れを帯びるため、[[えた]]（穢多）とも呼称
  :::
- 非人
  :::lead
  村などから排除された<ruby>乞食<rt>こつじき</rt></ruby>で、芸能・掃除・<ruby>物乞<rt>ものご</rt></ruby>いに従事
  :::

:::last
かわた・非人は居住地・衣服などの点で、他身分と区別されました。
:::

### 家と戸主の権限

:::top
武士や有力な百姓・町人の家では、家の筆頭戸主に強い権限がありました。
:::

:::last
戸主の立場や<ruby>家督<rt>かとく</rt></ruby>（家・財産など）は長子が相続し、原則的に女性は相続しませんでした。
:::

:::sup
家督
:::lead
時代によって意味が異なり、鎌倉時代の「家督」は一門の首長のこと
:::
