## 元禄文化

### 元禄文化とは

:::top
<ruby>元禄<rt>げんろく</rt></ruby>期（５代将軍徳川綱吉の治世）、幕政が安定しました。
:::

:::middle
生活に余裕ができ、色々な分野に多彩な様式をもつ文化が展開しました。
:::

:::last
この時期の文化を[[元禄]]文化と呼び、次の①～②の特色をもちます。
:::

- 現世を「浮き世（辛いことの多い世）」と捉えて表現
- 儒学以外の実証的・実用的な学問も発達

### 儒学とその一派

:::top
儒学は、身分ごとの職分・上下の秩序・「忠孝・礼儀」を重んじます。
:::

:::middle
特に儒学の一派朱子学は、大義名分論を基礎にします。
:::

:::last
幕府は儒学・朱子学が幕藩体制の維持に役立つと考え、これらを道徳的な教えを説く学問「教学」と位置づけて重視しました。
:::

:::gazo
![](/images/jh/img/86/2.svg)

儒学の一派の対立
:::

::arrow

:::top
明の<ruby>王陽明<rt>おうようめい</rt></ruby>が創始した儒学の一派[[陽明学]]は、<ruby>知行合一<rt>ちこうごういつ</rt></ruby>を基礎にします。
:::

:::last
陽明学者は知行合一の立場で現実や朱子学を批判したため、幕府に警戒されました。
:::

:::sup
知行合一
:::lead
知識は人間の経験・実践を通してこそ「本当の知」になるという思想
