## 東と西への進出

### ５世紀のヤマト政権の範囲

:::top
熊本県[[江田船山]]古墳の鉄刀・埼玉県[[稲荷山]]古墳の鉄剣には、倭王[[武]]（[[雄略]]天皇）の名があります。
:::

:::last
このことから、５世紀のヤマト政権の勢力範囲は、おおよそ下図（色塗り部分）のように示せます。
:::

![](/images/jh/img/17/1.svg)
:::

::arrow

:::top
また、『[[宋書]]』[[倭国伝]]には次の文が載ります。
:::

:::middle
「東は<ruby>毛人<rt>もうじん</rt></ruby>を征すること五十五国、西は<ruby>衆夷<rt>しゅうい</rt></ruby>を服すること六十六国、…」
:::

:::last
ヤマト政権は毛人・衆夷のような勢力範囲の外の者と争い、服属させて次第に勢力範囲の境界を東へ西へと広げていきました。
:::

### 東北への進出

:::top
ヤマト政権に属さない東国の人々「毛人」は、後に[[<ruby>蝦夷<rt>えみし</rt></ruby>]]と呼称されました。
:::

:::last
ヤマト政権は６世紀以降も進出を続け、蝦夷を服属させていきました。
:::

::arrow

:::top
孝徳天皇の御代647年に[[<ruby>渟足柵<rt>ぬたりのさく</rt></ruby>]]、648年に[[<ruby>磐舟柵<rt>いわふねのさく</rt></ruby>]]を設置しました。
:::

::arrow

:::top
斉明天皇の御代658年、[[阿倍比羅夫]]を派遣して、==日本海側の==蝦夷を服属させました。
:::

::arrow

:::top
712年、日本海側に[[出羽]]国を設置しました。
:::

:::last
724年、[[<ruby>陸奥<rt>むつの</rt></ruby>]]国に国府および[[鎮守府]]を兼ねる[[<ruby>多賀城<rt>たがじょう</rt></ruby>]]を設置した。
:::

:::sup
鎮守府
:::lead
蝦夷征討のための役所・拠点
:::

::arrow

:::top
上記のような蝦夷征討には「<ruby>夷<rt>い</rt></ruby>を<ruby>以<rt>もっ</rt></ruby>て夷を制す」の政策がとられました。
:::

:::last
そのような中で、780年、**<ruby>俘囚<rt>ふしゅう</rt></ruby>**の**<ruby>伊治呰麻呂<rt>これはりのあざまろ</rt></ruby>**に多賀城を焼かれました。
:::

:::sup
「夷を以て夷を制す」
:::lead
帰順した蝦夷で蝦夷を討つ政策
:::

:::sup
俘囚
:::lead
８世紀以降に使用された、帰順した蝦夷の呼称

### 九州南部の支配

:::top
<ruby>熊襲<rt>くまそ</rt></ruby>（衆夷）は早くに服属し、ヤマト政権の勢力範囲は九州南部まで及びました。
:::

:::last
服属した熊襲は[[<ruby>隼人<rt>はやと</rt></ruby>]]と呼称されました。
:::

::arrow

:::top
九州南部での律令の導入に対して、しばしば隼人が反乱を起こしました。
:::

:::last
隼人への支配強化のため、８世紀初めに[[<ruby>薩摩<rt>さつま</rt></ruby>]]国や[[<ruby>大隅<rt>おおすみ</rt></ruby>]]国が設置されました。
