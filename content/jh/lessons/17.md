---overview---
「日本」と聞いて、どれほどの地理的範囲を想像するでしょうか。想像した現代の範囲と比べ、古代の「日本（７世紀後半までヤマト政権）」の範囲は遙かに狭いものでした。５世紀でようやく東は北関東、西は九州中部に達しました。範囲の拡大は範囲外の人々の抵抗を伴い、日本は抵抗する人々を服属させることで、東と西の境界を拡大していきました。
---
## 東と西への進出

::top

::
::last

::top
また、『[[宋書]]』[[倭国伝]]には次の文が載ります。
::
::middle
「東は{{毛人|もうじん}}を征すること五十五国、西は{{衆夷|しゅうい}}を服すること六十六国、…」
::
::last
ヤマト政権は毛人・衆夷のような勢力範囲の外の者と争い、服属させて次第に勢力範囲の境界を東へ西へと広げていきました。
::
### 東北への進出
::top
ヤマト政権に属さない東国の人々「毛人」は、後に[[蝦夷|えみし]]と呼称されました。
::
::last
ヤマト政権は６世紀以降も進出を続け、蝦夷を服属させていきました。
::
---arrow---
::top
孝徳天皇の御代647年に[[渟足柵|ぬたりのさく]]、648年に[[磐舟柵|いわふねのさく]]を設置しました。
::
---arrow---
::top
斉明天皇の御代658年、[[阿倍比羅夫]]を派遣して、==日本海側の==蝦夷を服属させました。
::
---arrow---
::top
712年、日本海側に[[出羽]]国を設置しました。
::
::last
724年、[[陸奥|むつの]]国に国府および[[鎮守府]]を兼ねる[[多賀城|たがじょう]]を設置した。
