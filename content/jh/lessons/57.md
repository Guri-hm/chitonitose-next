#### 琉球の貿易の特徴

:::top
中継貿易
:::lead
自国の商品よりも、主に他国Aからの輸入品を他国Bへの輸出品にする貿易
:::

:::gazo
![](/images/jh/img/57/4.jpg)
:::explanation
蘇木（左）・香木（右）
東南アジア産の植物で、蘇木は
		染料に香木は香料に用いられました。琉球の中継で日本に輸入されました。
:::

## 蝦夷ヶ島

### 13世紀の蝦夷ヶ島

:::top
縄文文化以降も、蝦夷ヶ島では食料採取に頼る生活が続きました。
:::

:::middle
このような蝦夷ヶ島の文化を[[続縄文]]文化と呼びます。
:::

:::last
７世紀以降も食料採取中心でしたが、次の文化に発展しました。
:::

:::list{class="en"}
- 擦文さつもん文化（７世紀～、サハリン・蝦夷ヶ島一円から東北地方まで）
- オホーツク文化（９世紀～、オホーツク沿岸のみ）

### 13～15世紀の蝦夷ヶ島

:::top
13～14世紀、安東氏は、十三湊を拠点にアイヌと貿易をしました。
:::

:::last
この貿易でもたらされた鮭・昆布は、京都にも運ばれました。
:::

::arrow

:::top
15世紀、安東氏や<ruby>和人<rt>わじん</rt></ruby>が、蝦夷ヶ島の南部（<ruby>道南<rt>どうなん</rt></ruby>）に進出しました。
:::

:::last
道南の沿岸に和人の<ruby>館<rt>たて</rt></ruby>が築造され、<ruby>道南十二館<rt>どうなんじゅうにたて</rt></ruby>と総称されました。

### 補説―蝦夷とアイヌ

:::top
古代、日本の支配拡大に抵抗した東の人々を<ruby>蝦夷<rt>えみし</rt></ruby>と呼びました。
:::

:::last
蝦夷には、蝦夷ヶ島のアイヌも含まれました（蝦夷⊃アイヌ）。
:::

::arrow

:::top
日本の支配拡大で、蝦夷と呼ばれる範囲が狭まっていきました。
:::

:::last
中世、蝦夷はアイヌをほぼ指しました（蝦夷≒アイヌ）。
:::
