## 律令のなかの民衆

### 民衆の管理

:::top
複数の家族を成員25人程度にまとめて、<ruby>戸<rt>こ</rt></ruby>という単位を編成しました。
:::

:::last
民衆はいずれかの戸に所属する形で国家に管理されました。
:::

:::sup
戸
:::lead
50戸で[[里長]]が管轄する行政区分「**<ruby>里<rt>り</rt></ruby>**（後に**<ruby>郷<rt>ごう</rt></ruby>**
      と改称）」を編成
:::

::arrow

:::top
戸の詳細な情報は、２種類の文書にまとめられました。
:::

- 戸籍
  :::lead
  ==６年に１度作成され==、戸の構成員の名・年齢・性別を記載
  :::
  :::last
  ６年ごとの班田収授の対象者を一目で把握可能
  :::
- 計帳
  :::lead
  ==毎年作成され==、戸籍の情報に加えてホクロなど身体的特徴も記載
  :::
  :::last
  年齢（20歳以上か未満かなど）で負担量が変わる税を把握可能
  :::

### 民衆の尊卑

:::top
民衆の身分は[[良民]]と[[賤民]]に二分されました。
:::

:::last
賤民は==官有の==「<ruby>陵戸<rt>りょうこ</rt></ruby>・官戸・<ruby>公奴婢
    <rt>くぬひ</rt></ruby>」と==私有の==「<ruby>家人<rt>けにん</rt></ruby>・私奴婢」に分かれ、まとめて[[<ruby>五色の賤<rt>ごしきのせん</rt></ruby>]]とも総称されました。
:::

:::sup
良民
:::lead
貴族を含む官人も該当しますが、彼らは一般民衆と異なり==税負担なし==
:::

### 条里制

:::top
碁盤状の田地で、東西の列を「１条、２条、３条…」、南北の列を「１里、２里、３里…」と数えます。
:::

:::middle
すると、特定の田地の位置を「○条×里」と示せます。
:::

:::last
このような田地の位置管理を[[条里制]]と呼びます。
:::

:::sup
条坊制
:::lead
条坊制は、碁盤状の京（藤原京など）で使用南北の列を「１坊、２坊、３坊…」と数え、特定の区画の位置を「○条×坊」と表示
:::

## 税制度の様相

### 田地の配給

:::top
[[班田収授]]法に従い、一定の田地[[口分田]] が班給・収公されました。
:::

:::list{class="en"}
- 班給：６年ごとの戸籍作成時、６以上の男女を対象
- 収公：６年ごとの戸籍作成時、班給された者で既に死去している者を対象
:::

男性→２段（720歩）
    女性→男性の２/３（480歩＝１段120歩）
    家人・私奴婢の男性→良民男性の１/３（240歩）
    家人・私奴婢の女性→良民女性の１/３（160歩）

:::sup
口分田を班給した残りの田地を乗田と呼び、期限・利子付きで貸し出し（<ruby>賃租<rt>ちんそ</rt></ruby>）
:::

:::sup
田地の種類には、大臣などに支給される[[<ruby>職田
      <rt>しきでん</rt></ruby>]]（不輸租田の一種）や、
    五位以上の位階者に与えられる[[位田]]（輸租田の一種）などが存在
:::
