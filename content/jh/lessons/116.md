## 国会開設の請願

### 地方統治制度の整備

:::top
1878年、新政府は[[地方三新法]]と総称される３つの法を定めました。
:::

- 郡区町村編制法
  :::lead
  旧来の郡（大都市は区）・町・村を行政上の単位として復活させた法律
  :::
- 府県会規則
  :::lead
  地方議会[[府県会]]の議員の選挙（公選）・権限などについての統一規則
  :::
- 地方税規則
  :::lead
  府県税や民費などの複雑な諸税を==「地方税」に統一した規則==
  :::

### 国会開設の請願と強まる弾圧

:::top
1877年、[[<ruby>片岡健吉
      <rt>かたおかけんきち</rt></ruby>]]を総代とする政治結社（政社）[[立志社]]が、国会開設を求める意見書[[立志社建白]]を天皇に提出しようとしました。
:::

:::last
新政府に却下されましたが、自由民権運動を国会開設請願へと突き動かしました。
:::

::arrow

:::top
1878年、立志社を中心に[[愛国社]]の再興大会が大阪で開かれ、この頃から、士族以外に地主や都市の商工業者も自由民権運動に加わり始めました。
:::

:::sup
愛国社
:::lead
1875年の大阪会議後、板垣退助が新政府に復帰して事実上解散状態
:::

::arrow

:::top
1880年、**愛国社**が、国会開設請願の全国的団体[[国会期成同盟]]と改称し、８万7000人が署名した国会開設請願書を天皇宛で提出しました。
:::

:::last
新政府は請願書を受理せず、さらには、勢いづいた運動の弾圧を始めました。
:::

### 政府に先んじた憲法の準備

:::top
国会期成同盟は、1881年10月に各自の憲法草案を持ち寄ることを約束しました。
:::

:::last
次のような民間の憲法草案が作成され、[[<ruby>私擬憲法
      <rt>しぎけんぽう</rt></ruby>]]と総称されました。
:::

- 「私擬憲法案」
  :::lead
  福沢諭吉が設立した政治結社[[<ruby>交詢社
            <rt>こうじゅんしゃ</rt></ruby>]]の憲法草案
  :::
- 「東洋大日本国国憲按あん」
  :::lead
  『民権自由論』の著者[[<ruby>植木枝盛<rt>うえきえもり</rt></ruby>]]の憲法草案
  :::
  :::lead
  最も民主的な憲法草案と言われ、==抵抗権・革命権を認可==
  :::
- 「日本憲法見込案」
  :::lead
  高知の政治結社（政社）[[立志社]]の憲法草案
  :::
- 「<ruby>五日市<rt>いつかいち</rt></ruby>憲法草案」
  :::lead
  東京の五日市の==地域住民が共同討議でまとめた憲法草案==

### 再度の内紛と国会開設の勅諭

:::top
大久保利通を欠いた新政府は、自由民権運動を前にして、次の内紛を生じさせました。
:::

- 急進論（主張者：(<ruby>大隈重信<rt>おおくましげのぶ</rt></ruby>）
  :::lead
  国会を早期に開設し、イギリス流の議院内閣制で君主権を弱くするべきと主張
  :::
- 漸進論（主張者：伊藤博文・右大臣岩倉具視ともみ）
  :::lead
  国会をゆっくり開設し、ドイツ流の<ruby>欽定<rt>きんてい</rt></ruby>憲法で君主権を強くするべきと主張
  :::
