---overview---
秀吉が征服地に検地を施行し、全ての土地を面積ではなく米の標準収穫量「石高」で示すことが可能になりました。大名・武士・百姓たちは、石高の有無・多少で序列化されていきました。この社会の在り方を石高制と呼び、その形成過程で兵農分離が進行しました。もはや先祖伝来の土地を守りながら武装し、かつ農業経営にたずさわる存在は許されなくなりました。この点で中世と近世では性格に違いが出てきます。
---
## 国内政策
### 経済基盤の確保
::top
豊臣氏の経済基盤は次の２つでした。
::

- 約220万石の莫大な直轄領**{{蔵入地|くらいりち}}**からの収入
- 直轄鉱山（佐渡金山・生野銀山・{{石見|いわみ}}大森銀山など）からの収入

::last
これらを財源に[[天正大判]]などの貨幣を鋳造しました。
::
### 検地と刀狩と兵の確保
::top
羽柴秀吉の以下の政策を通して、[[兵農分離]]が進行しました。
::
#### 検地
::top
秀吉は、==屋敷地を含む==全国の土地に、土地政策[[太閤たいこう 検地]]を実施しました。
::

- 田畑の面積を１{{段|たん}}＝360{{歩|ぶ}}から1段＝[[300]]歩に変更
- 測量・調査した田畑を４つの等級（上・中・下・下々）で区別し、等級に応じた田畑１段あたりの生産量目安を**{{石盛|こくもり}}**（{{斗代|とだい}}）と呼称
- 田畑の総生産量（～石）は、「石盛×総面積」で算出した[[石高|こくだか]]で把握可能石高を基準に、大名・社会などを管理する在り方は石高制と呼称
- 宣旨枡などの使用を止め、[[京枡]]に統一宣旨枡
  :::lead
  後三条天皇が公定した枡
  :::
- 田畑一区画の耕作・所有・年貢負担者を１人とする[[一地一作人]]を確立し、その者から領主に年貢が納められる過程での中間搾取を否定収穫の取り分は二公一民（領主：耕作者＝２：１）
- 1591年、大名に分国の[[検地帳]]・国絵図の提出を命令検地帳
  :::lead
  土地台帳であり、田畑・屋敷地の所有はこの台帳に登録された者に許可
  :::

#### 刀狩
::top
1588年、秀吉は[[刀狩令]]を出して、==百姓から==武器を没収しました。
::
::last
百姓の身分の明確化、一揆防止による農業への専念化に繋がりました。
::
::sup
回収した武器は、**方広寺**の大仏などに利用
::
#### 兵の確保
::top
朝鮮侵略に備えて、兵と米の生産者の定数確保が必須でした。
::
::last
1591年、秀吉は[[{{人掃|ひとはらい}}令]]（[[身分統制令]]）を出して、武家奉公人（兵）が町人・百姓になること、百姓が商工業に従事することを禁じました。
::

## 対外政策
### 倭寇対策と海上の兵農分離
::top
1588年、豊臣秀吉は[[海賊取締令]]を出して、[[倭寇|わこう]]などの海賊行為を禁止しました。
::
::last
海上の安全を確保して堺・博多などの商人に貿易を奨励し、また、元海賊を漁業に従事させました。
::
### キリスト教対策
::top
当初、羽柴秀吉はキリスト教の布教を許可していました。
::
::middle
1587年の九州平定の際、国内初のキリシタン大名の[[大村{{純忠|すみただ}}]]が[[長崎]]を教会領としてイエズス会に寄付したと聞きました。
::
::middle
大名のキリスト教入信を許可制にするとともに、[[バテレン追放令]]を出して、宣教師の国外追放と教会領没収を命じました。
::
::last
一方で==南蛮貿易は従来通り奨励したため、キリスト教の取締りは不徹底==
::
::sup
1596年、土佐国に漂着した**サン＝フェリペ号**の乗組員の失言で、宣教師・信徒の処刑**26聖人{{殉教|じゅんきょう}}**が発生（秀吉の唯一の直接的迫害）
::
::sup
播磨国の大名高山{{右近|うこん}}は、信仰を捨てずに領地没収の処分
::
::sup
一般人の信仰は「その者の心次第」として黙認
::
