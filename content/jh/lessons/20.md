##### 税負担からの逃避

:::top
貧窮問答歌の下線部Ⓑは里長によるの税の取り立てです。
:::

:::middle
税負担は重く、下線部Ⓒのように農民は「飛んで逃げたい」と思うほどでした。
:::

:::last
実際に負担から逃げた農民もいました。
:::

::arrow

:::list{class="en"}
- 逃げた先の場所では、「浮浪」者と呼称
- 逃げられた場所では、「逃亡」者と呼称
:::

:::sup
浮浪・逃亡の違いは、呼ぶ側の視点
:::

### 田地の借用と利息

:::top
農民は、口分田班給後の余剰地**乗田**を借りること（<ruby>賃租<rt>ちんそ</rt></ruby>）ができました。
:::

:::middle
ただし、利息として収穫の20％を納入しなくてはなりませんでした。
:::

:::last
生活が貧しくても、容易には借りられませんでした。
:::

:::sup
口分田は租（広さ１段当たり稲２束２把で、収穫の３％に相当）を納入
:::

### 「浮浪」「逃亡」者の行方

:::top
浮浪・逃亡している農民は、寺院・豪族のもとに仕えました。
:::

:::middle
寺院・豪族は、農民を養って田地開墾のために使役しました。
:::

:::last
743年に施行された[[墾田永年私財法]]に従い、寺院・豪族は大規模な田地[[初期荘園]]を形成していきました。
:::

:::sup
初期荘園
:::lead
租が掛かる輸租田という扱い
:::

### 税負担を避ける別の方法

:::top
当初、日本では「受戒（僧になるための儀礼）」なしで僧になれました。
:::

:::last
そして、僧になると税を免除されました。
:::

::arrow

:::top
読経もできず、勝手に出家を宣言して僧となった**<ruby>私度僧<rt>しどそう</rt></ruby>**が続出しました。
:::

:::middle
「受戒」を義務化しようとしたのですが、それを執り行える僧が日本にいませんでした。
:::

:::last
唐から僧を招いたところ、[[<ruby>鑑真
			<rt>がんじん</rt></ruby>]]が渡来して戒律を伝えました。
:::
