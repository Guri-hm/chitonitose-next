---overview---
奈良時代の民衆の生活、特に農民の生活は、当時の和歌や住居跡から辛うじて推測できる程度です。ところが厄介なことに、実態を捉えにくい農民の動きが、後の土地問題に大きく関わってきます。①税負担が嫌で逃亡・浮浪した農民がその後にどうなったか②初期荘園とは何か、以上２点は確実に理解したいです。
---
## 住居と税負担

::top

::
::last

天地は広いと言うが、私にとっては狭くなってしまったのか。［中略］Ⓐ低くつぶれ、曲がって傾いた家の中に、地べたにじかに藁を敷いて［中略］カマドには火の気がなく、飯炊き用の土器には蜘蛛の巣がはって、飯を炊くことも忘れたふうで［中略］Ⓑ鞭をもった里長の呼ぶ声が、家の中にまで聞こえてくる。世の中を生きていくことはなぜこれほど難しいのか。Ⓒ世の中を辛く痩 やせ せるように耐えがたく思うが、鳥ではないから飛んでいくこともできない。

#### 貧窮問答歌が描写する生活
::top
貧窮問答歌の下線部Ⓐのように、貧しい農民の住居、竪穴住居でした。
::
::last
ただし、{{掘立柱住居|ほったてばしらじゅうきょ}}という住居が、西日本から次第に普及していました。
::
::sup
寺院などは、==礎石を用いた{{瓦葺|かわらぶき}}==建築
::

### 田地の借用と利息
::top
農民は、口分田班給後の余剰地**乗田**を借りること（{{賃租|ちんそ}}）ができました。
::
::middle
ただし、利息として収穫の20％を納入しなくてはなりませんでした。
::
::last
生活が貧しくても、容易には借りられませんでした。
::
::sup
口分田は租（広さ１段当たり稲２束２把で、収穫の３％に相当）を納入
::
### 「浮浪」「逃亡」者の行方
::top
浮浪・逃亡している農民は、寺院・豪族のもとに仕えました。
::
::middle
寺院・豪族は、農民を養って田地開墾のために使役しました。
::
::last
743年に施行された[[墾田永年私財法]]に従い、寺院・豪族は大規模な田地[[初期荘園]]を形成していきました。
::
::sup
初期荘園
:::lead
租が掛かる輸租田という扱い
::
### 税負担を避ける別の方法
::top
当初、日本では「受戒（僧になるための儀礼）」なしで僧になれました。
::
::last
そして、僧になると税を免除されました。
::
---arrow---
::top
読経もできず、勝手に出家を宣言して僧となった**{{私度僧|しどそう}}**が続出しました。
::
::middle
「受戒」を義務化しようとしたのですが、それを執り行える僧が日本にいませんでした。
::
::last
唐から僧を招いたところ、[[鑑真|がんじん]]が渡来して戒律を伝えました。
::

古代の日本の結婚は、夫婦別居で夫が妻の住む家に通う、あるいは妻の家に同居しました。
::
