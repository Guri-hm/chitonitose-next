## 執権政治の発展

### 合議制の確立

:::top
承久の乱後まもなく、[[北条<ruby>泰時
			<rt>やすとき</rt></ruby>]]が３代[[執権]]を引継ぎました。
:::

:::middle
御家人の支持獲得には、独裁的でない公平な合議による裁断が必要です。
:::

:::last
その合議制のために、次の２つを設置し、ともに最高政務・裁判にあたりました。
:::

- 連署
  :::lead
  執権の補佐役（もう１人の執権）
  :::
- <ruby>評定衆<rt>ひょうじょうしゅう</rt></ruby>
  :::lead
  11名の有力御家人
  :::

:::sup
連署
:::lead
北条氏一族で世襲し、初代は**北条<ruby>時房<rt>ときふさ</rt></ruby>**
:::

:::sup
最高政務・裁判にあたる執権・連署・評定衆の会議を<ruby>評定<rt>ひょうじょう</rt></ruby>と呼称
:::

### 裁判基準の成文化

:::top
幕府成立以来、御家人同士の紛争や、御家人と荘園領主の紛争が相次ぎました。
:::

:::gazo{size="half"}
![](/images/jh/img/44/1.svg)

土地に問題はつきもの
:::

::arrow

:::top
1232年、御成敗式目（貞永式目）制定
:::lead
[[北条泰時]]が作らせた最初の武家法典==後の武家法にも大きな影響==[[源頼朝]]以来の「先例」、武士の慣習・道徳である「[[道理]]」に基づき、地頭・守護の任務と権限や、紛争を裁く基準を成文化弟北条重時へ宛てた手紙のなかで、御成敗式目の内容のことや、御成敗式目が律令や貴族社会の規則である公家法を否定しないと説明全51カ条で、後に追加した条文を式目追加と呼称
:::

::arrow

:::top
当時、貴族社会の規則公家法や、荘園領主が荘園支配のために定めた規則**本所法**がありました。
:::

:::middle
御成敗式目は==律令・公家法・本所法を否定せず==、幕府の支配領域にのみ適用されました。
:::

:::last
幕府の支配が全国に拡大するにつれ、公平さを重視する御成敗式目の影響は広がり、効力をもつ範囲が拡大しました。
:::

### 迅速で公正な裁判

:::top
訴訟が増加し、評定による裁判が滞っていました。
:::

:::middle
北条時頼は、裁判の迅速化とそれによる御家人の信頼獲得を考えました。
:::

:::last
1249年、次の３段階を踏むことで、迅速で公正な裁判を確立しました。
:::

:::list{class="en"}
- 定員４～５人の引付衆を任命
- 引付衆で構成する会議引付が判決原案を評定に提出
- 判決原案にほぼ従い、評定が最終的な裁断

### 対北条氏の勢力消滅

:::top
1226年、[[藤原頼経]]が４代将軍に就任しました。
:::

:::middle
頼経は在職の間に一部の御家人（三浦氏など）と親近な関係をもっていきました。
:::

:::middle
それら一部の御家人は、北条氏が御家人を統制する上での障害になりました。
:::

:::last
北条時頼は、頼経の子[[藤原<ruby>頼嗣
			<rt>よりつぐ</rt></ruby>]]を５代将軍に就任させ、後に頼経を京に送還しました。
:::

:::sup
藤原頼経・藤原頼嗣
:::lead
[[摂家]]将軍と呼称
:::

:::sup
頼経を解任することで将軍と一部の御家人の親近な関係を消滅
:::

:::gazo{size="half"}
![](/images/jh/img/44/3.jpg)

藤原頼経
:::

::arrow

:::top
1247年、<ruby>宝治合戦<rt>ほうじかっせん</rt></ruby>
:::lead
時頼が[[三浦<ruby>泰村<rt>やすむら</rt></ruby>]]を滅ぼした戦い
:::

:::last
幕府内において、北条氏に対抗できる最後の勢力が消滅しました。
:::

### 朝廷との密接な関係

:::top
北条時頼は、朝廷に政治の刷新と制度の改革を求めました。
:::

:::last
朝廷に、幕府が認めた公家から選抜した評定衆を置かせ、評定衆で構成する評定を院政の最高議決機関とさせました。
:::

::arrow

:::

:::last
幕府は朝廷の内部に深く影響力を持つようになりました。
