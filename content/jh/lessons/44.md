---overview---
承久の乱後、３代執権の北条泰時は、合議で最高政務・裁判をおこなう体制を整えました。また、武家最初の体系的法典である御成敗式目を制定しました。次第に執権政治は確立されていき、幕府の政治は新しい段階に入りました。しかし、北条氏は５代執権の頃に独裁的な性格を強め、合議も北条氏出身の者のみでおこなわれていきました。
---

## 執権政治の盤石化
### ５代執権の登場
::top
５代執権である、北条泰時の孫[[北条時頼]]は以下の政策を実施して、北条氏の執権政治を強化し、次第に独裁的な性格を強めていきました。
::
::gazo
![](img/44/2.jpg){.half}
北条時頼
::
::middle
北条時頼は、裁判の迅速化とそれによる御家人の信頼獲得を考えました。
::
::last
1249年、次の３段階を踏むことで、迅速で公正な裁判を確立しました。
::

- 定員４～５人の[[引付衆]]を任命
- 引付衆で構成する会議[[引付]]が判決原案を評定に提出
- 判決原案にほぼ従い、[[評定]]が最終的な裁断

### 対北条氏の勢力消滅
::top
1226年、[[藤原頼経]]が４代将軍に就任しました。
::
::middle
頼経は在職の間に一部の御家人（三浦氏など）と親近な関係をもっていきました。
::
::middle
それら一部の御家人は、北条氏が御家人を統制する上での障害になりました。
::
::last
北条時頼は、頼経の子[[藤原{{頼嗣|よりつぐ}}]]を５代将軍に就任させ、後に頼経を京に送還しました。

### 朝廷との密接な関係
::top
北条時頼は、朝廷に政治の刷新と制度の改革を求めました。
::
::last
