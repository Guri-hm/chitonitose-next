## 社会運動

### 様々な運動

:::top
第一次世界大戦による産業発展・物価上昇・労働者数増加、そして、ロシア革命をうけ、日本では次の３つの動きが活発化しました。
:::

- 待遇改善を求める運動
  :::top
  労働争議
  :::lead
  労働者の権利拡張（賃金引上げなど）を求める運動
  :::
  :::top
  小作争議
  :::lead
  小作人が地主に小作料引下げを求める運動
  :::
- 社会主義運動（①に思想的な影響）
- 対等な関係を求める運動（身分差別解消・女性軽視打破）

## 待遇改善の希求

### 労働争議

:::top
1912年、労働団体[[友愛会]]が[[鈴木文治]]によって組織されました。
:::

:::middle
1919年、友愛会は大日本労働総同盟友愛会と改称し、1920年には、権利要求の祭典として==第１回メーデーを主催しました==。
:::

:::last
1921年、大日本労働総同盟友愛会はさらに[[日本労働総同盟]]と改称し、雇い主との協力関係の模索を止め、敵視し対立する路線に進みました。
:::

:::gazo{size="half"}
![](/images/jh/img/142/1.jpg)

鈴木文治
:::

### 小作争議

:::top
1922年、小作人の全国組織[[日本農民組合]]が、杉山元治郎・[[<ruby>賀川豊彦<rt>かがわとよひこ</rt></ruby>]]らによって組織されました。
:::

### 「冬の時代」の終わり

:::top
社会主義運動は、1910年の[[<ruby>大逆事件<rt>たいぎゃくじけん</rt></ruby>]]で大弾圧を受けて以降、「冬の時代」と呼ばれる不振期に入っていました。
:::

:::last
争議・社会運動の勢いが高まる中、社会主義者たちは活動を再開しました。
:::

::arrow

:::top
社会主義者の活動や社会主義の学問的研究は、政府から制限を受けました。
:::
