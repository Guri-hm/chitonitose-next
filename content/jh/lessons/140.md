## 政党内閣と非政党内閣

### 新たな政治思想

:::top
民衆の政治意識の高揚は、大正政変で第３次[[桂太郎]]内閣を退陣させ、さらに政治思想にも大きな影響を与えました。
:::

:::last
1916年、[[吉野<ruby>作造<rt>さくぞう</rt></ruby>]]がデモクラシーの訳語[[民本主義]]を提唱しました。
:::

:::sup
民本主義
:::lead
主権在君の下で国民の利益のために政治をおこなうことであり、
			主権在民の「民主主義」とは異なる考え総合雑誌『[[中央公論]]』で発表
:::

### ビリケン内閣と米騒動

#### 寺内正毅内閣｜1916年10月～1918年９月

:::top
1916年、陸軍大将で初代[[朝鮮<ruby>総督<rt>そうとく</rt></ruby>]]を務めた[[<ruby>寺内正毅<rt>てらうちまさたけ</rt></ruby>]]が、議会に支持政党をもたない内閣「超然内閣」を組織しました。
:::

:::last
そのため、寺内正毅内閣は全政党に協力を求める「挙国一致」を標榜しました。
:::

:::gazo{size="half"}
![](/images/jh/img/140/2.jpg)

寺内正毅
:::

::arrow

:::top
民衆が政治の民主化を求める中、このような内閣が登場したことは、「非立憲」だと非難されました。
:::

:::middle
1916年、立憲同志会が改称して憲政会となり、内閣に対抗しました。
:::

:::middle
1917年、寺内正毅は衆議院を解散し、第13回総選挙を実施しました。
:::

:::last
結果、内閣に接近中の立憲政友会が、衆議院第一党となりました。
:::

:::gazo
![](/images/jh/img/140/1.svg)

第13回総選挙
:::

::arrow

:::top
寺内正毅は、立憲政友会代表の原敬と立憲国民党代表の犬養毅を、臨時外交調査委員会に取り込み、外交方針での挙国一致を図りました。
:::

::arrow

:::top
1918年、米騒動
		
		
		
		
		責任を追及する世論に、寺内正毅内閣は退陣に追い込まれました。
:::lead
下記３点を背景とする米価==高騰==に、富山県の漁村の主婦たちが蜂起し、それを機に全国の==都市でも==人々が米価引き下げ・安売りを求めた暴動背景①：工業労働者の増加と人口の都市集中による米の消費増加背景②：寄生地主制のもとでの農業生産の停滞背景③：[[シベリア出兵]]を見越した米の買い占め
:::

### 期待の内閣と民衆の失望

#### 原敬内閣｜1918年９月～1921年11月

:::top
民主的な政治を求める民衆を前に、元老もようやく政党内閣を認めました。
:::

:::middle
1918年、立憲政友会の総裁[[<ruby>原敬
			<rt>はらたかし</rt></ruby>]]が内閣を組織しました。
:::

:::last
原敬は、華族でも藩閥でもない、衆議院に議席をもつ首相であったため、「平民<ruby>宰相<rt>さいしょう</rt></ruby>」と呼ばれ、民衆から歓迎されました。
:::

:::gazo{size="half"}
![](/images/jh/img/140/4.jpg)

原敬
:::

::arrow

:::top
1918年、大学令公布
:::lead
帝国大学の他に、公立・私立大学や単科大学の設立を認めた勅令慶応義塾・早稲田など専門学校が大学として昇格・認可
:::

::arrow

:::top
民衆は、原敬に普通選挙の導入を期待しました。
:::

:::middle
しかし、原敬自身は==導入に反対で==、次の２策の実施にとどまりました。
:::

- 選挙資格を納税３円以上に引き下げ（初期は15円以上）
- 小選挙区制の導入

:::last
そのため、1920年には普通選挙を要求する大運動が起こりました。
:::

:::sup
普通選挙
:::lead
成人した者に選挙権を認める制度で、対義語は制限選挙
:::

:::gazo
![](/images/jh/img/140/5.jpg)

普通選挙を要求する尾崎行雄
:::

::arrow

:::top
憲政会などの野党は、衆議院に男性普通選挙法案を提出しました。
:::

:::middle
原敬は、時期尚早と衆議院を解散させ、第14回総選挙を実施しました。
:::

:::last
立憲政友会は、鉄道拡充・高等学校増設を掲げ、総選挙で圧勝しました。
:::

:::gazo
![](/images/jh/img/140/3.svg)

第14回総選挙
:::

::arrow

:::top
1920年、大戦後の反動で恐慌が日本を襲い、汚職事件も多発しました。
:::

:::last
1921年、原敬は政党政治の腐敗に憤激した青年に暗殺されました。
:::
