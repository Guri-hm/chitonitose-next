## 貨幣制度の改革

### 不安定な不換紙幣

:::top
当初の新政府は、財源に乏しく、商人から臨時負担金である御用金を徴発しました。
:::

:::middle
1868年、<ruby>由利<rt>ゆり</rt></ruby><ruby>公正<rt>きみまさ</rt></ruby>の立案で最初の政府紙幣[[<ruby>太政官札<rt>だじょうかんさつ</rt></ruby>]]が、次いで1869年、<ruby>民部省札<rt>みんぶしょうさつ</rt></ruby>が発行されました。
:::

:::last
太政官札・民部省札は、金銀との引換え（<ruby>兌換<rt>だかん</rt></ruby>）ができない紙幣[[<ruby>不換<rt>ふかん</rt></ruby>紙幣]]で、増刷のたびに人々からの信用を失い、価値が下落しました。
:::

:::sup

:::lead
貨幣単位は、江戸時代と同じく「<ruby>両<rt>りょう</rt></ruby>・<ruby>分<rt>ぶ</rt></ruby>・<ruby>朱<rt>しゅ</rt></ruby>」を使用

### 金本位制の確立推進と失敗

:::top
江戸時代、金貨は「金１両＝４分」「１分＝４朱」と、４で位が１つ進みました。
:::

:::last
また、金貨・銀貨の交換は「金１両＝銀50<ruby>匁<rt>もんめ</rt></ruby>」と繁雑で、外国人から嫌がられました。
:::

::arrow

:::top
1871年、新貨条例
:::lead
「１[[円]]＝100[[銭]]」「１[[銭]]＝10[[厘]]」と、==10進法で==位を進める新たな貨幣単位を設定し、「１両＝１円」として旧単位を廃止純金1.5gを含有する１円金貨を鋳造し、貨幣への信用を「金」の希少価値で支える[[金本位]]を採用
:::

:::gazo{size="twice"}
![](/images/jh/img/112/5.jpg)

金本位（左：50銭銀貨／中：１円金貨／右：１厘銅貨）
:::

::arrow

:::top
しかし、新政府の純金の保有量は少なく、１円金貨の不足を１円銀貨の鋳造で補いました。
:::

:::middle
従って、貨幣への信用を「金」「銀」の希少価値で支える金銀複本位となりました。
:::

:::last
加えて、人々は金貨の使用を惜しみ、貿易では銀貨、国内では紙幣が主に使用されました。
:::

:::gazo
![](/images/jh/img/112/6.jpg)

１円銀貨の鋳造
:::

## 国立銀行の開業

### 兌換紙幣の発行

:::top
太政官札・民部省札は、旧単位で発行され、また、不換紙幣で信用を失いやすいものでした。
:::

:::middle
新単位で、金銀との引換え（兌換）ができる紙幣[[<ruby>兌換<rt>だかん</rt></ruby>紙幣]]の発行が必要となりました。
:::

:::last
しかし、新政府の純金・純銀の保有量は少なく、仮に発行しても引換えができませんでした。
:::

:::gazo{size="twice"}
![](/images/jh/img/112/8.jpg)

金準備の不足・銀準備の不足
:::

::arrow

:::top
1872年、国立銀行条例
:::lead
不換紙幣の回収整理のために、[[<ruby>渋沢栄一<rt>しぶさわえいいち</rt></ruby>]]が起草した条例純金・純銀を保有する有力商人に、[[国立銀行]]を４<ruby>行<rt>こう</rt></ruby>開業させて、兌換紙幣である兌換銀行券の発行と、兌換銀行券と金貨との引換えを義務化
:::

:::sup
渋沢栄一
:::lead
[[第一国立銀行]]を創立し、その初代の代表者
:::

:::sup
国立銀行
:::lead

:::

:::sup
政府と組んで富を得た三井や岩崎（三菱）などの有力商人を[[政商]]と呼称

### 兌換義務の廃止と紙幣価値の下落

:::top
次第に国立銀行の保有する金貨が底をつき、経営不振に陥りました。
:::

::arrow

:::top
1876年、国立銀行条例改正
:::lead
国立銀行の「紙幣と金貨との引換え義務」を廃止国立銀行に不換紙幣の発行を許可国立銀行の自由な開業を許可
:::

::arrow

:::top
華族・士族が、<ruby>金禄公債<rt>きんろくこうさい</rt></ruby>証書を元手に、国立銀行を==次々と開業しました==。
:::

:::last
これらの国立銀行が、不換紙幣の銀行券を発行し、紙幣の価値が下落しました。
:::

:::sup
1879年、153行目の国立銀行の開業で、開業許可打ち切り（行：銀行の単位）
