## 日露戦争が変えた国際関係

### 同盟と協商の変化

:::top
日露戦争後、ロシアは東アジアでの南下を諦め、矛先を再び西側へ向けました。
:::

:::last
ロシアはバルカン半島を狙い、同半島を狙うドイツと対立を深めました。
:::

::arrow

:::top
上記の動向に応じて、国際関係は次のように変化しました。
:::

- 英―独
  :::lead
  両国が海軍力増強を目指す「建艦競争」で対立
  :::
- 日―米
  :::lead
  アメリカが[[満州]]の市場に関心を示し、日本が共同経営を拒否して急速に対立
  :::
- 英―露
  :::lead
  ドイツとの対立から、英露協商を結んで急速に接近
  :::
- 日―露
  :::lead
  満州の互いの利権をアメリカから守るために、第１～３次[[日露協約]]を結んで急速に接近
  :::

## 桂園時代

### 政界の第一線からの引退

:::top
老齢の山県有朋や伊藤博文は、政界の第一線から退きました。
:::

:::last
彼らは、==非公式に==天皇を補佐する[[元老]]として首相の選任権を握りました。
:::

### 山県と伊藤の後継者

:::top
1901年から10年間以上、山県有朋の後継者[[桂太郎]]と伊藤博文の後継者[[<ruby>西園寺公望<rt>さいおんじきんもち</rt></ruby>]]が、首相を交互に務めました。
:::

:::last
この時期を２人の苗字から１字ずつとって<ruby>桂園<rt>けいえん</rt></ruby>時代と呼びます
:::

#### 第１次桂太郎内閣｜1901年６月～1906年１月

:::top
1901年、山県の後継者[[桂太郎]]が、藩閥官僚中心の内閣を組織しました。
:::

:::middle
日英同盟協約締結から日露戦争を経てポーツマス条約締結に至る長期政権となりました。
:::

:::last
しかし、日比谷焼打ち事件で支持を失い、総辞職しました。
:::

#### 第１次西園寺公望内閣｜1906年１月～1908年７月

:::top
伊藤博文の後継者[[<ruby>西園寺公望<rt>さいおんじきんもち</rt></ruby>]]は、[[立憲政友会]]の総裁となりました。
:::

:::last
西園寺公望は、鉄道・港湾の拡充を掲げ、着実に支持を集めていきました。
:::

::arrow

:::top
1906年、政党内閣である第１次西園寺公望内閣が組織され、次のことを実施しました。
:::

- 1906年、鉄道国有法制定
  :::lead
  ==軍事上・経済上の目的から==鉄道輸送を画一化するために鉄道を国有化した法令
  :::
- 1906年に片山<ruby>潜<rt>せん</rt></ruby>らが結成した日本社会党の存続を容認
  :::lead
  日本社会党内で、革命などで社会主義実現を目指す直接行動派が、議会を通じた実現を目指す議会政策派に勝ると、遂に強制解散
  :::

:::sup
社会主義
:::lead
計画的な生産と富の均等配分で、貧富の格差消滅を目指す思想
:::

:::sup
無政府主義
:::lead
国家の政治権力を一切否定し、個人の完全な自由を目指す思想
:::

:::sup
1901年に片山<ruby>潜<rt>せん</rt></ruby>・幸徳秋水・安部<ruby>磯雄<rt>いそお</rt></ruby>・木下<ruby>尚江<rt>なおえ</rt></ruby>らが結成した日本初の社会主義政党[[社会民主]]党は、第２次山県有朋内閣で成立した[[治安警察]]法で即座に強制解散
:::

#### 第２次桂太郎内閣｜1908年７月～1911年８月

:::top
第２次[[桂太郎]]内閣は、次のことを実施しました。
:::

- 1908年、<ruby>戊申詔書<rt>ぼしんしょうしょ</rt></ruby>発布
  :::lead
  日露戦争後に芽生えた享楽的民心の是正に、倹約・勤労を説いた天皇の文書
  :::
  :::lead
  発布後、疲弊した地方自治体の財政再建・民心の向上などを目指し、政府（**内務省**）は[[地方改良運動]]を実施
  :::
  :::lead
  地方改良運動の一環として、近世の共同体の名残（青年の組織など）を、新しい行政単位で再編成（青年会など）
  :::
  :::lead
  地方改良運動の一環として、軍隊の地域社会での基盤を強化するために、1910年、予備役・後備役・退役の軍人の団体[[帝国在郷軍人会]]を設立
  :::
- 1910年、大逆事件
  :::lead
  天皇暗殺計画の発覚を機に、全国の社会主義者・無政府主義者を検挙し、うち[[幸徳秋水]]ら26名を大逆罪とし、翌年、その死刑を執行
  :::
  :::lead
  以後、社会主義運動は「冬の時代」と呼ばれる不振期に突入
  :::
  :::lead
  事件後、思想犯・政治犯を取り締まる警察[[特別高等警察]]を設置
  :::
- 1910年、韓国併合
- 1911年、工場法制定
  :::lead
  工場労働者保護のため、事業主に義務を課す法律
  :::
- 1911年、外相小村寿太郎が関税自主権を完全回復
