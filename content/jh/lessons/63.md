## 庶民を担い手とする文化

### 庶民文芸の流行

#### 連歌

:::top
[[宗鑑]]が『**犬筑波集**
		』を編集して、滑稽を主とする庶民的な[[<ruby>俳諧<rt>はいかい</rt></ruby>]]連歌（俳諧）を確立しました。
:::

:::last
宗鑑のような連歌師が各地を遍歴して普及させ、民衆の間で連歌が流行しました。
:::

:::gazo{size="half"}
![](/images/jh/img/62/18.jpg)

宗鑑（山崎宗鑑）
 室町幕府９代足利義尚に仕え、後に出家しました。俳諧連歌を興隆し、その奔放で滑稽味のある気風は、江戸時代初期の談林俳諧に影響を与えました。生没年不明。
:::

#### 芸能

:::top
田楽・猿楽から発達した[[能]]は庶民にも好まれました。
:::

:::last
能の<ruby>幕間<rt>まくあい</rt></ruby>に行われる[[狂言]]には日常的な言葉が用いられ、庶民の生活がその題材に選ばれました。

## 鎌倉仏教の動向

### 世俗を嫌った禅宗の一派

:::top
幕府の保護を受ける臨済宗の一派（五山派あるいは
		<ruby>叢林<rt>そうりん</rt></ruby>）に対して、世俗を嫌った臨済宗の一派および曹洞宗は[[<ruby>林下
			<rt>りんか</rt></ruby>]]と呼ばれました。
:::

:::last
林下は地方で布教をおこない、地方の武士や庶民の支持を得ました。
:::

::arrow

:::top
林下の寺院には、臨済宗では[[<ruby>妙心<rt>みょうしん</rt></ruby>寺]]・**大徳寺**、曹洞宗では永平寺でありました。
:::

:::last
==臨済宗==大徳寺の僧[[一休宗純]]は、五山の腐敗や堕落を批判しました。
:::

:::gazo{size="half"}

### 日蓮宗の展開

:::top
関東に留まっていた布教が、室町時代には京にも及ぶようになりました。
:::

:::middle
６代将軍足利義教の時代に現れた[[日親]]は、==他宗と激しい論戦をして、しばしば迫害を受けました==。
:::

:::middle
日蓮宗は京で信者を徐々に増やし、[[法華一揆]]を結んで他宗と衝突しました。
:::

:::last
1536年、天文法華の乱

### 浄土真宗（一向宗）の展開

:::top
応仁の乱の頃、本願寺の[[蓮如]]が現れ、[[御文]]という平易な文章で浄土真宗の教義を説きました。
:::

:::middle
また、[[講]]という門徒の集会を惣村ごとに開かせ、御文を送って信仰を固めさせました。
:::

:::middle
門徒の結束力は強大になり、各地で[[一向一揆]]が起きました。
:::

:::last
代表的な一向一揆は、1488年の[[加賀の一向一揆]]です。
:::

:::gazo{size="half"}
![](/images/jh/img/63/7.jpg)

蓮如
:::

				</rt></ruby>]]が[[足利学校]]を再興しました。
:::

:::last
足利学校は関東の学問の拠点となり、全国から人々が集まりました。
:::

![](/images/jh/img/63/8.jpg)

足利学校（所在地：栃木県）
足利学校は日本で最も古い学校として知られます。しかし、その創建については未だ不明で、明らかな歴史は上杉憲実の再興以降のみです。
