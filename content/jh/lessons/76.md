## 近隣諸国との外交

### 琉球王国の二重外交

:::top
1609年、<ruby>薩摩<rt>さつま</rt></ruby>藩主島津家久は[[琉球王国]]（1423年～）を征服しました。
:::

:::middle
薩摩藩は、琉球王国を支配下に置きながらも、[[尚]]氏をその王位につかせて、名目上は独立王国として存続させました。
:::

:::middle
そして薩摩藩は、琉球王国に明（後に<ruby>清<rt>しん</rt></ruby>）との朝貢貿易をさせ、その貿易で得た中国の産物を送らせるなど利益を吸い上げました（密貿易）。
:::

:::last
また薩摩藩は、琉球王国に琉球産黒砂糖の上納を要求しました。
:::

:::top

### 朝鮮との外交

:::top
徳川家康は朝鮮（1392年～）との講和に意欲を示しました。
:::

:::middle
1609年、<ruby>対馬<rt>つしま</rt></ruby>藩[[宗]]氏と朝鮮との間に、[[<ruby>己酉<rt>きゆう</rt></ruby>約条]]が結ばれました。
:::

:::last
この条約を両国の関係の基本とし、[[<ruby>釜山<rt>プサン</rt></ruby>]]に[[倭館]]が設置されました。
:::

:::sup
15世紀以降、宗氏は朝鮮に対する窓口として、特権的地位を確立
:::

::arrow

:::top
朝鮮からは、計12回の使節が江戸時代を通じて来日しました。
:::

:::middle
初期３回は文禄・慶長の役の朝鮮人捕虜返還目的の使節でした。
:::

:::last
４回以降は<ruby>信<rt>よしみ</rt></ruby>を通じる修好目的の使節で、[[通信使]]と呼ばれました。

  :::lead

  :::
