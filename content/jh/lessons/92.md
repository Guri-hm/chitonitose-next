---overview---
18世紀、全国に様々な教育施設が設立されました。中でも寺子屋の貢献は大きいものでした。一般民衆が「読み・書き・そろばん」を学び、文字文化の享受が可能になりました。また、同時期の出版業の貢献も欠かせません。出版業は教科書用の書物を刊行し、学習活動を促進させました。そして、読み手となった民衆が出版業を支えて相関しました。
---
## 学問の発展と普及
### 正学と異学
::top
18世紀、新たな学問・思想の台頭に対し、幕府は儒学による武士の教育を奨励しました。
::
::middle
18世紀末、幕府は[[寛政異学の禁]]を発して[[朱子学]]を“正学”とし、[[湯島聖堂]]に付属する{{林家|りんけ}}の私塾で朱子学以外の学派“異学”の講義を禁じました。
::
::last
1797年、この私塾は切り離され、幕府直轄の学問所[[昌平坂学問所]]となりました。

### 教育施設
#### 藩校
::top
全国の藩に、藩士や子弟の教育施設[[藩校]]（[[藩学]]）が設立されました。
::
::last
当初、藩校は朱子学や武術を学ばせましたが、のちには蘭学・国学も取り入れました。
::
#### 郷校
::top
藩の援助を得て、藩士や==庶民の==教育施設[[郷校]]（[[郷学]]）が設立されました。
::
::last
岡山藩主[[池田光政]]が建てた[[{{閑谷|しずたに}}学校]]は最古の郷校です。
::
::gazo
::
#### 私塾
::top
武士・学者・町人などは、各地に私塾を開き、儒学・国学・蘭学などを講義しました。
::
::gazo
::
::gazo
::
::last
浪人（{{牢|ろう}}人）・僧侶・神職・町人・女性が、「読み・書き・そろばん」を主に教えました。
::
::gazo
::
::top
教授用の教科書には、出版された書物が用いられました。
::
::last
[[貝原益軒|かいばらえきけん]]の著作をもとにした『[[女大学]]』を用いた女子教育が進展しました。
::
