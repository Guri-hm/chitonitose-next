## 源氏と平氏の武名並ぶ

### 東の源氏

:::top
清和源氏は源満仲以来、摂関家と結びつき、また、武勲を得て勢力を強めました。
:::

:::top
源[[頼信]]―[[平忠常の乱]]（1028年）
:::

:::top
源[[頼義]]―[[前九年合戦]]（1051～62年）
:::

:::top
源[[義家]]―[[後三年合戦]]（1083～87年）
:::

::arrow

:::top
義家は東国の武士の功労に私財で報い、その信頼を集めました。
:::

:::middle
これら武士は、義家の武力・地位を頼りに田地を寄進しました。
:::

:::last
白河上皇は、諸国の武士・農民が義家に寄進することを禁止しました。
:::

:::sup
寄進地系荘園は、寄進する相手次第で没収・消滅
:::

:::last
開発領主は、寄進する相手により強力で安定した者を選択
:::

### 西の平氏

:::top
白河上皇は清和源氏への対抗勢力として、伊勢平氏に目をつけました。
:::

:::last
伊勢平氏の平正盛は北面の武士に任命され、また、1108年に源義家の子源<ruby>義親<rt>よしちか</rt></ruby>が出雲国で起こした乱を鎮圧しました。
:::

:::sup

## 武士に頼った２つの乱

### 嫌われる「叔父子」

:::top
白河上皇は養子の藤原<ruby>璋子<rt>しょうし</rt></ruby>を鳥羽天皇に<ruby>入内<rt>じゅだい</rt></ruby>させました。
:::

:::middle
璋子に子が生まれましたが、実は白河と璋子の子であり、鳥羽は子を「<ruby>叔父子<rt>おじご</rt></ruby>」と呼んで忌み嫌いました。
:::

:::last
白河は鳥羽に迫って譲位させ、叔父子[[崇徳]]天皇を即位させました。
:::

:::gazo
![](/images/jh/img/38/1.svg)

崇徳天皇関係の系図
:::

::arrow

:::top
白河の死後、鳥羽が院政を始めると、鳥羽は崇徳に迫って譲位させ、やがて[[後白河]]天皇が即位しました。
:::

:::middle
崇徳はこの皇位継承に不満を抱き、後白河と対立しました。
:::

:::last
この対立に摂関家の内部対立が加わり、両者の激突は一触即発となりました。
:::

### 上皇と天皇の対立

:::top
鳥羽の死後、上皇と天皇の対立に一挙に火がつき、武力衝突に発展しました。
:::
