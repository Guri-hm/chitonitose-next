## 軍拡の要求と時代の転換

### 多方面からの要求と伝家の宝刀

#### 第２次西園寺公望内閣｜1911年８月～1912年12月

:::top
1911年、国家財政の悪化の中、第２次[[<ruby>西園寺公望
			<rt>さいおんじきんもち</rt></ruby>]]内閣が成立しました。
:::

:::last
西園寺公望内閣は、次の方面から財政に関する要求を受け、困難に立たされました。
:::

- 与党立憲政友会：経済成長のための支出増加
- 陸軍：辛亥革命の朝鮮半島への波及を防ぐための２個師団増設
- 海軍：戦艦・装甲巡洋艦各８隻の建造計画八・八艦隊

:::gazo
![](/images/jh/img/137/1.jpg)

陸軍・海軍の軍拡要求

＊前：草刈り（財政整理）する西園寺

＊後：勝手に杭を打つ陸軍・海軍
:::

::arrow

:::top
西園寺公望内閣は、陸軍拡張よりも海軍拡張を優先しました。
:::

:::middle
陸軍およびその巨頭である<ruby>山県有朋<rt>やまがたありとも</rt></ruby>は、２個師団増設を強く求めました。
:::

:::last
西園寺公望内閣は、師団増設を財政上の理由から拒否しました。
:::

::arrow

:::top
陸軍大臣**上原<ruby>勇作<rt>ゆうさく</rt></ruby>**が、単独で辞表を天皇に提出しました。
:::

:::last
西園寺公望内閣は、陸軍が上原勇作の後任の陸相を出すことを拒否したため、軍部大臣現役武官制を守れず、1912年に総辞職しました。

### 時代の転換

#### 新時代の到来

:::top
1912年７月、明治天皇の死去に伴い、[[大正天皇]]が即位しました。
:::

:::last
天皇の最高顧問元老の山県有朋は、大正天皇の内大臣兼侍従長に、長州出身で陸軍の長老である桂太郎を選びました。
:::

:::sup
元老
:::lead
==憲法に規定されていないが==、天皇を補佐して首相の選任権を掌握
:::

#### 天皇の在り方

:::top
大日本帝国憲法下での天皇の位置づけには、統一見解がありませんでした。
:::

:::middle
1912年、東京帝国大学教授[[<ruby>美濃部達吉
			<rt>みのべたつきち</rt></ruby>]]が、『憲法講話』を刊行し、国家という枠の下に天皇を位置づける学説[[天皇機関説]]を唱えました。
:::

:::last
天皇機関説は、憲法解釈の定説として支持されました。
:::

:::sup
天皇機関説
:::lead
国家が天皇の上にあり、たとえ天皇でも憲法に従うとする説
:::

:::sup
天皇主権説
:::lead
国家が天皇の下にあり、天皇は憲法の制限を受けないとする説
:::

## 民衆の力と政治意識

### 大正政変

#### 第３次桂太郎内閣｜1912年12月～1913年２月

:::top
元老らの間で、次の首相に[[桂太郎]]の名があがりました。
:::

:::last
内大臣兼侍従長の桂太郎が首相に就任することは、宮中と政府（府中）の境界を乱すと非難されました。
:::

::arrow

:::top
立憲政友会の[[尾崎<ruby>行雄<rt>ゆきお</rt></ruby>]]や**立憲国民党**の[[<ruby>犬養毅
			<rt>いぬかいつよし</rt></ruby>]]に民衆が加わり、次の２つを訴えて倒閣を目指す運動[[第一次護憲運動]]を始めました。
:::

- <ruby>閥族<rt>ばつぞく</rt></ruby>打破
  :::lead
  薩長の出身者が首相の選任などに力をもつ現状の打破
  :::
- 憲政擁護
  :::lead
  憲法に従った、民衆による真の政治の実現
  :::

:::gazo{size="half"}
![](/images/jh/img/137/5.jpg)

尾崎行雄
:::

:::gazo{size="half"}
![](/images/jh/img/137/6.jpg)

犬養毅
:::

:::gazo
![](/images/jh/img/137/7.jpg)

尾崎行雄の弾劾演説
:::

::arrow

:::top
桂太郎は、批判の原因「元老の後ろ盾を得た政治」を止め、支持母体となる新たな政党の結成を進めました。
:::

:::middle
立憲政友会・立憲国民党が、内閣不信任案を議会に提出し、それを支持する民衆が議会を包囲しました。
:::

:::middle
1913年２月、内閣は50日余りで退陣しました。
:::

:::last
退陣に追い込んだ一連の動きを[[大正政変]]と呼びます。
:::

:::sup
桂太郎の支持母体に、加藤高明を総裁とする[[立憲同志会]]が結成
:::

### 寛容な内閣の登場と退陣

#### 第１次山本権兵衛内閣｜1913年２月～1914年４月

:::top
海軍大将[[山本<ruby>権兵衛<rt>ごんべえ</rt></ruby>]]が、[[立憲政友会]]を与党に内閣を組織しました。
:::

:::last
山本権兵衛内閣は、次のことを実施し、軍部・官僚に対する政党の影響力を強めました。
:::

- 1913年、軍部大臣現役武官制改正
  :::lead
  陸軍大臣・海軍大臣の候補に、予備・後備役の大将・中将を追加した改正
  :::
- 1913年、文官任用令改正
  :::lead
  任用の制限を緩和して、政党員が官僚（文官）になることを容認した改正
  :::

:::gazo{size="half"}
![](/images/jh/img/137/8.jpg)

山本権兵衛

:::top
1914年、ジーメンス事件

:::

:::last
この事件は、海軍出身の山本権兵衛に対する非難を招き、内閣を退陣させました。

### 民衆に人気の内閣

:::top
元老らは、民衆の非難による内閣の退陣をみて、民衆に人気のある[[大隈重信]]を次の首相に起用しました。
:::

:::last
1914年、大隈重信内閣は**立憲同志会**を与党に内閣を組織しました。
:::

::arrow

:::top
