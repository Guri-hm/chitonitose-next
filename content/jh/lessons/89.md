---overview---
享保の改革の後、すなわち18世紀後半、村・町における貧富の差が拡大し、幕藩体制は大きく動揺しました。村・町では争いが急増し、都市部では出稼ぎ目的の流入民が治安悪化を招きました。また、自然災害やそれに伴う大飢饉も起き、幕政はその対応に追われました。人心はますます不安定になり、民衆運動が増加していきました。
---
## 社会の変容
### 村の変容
#### 田地を収集した百姓
::top
[[村方三役]]（[[名主なぬし ]]・[[組頭]]・[[百姓代]]）を務める有力な百姓は、困窮する百姓に金を貸し、質にとった田地を集めて地主に成長しました。
::
---arrow---
::top
地主は次の２つの耕作形態で田地を経営した。
::

- 地主{{手作|てづくり}}
  :::lead
  期間を定めて雇った{{零細|れいさい}}な百姓とともに田地を直接経営
  :::
- 零細な百姓に田地を貸し、{{小作|こさく}}人として経営させて小作料（土地使用料）を徴収

---arrow---
::top
一部の地主は、村において商業行為に{{携|たずさ}}わり、流通・金融の担い手となりました。
::
::last
このような地主を豪農（{{在郷|ざいごう}}商人）と呼びました。
::

#### 17世紀末からの百姓一揆
::top
武力蜂起を伴う全村民による[[惣百姓一揆]]が各地で起こりました。
::
::middle
規模が藩全域に及んだ場合、これを{{全藩|ぜんぱん}}一揆と呼びます。
::
::last
全藩一揆としては、1686年の{{信濃|しなの}}松本藩の{{嘉助|かすけ}}騒動、1738年の{{陸奥|むつ}}{{磐城|いわき}}{{平|たいら}}藩の元文一揆が有名です。
::
::gazo
![](img/89/5.svg)

::last

1733年、江戸の民衆が米価急騰の原因を有力な米問屋の仕業と考え、江戸時代最初の打ちこわしが起こりました。
