## 社会の変容

### 村の変容

#### 田地を収集した百姓

:::top
[[村方三役]]（[[<ruby>名主
				
			<rt>なぬし</rt></ruby>]]・[[組頭]]・[[百姓代]]）を務める有力な百姓は、困窮する百姓に金を貸し、質にとった田地を集めて地主に成長しました。
:::

::arrow

:::top
地主は次の２つの耕作形態で田地を経営した。
:::

- 地主手作てづくり
  :::lead
  期間を定めて雇った<ruby>零細<rt>れいさい</rt></ruby>な百姓とともに田地を直接経営
  :::
- 零細な百姓に田地を貸し、小作こさく人として経営させて小作料（土地使用料）を徴収

::arrow

:::top
一部の地主は、村において商業行為に<ruby>携<rt>たずさ</rt></ruby>わり、流通・金融の担い手となりました。
:::

:::last
このような地主を豪農（<ruby>在郷<rt>ざいごう</rt></ruby>商人）と呼びました。
:::

#### 田地を失った百姓

:::top
田地を失った百姓は、雇用されるか小作人となるか、都市部に流出するしかありませんでした。
:::

::arrow

:::top
村において、村方三役を兼ねる豪農と貧しい百姓が、貧富の差から対立を深めました。
:::

:::last
貧しい百姓は、村方三役に不正追及や要求をおこなう運動[[村方騒動]]を起こしました。
:::

### 町の変容

:::top
町人地では、自分の宅地と家をもつ者[[家持]]が減少しました。
:::

:::last
町人のほとんどが、借地に家を建てる者[[地借]]、家やその一部を借りる者[[店借]]（[[借家]]）でした。
:::

::arrow

:::top
農村部からの流入民、日用稼ぎなどに従事する貧しい人々は、町の<ruby>裏長屋<rt>うらながや</rt></ruby>（<ruby>裏店
		<rt>うらだな</rt></ruby>）や城下町の<ruby>場末<rt>ばすえ</rt></ruby>に多数居住しました。
:::

:::last
特に貧しい人々は<ruby>棟割<rt>むねわり</rt></ruby>長屋に居住し、飢饉・災害などの時にはたちまち生活を破壊されました。
:::

:::sup
延焼を防ぐため、災害時に棟割長屋は破壊消火の対象
:::

:::gazo
![](/images/jh/img/89/2.svg)

町人地の住居の簡易モデル
:::

## 百姓一揆と打ちこわし

### 百姓一揆

:::top
幕府や藩の支配が原因で暮らし・生産活動が脅かされた時、百姓は広範囲で結集して反抗する運動[[百姓一揆]]を起こしました。
:::

:::last
同時に、藩の悪政に協力したとみなされた==商人や村方三役は打ちこわしにあいました==。
:::

:::sup
団結の意思表示の署名には、首謀者を隠せる[[<ruby>傘連判状
			<rt>からかされんばんじょう</rt></ruby>]]を使用
:::

:::gazo{size="half"}
![](/images/jh/img/89/1.jpg)

傘連判状
:::

:::top
土豪が中心となり、幕府や藩の支配に武力蜂起や逃散で抵抗しました。
:::

:::last
一揆の担い手には、中世の一揆の名残がまだ見られました。

#### 17世紀後半からの百姓一揆

:::top
村方三役が百姓全体の要求をまとめ、正規の手続きを無視して、将軍・藩主に直訴する一揆[[<ruby>代表越訴<rt>
					だいひょうおっそ</rt></ruby>型一揆]]が増えました。
:::

:::middle
直訴した代表者は厳罰に処され、「[[<ruby>義民
			<rt>ぎみん</rt></ruby>]]」としてよく伝説化されました。

			<rt>さくらそうごろう</rt></ruby>]]や<ruby>上野<rt>こうずけ</rt></ruby>の<ruby>磔<rt>はりつけ</rt></ruby><ruby>茂左衛門

### 打ちこわし

:::top
町人や百姓は、不正を働いたとみなした商人・役人に対して、その家屋や家財を破壊する行為[[打ちこわし]]に及びました。
:::

:::last
百姓一揆では、領主の悪政と結びついた商人・村方三役に対しておこないました。
:::

:::gazo

#### 享保期の打ちこわし

:::top
1732年、天候不順と西日本一帯でのイナゴ・ウンカの大量発生で大凶作となり、[[享保の飢饉]]と呼ばれる全国的な<ruby>飢饉<rt>ききん</rt></ruby>となりました。
:::

:::last
1733年、江戸の民衆が米価急騰の原因を有力な米問屋の仕業と考え、江戸時代最初の打ちこわしが起こりました。
:::

:::sup
享保の飢饉
:::lead
甘藷の有用性が見出され、８代将軍[[徳川吉宗]]がその栽培を奨励
:::

#### 天明期の打ちこわし

:::top
1782年の冷害、翌年の<ruby>浅間山<rt>あさまやま</rt></ruby>の大噴火などが重なり、[[天明の飢饉]]と呼ばれる東北・関東を中心とした全国的な飢饉となりました。
:::

:::middle
全国で百姓一揆、江戸や大坂などで打ちこわしが発生しました。
:::

:::last
10代将軍[[徳川家治]]のもとで実権を握った[[田沼意次]]は、噴火や飢饉をその政治のせいとされ、やがて失脚しました。
:::
