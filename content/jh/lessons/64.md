## 戦国時代へ

### 東国のひと足早い戦国時代

:::top
将軍と鎌倉公方の、鎌倉公方と関東管領の対立を契機に、1438年に[[永享の乱]]、1440年に結城合戦が起きました。
:::

:::middle
足利持氏の子足利<ruby>成氏<rt>しげうじ</rt></ruby>が鎌倉公方に就任しました。
:::

:::middle
成氏と上杉氏が対立し、1454年に[[享徳の乱]]が始まりました。
:::

:::last
幕府が上杉氏を支援したため、成氏は幕府とも対立しました。
:::

::arrow

:::top
[[足利成氏]]は幕府の勢力に押されて下総国[[<ruby>古河<rt>こが</rt></ruby>]]に逃れ、[[古河公方]]と呼ばれました。
:::

:::last
対抗上、８代将軍の兄足利<ruby>政知<rt>まさとも</rt></ruby>が新たな鎌倉公方に任じられたが、伊豆国[[<ruby>堀越<rt>ほりごえ</rt></ruby>]]にとどまり、[[堀越公方]]と呼ばれました。
:::

:::sup
鎌倉は争いで焼け、また、反対勢力のせいで入れない状態
:::

:::gazo
![](/images/jh/img/64/1.svg)

西から順に堀越・鎌倉・古河
:::

::arrow

:::top
上杉氏も[[<ruby>山内<rt>やまのうち</rt></ruby>]]・[[<ruby>扇谷<rt>おうぎがやつ</rt></ruby>]]に分裂して争い始めました。
:::

:::last
他の勢力も争いを始め、東国は西国よりも早く戦国時代へ突入しました。
:::

:::sup
西国は15世紀末（応仁の乱の終結後）から戦国時代へ突入
:::

## 戦国大名

### 戦国大名の登場

:::top
戦国時代には、既成の権力に頼らず、実力で成り上がる者が増えました。
:::

:::middle
なかには自ら支配する領域[[分国（領国）]]を作りあげ、独自の支配をおこなう者も現れました。
:::

:::last
こうした者を[[戦国大名]]と呼びます。
:::

:::sup
戦国大名
:::lead
大半は守護・守護代・国人出身の者

:::top

:::

:::middle
