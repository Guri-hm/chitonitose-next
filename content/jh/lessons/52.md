### 後醍醐天皇の悩み

:::top
1318年、[[大覚寺]]統から[[後醍醐]]天皇が即位しました。
:::

:::middle
後醍醐は朱子学の大義名分論に影響を受け、幕府の存在自体に批判的でした。
:::

:::last
朝廷の権限回復のため、意欲的に諸改革に取り組みました。
:::

:::sup
後醍醐天皇
:::lead
<ruby>諡号<rt>しごう</rt></ruby>を自ら「後醍醐」としたように、[[延喜・天暦の治]]を理想視
:::

::arrow

:::top
しかし、後醍醐の最大の悩みは、皇位継承が天皇の意思に反することでした。
:::

:::last
後醍醐は鎌倉幕府の政情不安を見て、討幕計画を開始しました。
:::

:::gazo
![](/images/jh/img/52/6.jpg)
:::explanation
後醍醐天皇（尊治親王）

		絵画に描かれるヒゲと密教法具をもつ姿は、他の天皇にない異形さを示します。建武の新政では急進的な改革を目指したため、多くの者の離反を招いてしまいました。
:::

### 幕府の政情不安

:::top
得宗北条高時のもと、内管領長崎<ruby>高資<rt>たかすけ</rt></ruby>が権勢を握っていました。
:::

:::last
この状況に加え、窮乏などで御家人の不満は高まっていました。
:::

:::gazo
![](/images/jh/img/52/7.jpg)
:::explanation
北条高時（14代執権）

		闘犬・田楽に興じ、享楽的生活を送った暗愚と、後の記録で多く伝えられています。後世の誇張はありますが、少なくとも執権という自覚をもたなかった人物と言えます。
:::

## 鎌倉幕府の倒壊

### 高まる討幕の気運

:::top
後醍醐天皇は、幕府に不満を抱く御家人・僧兵・悪党を集め、密かに討幕の計画を進めました。
:::

::arrow

:::top
1324年、正中の変
:::lead
計画の露見により後醍醐の近臣が罪に問われた事件
:::

::arrow

:::top
1331年、元弘の変
:::lead
再度の計画露見で後醍醐が挙兵し、[[隠岐]]に流された事件[[持明院]]統から[[光厳]]天皇が即位
:::

::arrow

:::top
元弘の変以後、後醍醐側・幕府側の戦いが続きました。
:::

:::middle
後醍醐天皇側では、後醍醐の皇子[[<ruby>護良
			<rt>もりよし</rt></ruby>親王]]と、悪党とされた河内の[[<ruby>楠木正成
			<rt>くすのきまさしげ</rt></ruby>]]が奮闘しました。
:::

:::last
後に後醍醐は隠岐を脱出し挙兵に加わりました。
:::

### 武将の離反と幕府の倒壊

:::top
幕府側では、軍の指揮官であった[[足利高氏]]と、高氏に同調した[[<ruby>新田義貞<rt>にったよしさだ</rt></ruby>]]が奮闘しました。
:::

:::last
後に２人は離反して、幕府に決定的打撃を与えました。
:::

::arrow

:::top
高氏は京都の[[六波羅探題]]を攻め落とし、義貞は鎌倉を攻めて得宗[[北条高時]]ら北条氏と、内管領[[長崎高資]]を滅ぼしました。
:::

:::last
1333年、鎌倉幕府は倒壊しました。
:::

:::gazo
![](/images/jh/img/52/8.jpg)
:::explanation
足利高氏（尊氏）

		源義家の時に枝分かれした源氏の分流出身。鎌倉幕府は源氏が打ち立てた政治組織であり、足利氏は自らを北条氏に取って代わる存在だと、当然自覚したでしょう。なお、高氏は後醍醐天皇の「尊治」の１字をもらい、「尊氏」と後に名乗りました。
:::

:::gazo
![](/images/jh/img/52/9.jpg)
:::explanation
新田義貞

		源義家の時に枝分かれした源氏の分流出身。足利氏と祖先を同じくしながら、鎌倉時代を通じて重んぜられませんでした。北条氏への不満は強かったと思われます。
:::
