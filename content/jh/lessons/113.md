## 文明開化

### 文明開化とは

:::top
富国強兵のための具体的政策の１つは、思想・学問・教育・生活様式など、様々な西洋文明を急速に摂取する[[文明開化]]でした。
:::

### 思想

:::top
儒学・神道の考え方や慣習が排斥され、自由主義・個人主義などの西洋思想が流入しました。
:::

:::last
人は生まれながらに自由・平等で、幸福追求の権利をもつという[[<ruby>天賦<rt>てんぷ</rt></ruby>人権]]思想は、後の自由民権運動にも大きな影響を与えました。
:::

### 教育

:::top
1871年、文部省が新設され、教育・学芸関連を司りました。
:::

:::middle
1872年、==フランスに倣った==学校制度の法令[[学制]]を公布し、序文「<ruby>被仰出書<rt>おおせいだされしょ</rt></ruby>」で「学問は身を立てる財本」と立身出世主義を強調しました。
:::

:::last
全国を大・中・小の３段階の学区に分け、小学区に初等教育機関[[小学校]]を置きました。
:::

::arrow

国民皆学を目指す理想が現実と<ruby>乖離<rt>かいり</rt></ruby>し、就学率は、==授業料の徴収==などを理由に低いものでした。

### 宗教

#### 神道の純化と国教化

:::top
新政府は、宗教権威者（天皇）が政治権力者を兼ねる「祭政一致」の立場をとりました。
:::

::arrow

:::top
1868年、神仏分離令制定
:::lead
==神道の国教化を==目的として、神道の純化のために神仏習合を禁じた法令堕落した僧侶への怒りから、仏教を排斥する行動[[<ruby>廃仏毀釈<rt>はいぶつきしゃく</rt></ruby>]]が全国で発生
:::

:::top
1870年、<ruby>大教宣布<rt>たいきょうせんぷ</rt></ruby>の<ruby>詔<rt>みことのり</rt></ruby>発布
:::lead
神道の国教化の方針を示し、これに沿った神社制度・祝祭日などを制定させた詔祝祭日には、[[紀元節]]（神武天皇即位の２月11日）、[[天長節]]（明治天皇誕生の11月３日）などを制定
:::

:::sup
戦後、明治時代以降の国家中心で形成・振興した神道を国家神道と呼称
:::

#### キリスト教の黙認

:::top
新政府は、江戸幕府同様の禁教政策をとり、隠れキリシタンを迫害しました。
:::

::arrow

:::top
1868～73年、<ruby>浦上信徒<rt>うらかみしんと</rt></ruby>弾圧事件
:::lead
長崎浦上のキリシタンが公然と信教を表明し、新政府に捕らえられた事件新政府欧は米列強に抗議され、1873年、禁教の高札を撤廃してキリスト教を黙認
:::

### 新聞と雑誌

:::top
1869年、<ruby>本木昌造<rt>もとぎしょうぞう</rt></ruby>が鉛製活字の量産に成功し、日本での活版印刷が再開しました。
:::

:::last
印刷技術に助けられ、東京を中心に日刊新聞や雑誌が創刊されました。
:::

:::sup
活版印刷は、江戸時代の直前に日本に伝来したが、幕府の禁教で消滅
:::
