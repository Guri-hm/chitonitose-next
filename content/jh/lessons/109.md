## 軍事制度と警察制度

### 職業軍人の配備

### 国民皆兵のための四民平等

:::top
長州藩の[[大村<ruby>益次郎<rt>ますじろう</rt></ruby>]]は、国民皆兵による近代的軍隊の創設を発案しました。
:::

:::middle
しかし、江戸時代以降は、「武士―軍事・政治」「百姓―生産」というように、身分ごとに負担すべき役割が固定されていました。
:::

:::last
例えば、「百姓」が軍事に手を出すことは許されませんでした。
:::

:::sup
大村益次郎
:::lead
後に暗殺され、案は長州藩の奇兵隊指揮官[[<ruby>山県有朋<rt>やまがたありとも</rt></ruby>]]が実現
:::

:::gazo{size="half"}
![](/images/jh/img/109/2.jpg)

大村益次郎
:::

:::gazo
![](/images/jh/img/109/2.svg)

役割の固定
:::

::arrow

:::top
1869～71年、新政府は人々を次の３つの身分（族籍）に再編しました。
:::

- 華族
  :::lead
  公家・旧藩主に与えた呼称
  :::
- 士族
  :::lead
  旧幕臣・旧藩士に与えた呼称
  :::
- 平民
  :::lead
  百姓・商人など庶民に与えた呼称
  :::

:::middle
平民には、華族・士族との結婚、==苗字の公的使用、職業・移転の自由==を認めました。
:::

:::last
再編により、あらゆる人々（四民）が役割を平等に負担できる、<ruby>所謂<rt>いわゆる</rt></ruby>[[四民平等]]の世が実現しました。
:::

:::gazo
![](/images/jh/img/109/3.svg)

役割の平等負担
:::

::arrow

:::top
1871年、解放令（賤称せんしょう廃止令）
:::lead
江戸時代の下位の身分「えた（かわた）」「非人」を廃止し、制度上の身分を平民とした法令新政府は、実質的な差別是正をせず、人々の差別意識は継続
:::

### 国民皆兵の実現

:::top
1872年、徴兵告諭
:::lead
四民平等に基づく国民皆兵を人々に告知したこと「兵役で国に命（血）を捧げること」は、仏語で「impôt du sang」と表現され、新政府がこれを「<ruby>血税<rt>けつぜい</rt></ruby>」と直訳したため、<ruby>生血<rt>いきち</rt></ruby>を採られるという誤解が発生
:::

::arrow

:::top
1873年、徴兵令
:::lead
徴兵告諭を具体化し、国民の兵役を定めた法令==身分の区別なく、満20歳に達した男性から選抜し==、３年間の兵役を義務化==兵役免除の規定があり==、戸主と跡継ぎ、官吏や学生、==代人料上納==（270円）など
:::

## 士族の没落

### 財政負担と秩禄処分

:::top
版籍奉還・廃藩置県を経て、華族（旧藩主）・士族（旧藩士）の収入源は、新政府から支給される次の２つになりました。
:::

- 家禄
  :::lead
  華族には藩の年貢収入の10％相当を、士族には俸禄を大幅に減らした額を米で支給したもの
  :::
- 賞典禄しょうてんろく
  :::lead
  明治維新の功労者に米で支給したもの
  :::

:::last
①②を合わせて[[<ruby>秩禄<rt>ちつろく</rt></ruby>]]と呼び、新政府の毎年の歳出の約30％を占めました。
:::

### 士族の没落と救済

:::top
多くの士族が、わずかな受給額を元手に商売に手を出しました。
:::

:::last
慣れない商売で失敗する士族が多く、その失敗を「士族の商法」と呼びました。
:::

:::gazo
![](/images/jh/img/109/5.jpg)

｢士族の商法｣の風刺画
:::

::arrow

:::top
新政府は、困窮する士族の救済に、就業奨励策[[士族授産]]を講じました。
:::

:::middle
士族授産の一例は、北海道開拓と北辺防備にあたる農兵[[<ruby>屯田兵<rt>とんでんへい</rt></ruby>]]です。
:::

:::last
しかし、ほとんど失敗に終わり、新政府に反発する不平士族が増加しました。
:::
