## 天保の改革

### 質素倹約と風俗取締り

:::top
倹約令や風俗取締令を発令して、将軍・<ruby>大奥<rt>おおおく</rt></ruby>も対象に次のことを決めました。
:::

- 贅沢ぜいたく品や華美な衣服、高価な菓子・料理などの禁止
- 江戸の寄席よせ211軒を15軒まで減少
- 歌舞伎かぶきを浅草のはずれに移動し、町を歩く役者に編笠あみがさ着用を義務化
- 『<ruby>春色梅児誉美<rt>しゅんしょくうめごよみ</rt></ruby>』など人情本の作家為永春水と、『<ruby>偐紫田舎源氏<rt>にせむらさきいなかげんじ</rt></ruby>』など合巻の作家<ruby>柳亭種彦<rt>りゅうていたねひこ</rt></ruby>を処罰

### 物価騰貴の抑制

#### 原因への誤った対処

:::top
物価騰貴の原因は、<ruby>十組問屋<rt>とくみといや</rt></ruby>など幕府に公認された仲間[[株仲間]]が、<ruby>上方<rt>かみがた</rt></ruby>市場からの商品流通を独占しているからと考えられました。
:::

:::middle
物価を引き下げるために、**株仲間解散令**を発令し、株仲間の解散を命じました。
:::

:::last
実際の原因は生産地からの供給不足にあり、解散はむしろ江戸への輸送量を乏しくしました。
:::

#### 騰貴による問題への対処

:::top
物価騰貴は、旗本・御家人の生活を圧迫していました。
:::

:::last
<ruby>棄捐令<rt>きえんれい</rt></ruby>を発令して、<ruby>札差<rt>ふださし</rt></ruby>に<ruby>貸金
		<rt>かしきん</rt></ruby>返済の帳消しと低利の貸金を命じました。
:::

### 海岸防備と救済の失敗

:::top
川越藩は<ruby>相模<rt>さがみ</rt></ruby>の海岸防備の負担に苦しみ、領地の移動（<ruby>転封<rt>てんぽう</rt></ruby>）を嘆願しました。
:::

:::middle
幕府は願いを聞き入れ、川越藩主の領地を庄内藩に、庄内藩主の領地を長岡藩に、長岡藩主の領地を川越藩に移動しようとしました。
:::

:::middle
このように３藩を同時に入れ替えることを[[三方領知（領地）替え]]と呼びます。
:::

:::last
三方領知（領地）替えは、[[庄内]]藩の領民の強い反対が起きたことで撤回されました。
:::

### 幕府権力の再強化と不満

:::top
1843年、徳川家慶は幕府権力の再強化を意図し、日光東照宮の参詣を実行しました。
:::

:::last
莫大な費用と人手を要したため、財政悪化と動員された人々の不満を招きました。
:::

### 最大の失策と水野忠邦の退場

:::top
1843年、水野忠邦は財政の再建や対外の防備強化を意図し、[[上知令]]を発令して、江戸・大坂周辺の約50万<ruby>石<rt>ごく</rt></ruby>の土地を直轄地にしようとしました。
:::

:::last
強い反対によって上知令は撤回されました。
:::

::arrow

:::top
1843年、上知令の撤回を契機に、水野忠邦は退陣させられました。
:::

:::last
天保の改革は失敗に終わり、印旛沼の干拓も中止されました。
:::

## 改革失敗の２つの背景

### 農村の変化

:::top
19世紀初め、北関東の人口は１世紀前と比べて3割減り、田地は荒廃していました。
:::

:::last
つまり、農村の年貢米を財政基盤とした幕藩体制は、すでに行き詰まっていました。
:::

::arrow

:::top
19世紀前半、荒廃対応の農業指導者として次の２人が現れました。
:::

- 二宮尊徳
  :::lead
  勤労・倹約を中心とする事業法[[報徳仕法]]を実施した人物
  :::
- 大原幽学
  :::lead
  人の本性を論じる実践学問「性学」を説いた人物
  :::

### 経済の変化

:::top
18世紀、問屋は、産地に資金・原料を==貸与して==生産させる[[問屋制家内工業]]を始めました。
:::

::arrow

:::top
19世紀、一部の問屋・地主は、作業場を設けてそこに労働者を集め、分担・協力による手作業で生産させる[[工場制手工業]]（[[マニュファクチュア]]）を始めました。
:::

:::last
工場制手工業は、尾張の綿織物業、北関東の絹織物業、池田・<ruby>灘<rt>なだ</rt></ruby>などの酒造業で見られました。
:::

::arrow

:::top
諸藩は、19世紀の新しい経済活動を取り入れ、次の改革をおこないました。
:::

- 藩営工業
  :::lead
  諸藩が直営したマニュファクチュアの総称
  :::
- 藩専売制
  :::lead
  諸藩がおこなった特産物などの専売制度
  :::
