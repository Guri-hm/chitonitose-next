## ５代将軍の治世

### 側用人の設置と台頭

:::top
1680年、４代将軍[[徳川家綱]]が病没しました。
:::

:::last
徳川家綱の弟[[徳川綱吉]]が５代将軍に就任しました。
:::

:::gazo
![](/images/jh/img/82/1.jpg)

徳川綱吉
:::

::arrow

:::top
当初の補佐役は、大老の**堀田<ruby>正俊<rt>まさとし</rt></ruby>**でした。
:::

:::middle
後に堀田正俊が老中らの勤務部屋（将軍の居室の
		<ruby>傍<rt>そば</rt></ruby>）の近くで殺され、万が一に備え、この部屋を将軍の居室から遠く離れた場所に移しました。
:::

:::last
これによって、将軍と老中の間を取り次ぐ役職**<ruby>側用人<rt>そばようにん</rt></ruby>**
		が設置されました。
:::

::arrow

:::top
将軍の側に仕え、取り次ぎをする側用人は、時に政務に介入する力をもちました。
:::

:::last
特に側用人の**<ruby>柳沢吉保<rt>やなぎざわよしやす</rt></ruby>**は、堀田正俊の死後の補佐役を務めました。
:::

:::gazo
![](/images/jh/img/82/2.jpg)

柳沢吉保

### 朱子学に影響された政策

#### 朱子学の重視と第１条の文言

:::top
徳川綱吉は朱子学を重視し、朱子学者木下順庵に学びました。
:::

:::last
綱吉は、朱子学が説く主君への忠、父祖への孝、社会規範を意味する礼で秩序を安定させようとしました。
:::

:::gazo{size="half"}
![](/images/jh/img/82/3.jpg)

木下順庵
:::

::arrow

#### 朝廷の重視

:::top
朱子学の本来の教えに従えば、朝廷を重んじることが秩序の在り方です。
:::

:::last
綱吉は朝廷政策を改め、儀礼復興と天皇家の領地[[禁裏御料]]の増加を決めた。
:::

#### 忠と法の矛盾

:::top
1701年、赤穂あこう事件
:::lead
赤穂藩主浅野<ruby>長矩<rt>ながのり</rt></ruby>が<ruby>吉良義央<rt>きらよしなか</rt></ruby>に恥辱を受けたとして、赤穂藩浪士らが吉良を討った事件浪士らの起こした騒動を、「忠」で評するか「法」で裁くかで一<ruby>悶着<rt>もんちゃく</rt></ruby>
:::

:::lead

:::

### 仏教に影響された政策

:::top
1685年、生類憐みの令
:::lead
<ruby>捨子<rt>すてご</rt></ruby>や生類<ruby>殺生<rt>せっしょう</rt></ruby>などを禁止する保護法令の総称犬は極端な保護対象とされ、野犬の横行減少に効果あり
:::

:::gazo{size="half"}
![](/images/jh/img/82/5.jpg)
:::explanation
大事な「御犬様」―徳川綱吉
疑問の余地もありますが、生類憐みの令は、跡継ぎの子を求める綱吉に、僧が「生類を愛し、戌年生まれの綱吉は特に犬を大事にすること」と助言したためと言われます。

		犬食や野犬被害の減少に一役買いましたが、人々は悪令と呼び、廃止を大変喜びました。
:::

### 財政難と貨幣改鋳の「出目」

:::top
幕府の財政は、次の①②で<ruby>破綻<rt>はたん</rt></ruby>に追い込まれました。
:::

:::list{class="en"}
- 支出増：1657年の明暦の大火後の江戸復興費用
- 収入減：鉱山の金銀産出量の減少（徳川綱吉の治世期以降）
:::

::arrow

:::top
綱吉は、<ruby>勘定吟味役<rt>かんじょうぎんみやく</rt></ruby>の[[<ruby>荻原重秀<rt>おぎわらしげひで</rt></ruby>]]が意見した収入増の方策を採用しました。
:::

:::middle
荻原重秀は、小判の金<ruby>含有率<rt>がんゆうりつ</rt></ruby>を減らすことで、より多くの小判<ruby>鋳造<rt>ちゅうぞう</rt></ruby>を可能にしました。
:::

:::last
「出目（改鋳差<ruby>益金<rt>えききん</rt></ruby>）」は幕府の財政を一時潤しましたが、貨幣価値の下降と物価の上昇を招き、人々の生活を苦しませました。
:::

### ３歳の将軍と朱子学者の政策

:::top
1712年、６代将軍徳川家宣は、在職３年で死去しました。
:::

:::last
家宣の子[[徳川<ruby>家継<rt>いえつぐ</rt></ruby>]]が３歳で７代将軍に就任しました。
:::

:::gazo
![](/images/jh/img/82/9.jpg)

徳川家継
:::

::arrow

:::top
[[新井白石]]が、**正徳の治**
		と呼ばれる次の政策をおこないました。
:::

- <ruby>閑院宮<rt>かんいんのみや</rt></ruby>家の創設、将軍と皇女の婚約
  :::lead
  天皇家との関係を強化し、将軍個人の幼弱に影響されない将軍職の権威を確立
  :::
- 丁重すぎた通信使の待遇の簡素化
- 朝鮮からの国書に記す将軍の称号を「日本国大君」から「日本国王」に変更
- 正徳小判の鋳造
  :::lead
  物価上昇の抑制に、==<ruby>慶長<rt>けいちょう</rt></ruby>小判と同じ金含有率（84.3％）の==小判を鋳造
  :::
- 1715年、海舶互市新例の発布
  :::lead
  金銀が長崎貿易で大量流出したため、年間の貿易額と貿易隻数を制限
  :::
  :::lead
  清船は年間30隻（1688年の70隻から）、オランダ船は年間２隻
  :::

:::sup
通信使
:::lead
対馬藩で朝鮮との外交を担当した儒学者**<ruby>雨森芳洲<rt>あめのもりほうしゅう</rt></ruby>**は、待遇の簡素化を批判
:::
