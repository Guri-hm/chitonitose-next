## 摂関政治

### 天皇の政務

:::top
天皇の政務は、次の３つです。
:::

- 報告・陳情への対応決裁
  :::lead
  諸国からの報告・陳情への対応は、<ruby>公卿<rt>くぎょう</rt></ruby>（大臣・納言の総称）の会議で決め、それを天皇が決裁
  :::
- 重要課題（外交・財政など）の提議と決裁
  :::lead
  天皇が公卿に課題への意見を出させる会議**<ruby>陣定<rt>じんのさだめ</rt></ruby>**をひらき、意見を参考に決裁
  :::
- 人事の決裁
  :::lead
  叙位（位階を授けること）と<ruby>除目<rt>じもく</rt></ruby>（官職に任じること）の決裁
  :::

### 摂政と関白

:::top
摂政
:::lead
幼少の天皇に代わり、天皇の政務を執行する担当役
:::

:::top
関白
:::lead
成人した天皇の補佐役で、実質摂政と同じ
:::

::arrow

:::top
北家の者は天皇の外戚として、**<ruby>氏長者<rt>うじのちょうじゃ</rt></ruby>**や摂政・関白の地位を獲得しました。
:::

:::middle
天皇の政務の代行、特に人事権の掌握（上記「天皇の政務③」）は絶大な効果をもちました。
:::

:::last
摂関家は高位の官職を独占し、その高収入を経済的基盤にしました。
:::

:::sup
氏長者
:::lead
氏族のなかで位階第一の者が就任、氏族を管理・統率
:::

:::sup
摂関家
:::lead
北家のなかでも、特に摂政・関白を出す家柄
:::

:::last
中・下級貴族は、官職のなかでも経済的に有利な国司になりたがりました。
