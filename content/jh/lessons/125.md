## 憲政党の結成

### 内閣と政党の提携

#### 第２次伊藤博文内閣｜1892年８月～96年９月

:::top
1894～95年の日清戦争は、奇しくも内閣と政党の関係を一変させました。
:::

:::middle
第２次[[伊藤博文]]内閣は、自由党と提携し、板垣退助を内相として迎えました。
:::

:::last
また、自由党は帝国議会で軍備拡張予算の成立に協力しました。
:::

:::

#### 第２次松方正義内閣｜1896年９月～98年１月

:::top
第２次[[<ruby>松方正義
			<rt>まつかたまさよし</rt></ruby>]]内閣は、1896年に結成した進歩党と提携し、<ruby>大隈重信<rt>おおくましげのぶ</rt></ruby>を外相として迎えました。
:::

:::last
第２次松方正義内閣は、各省の官僚にも進歩党出身者を迎えたため、<ruby>松隈<rt>しょうわい</rt></ruby>内閣とも呼ばれました。
:::

### 憲政党の結成

#### 第３次伊藤博文内閣｜1898年１月～98年６月

:::top
1898年、第３次[[伊藤博文]]内閣が成立しました。
:::

:::middle
第３次伊藤博文内閣は、地租増税を目指しましたが、政党と提携できませんでした。
:::

:::last
そのため第３次伊藤博文内閣は、超然主義の立場をとりました。
:::

:::gazo
![](/images/jh/img/125/3.svg)

第５回総選挙
:::

::arrow

:::top
この立場に対して、[[自由]]党・[[進歩]]党は合同し、[[憲政]]党を結成しました。
:::

:::middle
憲政党は、衆議院に多数の議席をもち、帝国議会で強い力を示しました。
:::

:::last
第３次伊藤博文内閣は、議会を制する自信を失って総辞職しました。

## 政党内閣の試み

### 政党内閣の登場と早期の崩壊

#### 第１次大隈重信内閣｜1898年６月～98年11月

:::top
次の内閣では、首相に[[大隈重信]]が、内相に[[板垣退助]]が就任しました。
:::

:::middle
他の国務大臣も、陸・海軍大臣を除き、[[憲政]]党出身者が占めました。
:::

:::middle
従って、第１次大隈重信内閣は、政党を母体とする日本初の政党内閣であり、首相・内相の１字をとって[[<ruby>隈板
				
			<rt>わいはん</rt></ruby>]]内閣とも呼ばれる。
:::

:::last
しかし、母体の憲政党内で旧自由党・旧進歩党が対立し、安定に欠いていました。
:::

:::gazo
![](/images/jh/img/125/5.svg)

第６回総選挙
:::

::arrow

:::top
1898年、共和演説事件
:::lead
文相尾崎行雄が、発言を天皇への不敬と攻撃された騒動「仮に日本が共和制の国家なら、三井・三菱が大統領候補だ」と、金と政治の<ruby>癒着
			<rt>ゆちゃく</rt></ruby>の批判に、天皇のいない共和制を想定したことで、不敬と攻撃できる格好の材料を提供
:::

:::gazo{size="half"}
![](/images/jh/img/125/1.jpg)

尾崎行雄
:::

::arrow

:::top
失脚した尾崎行雄の後任をめぐり、憲政党は次の２党に分裂しました。
:::

:::list{class="en"}
- 旧自由党系の憲政党
- 旧進歩党系の憲政本党
:::

:::last
母体の[[憲政]]党が分裂した結果、第１次大隈重信内閣は総辞職しました。

### 政党内閣と政党への警戒

#### 第２次山県有朋内閣｜1898年11月～1900年10月

:::top
第２次[[<ruby>山県有朋<rt>やまがたありとも</rt></ruby>]]内閣は、[[憲政]]党と提携し、地租増税を実現しました。
:::

:::last
一方、次の改正・制定によって、==政党員が官僚になることを防ぎました==。
:::

- 1899年、文官任用令改正
  :::lead
  1893年公布の文官任用令の改正で、勅任官（天皇の勅命による任命）も、自由任用から試験任用に変更し、文官になることを難化
  :::
- 1900年、軍部大臣現役武官制制定
  :::lead
  現役の武官である大将・中将から、陸・海軍大臣を任用する制度
  :::
  :::lead
  政党員が陸・海軍大臣になることを防ぎ、軍部に対する政党の影響を阻止
  :::

:::sup
官僚
:::lead
行政官である「文官」と、上位の軍人である「武官」に大別

### 政党内閣の再登場と早期の崩壊

#### 第４次伊藤博文内閣｜1900年10月～1901年６月

:::top
1900年、[[立憲政友会]]を母体とする政党内閣として、第４次[[伊藤博文]]内閣が成立しました。
:::

:::last
しかし、帝国議会において貴族院の反対に苦しみ、間もなく総辞職しました。
:::

### 山県と伊藤の後継者

:::top
1901年、第１次[[桂太郎]]内閣が成立しました。
:::

:::last
以後、山県の後継者桂太郎と、伊藤の後継者[[<ruby>西園寺公望
			<rt>さいおんじきんもち</rt></ruby>]]が政界を二分しました。
:::

:::sup
山県・伊藤らは、==非公式に==天皇を補佐する[[元老]]として首相の選任権を掌握
