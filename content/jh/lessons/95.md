## 寛政の改革―外患

### ロシアの来航

:::top
伊勢の<ruby>船頭<rt>せんどう</rt></ruby>[[<ruby>大黒屋光太夫<rt>だいこくやこうだゆう
				</rt></ruby>]]は、嵐で漂流し、アムチトカ島に漂着しました。
:::

:::middle
光太夫はロシア人に救われ、1791年、ロシアの首都サンクトペテルブルクで、女帝エカチェリーナ２世に<ruby>拝謁<rt>はいえつ</rt></ruby>して帰国を許されました。
:::

:::middle
1792年、ロシア使節[[ラクスマン]]が <ruby>蝦夷地
		<rt>えぞち</rt></ruby>の[[<ruby>根室<rt>ねむろ</rt></ruby>]]に来航し、光太夫を届けるとともに通商を求めました。
:::

:::last
ラクスマンは通商と江戸入港を拒否され、長崎入港の許可証<ruby>信牌<rt>しんぱい</rt></ruby>
		を受けて帰国しました。
:::

:::sup
光太夫の見聞をもとに、桂川<ruby>甫周<rt>ほしゅう</rt></ruby>が『**<ruby>北槎聞略<rt>ほくさぶんりゃく</rt></ruby>**』を著述

### 松平定信の退陣

:::top
1793年、老中[[松平定信]]は次の出来事で対立を生み、退陣させられました。
:::

- 寛政の改革の統制や倹約令で、民衆が強く反発
- 1789～1793年、尊号一件（尊号事件）
  :::lead
  <ruby>閑院宮<rt>かんいんのみや</rt></ruby>家出身の[[光格]]天皇が、実父<ruby>典仁<rt>すけひと</rt></ruby>親王に、太上天皇の称号を贈ろうとし、松平定信が反対した事件
  :::
  :::lead
  本来幕府側に立つべき<ruby>武家伝奏<rt>ぶけてんそう</rt></ruby>の公家が、再三の同意要求をしたため、松平定信はこの公家を処分
  :::
  :::lead
  事件後、==幕府と朝廷の協調関係が緊張==
  :::
- 11代将軍徳川<ruby>家斉
				<rt>いえなり</rt></ruby>が、実父を江戸城に迎えて大御所と崇めようとし、松平定信の反対で失敗

:::sup
松平定信
:::lead
引退後、随筆『花月草紙』や自伝『**<ruby>宇下人言<rt>うげのひとごと</rt></ruby>**』を著述

## 異国の接近

### 蝦夷地政策

:::top
松平定信の退陣の頃、ロシア人が<ruby>択捉島<rt>えとろふとう</rt></ruby>に現れ、現地のアイヌと交易していました。
:::

:::last
1798年、幕府は幕臣[[近藤重蔵]]・[[最上徳内]]らを択捉島調査に派遣し、「大日本<ruby>恵登呂府<rt>えとろふ</rt></ruby>」の標柱を建てさせて日本の領土と示しました。
:::

:::gazo
![](/images/jh/img/95/2.svg)

東蝦夷地
:::

::arrow

:::top
1799年、幕府は[[東]]蝦夷地（北海道東岸部から千島）を永久直轄地とし、蝦夷地の直接経営に乗り出しました。
:::

:::last
居住するアイヌには、和人同様の風俗を強制する同化政策を進めていきました。
:::

### 対外的緊張の高まり

#### 対ロシア

:::top
1804年、ロシア使節[[レザノフ]]が、ラクスマンのもち帰った信牌を携え、[[長崎]]に来航して通商開始を求めました。
:::

:::middle
幕府は要求を全て拒んだ上で、信牌を再交付せずに使節を帰航させました。
:::

:::last
冷淡な幕府の対応に、ロシア船が報復として択捉島などを攻撃しました。
:::

### 対ロシアの緊張緩和

:::top
1811～1813年、ゴローウニン事件
:::lead
1811年、幕府が==<ruby>国後<rt>くなしり</rt></ruby>島に上陸した==ロシア艦長[[ゴローウニン]]を監禁し、1812年、対するロシアが淡路の商人[[<ruby>高田屋嘉兵衛<rt>たかだやかへえ</rt></ruby>]]を抑留した事件幕府とロシアは今後の誓約を結び、ゴローウニン・高田屋嘉兵衛が釈放されて事件解決
:::

:::last

### 対外政策の方針決定

:::top
ロシアとの緊張緩和後も、幕府は日本近海に出没する異国船に悩み、次の対策を講じました。
:::

- 全国各地の海岸線に、大砲を据え付ける砲台である台場を建設
- 1825年、異国船打払令（無二念打払令）を発令し、
			清・朝鮮・琉球王国の船以外の異国船を無差別に撃退するように指示
- 1811年、朝鮮からの使節通信使の歓迎を、対馬つしまに迎える易地聘礼
				えきちへいれいに変更

:::sup
異国船打払令
:::lead
発令以前は、衝突回避のために薪水・食糧を供給して帰国させる方針
:::

:::sup
通信使
:::lead
江戸まで迎えると莫大な費用が掛かり、日本の地理・実情が異国に<ruby>漏洩<rt>ろうえい</rt></ruby>する恐れあり
:::
