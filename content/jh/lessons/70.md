### 絵画

:::top
城郭建築の内部は、権威を視覚的に高めるために装飾されました。
:::

:::middle
<ruby>欄間<rt>らんま</rt></ruby>には<ruby>透<rt>すか</rt></ruby>し<ruby>彫
		<rt>ぼり</rt></ruby>の彫刻である欄間彫刻が施され、また、壁・<ruby>屏風<rt>びょうぶ</rt></ruby>などには[[障壁画]]が描かれました。
:::

:::middle
障壁画は、<ruby>金箔地<rt>きんぱくじ</rt></ruby>に青・緑を彩色する[[<ruby>濃絵<rt>
					だみえ</rt></ruby>]]でした。
:::

:::last
障壁画の題材には、深い教養をもたずに理解できる花鳥や風景が好まれました。
:::

:::sup
障壁画
:::lead
水墨のものと金碧濃彩のものに分かれ、特に後者を濃絵と呼称
:::

:::sup
濃絵
:::lead
視覚的に訴えやすいため、大名が見栄えや威厳強調に利用
:::

::arrow

:::top
障壁画の担い手の中心は[[狩野]]派でした。
:::

:::last
==安土城などの障壁画を描いた==[[狩野永徳]]は、水墨画と大和絵の技法を融合して障壁画を大成しました。
:::

## 民衆の生活

### 茶

:::top
[[千利休]]は茶の儀礼を定め、茶の湯（茶道）、特に佗茶を大成しました。
:::

:::last
千利休の佗茶は簡素を重んじ、華やかな桃山文化でも異色を放ちました。
:::

:::sup
妙喜庵茶室
:::lead
豊臣秀吉の命を受けて千利休が考案・造作した茶室
:::

:::gazo{size="half"}
![](/images/jh/img/70/14.jpg)

千利休
:::

::arrow

:::top
茶の湯は、諸大名の保護を受けて大いに流行しました。
:::

:::last
1587年、秀吉は京都北野で茶会（北野<ruby>大茶湯<rt>おおちゃのゆ</rt></ruby>）を開き、千利休を含む茶人を中心に、貧富・身分の別なく民衆を参加させました。
:::
