## 1946～48年の内閣

### 経済の再建政策

#### 幣原喜重郎内閣｜1945年10月～1946年５月

:::top
1946年２月、金融緊急措置令制定
:::lead
流通中の旧円を==強制的に預け入れさせ（預金封鎖）==、新円にて一部引き出しを許可
:::

:::last
貨幣流通量の減少、つまり==物価上昇の抑制を狙ったが、効果は一時的でした==。
:::

![](/images/jh/img/162/1.jpg)

:::

#### 第１次吉田茂内閣｜1946年５月～1947年５月

:::top
1947年、基幹産業の==石炭==・鉄鋼の生産==拡大==を優先する[[傾斜生産方式]]を採用し、[[復興金融金庫]]を設立して資金・資材の供給を開始しました。
:::

### 二・一ゼネストの中止

#### 第１次吉田茂内閣｜1946年５月～1947年５月

:::top
敗戦後、労働組合の結成が奨励され、民間企業の賃金の値上がりが実現しました。
:::

::arrow

:::top
物価上昇の進行のなか、賃金の値上げが遅れる公務員らは、==その値上げを要求しました==。
:::

:::middle
吉田茂首相が運動指導者を「<ruby>不逞<rt>ふてい</rt></ruby>の輩」と非難し、大きな反発を招きました。
:::

:::last
打倒吉田内閣を目指す[[ゼネラル＝ストライキ]]（[[二・一ゼネスト]]）が、1947年２月１日に予定されましたが、==ＧＨＱの==中止命令で==不発に終わりました==。
:::

:::sup
1948年、芦田均内閣が公布した[[政令２０１号]]で、公務員の争議行動禁止

### 新憲法下初の内閣

:::top
1947年４月、新憲法下の新内閣を組織するため、衆参両議院議員選挙が実施されました。
:::

:::last
日本社会党が、日本自由党・民主党を僅差で破り、衆議院第一党となりました。
:::

#### 片山哲内閣｜1947年５月～1948年３月

:::top
日本社会党委員長[[片山哲]]が、新憲法下の初の首相に就任し、民主党・国民協同党との３党連立内閣を組織しました。
:::

:::last
新内閣は、保守でも急進でもない「中道」とＧＨＱから一定の評価を受けましたが、連立内閣ゆえの対立に悩まされ、炭鉱国家管理法案で揉めて総辞職しました。
:::

### 贈収賄事件による総辞職

#### 芦田均内閣｜1948年３月～1948年10月

:::top
民主党総裁[[<ruby>芦田均<rt>あしだひとし</rt></ruby>]]が、日本社会党・国民協同党との３党連立内閣を組織しました。
:::

:::last
しかし、復興金融金庫から不当融資を得た贈収賄事件[[昭和電工事件]]で総辞職しました。
