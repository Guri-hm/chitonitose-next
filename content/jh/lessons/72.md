## 幕藩体制

### 大名の分類

:::top
将軍と主従関係を結んだ石高[[1]]万石以上の武士を[[大名]]（藩主）と呼びます。
:::

:::last
大名は将軍との親疎関係で３つに分類されます。
:::

- 親藩
  :::lead
  徳川氏一門の大名
  :::
  :::lead
  御三家の[[尾張]]家・[[紀伊]]家・[[水戸]]家が代表的
  :::
- 譜代
  :::lead
  関ヶ原の戦い以前から徳川氏に従う大名
  :::
  :::lead
  石高は小さいが要所に置かれ、要職に就任
  :::
- 外様
  :::lead
  関ヶ原の戦い以後に徳川氏に従った大名
  :::
  :::lead
  石高は大きいが辺境に配置
  :::

:::sup
藩
:::lead
明治時代に成立した、江戸時代の大名の領地に対する呼称（幕府開設時は約**180**藩）
:::

### 大御所家康の政策

:::top
大御所となった徳川家康は、次の①②の政策をおこないました。
:::

- 1615年、一国一城令
  :::lead
  大名の居城を１つに限定させる法令
  :::
- 1615年、武家諸法度（元和げんな令）
  :::lead

### ３代将軍家光の政策

:::top
秀忠の死後、３代将軍[[徳川家光]]は、次の①～③の政策をおこないました。
:::

- 1632年、肥後の外様大名加藤氏の処分
- 1634年、30万人の軍勢を率いて上洛
  :::lead
  統一した[[軍役]]を大名に賦課し、軍事指揮権の所有を明示
  :::
- 1635年、武家諸法度（寛永かんえい令）
  :::lead
  大名に<ruby>国元<rt>くにもと</rt></ruby>と==江戸==とを１年交代で往復する[[参勤交代]]を義務づけ、==大名の妻子の江戸定住を強制==（諸経費は全て==大名の負担==）
  :::
  :::lead
  海外渡航の制限に関連し、500石以上の船の建造を禁止
  :::

:::gazo
![](/images/jh/img/72/10.jpg)

徳川家光

## 史料

### 武家諸法度（元和令）

#### 原文

:::top
一文武弓馬ノ道、専ラ相嗜ムベキ事。……
:::

:::top
一諸国ノ居城修補ヲ為スト雖モ、必ズ言上スベシ。況ンヤ新儀ノ構営堅ク停止令ムル事。……
:::

#### 現代語訳

:::top
一、文武弓馬の道に、ひたすら励むようにせよ。……
:::

:::top
一、諸国の居城はたとえ修理であっても必ず幕府に報告せよ。まして、新規に築城することは厳重に禁止する。……
:::

#### 解説

### 武家諸法度（寛永令）

#### 原文

:::top
一大名小名、在江戸交替、相定ル所也。毎歳夏四月中参勤致スベシ。従者ノ員数近来甚ダ多シ、且ハ国郡ノ費、且ハ人民ノ労也。向後其ノ相応ヲ以テ、之ヲ減少スベシ。……
:::

:::top
一私ノ関所、新法ノ津留、制禁ノ事。
:::

:::top
一五百石以上ノ船停止ノ事。
:::

#### 現代語訳

:::top
一、大名・小名が国元と江戸とを参勤交代するよう定めるものである。毎年夏の四月中に江戸へ参勤せよ。しかし、参勤に来る大名の供の者の人数がたいへん多くなっている。これは治めている国や郡の無駄な出費であり、人々の苦しむところとなる。これからは供の人数をよく考えて減らすことが必要である。……
:::

:::top
一、大名が私設の関所をつくったり、津留といって新たに港や川に関所をつくったりすることを禁止する。
:::

:::top
一、五〇〇石積み以上の船をつくることは禁止する。
:::

#### 解説
