## 幕府の支配体制

### 封建制度―土地を介した主従関係

:::top
将軍（頼朝）は開発領主（田地開墾に及んだ土着の軍事貴族や有力農民）と主従関係を結びました。
:::

:::last
将軍と主従関係を結んだ者を[[御家人]]と呼びます。
:::

#### 御恩（将軍が御家人に与える恩恵）

:::top
本領安堵
:::lead
先祖伝来の田地の支配を将軍が保障すること
:::

:::top
新恩給与
:::lead
新たな田地を将軍から与えられること（<ruby>平家没官領<rt>へいけもっかんりょう</rt></ruby>などから分与）
:::

:::sup
平家没官領
:::lead
平氏に寄進された大量の荘園群で、平氏滅亡後に朝廷に没収され、全て頼朝に付与
:::

#### 奉公（御家人が将軍に与える恩恵）

:::top
戦時
:::lead
軍役
:::

:::top
平時
:::lead
[[京都大番役]]（京の警護）・[[鎌倉番役]]（鎌倉の警護）
:::

### 地方の機関

#### 守護

:::top
諸国に１人置かれ、国の治安維持や地頭の指揮統率を担当しました。
:::

:::last
有力御家人が任命され、基本的な権限は[[大犯三ヵ条]]と総称します
:::

:::sup
大犯三ヵ条
:::lead
地頭に京都大番役の催促／謀叛人の逮捕／殺害人の逮捕
:::

#### 地頭

:::top
公領・荘園に置かれ、現地で直接支配をおこない、名主からの徴税を担当しました。
:::

![](/images/jh/img/42/2.svg)

##### 知行国主・荘園領主が朝廷関係者の場合（上図）

:::top
将軍が源頼朝の頃、朝廷関係者の力はまだ強く残っていました。
:::

:::last
地頭は公領で将軍と受領の、荘園で将軍と荘園領主の２元的支配を受けました。
