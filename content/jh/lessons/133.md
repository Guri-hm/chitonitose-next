## 文化の特色

### 文化の担い手の変化

:::top
明治時代初め、新政府は強大な欧米列強への対抗を急ぎました。
:::

:::last
新政府が先頭に立ち、富国強兵・殖産興業・文明開化などの標語を掲げ、西洋文明の移植による急速な発展を進めました。
:::

::arrow

:::top
明治時代中頃、教育の普及や通信・出版の発達で、国民自身の手による文化の発展を見せ始めました。
:::

### 文化の二元性

:::top
人々の精神の変化は緩やかで、また、農村の発展は都市よりも遅れました。
:::

:::last
そのため、明治時代の文化は、新旧・和洋の文化が混在・併存しました。
:::

## 思想と信教

### 思想

#### 国民の権利と国家の権力

:::top
明治時代初めの西洋思想の流入は、国民の権利・自由が保障されて、初めて国家の権力が強化されるという民権論を形成しました。
:::

::arrow

:::top
朝鮮半島で1882年に壬午軍乱、1884年に甲申事変が起きました。
:::

:::last
これらの事件は、国家の権力の強化・拡張が実現すれば、国民の権利・自由が保たれるという国権論を形成しました。
:::

:::gazo
![](/images/jh/img/133/2.svg)

民権論と国権論
:::

:::top

:::

:::last
鹿鳴館外交に代表される極端で上からの欧化政策は、激しい反発を招きました。
:::

:::top
次の思想が、近代的民族主義と総称されました。

  :::lead
  日本的な伝統や美意識を強調する思想
  :::

  :::lead
  国家の独立維持と、日本人として自覚向上および全体の福利を主張する思想
  :::
  :::lead
  [[<ruby>陸羯南<rt>くがかつなん</rt></ruby>]]が、新聞「[[日本]]」を創刊するとともに主張した思想

#### 達成感と燃え尽き

:::top
日露戦争の勝利は、人々に大きな達成感を与えた反面、国家主義への疑問も抱かせました。
:::

::arrow

:::top
1908年、政府は、勤倹節約と皇室尊重を国民に求める[[戊申詔書]]を発し、列強の一員としての日本を支える国民の道徳強化に努めました。
:::

### 信教

#### 神道

:::top
明治時代初め、新政府は神道の国教化を試みましたが、十分な成果をあげられませんでした。
:::

:::last
一方、政府の公認を受けた民間の神道[[教派神道]]は、庶民の間にかなり広まりました。
:::

#### 仏教

:::top
1868年、神仏分離令制定
:::lead
神道の国教化を前提として、神道の純化のために神仏習合を禁じた法令堕落した僧侶への怒りから、仏教を排斥する行動[[<ruby>廃仏毀釈<rt>はいぶつきしゃく</rt></ruby>]]が全国で発生
:::

::arrow

:::top
仏教の勢力は廃仏毀釈で弱まりましたが、[[<ruby>島地黙雷<rt>しまじもくらい</rt></ruby>]]のように信教の自由を説く者の活躍で、次第に勢力を回復していきました。
:::

![](/images/jh/img/133/5.jpg)

島地黙雷
:::

#### キリスト教

:::top
次の外国人教師、<ruby>所謂<rt>いわゆる</rt></ruby>[[お雇い外国人]]の影響で、キリスト教が知識人に広まりました。
:::

:::list{class="en"}
- ヘボン：日本語のローマ字表記方法の考案、日本初の和英辞典を編纂した宣教師
- クラーク：札幌農学校で教鞭をとった人物
- ジェーンズ：
			熊本洋学校で教鞭をとり、生徒がキリスト教徒の団体「熊本バンド」を結成したため解任された人物
:::

:::middle
日本人の信徒のなかからは、札幌農学校で学んだ内村鑑三・新渡戸稲造や、熊本バンドに参加した[[海老名弾正]]が現れ、人々の心をとらえました。
:::

:::last
キリスト教会は、売春廃止に取り組む運動<ruby>廃娼<rt>はいしょう</rt></ruby>運動をおこないました。
