## 田沼時代

### 田沼時代の成立

:::top
1760年、９代将軍**徳川<ruby>家重<rt>いえしげ</rt></ruby>**の後、10代将軍に[[徳川<ruby>家治<rt>いえはる</rt></ruby>]]が就任しました。
:::

:::middle
1772年、[[<ruby>田沼意次<rt>たぬまおきつぐ</rt></ruby>]]が<ruby>側用人<rt>そばようにん</rt></ruby>
		から[[老中]]となり、幕政の実権を握りました。
:::

:::last
意次が幕政の実権を握り、財政の再建に臨んだ時代を田沼時代と呼びます。
:::

#### 商業奨励による財政再建

:::top
商人・職人の仲間を広く公認して[[株仲間]]を==増加させました==。
:::

:::last
そして、公認の代わりに営業税[[運上]]・[[<ruby>冥加<rt>みょうが</rt></ruby>]]を課すことで財源を増加させました。
:::

:::sup
==銅・鉄・真鍮・朝鮮人参などを幕府の専売とし==、直営の施設[[座]]を設置
:::

#### 貨幣統一による商業円滑化

:::top
<ruby>秤量<rt>しょうりょう</rt></ruby>貨幣である銀貨を計数貨幣化することで金貨に連結しようとしました。
:::

:::last
1772年に鋳造された、金２<ruby>朱<rt>しゅ</rt></ruby>と等価の[[<ruby>南鐐<rt>なんりょう</rt></ruby>二朱銀]]が有名です。
:::

#### 蝦夷地の調査・開発と長崎貿易

:::top
[[<ruby>蝦夷地<rt>えぞち</rt></ruby>]]の調査で、ロシア人との交易の可能性や進出への対処策を探ろうとしました。
:::

:::last
また、蝦夷地の産物「干し<ruby>鮑<rt>あわび</rt></ruby>・いりこ・ふかひれ」などにも注目し、長崎貿易でこれらを[[俵物]]として清に輸出し、金銀の獲得を目指しました。
:::

::arrow

### 田沼時代の終焉

:::top
1782年からの[[天明の飢饉]]は社会不安を増大させ、全て田沼意次の責任にされました。
:::

:::middle
1784年、意次の子田沼<ruby>意知<rt>おきとも</rt></ruby>が江戸城内で佐野<ruby>政言<rt>まさこと</rt></ruby>に刺殺されました。
:::

:::last
強まる非難に意次は追い込まれ、1786年の徳川家治の死後、老中を<ruby>罷免<rt>ひめん</rt></ruby>されました。
:::

:::sup
佐野政言…刺殺後、偶然米価が下落したため、民衆は彼を「世直し大明神」と評価
:::
