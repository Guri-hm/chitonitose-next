---overview---
10代将軍徳川家治は、田沼意次を側用人、老中と栄進させ、その手腕・開明性に期待しました。意次は、経済活動の活発化を幕府財政の再建の活路と考え、正統的で当を得た政策を進めました。しかし、政策に伴う商人の富獲得は、自然災害で不安定になった民衆の不満を増大させ、意次の贈収賄疑惑を浮上させる隙を生んでしまいました。
---
## 田沼時代
### 田沼時代の成立
::top
1760年、９代将軍**徳川{{家重|いえしげ}}**の後、10代将軍に[[徳川{{家治|いえはる}}]]が就任しました。
::
::middle
1772年、[[田沼意次|たぬまおきつぐ]]が側用人そばようにん  から[[老中]]となり、幕政の実権を握りました。
::
::last
意次が幕政の実権を握り、財政の再建に臨んだ時代を田沼時代と呼びます。
::
::gazo
::
::top
商人・職人の仲間を広く公認して[[株仲間]]を==増加させました==。
::
::last
そして、公認の代わりに営業税[[運上]]・[[冥加|みょうが]]を課すことで財源を増加させました。
::
::sup
==銅・鉄・真鍮・朝鮮人参などを幕府の専売とし==、直営の施設[[座]]を設置
::
#### 貨幣統一による商業円滑化
::top
{{秤量|しょうりょう}}貨幣である銀貨を計数貨幣化することで金貨に連結しようとしました。
::
::last
1772年に鋳造された、金２{{朱|しゅ}}と等価の[[{{南鐐|なんりょう}}二朱銀]]が有名です。
::
::gazo
::
::last
また、蝦夷地の産物「干し{{鮑|あわび}}・いりこ・ふかひれ」などにも注目し、長崎貿易でこれらを[[俵物]]として清に輸出し、金銀の獲得を目指しました。
::
