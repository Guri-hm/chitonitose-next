## 蒙古襲来後の政治

### 幕府の支配権強化

:::top
蒙古襲来後、次の①②で幕府の支配権は強化されました。
:::

:::list{class="en"}
- 襲来の際に動員した非御家人を以後も動員する権限獲得
- 筑前ちくぜん国に<ruby>鎮西探題
				<rt>ちんぜいたんだい</rt></ruby>の設置
:::

:::sup
鎮西探題
:::lead
北条氏が務めた、九州の政務・裁判・御家人の統率を担う機関
:::

### 政治体制の変化―合議から独断へ

:::top
蒙古襲来に際して、幕府内の権力が得宗かつ執権の北条時宗に集中しました。
:::

:::last
得宗の従者を意味する[[御内人]]の発言力が高まり、御家人と対立しました。
:::

:::sup
御内人の代表は[[内管領]]と呼ばれ、幕府の政治を度々主導
:::

::arrow

:::top
1285年、[[霜月騒動]]
:::

:::lead
時宗の子**北条<ruby>貞時<rt>さだとき</rt></ruby>**
		が得宗となった翌年の出来事
:::

:::lead
内管領の[[平頼綱]]が有力御家人[[安達泰盛]]を滅ぼした事件
:::

:::last
やがて、北条貞時は平頼綱を滅ぼして、幕府の全権を握りました。
:::

::arrow

:::top
以後、執権を務める得宗の独断で、幕府の政治がおこなわれました。
:::

:::last
幕府の政治は、合議を重んじる執権政治から[[得宗専制政治]]に変化しました。
:::
