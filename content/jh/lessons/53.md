---overview---
鎌倉幕府の倒壊後、後醍醐天皇は幕府が擁立した光厳天皇の存在を否定し、再度政治を開始しました。以後の後醍醐の治世を「建武の新政」と称します。延喜・天暦の治を理想として天皇の権限集中を図りましたが、貴族・武士・農民の反発を招きました。やがて中先代の乱が起こると、足利尊氏は公然と後醍醐に反旗を翻しました。
---
## 束の間の新政
### 新しい政治の開始
::top
[[後醍醐]]天皇は光厳天皇を否定し、自分が即位中だと主張しました。
::
::last
年号を「建武」とし、[[建武の新政]]と呼ばれる政治を始めました。
::

#### 建武の新政
::top
天皇に権限を集中しました（院・摂政関白否定）。
::
::middle
重要政務を担う[[記録所]]や、土地問題を取り扱う[[雑訴決断所]]が、中央に再興・設置されました。
::
::last
訴訟・土地所有の確認なども含めて、最終決定に天皇の[[綸旨|りんじ]]を必要としました。

### 新政への不満
::top
貴族・武士・農民など広い層から批判・不満の声が出ました。
::
::last
民衆の思いを代弁する800余字の落書（風刺）として、[[二条河原落書]]が掲げられました。
::
### 新政の崩壊
::top
1335年、[[中先代の乱]]
::
::last
[[足利尊氏]]は反乱を鎮圧すると、鎌倉で新政権の組織化を進め、新政に公然と反旗を翻しました。
::
::gazo
![](img/53/2.svg){.twice}
建武の新政の機構
::
::top
1336年、尊氏は京都を制圧して[[持明院]]統から光厳上皇の弟[[光明]]天皇を即位させました。
::
::last
他方、後醍醐天皇は奈良の[[吉野]]へ逃れ、自らの皇位を主張しました。
::
---arrow---
::top
持明院統による京都の朝廷[[北朝]]と、大覚寺統による吉野の朝廷[[南朝]]が並び立ちました。
::
::last
以後、２つの朝廷が対立する南北朝の動乱が約60年間続きました。
