## ８代将軍の治世

### 三家からの将軍

:::top
７代将軍[[徳川家継]]が<ruby>夭折<rt>ようせつ</rt></ruby>し、家康以来の宗家（本家）が途絶えました。
:::

:::last
1716年、**三家**の１つ[[紀伊]]藩の藩主[[徳川吉宗]]が８代将軍になりました。
:::

::arrow

:::top
５代将軍徳川綱吉以来の側用人の政治は反発を招いていました。
:::

:::last
吉宗は側用人の廃止で反発を鎮めた後、同機能の御用取次を新設し、これを介して[[享保の改革]]という将軍主導の幕政改革に取り組みました。
:::

:::

### 改革のための人材登用

#### 儒学者の登用

:::top
『**政談**』を著した[[<ruby>荻生徂徠<rt>おぎゅうそらい</rt></ruby>]]や木下順庵に学んだ[[<ruby>室鳩巣<rt>むろきゅうそう</rt></ruby>]]などを登用しました。
:::

#### 有能な人材の登用

:::top
武士の収入<ruby>俸禄<rt>ほうろく</rt></ruby>（石高は俸禄の<ruby>多寡<rt>たか</rt></ruby>を示すもの）は世襲され、増減しませんでした。
:::

:::last
役職ごとに石高の基準([[役高]]）があり、満たさぬ者は上位の役職に就けませんでした。
:::

:::gazo{size="half"}
![](/images/jh/img/88/1.svg)

俸禄と役高
:::

::arrow

:::top
吉宗は、役高に対する石高の不足分を==在職中のみ==[[役料]]で補いました。
:::

:::middle
この制度を[[<ruby>足高<rt>たしだか</rt></ruby>の制]]といい、少ない財政負担で有能な人材を登用できました。
:::

:::last
例えば、川崎宿の田中<ruby>丘隅<rt>きゅうぐ</rt></ruby>や**町奉行**として活躍した旗本[[大岡<ruby>忠相<rt>ただすけ</rt></ruby>]]を登用しました。
:::

:::sup
田中丘隅
:::lead

## 享保の改革

### 最重要課題―財政の再建

#### 財政窮乏対策

:::top
1722～1730年、徳川吉宗は財政窮乏対策に[[上げ米]]を実施し、大名から石高１万石につき米100石を臨時に上納させました。
:::

:::last
実施中、上納の代償に[[参勤交代]]の在江戸を==１年から半年に短縮しました==。
:::

:::sup
参勤交代
:::lead
大名が経費調達に裕福な商人からの貸付**大名<ruby>貸<rt>がし</rt></ruby>**
			を頼って問題化
:::

:::gazo{size="half"}
![](/images/jh/img/88/2.jpg)

恥辱の上げ米
:::

#### 収入の増大

- 定免法の採用
  :::lead
  **<ruby>検見<rt>けみ</rt></ruby>法**を改め、年貢率を一定期間固定
  :::
- 新田開発の奨励
  :::lead
  商人の協力を得て米の増産
  :::
- 商品作物生産の推進
  :::lead
  西日本の幕領での綿作などで富の形成
  :::

:::last
結果、幕領の石高は１割以上増え、幕府の財政はやや立ち直りました。
:::

:::gazo
![](/images/jh/img/88/3.svg)

幕領の石高
:::

#### 米価引上げ

:::top
米の増産の一方、供給量の過多が米価下落を招くことも懸念されました。
:::

:::middle
吉宗は、米価上昇（武士の収入増加）で、武家の財政を安定化しようとしました。
:::

:::last
大坂[[堂島]]の米市場の<ruby>先物<rt>さきもの</rt></ruby>取引を公認し、仮需要での米価上昇を試みました。
:::

:::sup
米価調節に<ruby>奔走<rt>ほんそう</rt></ruby>したため、吉宗の俗称は米将軍
:::

:::gazo{size="twice"}
![](/images/jh/img/88/10.jpg)

先物取引
:::

#### 財政支出の抑制

:::top
吉宗は倹約令をしばしば出して、大名やその領民に質素倹約を命じました。
:::

### その他の諸改革

#### 裁判事務の簡素化

:::top
1719年、吉宗は[[<ruby>相対済<rt>あいたいすま</rt></ruby>し]]令を出し、金銀貸借の争いを当事者間で解決させました。
:::

:::sup
相対済し令
:::lead
訴訟の減少が目的で、裁判借金銀に悩む旗本・御家人の救済策とするのは誤説
:::

#### 新しい産業の奨励

:::top
吉宗は<ruby>甘藷<rt>かんしょ</rt></ruby>・さとうきび・<ruby>櫨<rt>はぜ</rt></ruby>・朝鮮人参などの栽培を奨励しました。
:::

:::last
特に甘藷に関しては、[[青木<ruby>昆陽<rt>こんよう</rt></ruby>]]を登用して、救荒用に普及を実現させました。
:::

![](/images/jh/img/88/3.jpg)

:::

#### 学問の奨励

:::top
1720年、吉宗は**漢訳洋書**の==輸入制限を緩和し==、西洋の技術・学問の吸収を図りました。
:::

:::sup
1740年、**青木昆陽**・**<ruby>野呂元丈<rt>のろげんじょう</rt></ruby>**に==オランダ語を学ばせ==、後の<ruby>蘭学<rt>らんがく</rt></ruby>
		興隆の基礎を構築
:::

### 吉宗政権末期の政策

#### 司法基準の明確化

:::top
1742年、幕府の成文法[[公事方御定書]]が完成し、裁判の公正な基準が定まりました。
:::

#### 将軍家の断絶防止策

:::top
徳川吉宗は将軍家の断絶を防止するため、三家のほかに新たな分家を２つ設けました。
:::

:::last
１つ目が吉宗の次男徳川<ruby>宗武<rt>むねたけ</rt></ruby>を始祖とする田安家、２つ目が吉宗の四男徳川<ruby>宗尹<rt>むねただ</rt></ruby>を始祖とする[[一橋]]家です。
:::

:::sup
後に９代将軍徳川<ruby>家重<rt>いえしげ</rt></ruby>の次男徳川<ruby>重好<rt>しげよし</rt></ruby>を始祖とする清水家も加え、田安家・一橋家・清水家を<ruby>三卿<rt>さんきょう</rt></ruby>と総称
