---overview---
徳川家継の死をもって徳川宗家の血は完全に絶えました。三家が８代将軍の座をめぐって争い、紀伊の吉宗が勝ち抜きました。吉宗自身の資質・人望や彼を支える有能な人材があった結果です。吉宗は、財政再建のために「上げ米」という恥辱に甘んずる一方で、65年ぶりの日光社参（軍役）を命じて強い将軍像も誇示しました。
---
## ８代将軍の治世
### 三家からの将軍
::top
７代将軍[[徳川家継]]が{{夭折|ようせつ}}し、家康以来の宗家（本家）が途絶えました。
::
::last
1716年、**三家**の１つ[[紀伊]]藩の藩主[[徳川吉宗]]が８代将軍になりました。
::
---arrow---
::top
５代将軍徳川綱吉以来の側用人の政治は反発を招いていました。
::
::last
吉宗は側用人の廃止で反発を鎮めた後、同機能の御用取次を新設し、これを介して[[享保の改革]]という将軍主導の幕政改革に取り組みました。

#### 有能な人材の登用
::top
武士の収入{{俸禄|ほうろく}}（石高は俸禄の{{多寡|たか}}を示すもの）は世襲され、増減しませんでした。
::
::last
役職ごとに石高の基準([[役高]]）があり、満たさぬ者は上位の役職に就けませんでした。
::
::gazo
![](img/88/1.svg){.half}
俸禄と役高
::
::top
吉宗は、役高に対する石高の不足分を==在職中のみ==[[役料]]で補いました。
::
::middle
この制度を[[{{足高|たしだか}}の制]]といい、少ない財政負担で有能な人材を登用できました。
::
::last
例えば、川崎宿の田中{{丘隅|きゅうぐ}}や**町奉行**として活躍した旗本[[大岡{{忠相|ただすけ}}]]を登用しました。
::

### その他の諸改革
#### 裁判事務の簡素化
::top
1719年、吉宗は[[{{相対済|あいたいすま}}し]]令を出し、金銀貸借の争いを当事者間で解決させました。
::
::sup
相対済し令
:::lead
訴訟の減少が目的で、裁判借金銀に悩む旗本・御家人の救済策とするのは誤説
::
::top
吉宗は{{甘藷|かんしょ}}・さとうきび・{{櫨|はぜ}}・朝鮮人参などの栽培を奨励しました。
::
::last
特に甘藷に関しては、[[青木{{昆陽|こんよう}}]]を登用して、救荒用に普及を実現させました。
::
::top
1720年、吉宗は**漢訳洋書**の==輸入制限を緩和し==、西洋の技術・学問の吸収を図りました。
::
::sup
1740年、**青木昆陽**・**{{野呂元丈|のろげんじょう}}**に==オランダ語を学ばせ==、後の{{蘭学|らんがく}} 興隆の基礎を構築
::
