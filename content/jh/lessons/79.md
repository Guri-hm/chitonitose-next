## 村と百姓

### 近世の村の形成

:::top
豊臣政権下での全国的な検地[[太閤検地]]は、惣村の境界の画定<ruby>村切
		<rt>むらぎり</rt></ruby>もおこない、惣村の分割・消滅を促しました。
:::

:::middle
また、中世末以来[[新田]]開発が急速に進み、新しい村が生まれました。
:::

:::last
17世紀末、全国には６万3000余りの村が存在しました。
:::

:::gazo
![](/images/jh/img/79/1.svg)

村の構造
:::

::arrow

:::top
村の大半は、農業を主とする農村であったが、漁村・山村、あるいは定期市などを中心に村が都市化した<ruby>在郷町<rt>ざいごうまち</rt></ruby>もありました。
:::

:::last
村は一つ一つ個性的であったが、ほぼ共通する特徴も見られました。
:::

### 共通する村の特徴｜構成員と税

#### 村の構成員と階層

:::top
村の構成員は、次のように大別されます。
:::

- 本百姓
  :::lead
  田・畑の所持を検地帳に登録され、年貢・諸役の負担がある百姓
  :::
- <ruby>水呑<rt>みずのみ</rt></ruby>百姓
  :::lead
  田・畑を所持せず、小作（借地での耕作）や日雇いに従事した百姓
  :::
- 名子なご（別称：被官ひかん・譜代ふだい）
  :::lead
  有力な本百姓に隷属する者
  :::

:::sup
漁村には、経営者・労働者の関係をもつ<ruby>網元<rt>あみもと</rt></ruby>・<ruby>網子<rt>あみこ</rt></ruby>という階層あり
:::

:::sup
厳密には、僧侶・神職などの宗教者や職人なども構成員に含まれる場合あり
:::

:::gazo{size="half"}
![](/images/jh/img/79/2.svg)

村の構成員
:::

#### 年貢と諸役

:::top
田・畑を所持するため、本百姓には主に次の５種類の課税がありました。
:::

- <ruby>本途物成<rt>ほんとものなり</rt></ruby>
  :::lead
  土地の総生産量[[<ruby>石高<rt>こくだか</rt></ruby>]]を基準に米で納める年貢
  :::
  :::lead
  通常石高の40～50％の米を納入
  :::
  :::lead
  上記の比率の決め方は、当初その年の収穫に応じた[[<ruby>検見法
  					<rt>けみほう</rt></ruby>]]で、後に一定期間は同じ比率を続ける[[定免法]]に変更
  :::
- 小物成
  :::lead
  <ruby>山野河海<rt>さんやかかい</rt></ruby>の利用や農業以外の副業などに掛かる税
  :::
- 国役
  :::lead
  河川の土木工事などの労働
  :::
- <ruby>伝馬役<rt>てんまやく</rt></ruby>
  :::lead
  街道での公用交通で使う人馬を差し出す負担
  :::
- <ruby>助郷役<rt>すけごうやく</rt></ruby>
  :::lead
  公用交通で不足した人馬を差し出す負担（街道近辺の村に住む場合）
  :::
  :::lead
  村全体で負担し、人馬を徴発された村を[[<ruby>助郷
  					<rt>すけごう</rt></ruby>]]と呼称
  :::

::arrow

:::top
本百姓の①～⑤の負担は、個人ではなく村全体で一括して負担しました。
:::

:::middle
このような負担の方法を[[<ruby>村請<rt>むらうけ</rt></ruby>制]]を呼びます。
:::

:::last
本百姓が５戸で編成する[[五人組]]は、年貢納入などに連帯責任をもち、村請制を支えました。
:::

### 共通する村の特徴｜村の運営

:::top
村は、==本百姓から==選出した次の３役[[村方三役]]（[[村役人]]）が運営しました。
:::

- <ruby>名主<rt>なぬし</rt></ruby>（庄屋）
  :::lead
  村の長にあたり、村政全般を担当
  :::
- 組頭
  :::lead
  ２～３人から成る、名主の補佐役
  :::
- 百姓代
  :::lead
  ２～３人から成る、名主・組頭の監査役
  :::

:::last
運営は[[村法]]（[[村掟]]）に基づいておこなわれ、違反には背くと村八分などの制裁がありました。
:::

:::gazo
![](/images/jh/img/79/3.svg)

村請制と村政運営
:::

::arrow

:::top
治安維持や[[<ruby>入会地
			<rt>いりあいち</rt></ruby>]]（山・野原などの共同利用地）の管理は村で共同して担いました。
:::

:::last
その経費は[[村入用]]と呼ばれ、==村の構成員で負担し合いました==。
:::

:::sup
多大な労力を要する田植えなどに、共同で取り組むことを[[<ruby>結
			<rt>ゆい</rt></ruby>]]（[[もやい]]）と呼称
:::

### 幕府の農村政策

:::top
年貢・諸役の徴収を確実にするため、幕府は以下の政策をおこないました。
:::

::arrow

:::top
1643年、田畑永代売買の禁止令
:::lead
富農への土地集中と本百姓没落を防止
:::

:::top
1643年、田畑勝手作りの禁
:::lead
たばこ・木綿・菜種など商品作物の自由栽培を禁止し、百姓が貨幣経済に巻き込まれて没落することを防止
:::

:::top
1649年、慶安の触書
:::lead
百姓の日常生活の心得などを細かく指示
:::

:::top
1673年、分地制限令
:::lead
分割相続による田畑の細分化を防止
:::

:::sup
慶安の触書
:::lead
32ヵ条からなる法令の総称で、存在を疑問視する説が有力
:::

## 町と町人

### 城下町と地区

:::top
近世、多数の都市が現れ、特に城下町が中心都市でした。
:::

:::last
城下町は将軍・大名の城郭を核とし、次のような地区に分けられました。
:::

- 武家地
  :::lead
  政治・軍事の施設があり、家臣団・足軽らが居住する地区
  :::
- 町人地
  :::lead
  商人・手工業者などが居住し、経営・生産をおこなう地区
  :::
- 寺社地
  :::lead
  寺院・神社が集められ、領内の宗教統制の機能をもつ地区
  :::

:::sup
城下町
:::lead
町奉行が行政・司法・警察の全てを監督
:::

![](/images/jh/img/79/4.svg)

:::

### 町人地の構成と税

:::top
町人地には、自治組織をもつ共同体<ruby>町<rt>まち</rt></ruby>が多数存在しました。
:::

:::middle
町の住民は町人と総称され、次のように大別されます。
:::

- 家持
  :::lead
  自分の宅地と家をもち、町の運営に参加できる者
  :::
- 地借
  :::lead
  借りた宅地に家を建てて住む者
  :::
- 店借（借家）
  :::lead
  家やその一部を借りて住む者
  :::

:::last
家持は町人足役という夫役を負い、清掃・防災などに努めました。
:::

![](/images/jh/img/79/5.svg)

町政運営
:::
