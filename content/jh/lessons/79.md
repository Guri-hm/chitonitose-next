---overview---
幕藩体制は、人口の大多数を占める百姓の負担で支えられました。百姓は村に住み、村全体で共同して作業にあたって税を領主に納めました。全国の村は共通点をもつ一方で、１つ１つ個性的でもありました。このような村に対する存在として、城下町の町人地を構成する町が挙げられます。町もまた、多様な特徴をもち、非常に複雑でした。
---
## 村と百姓
### 近世の村の形成
::top
豊臣政権下での全国的な検地[[太閤検地]]は、惣村の境界の画定{{村切|むらぎり}}もおこない、惣村の分割・消滅を促しました。
::
::middle
また、中世末以来[[新田]]開発が急速に進み、新しい村が生まれました。
::
::last
17世紀末、全国には６万3000余りの村が存在しました。
::
::gazo
::
::top
村の大半は、農業を主とする農村であったが、漁村・山村、あるいは定期市などを中心に村が都市化した{{在郷町|ざいごうまち}}もありました。
::
::last
村は一つ一つ個性的であったが、ほぼ共通する特徴も見られました。
::
### 共通する村の特徴｜構成員と税
#### 村の構成員と階層
::top
村の構成員は、次のように大別されます。
::

- [[本百姓]]
  :::lead
  田・畑の所持を検地帳に登録され、年貢・諸役の負担がある百姓
  :::
- [[{{水呑|みずのみ}}百姓]]
  :::lead
  田・畑を所持せず、小作（借地での耕作）や日雇いに従事した百姓
  :::
- {{名子|なご}}（別称：{{被官|ひかん}}・{{譜代|ふだい}}）
  :::lead
  有力な本百姓に隷属する者
  :::

::sup
漁村には、経営者・労働者の関係をもつ{{網元|あみもと}}・{{網子|あみこ}}という階層あり
::
::sup
厳密には、僧侶・神職などの宗教者や職人なども構成員に含まれる場合あり
::
::gazo
::

- [[本途物成|ほんとものなり]]
  :::lead
  土地の総生産量[[石高|こくだか]]を基準に米で納める年貢
  :::
- [[小物成]]
  :::lead
  {{山野河海|さんやかかい}}の利用や農業以外の副業などに掛かる税
  :::
- [[国役]]
  :::lead
  河川の土木工事などの労働
  :::
- [[伝馬役|てんまやく]]
  :::lead
  街道での公用交通で使う人馬を差し出す負担
  :::
- [[{{助郷役|すけごうやく}}]]
  :::lead
  公用交通で不足した人馬を差し出す負担（街道近辺の村に住む場合）
  :::

## 町と町人
### 城下町と地区
::top
近世、多数の都市が現れ、特に城下町が中心都市でした。
::
::last
城下町は将軍・大名の城郭を核とし、次のような地区に分けられました。
::

- [[武家地]]
  :::lead
  政治・軍事の施設があり、家臣団・足軽らが居住する地区
  :::
- [[町人地]]
  :::lead
  商人・手工業者などが居住し、経営・生産をおこなう地区
  :::
- 寺社地
  :::lead
  寺院・神社が集められ、領内の宗教統制の機能をもつ地区
  :::

::sup
城下町
:::lead
城下町
:::
iv class="lead">町奉行が行政・司法・警察の全てを監督
::
::gazo
::
::middle
町の住民は町人と総称され、次のように大別されます。
::

- [[家持]]
  :::lead
  自分の宅地と家をもち、町の運営に参加できる者
  :::
- [[地借]]
  :::lead
  借りた宅地に家を建てて住む者
  :::
- [[店借]]（[[借家]]）
  :::lead
  家やその一部を借りて住む者
  :::

::last
家持は町人足役という夫役を負い、清掃・防災などに努めました。
::
::gazo
