### 貴族院の内閣

#### 清浦奎吾内閣｜1924年１月～1924年６月

:::top
1924年、松方正義・西園寺公望ら元老は、政党と距離を置く者を首相に望み、<ruby>枢密院<rt>すうみついん</rt></ruby>の議長[[清浦<ruby>奎吾<rt>けいご</rt></ruby>]]を選びました。
:::

:::last
清浦奎吾は、陸相・海相を除く大臣を貴族院から選び、非政党内閣を組織しました。
:::

:::gazo{size="half"}
![](/images/jh/img/143/4.jpg)

清浦奎吾
:::

::arrow

:::top
**憲政会**・立憲政友会・**革新<ruby>倶楽部<rt>くらぶ</rt></ruby>**
		の護憲三派は、清浦奎吾内閣を超然内閣と捉え、その打倒運動[[第二次護憲運動]]を始めました。
:::

:::middle
内閣は、立憲政友会内の反高橋是清派が組織した政党政友本党を味方にし、議会を解散して総選挙に臨みました。
:::

:::last
総選挙が護憲三派の圧勝に終わり、内閣は総辞職しました。
:::

:::gazo
![](/images/jh/img/143/5.jpg)

護憲三派と憲政本党

＊台の上が清浦奎吾

## 憲政の常道へ

### 護憲三派内閣の成立

#### 加藤高明内閣｜1924年６月～1926年１月

:::top
衆議院第一党[[憲政会]]の総裁[[加藤高明]]が、３党の==連立内閣を==組織しました。
:::

:::last
加藤高明内閣は、外相[[<ruby>幣原喜重郎<rt>しではらきじゅうろう</rt></ruby>]]による協調外交、所謂[[<ruby>幣原<rt>しではら</rt></ruby>外交]]を基本としました。
:::

:::gazo{size="half"}
![](/images/jh/img/143/6.jpg)

加藤高明
:::

:::gazo{size="half"}
![](/images/jh/img/143/7.jpg)

幣原喜重郎
:::

::arrow

:::top
幣原外交下で、加藤高明内閣は、国外に対して次の２つを実施・経験しました。
:::

- 1925年、日ソ基本条約調印、日ソ国交樹立
  :::lead
  日本と社会主義国家ソヴィエト連邦（ソ連）との国交を樹立させた条約
  :::
- 1925年、五・三〇事件
  :::lead
  中国に建設された日本の紡績工場<ruby>在華紡<rt>ざいかぼう</rt></ruby>でのストライキを機に、反帝国主義運動が中国全土に広がったこと
  :::

::arrow

:::top
内閣は、国内に対して次の２策を実施しました。
:::

- 1925年、普通選挙法公布
  :::lead
  満[[25]]歳以上の==男性==に、納税額に関係なく衆議院議員の選挙権を与えた法令
  :::
  :::lead
  結果、有権者が従前の４倍に増加
  :::
- 1925年、治安維持法公布
  :::lead
  ==国体（天皇制）の変革、私有財産の否認を目的とする結社を==禁止した法令
  :::
  :::lead
  目的は、日ソ国交樹立や普通選挙実施による社会主義の活発化抑制
  :::

:::sup
第一次護憲運動から男子普通選挙制の成立までの風潮を大正デモクラシーと呼称
:::

:::gazo{size="half"}
![](/images/jh/img/143/1.svg)

選挙人率の推移

＊男性・満25歳以上
:::

::arrow

:::top
1925年、立憲政友会が、陸軍・長州閥の田中義一を総裁に迎え、また、他政党の革新<ruby>倶楽部<rt>くらぶ</rt></ruby>を吸収しました。
:::

:::last
結果、護憲三派の提携が崩れ、加藤高明の病死で内閣は総辞職しました。
:::

:::gazo{size="half"}
![](/images/jh/img/143/9.jpg)

田中義一
:::

### 憲政の常道

:::top
加藤高明内閣の成立から1932年の[[<ruby>犬養毅<rt>いぬかいつよし</rt></ruby>]]内閣の崩壊まで、憲政会（後の立憲民政党）と立憲政友会の総裁が交代で政党内閣を組織しました。
:::

:::last
この８年間は、==衆議院に議席を多くもつ政党が内閣を担当する慣例==が守られ、その慣例を[[憲政の常道]]と呼びます。
