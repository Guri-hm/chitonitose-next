## 太上天皇と天皇―淳仁天皇の代

### あくなき権勢追求

:::top
藤原仲麻呂は光明子の協力で孝謙天皇を譲位させ、義理の息子である[[<ruby>淳仁
			<rt>じゅんにん</rt></ruby>]]天皇を即位させました。
:::

:::middle
即位の日、仲麻呂は淳仁から恵美押勝の名を与えられました。
:::

:::last
この他にも経済的特権を与えられ、果てには<ruby>太政大臣<rt>だいじょうだいじん</rt></ruby>に就任しました。
:::

:::sup
淳仁は義父仲麻呂の言いなり
:::

:::sup
譲位後の天皇は「<ruby>太上天皇<rt>だいじょうてんのう</rt></ruby>（上皇）」と表記

:::top
孝謙<ruby>太上天皇<rt>だいじょうてんのう</rt></ruby>が自身を看病した僧[[道鏡]]を<ruby>寵愛<rt>ちょうあい</rt></ruby>しました。
:::

:::middle

:::

:::last

:::sup

## 皇位継承問題―称徳天皇の代

### 道鏡の台頭と宇佐八幡神託事件

:::top
孝謙太上天皇は再び即位して（<ruby>重祚<rt>ちょうそ</rt></ruby>）、[[称徳]]天皇となりました。
:::

:::gazo{size="half"}
![](/images/jh/img/19/5.jpg)

称徳天皇
:::

::arrow

:::top
道鏡は称徳の寵愛を受け、異例の早さで出世しました。
:::

:::middle
765年、太政大臣に準じる太政大臣禅師に任じられました。
:::

:::middle
766年、皇太子に準じる[[法王]]にまで上り詰めました。
:::

:::last
称徳は道鏡への譲位を望みましたが、周囲を納得させられる理由がありませんでした。
:::

::arrow

:::top
769年、**宇佐八幡宮**の神官が称徳と結託し、神から「道鏡を天皇にすべき」とのお告げがあったと報告しました。
:::

:::last
称徳は神の意志の再確認するために、[[<ruby>和気清麻呂
			<rt>わけのきよまろ</rt></ruby>]]を派遣しました。
:::

:::gazo{size="half"}
![](/images/jh/img/19/6.jpg)

和気清麻呂
:::

:::gazo{size="half"}
![](/images/jh/img/19/7.jpg)

藤原百川
:::

::arrow

:::top
和気清麻呂が、反道鏡派の[[藤原<ruby>百川
			<rt>ももかわ</rt></ruby>]]（式家、宇合の子）らと示し合わせ、神の意志は「跡継ぎは必ず皇族にすべき」であった報告しました。
:::

:::last
神託を利用した称徳の譲位計画は失敗に終わってしまいました。
:::

:::sup
清麻呂は称徳の怒りを買い、姉和気広虫とともに流刑処分
:::
