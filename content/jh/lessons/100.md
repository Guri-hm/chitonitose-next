## 文学

### 小説

:::top
19世紀初め、小説は次の４様式に分かれていました。
:::

- 滑稽本
  :::lead
  庶民生活の滑稽を会話中心に絵入りで描いた小説
  :::
- 人情本
  :::lead
  庶民の恋愛情痴を絵入りで描いた小説
  :::
- <ruby>読本<rt>よみほん</rt></ruby>
  :::lead
  歴史・伝説を文章主体で描いた小説
  :::
- 合巻
  :::lead
  黄表紙の流行・内容膨張に合わせ、数冊分を<ruby>綴<rt>と</rt></ruby>じた小説群
  :::

:::last
[[天保の改革]]で、人情本の代表的作家[[<ruby>為永春水<rt>ためながしゅんすい</rt></ruby>]]や合巻の代表的作家[[<ruby>柳亭種彦<rt>りゅうていたねひこ</rt></ruby>]]が処罰されました。
:::

### 和歌と狂歌

:::top
香川景樹が起こした古今調の優雅で平明な歌風の桂園派は、あまり浸透しませんでした。
:::

:::last
その他、越後の禅僧良寛が生活歌を詠み、わずかに活躍しただけであった。
:::

::arrow

:::top
和歌よりも、滑稽味を取り入れた短歌[[狂歌]]が盛んに詠まれました。
:::

:::last
[[大田<ruby>南畝<rt>なんぽ</rt></ruby>]]（蜀山人）は、滑稽味のなかに風刺も込めた狂歌を詠みました。
:::
