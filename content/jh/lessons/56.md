### 中国との貿易

:::top
日本と[[元]]の間に正式な国交はなく、==私的な商船の往来のみありました==。
:::

:::last
蒙古襲来後も、次の２例のような貿易船が元へ派遣されました。
:::

- 1325年、鎌倉幕府が建長寺の修造費調達のために派遣
- 1342年、足利尊氏が天龍寺の造営費調達のために派遣

:::sup
天龍寺
:::lead
[[後醍醐]]天皇の<ruby>冥福
			<rt>めいふく</rt></ruby>に、<ruby>夢窓疎石<rt>むそうそせき</rt></ruby>が足利尊氏に造営を勧めた寺院
:::

::arrow

:::top
1368年、[[<ruby>朱元璋<rt>しゅげんしょう</rt></ruby>]]が元を退けて[[明]]を建国しました。
:::

:::last
明は近隣諸国に朝貢を要求しました。
:::

		元打倒を目指す反乱軍に加わり、一兵士から統率者となりました。洪武帝の名で親しまれます。

:::gazo
![](/images/jh/img/56/2.svg)

冊封と朝貢

３代将軍[[足利義満]]は、朝貢によって許される貿易の利益を優先し、1401年、同朋衆**<ruby>祖阿<rt>そあ</rt></ruby>**を正使、博多商人**<ruby>肥富<rt>こいつみ</rt></ruby>**を副使として明に派遣しました。

:::last

#### 日明貿易の別称と特徴

:::top
別称①：朝貢貿易
		
		滞在費と運搬費は皇帝側の負担で、==日本の利益莫大==
:::lead
国王が皇帝に<ruby>貢物<rt>こうもつ</rt></ruby>を献じて、皇帝から返礼の品を<ruby>賜<rt>たまわ</rt></ruby>るという形で行われる貿易
:::

:::top
別称②：勘合貿易
		
		この証票で倭寇か否かを判断
:::lead
[[勘合]]という証票を使用する貿易
:::

:::

### 日明貿易の展開と断絶

:::top
４代将軍[[足利<ruby>義持
			<rt>よしもち</rt></ruby>]]は朝貢形式を嫌い、貿易を中止しました。
:::

:::gazo
![](/images/jh/img/56/5.jpg)

足利義持（４代将軍）

		父義満とは不和であり、義満の急死によりようやく実権を握れました。
		その反動か義持の政策には、義満の政治姿勢と乖離する点が多く見られます。
:::

::arrow

:::top
６代将軍[[足利<ruby>義教<rt>よしのり</rt></ruby>]]は貿易を再開しました。
:::

:::gazo
![](/images/jh/img/56/8.jpg)

足利義教（６代将軍）
くじ引きで将軍に選ばれました。それゆえに、義教は将軍権力の確立を急ぎ、時に神意で選ばれたことを盾に恐怖政治をおこないました。結果、赤松満祐に謀殺されてしまいました。

		容赦ない弾圧に民衆から「悪将軍」とも呼ばれました。
:::

::arrow

:::top
応仁の乱以降は、[[堺]]の商人と結ぶ[[細川]]氏と、[[博多]]の商人を結ぶ[[大内]]氏が貿易の実権を握りました。
:::

:::middle
1523年、寧波の乱
:::lead
両氏が主導権をめぐって中国の<ruby>寧波<rt>ニンポー</rt></ruby>で衝突した事件
:::

:::last
[[大内]]氏が勝利して、以後貿易を独占しました。
:::

:::gazo{size="twice"}
![](/images/jh/img/56/4.svg)
:::

::arrow

:::top
16世紀半ば、大内氏の滅亡で貿易が断絶し、後期倭寇が活発化しました。
:::

:::last
1588年、豊臣秀吉が[[海賊取締令]]を出し、倭寇は沈静化しました。
:::

### 朝鮮との貿易

:::top
1392年、[[李成桂]]が[[高麗]]を倒して、[[朝鮮]]を建国しました。
:::

:::last
朝鮮は日本に通交と==倭寇の禁止を求めました==。
:::

:::gazo{size="half"}
![](/images/jh/img/56/6.jpg)

李成桂（朝鮮の初代国王）

		北方で他民族を平定し、倭寇討伐にも勲功がありました。後に政権を握り、朝鮮（李氏朝鮮）を建国しました。
:::

::arrow

:::top
３代将軍[[足利義満]]は求めに応じて、日朝貿易を開始しました。
:::

#### 日朝貿易の特徴

- <ruby>対馬<rt>つしま</rt></ruby>の宗氏
			が朝鮮との窓口役
- 朝鮮の３つの貿易港は三浦（塩浦えんぽ・富山浦ふざんぽ・乃而浦ないじほと総称
- 三浦に<ruby>倭館<rt>わかん</rt></ruby>（日本人使節接待の客館、日本人の居留地）を設置

### 朝鮮の展開と衰退

:::top
1419年、応永の外寇
:::lead
朝鮮が[[対馬]]を倭寇の本拠地と考えて襲撃した事件
:::

:::last
貿易は一時中断されたが、後に再開されました。
:::

::arrow

:::top

:::lead
朝鮮の特権制限を不満とした日本人による暴動事件
:::

:::last
鎮圧されたが、以後日朝貿易は衰えました。
