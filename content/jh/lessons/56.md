---overview---
14～15世紀にかけての時期は、アジアで新旧の勢力が激しい勢いで交替して、新しい国際秩序が生み出された時代でした。倭寇と呼ばれる海賊集団が南北朝の動乱期に現れ、日本の東アジアでの孤立を許さない環境をつくりました。明・朝鮮が日本に倭寇の禁止・通交を求め、足利義満がこれに応じて貿易は始まりました。
---

### 中国との貿易
::top
日本と[[元]]の間に正式な国交はなく、==私的な商船の往来のみありました==。
::
::last
蒙古襲来後も、次の２例のような貿易船が元へ派遣されました。
::

- 1325年、鎌倉幕府が[[建長寺]]の修造費調達のために派遣
- 1342年、[[足利尊氏]]が[[天龍寺]]の造営費調達のために派遣

:::lead
天龍寺

---arrow---

1368年、[[朱元璋|しゅげんしょう]]が元を退けて[[明]]を建国しました。

::last
明は近隣諸国に朝貢を要求しました。
::
::gazo
::
::top
３代将軍[[足利義満]]は、朝貢によって許される貿易の利益を優先し、1401年、同朋衆**{{祖阿|そあ}}**を正使、博多商人**{{肥富|こいつみ}}**を副使として明に派遣しました。
::
::last
明の皇帝は義満に「[[日本国王]]」の称号を与え（冊封）、ここに日明貿易が開始しました。

### 朝鮮との貿易
::top
1392年、[[李成桂]]が[[高麗]]を倒して、[[朝鮮]]を建国しました。
::
::last
朝鮮は日本に通交と==倭寇の禁止を求めました==。
::
---arrow---
::top
３代将軍[[足利義満]]は求めに応じて、日朝貿易を開始しました。

#### 日朝貿易の特徴

- [[対馬|つしま]]の**宗氏**が朝鮮との窓口役
- 朝鮮の３つの貿易港は[[三浦]]（{{塩浦|えんぽ}}・{{富山浦|ふざんぽ}}・{{乃而浦|ないじほ}}と総称
- 三浦に[[倭館|わかん]]（日本人使節接待の客館、日本人の居留地）を設置
