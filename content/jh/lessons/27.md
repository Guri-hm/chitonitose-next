## ２大仏教の対比

### ２大仏教の２つの特徴

#### 特徴１

:::top
[[<ruby>加持祈禱<rt>かじきとう</rt></ruby>]]をおこない、現世利益を求めます。
:::

:::middle
加持祈禱は一部の限られた弟子のみに伝えられました。
:::

:::last
このような、教えを外部に隠す宗派を[[密教]]と総称します。
:::

:::sup
加持祈禱
:::lead
印を結んで呪文を唱え、仏の加護を祈る呪術
:::

:::sup
密教に対して、南都六宗は誰でも教えを経典から学習可能
:::

:::last
教えを公開する宗派を顕教と総称
:::

:::

## 弘仁・貞観文化―密教美術

### 像の４大特徴

#### ①木材の利用

:::top
平安時代には、粘土と同等以上に安価な「木材」を造像に利用しました。
:::

:::last
造像の技法は、１本の木材から像を彫る[[一木造]]が主流でした。
:::

![](/images/jh/img/27/4.jpg)

一木造
:::

#### ②密教に関わる仏像

:::top
密教に関わる如意輪観音・不動明王などの仏像が多数造られました。
:::

#### ③神像の登場

:::top
自然から転じた日本の神は本来目に見えず、像が造られませんでした。
:::

:::last
８世紀から進んだ神仏習合の影響で、神像も造られるようになりました。
