## 室町幕府の動揺

### ４代将軍の政治

:::top
1394年、４代将軍[[足利義持]]の就任
:::

:::top
1411年、[[日明貿易]]の中止
:::

:::top
1416年、上杉禅秀の乱
:::lead
前関東管領の反乱
:::

:::top
1419年、応永の外寇朝鮮の対馬襲撃
:::

:::top
＊対馬を倭寇の本拠地を誤解
:::

### ６代将軍の政治

:::top
1425～28年、義持が政治を再度担当（理由：５代将軍の早死に）
:::

:::top
1428年、義持の死去
:::last
次の将軍の就任の気運
:::
:::

:::top
1428年、正長の徳政一揆
:::lead
近江国の馬借が「代始めの徳政」を要求して蜂起した一揆鎮圧されたが、私徳政（実力による債務破棄）を実現
:::

:::top
1429年、６代将軍[[足利義教]]の就任
		
:::last
将軍専制、鎌倉公方・有力守護と対立
:::
:::

:::top
1432年、[[日明貿易]]の再開
:::

:::top
1438～39年、永享の乱
:::lead
幕府に反抗する鎌倉公方[[足利持氏]]の討伐
:::

:::top
1441年、嘉吉の変
		
		将軍の権威の失墜
		次の将軍の就任の気運
:::lead
播磨などの守護[[赤松満祐]]が義教を謀殺
:::

:::top
1441年、嘉吉の徳政一揆
:::lead
「代始めの徳政」を要求して起こった一揆初めて幕府が徳政令を出して沈静化
:::

### ８代将軍の政治

:::top
1449年、８代将軍[[足利義政]]の就任
:::

:::top
1454年～、享徳の乱
:::lead
鎌倉公方の足利成氏と関東管領の対立成氏が古河に敗走（古河公方）、堀越公方と対立
:::

### 応仁の乱（1467～1477）

:::top
背景：将軍継嗣問題、管領家の家督相続争いに端を発した内乱
:::

:::top
結果：京都の荒廃、室町幕府（将軍）の権威の失墜、守護の没落、守護代・国人の台頭（戦国大名）＝下剋上の風潮
:::

### 下剋上の風潮

:::top
1485～93年、山城の国一揆
		
		以後、８年間自治
:::lead
国人が一揆を結び、応仁の乱後も国内で畠山氏の軍の国外退去を実現
:::

:::top
1488～1580年、加賀の一向一揆
		
		以後、約100年間一向宗門徒が自治（織田信長に制圧される）
:::lead
加賀国の一向宗門徒が、守護[[富樫政親]]を殺害
:::

## 戦国大名

### 戦国大名の分国支配

#### 分国法の制定

:::top
各戦国大名が分国支配の基準を示す＝[[喧嘩両成敗]]、他国との婚姻の禁止
:::

##### 例

:::top
『[[甲州法度之次第]]』（武田氏）
:::

:::top
『[[今川仮名目録]]』（今川氏）
:::

:::top
『[[塵芥集]]』（伊達氏）
:::

:::top
『[[早雲寺殿廿一箇条]]』（後北条氏）
:::

#### 指出検地の実施

:::top
領主に支配地・耕作地の面積・収入を申告
:::

:::top
検地で把握した収入（年貢量）を銭に換算して申告
:::

:::last
[[貫高]]制に基づき課役の賦課
:::

#### 軍事力

:::top
国人・地侍の家臣化＝[[寄親･寄子制]]で家臣団を組織
:::

:::top
＊疑似親子関係による統制＝有力家臣が親、下級家臣が子
:::

#### 分国経営

:::top
城下町の建設、交通制度の整備、関所の廃止、楽市楽座、撰銭令の発布、鉱山の開発（金・銀）
:::
