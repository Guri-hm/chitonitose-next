### 中央集権体制の整備
#### 版籍奉還（1869）
:::top
藩主が土地（版）と人民（籍）を朝廷に返上
:::
:::top
旧藩主（大名）は旧領地の知藩事に任命され、家禄が支給される<div class="lead">旧藩主の実質的温存、徴税権・軍事権は各藩に所属
:::
#### <span onclick="chg(this)" class="all">廃藩置県</span>（1871）
:::top
薩摩・長州・[[土佐]]の３藩で御親兵を組織
:::
:::last
藩を廃止し、府・県を設置、[[知藩事]]を罷免
:::
:::last
府知事・県令を中央政府より派遣
:::
:::top
意義<div class="lead">幕藩体制解体、権力が中央政府に集中
:::
:::last
国内の政治的統一が完成
:::

## 諸政策の改革

:::top

:::
:::equal
３族籍 
:::
:::lead

:::
:::lead

:::
:::lead
百姓・町人⇒[[平民]]
:::

:::

:::
### 士族の特権処分
#### 背景
:::top
廃藩置県後も、政府支出の約30％が秩禄（家禄+賞典禄）<div class="last">国家財政圧迫
:::
:::top
家禄<div class="lead">政府から華族・士族に支給
:::
:::top
賞典禄<div class="lead">王政復古・維新の功労者への賞与
:::
#### 秩禄処分
:::top
金禄公債証書を支給、華族・士族への秩禄支給を廃止（1873）<div class="lead">すべての支給者に秩禄５～14年分の額の公債を与える
:::
#### 士族の特権剥奪
:::top
[[廃刀令]]（1876）・秩禄処分（1876）
:::
:::last
士族は全ての特権を失って困窮
:::
:::last
政府は士族授産で対応（事業資金の貸付、北海道開拓事業など）
:::
:::last
成功例は少なく、没落する士族が多数＝士族の反乱の一因
:::
:::top
＊士族の商法<div class="lead">公債を元手に慣れない商売に手を出して失敗
:::
