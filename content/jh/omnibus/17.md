### 中央集権体制の整備

#### 版籍奉還（1869）

:::top
藩主が土地（版）と人民（籍）を朝廷に返上
:::

:::top
旧藩主（大名）は旧領地の知藩事に任命され、家禄が支給される
:::lead
旧藩主の実質的温存、徴税権・軍事権は各藩に所属
:::

#### 廃藩置県（1871）

:::top
薩摩・長州・[[土佐]]の３藩で御親兵を組織
    
:::last
藩を廃止し、府・県を設置、[[知藩事]]を罷免
:::

    
:::last
府知事・県令を中央政府より派遣
:::
:::

:::top
意義
    国内の政治的統一が完成
:::lead
幕藩体制解体、権力が中央政府に集中
:::

### 政治組織の整備―太政官制の変遷―

#### 1869年（版籍奉還後）

:::top
二官六省に再編成、二官＝太政官・[[神祇官]]
:::

:::top
＊祭政一致の方針で、神祇官が太政官の上
:::

#### 1871年（廃藩置県後）

:::top
三院制となる（正院・左院・右院）
:::

## 諸政策の改革

### 封建的身分制度の撤廃

:::top
江戸時代の身分制を廃止し、身分の再編を実施
    ３族籍
:::lead
藩主・公家⇒[[華族]]藩士・旧幕臣⇒[[士族]]百姓・町人⇒[[平民]]
:::

:::top
新たな族籍に基づき壬申戸籍を編成（1872）
:::

:::top
平民
:::lead
苗字許可、華族・士族との結婚の自由、移住・職業選択の自由
:::

### 士族の特権処分

#### 背景

:::top
廃藩置県後も、政府支出の約30％が秩禄（家禄+賞典禄）
:::last
国家財政圧迫
:::
:::

:::top
家禄
:::lead
政府から華族・士族に支給
:::

:::top
賞典禄
:::lead
王政復古・維新の功労者への賞与
:::

#### 秩禄処分

:::top
金禄公債証書を支給、華族・士族への秩禄支給を廃止（1873）
:::lead
すべての支給者に秩禄５～14年分の額の公債を与える
:::

#### 士族の特権剥奪

:::top
[[廃刀令]]（1876）・秩禄処分（1876）
    
:::last
士族は全ての特権を失って困窮
:::

    
:::last
政府は士族授産で対応（事業資金の貸付、北海道開拓事業など）
:::

    
:::last
成功例は少なく、没落する士族が多数＝士族の反乱の一因
:::
:::

:::top
＊士族の商法
:::lead
公債を元手に慣れない商売に手を出して失敗
:::
