:::top
大和（奈良）に古墳が出現・普及＝畿内を中心とした政治連合の存在＝ヤマト政権
:::

:::top
大王(おおきみ)＝ヤマト政権の支配者の称号
:::

## 古墳時代中期（４ｃ後半～５ｃ）

### 朝鮮半島との関係（４ｃ後半）

:::top
加耶（加羅）諸国＝半島南部の鉄資源確保のため、密接な関係
:::

:::top
高句麗＝半島南部の支配拡大をめぐって衝突⇒[[好太王碑]]の碑文に記録あり
:::

#### 日本列島への影響

:::top
高句麗の騎馬軍団との交戦⇒乗馬の風習の伝来＝副葬品として馬具の登場
:::

### 中国との関係（５ｃ）

:::top
『宋書』倭国伝
:::lead
倭の五王（讃・珍・済・興・武）の遣使倭王武の上表文（「安東大将軍」の称号獲得）
:::

### ヤマト政権の支配拡大（５ｃ～）

#### 支配地域

:::top
遠方での出土物に「ワカタケル大王（＝[[雄略]]天皇、倭王武）」の文字
:::

:::top
＝埼玉県[[稲荷山]]古墳出土鉄剣銘・熊本県[[江田船山]]古墳出土鉄刀銘
:::

:::top
＝支配の拡大、漢字の使用
:::

#### 政治体制

:::top
中央豪族＝[[大臣]]・[[大連]]に就任して政権を左右
:::

:::top
地方豪族＝[[<ruby>国造<rt>くにのみやつこ</rt></ruby>]]に就任して人民支配
:::

:::top
私地私民制＝[[屯倉]]（大王家の私有地）・[[名代・子代]]（大王家の私有民）・[[田荘]]（豪族の私有地）・[[部曲]]（豪族の私有民）
:::

## 古墳時代後期（６ｃ～）

### ヤマト政権の動揺

#### 地方豪族の反抗

:::top
527年、磐井の乱
:::lead
筑紫国造磐井が[[新羅]]と結び、ヤマト政権の加耶諸国救援軍の派遣を阻止
:::

#### 朝鮮半島の動向

:::top
高句麗の圧迫⇒新羅・百済の勢力南下⇒加耶諸国の滅亡（562年）
:::

#### 政権内部の抗争

:::top
[[大伴金村]]の失脚…朝鮮半島南部との外交失敗を責められたため
:::

:::top
蘇我氏の政権掌握＝587年、[[蘇我馬子]]が[[物部守屋]]を滅ぼす
:::
