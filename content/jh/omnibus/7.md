#### 守護と地頭

##### 守護

:::top
各国に1人を任命
    主に東国の有力御家人から選出
:::

:::top
職務：大犯三ヵ条
:::lead
京都大番役の催促、<ruby>謀叛<rt>むほん</rt></ruby>人の逮捕、殺害人の逮捕
:::

##### 地頭

:::top
荘園・公領（国衙領）に設置
    御家人から任命
:::

:::top
職務
:::lead
年貢徴収と納入、土地管理、治安維持
:::

#### 主従関係

:::top
将軍と御家人が御恩と奉公の関係で結ばれる
:::

:::top
＊封建制度
:::lead
土地の給与を通じて、主人と従者の関係が成り立つ制度
:::

##### 御恩

:::top
[[本領安堵]]（御家人の先祖伝来の土地所有権を保証）
:::

:::top
[[新恩給与]]（御家人に新たな領地や守護職・地頭職を与える）
:::

##### 奉公

:::top
平時：京都大番役・鎌倉番役
:::

:::top
戦時：軍役
:::

### 北条氏の台頭と他氏排斥

#### 北条時政

:::top
1203年、２代将軍源頼家の外戚の比企能員を滅ぼす
:::last
政所別当となる
:::
:::

#### 北条義時

:::top
1213年、侍所別当[[和田義盛]]を滅ぼす（和田合戦）
:::last
侍所別当と政所別当を兼務、執権の地位を確立
:::
:::
