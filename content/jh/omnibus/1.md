# 旧石器時代～弥生時代

## 旧石器時代
### 日本列島の様子
:::top
更新世（～約１万年前）＝日本列島と大陸が陸続き
:::
:::top
⇒大型動物の渡来
:::
:::top
（長野県[[野尻湖]]でナウマンゾウの化石出土）
:::
:::top
人が生活＝人骨の発見（沖縄県の[[港川人]]・静岡県の[[浜北人]]）
:::
:::top
⇒新人段階の人の骨
:::
### 旧石器時代の生活
:::top
[[関東ローム層]]から打製石器を発見
:::
:::top
⇒群馬県の[[岩宿]]遺跡の調査（旧石器時代の解明開始）
:::
#### 道具
:::top
打製石器（ナイフ形石器・[[尖頭器]]・[[細石器]]）
:::


## 縄文時代（新石器時代）
### 日本列島の様子
:::top
完新世（約１万年前～現在）＝温暖化
:::
:::top
⇒①大型動の絶滅、中小型動物の増加②針葉樹林の減少③海面の上昇
:::
### 縄文時代の生活
:::top
狩猟・採取・[[漁労]]⇒生活の安定、集落を形成して定住
:::
#### 集落
:::top
主な住居は[[竪穴住居]]で、ゴミ捨て場の[[貝塚]]が出現
:::
:::top
＊代表的な遺跡は青森県の[[三内丸山遺跡]]
:::
#### 道具
:::top
狩猟…弓矢（矢の先に[[石鏃]]）、皮をはぐための[[石匙]]
:::
:::top
採取…石皿・すり石で木の実をすりつぶし、[[縄文土器]]（厚手で黒褐色）で煮て<ruby>灰汁<rt>あく
:::
:::top
漁労…骨角器（釣り針や銛など）
:::
#### 信仰
:::top
[[アニミズム]]（自然現象に霊威が存在するという考え）⇒[[屈葬]]で埋葬
:::
:::top
[[土偶]]（女性像）、抜歯
:::
#### 交易の存在
:::top
[[黒曜石]]（産地：北海道[[白滝]]、長野県[[和田峠]]）
:::


## 弥生時代
### 弥生時代の生活
#### 水稲耕作
:::top
九州北部で開始＝早期の水田跡（佐賀県の[[菜畑]]遺跡、福岡県の[[板付]]遺跡）
:::
:::top
東は本州北部、西は九州南部まで普及＝外側では別の文化が成立
:::
:::top
（南西諸島の[[貝塚]]文化、北海道の[[続縄文]]文化）
:::
:::top
水田の種類：[[湿田]]（前期）⇒[[乾田]]（中・後期）
:::
:::top
田植え（田下駄）⇒収穫（[[石包丁]]で穂首刈り）⇒保存（施設：[[高床倉庫]]）
:::
:::top
⇒脱穀（木臼・<ruby>竪杵<rt>たてぎね</rt></ruby>）⇒炊く（<ruby>甕<rt>かめ</rt></ruby>）
:::
#### 道具
:::top
[[弥生土器]]（薄手で赤褐色、様々な器形）＝保存用の[[壺]]、煮炊き用の[[甕]]
:::
#### 集落
:::top
[[環濠集落]]…まわりに深い濠などをめぐらした防御集落
:::
:::top
＊佐賀県の[[吉野ヶ里遺跡]]
:::
:::top
高地性集落…山上に営まれた防御集落
:::
#### 墓制・信仰
:::top
多様な墓の登場＝墳丘墓・方形周溝墓・[[甕棺墓]]・[[支石墓]]など
:::
:::top
⇒身分差の発生、首長の登場
:::
:::top
農耕儀礼＝青銅製祭器の利用（銅剣・銅矛・[[銅鐸]]）
:::
:::top
＊祭器が多量に発見された遺跡は島根県の[[神庭荒神谷遺跡]]
:::
### 紀元前１世紀からの倭国
#### 紀元前１世紀
:::top
『[[漢書]]』[[地理志]]
:::
:::top
…百余国が分立、漢の[[楽浪郡]]に定期的に使者を派遣
:::
#### １世紀中頃〜２世紀
:::top
『[[後漢書]]』[[東夷伝]]
:::
:::top
…57年、奴国の王が後漢に朝貢、光武帝より「漢委奴国王(かんのわのなのこくおう)」の印綬を賜る
:::
:::top
＊金印は福岡県の志賀島から出土
:::
:::top
…107年、倭国王帥升等、後漢の安帝に生口（奴隷？）を献上
:::
:::top
…２世紀後半から、倭国大乱
:::
#### ３世紀
:::top
『[[魏志]]』[[倭人伝]]
:::
:::top
…女王の卑弥呼が鬼道（呪術）を使って邪馬台国を統治（弟の助けあり）
:::
:::top
…狗奴国と交戦、邪馬台国には身分制度あり（大人－下戸）
:::
:::top
…卑弥呼が魏に使者を派遣、「[[親魏倭王]]」の称号や銅鏡などを賜る
:::
:::top
…卑弥呼の死後、男王では統治できず、壱与（台与）が新たな女王になる
:::
:::top
＊邪馬台国の所在地は不明（九州説・近畿説）
:::
