## 縄文時代（新石器時代）

### 日本列島の様子

:::top
完新世（約１万年前～現在）＝温暖化
:::

- 大型動の絶滅

- 中小型動物の増加

- 海面の変化


### 縄文時代の生活

:::top
狩猟・採取・[[漁労]]
:::

:::last
生活の安定、集落を形成して定住
:::

#### 集落

:::top
主な住居は[[竪穴住居]]で、ゴミ捨て場の[[貝塚]]が出現
:::

:::top
＊代表的な遺跡は青森県の[[三内丸山遺跡]]
:::

#### 道具

:::top
狩猟
:::lead
弓矢（矢の先に[[石鏃]]）、皮をはぐための[[石匙]]
:::

:::top
採取
:::lead
石皿・すり石で木の実をすりつぶし、[[縄文土器]]（厚手で黒褐色）で煮て<ruby>灰汁<rt>あく</rt></ruby>抜き
:::

:::top
漁労
:::lead
骨角器（釣り針や銛など）
:::

#### 信仰

:::top
[[アニミズム]]（自然現象に霊威が存在するという考え）⇒[[屈葬]]で埋葬
:::

:::top
[[土偶]]（女性像）、抜歯
:::

#### 交易の存在

:::top
[[黒曜石]]（産地：北海道[[白滝]]、長野県[[和田峠]]）
:::

::blank

:::top
東は本州北部、西は九州南部まで普及＝外側では別の文化が成立
:::

:::top
（南西諸島の[[貝塚]]文化、北海道の[[続縄文]]文化）
:::

::blank

:::top
水田の種類：[[湿田]]（前期）
:::last
[[乾田]]（中・後期）
:::

::blank

:::top

:::middle

### 紀元前１世紀からの倭国

#### 紀元前１世紀

:::top
『[[漢書]]』[[地理志]]
:::

:::top
:::lead
百余国が分立、漢の[[楽浪郡]]に定期的に使者を派遣
:::

#### １世紀中頃〜２世紀

:::top
『[[後漢書]]』[[東夷伝]]
:::

:::top
:::lead
57年、奴国の王が後漢に朝貢、光武帝より「漢委奴国王(かんのわのなのこくおう)」の印綬を賜る
:::

:::top
＊金印は福岡県の志賀島から出土
:::

:::top
:::lead
107年、倭国王帥升等、後漢の安帝に生口（奴隷？）を献上
:::

:::top
:::lead
２世紀後半から、倭国大乱
:::

#### ３世紀

:::top
『[[魏志]]』[[倭人伝]]
:::

:::top
:::lead
女王の卑弥呼が鬼道（呪術）を使って邪馬台国を統治（弟の助けあり）
:::

:::top
:::lead
狗奴国と交戦、邪馬台国には身分制度あり（大人－下戸）
:::

:::top
:::lead
卑弥呼が魏に使者を派遣、「[[親魏倭王]]」の称号や銅鏡などを賜る
:::

:::top
:::lead
卑弥呼の死後、男王では統治できず、壱与（台与）が新たな女王になる
:::

:::top
＊邪馬台国の所在地は不明（九州説・近畿説）
:::
